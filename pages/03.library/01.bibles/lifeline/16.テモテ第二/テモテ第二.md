# テモテ第二

### テモテへの手紙　第二

## 第1章

**神**のご意志により、キリスト・イエスにある命の約束に従い、イエス・キリストの使徒であるパウロより、  
<span class="verse-number">2&ensp;</span>愛する子であるテモテへ、**父**なる**神**、および、私たちの**主**キリスト・イエスより、恵み、哀れみ、そして平安がありますように。  
<span class="verse-number">3&ensp;</span>夜も昼も祈りの中で、私は絶えずあなたを思い、私の父祖たちが仕えたように、清い良心で仕える**神**に感謝しています。  
<span class="verse-number">4&ensp;</span>あなたの涙が私の心にあり、私が喜びで満たされるため、あなたに会うことを切に願っています。  
<span class="verse-number">5&ensp;</span>それは、偽りのない、あなたの中にある信仰に思い至るからです。その信仰は最初にあなたの祖母ロイスと母ユニケにあった、そしてあなたにもあると私は確信しています。  
<span class="verse-number">6&ensp;</span>そういう訳で、私の按手を通してあなたの中にある**神**の賜物を再び燃え上がらせることを、心に留めて欲しいです。  
<span class="verse-number">7&ensp;</span>なぜなら、**神**は私たちに小心者の霊ではなく、力の、そして、愛の、そして慎みの霊を与えて下さったからです。  
<span class="verse-number">8&ensp;</span>したがって、私たちの**主**の証、また、**主**の囚人である私を恥じてはいけません。むしろ**神**の力に従い、福音のために私と共に苦しみなさい。  
<span class="verse-number">9&ensp;</span>**神**は私たちを救い、そして、聖なる召命で私たちを召されました。それは私たちの行ないによるのではなく、永遠の時の以前からが始まる以前、キリスト・イエスによって、**神**ご自身のご意志と私たちに与えられた御恵みによります。  
<span class="verse-number">10&ensp;</span>そして、この御恵みは今、私たちの**救い主**イエス・キリストのお現れになったことにより啓示されました。イエスは死を無力にし、命と不死に、福音で光を当てられました。  
<span class="verse-number">11&ensp;</span>私はその福音の説教者、使徒、異邦人の教師として任命されました。  
<span class="verse-number">12&ensp;</span>そのため、私はこれらの苦痛を受けているが、恥じてはいません。信じている方を私は知っており、その方は、私がお任せしたものを、例のその日まで守ってくださることができると、私は確信しているからです。  
<span class="verse-number">13&ensp;</span>キリスト・イエスにある信仰と愛をもって、私から聞いた健全な言葉の模範をしっかり握りなさい。  
<span class="verse-number">14&ensp;</span>あなたに任されたあの良いものを、私たちの中に住んでおられる**聖霊**で守りなさい。  
<span class="verse-number">15&ensp;</span>アジアにいる者は皆、私を拒否したのは知っての通りです。その中にフゲロとヘルモゲネもいます。  
<span class="verse-number">16&ensp;</span>オネシポロの家の者全員に**主**の哀れみがありますように。彼は幾度も私を励まし、私の鎖を恥としなかったからです。  
<span class="verse-number">17&ensp;</span>それどころか、ローマに着いた時、オネシポロは非常に熱心に捜し、私を見つけ出しました。  
<span class="verse-number">18&ensp;</span>**主**のもとで哀れみを、例のその日に**主**がオネシポロに与えられますように。彼はエペソで私にいろいろな奉仕をどれほどしてくれたかを、あなたがよく知っての通りです。  

## 第2章

したがって、我が子よ、キリスト・イエスにある恵みによって、強くありなさい。  
<span class="verse-number">2&ensp;</span>そして、多くの証人の間で私からあなたが聞いたことを、他人にも教え得る真実な男たちに委ねなさい。  
<span class="verse-number">3&ensp;</span>したがって、イエス・キリストの良い兵士として、苦難を耐え忍びなさい。  
<span class="verse-number">4&ensp;</span>戦いに臨んでいる兵士は、生活のことに巻き込まれることはありません。入隊させてくれた人を喜ばせるためです。  
<span class="verse-number">5&ensp;</span>また、運動競技を争う者は、規則に従って競技しなければ、冠は与えられません。  
<span class="verse-number">6&ensp;</span>よく働く農家が先ず収穫の分配を取るべきです。  
<span class="verse-number">7&ensp;</span>私の言うことを良く考えなさい。なぜなら、**主**があなたに、すべてのことに関して、理解力を与えてくださるのです。  
<span class="verse-number">8&ensp;</span>私の福音により、ダビデの種である、死人たちの中から復活させられたイエス・キリストを忘れてはいけません。  
<span class="verse-number">9&ensp;</span>私はこの福音のため鎖に繋がれ、犯罪人として苦しんでいます。しかし、**神**の御言葉は繋がれていません。  
<span class="verse-number">10&ensp;</span>したがって、選ばれた者たちのため、私はすべてを堪え忍んでいます。永遠の栄光と共に、キリスト・イエスにある救いを彼らも頂くためです。  
<span class="verse-number">11&ensp;</span>これは真の言葉です。すなわち、私たちはキリストと共に死んだなら、私たちはキリストと共に生きます。  
<span class="verse-number">12&ensp;</span>私たちは堪え忍ぶなら、キリストと共に支配します。キリストを否定すれば、キリストも私たちを否定されます。  
<span class="verse-number">13&ensp;</span>私たちが忠実でなくとも、キリストは忠実のままであられます。キリストはご自分を否定されることができません。  
<span class="verse-number">14&ensp;</span>これらのことを人々に覚えさせ、何の役にも立たず、聞き手を破滅させるような言葉の論争しないように、**主**の御前で彼らに命じなさい。  
<span class="verse-number">15&ensp;</span>真理の御言葉を正しく解釈する働き人として、恥じることなく、**神**に自らを捧げるよう励みなさい。  
<span class="verse-number">16&ensp;</span>そうして、邪悪な無駄話は避けなさい。そう言うものはますます神聖でないものを増やすことになります。  
<span class="verse-number">17&ensp;</span>さらに、彼らの言葉はガンのように広がります。ヒメナオとピレトはこの類の者です。  
<span class="verse-number">18&ensp;</span>この者たちは、真理から外れ、復活はもう終わったと言って、ある者たちの信仰を覆します。  
<span class="verse-number">19&ensp;</span>それでも、**神**の強固な土台は立っています。これに、「**主**はご自分の者を知っている」、と言うしるしがあり、また、「キリストの御名を呼ぶすべての者は、不義から離れよ。」とあります。  
<span class="verse-number">20&ensp;</span>そして、立派な家庭には、金銀の器だけではなく、木や粘土の器もあり、誉れのため、また不浄のための器もあります。  
<span class="verse-number">21&ensp;</span>もし誰であれ、これらの不義を離れ、自らを清めれば、その者は**主**のために聖別され、役に立つ器となり、すべての善い業のために備えられ、誉れの器になります。  
<span class="verse-number">22&ensp;</span>また、若い時の欲望から逃れ、清い心から**主**を呼び求める人たちと共に義、信仰、愛、平和を追求しなさい。  
<span class="verse-number">23&ensp;</span>しかし、愚かな、無知な論争は争いを起こすことをあなたは知っているのだから、避けなさい。  
<span class="verse-number">24&ensp;</span>そして、**主**の僕は争ってはいけません。誰にでも柔和であり、教えることができ、堪え忍び、  
<span class="verse-number">25&ensp;</span>穏やかに敵対する者を正しなさい。もしかして、その者たちが真理を知るため、**神**は悔い改めの心をくださるでしょう。  
<span class="verse-number">26&ensp;</span>そうすると、悪魔の意を行なうため、囚われた者たちは、悪魔の罠を逃れ、正気に戻ることはあり得ます。  

## 第3章

しかし、最後の日々に危険な時が来ることを知っておきなさい。  
<span class="verse-number">2&ensp;</span>それは、人々は己を愛する者になり、金銭を愛する者になり、大言をはく者になり、高慢な者になり、冒涜者になり、両親に不従順になり、感謝しない者になり、不義になり、  
<span class="verse-number">3&ensp;</span>愛のない者になり、和解できない者になり、悪口を言うようになり、自制心がなくなり、粗暴な者となり、善いものをさげすみ、  
<span class="verse-number">4&ensp;</span>裏切り者になり、勝手気ままの者になり、不遜な者になり、**神**を愛するより、快楽を愛する者になるからです。  
<span class="verse-number">5&ensp;</span>敬虔な姿形をしながら、敬虔の力を否定する者となります。そういう者たちから離れよ。  
<span class="verse-number">6&ensp;</span>なぜなら、この類の者たちは、家人たちの中に忍び込み、数々の欲望に引き渡されている女たち、つまり罪を負った愚かな女たちを虜にします。  
<span class="verse-number">7&ensp;</span>常に学んではいても、真理に関わる知識に達することができない者たちです。  
<span class="verse-number">8&ensp;</span>正に、ヤンネとヤンブレがモーセに敵対したように、この者たちも真理に敵対します。心の腐った者たちであり、信仰に関しては不合格者です。  
<span class="verse-number">9&ensp;</span>しかし、ヤンネとヤンブレの愚かさもすべての人に明らかになったように、そういう者たちの愚行も明らかになるので、これ以上進みません。  
<span class="verse-number">10&ensp;</span>しかし、あなたは私の教理、生き様、目的、信仰、寛容、愛、忍耐についてきました。  
<span class="verse-number">11&ensp;</span>アンテオケで、イコニオムで、ルステラで起きた私への迫害と苦悩、私が耐えた迫害をあなたはついてきました。私は何という迫害を耐えたでしょう。そのすべての迫害から、**主**は私を救い出してくださいました。  
<span class="verse-number">12&ensp;</span>そして、すべてキリスト・イエスにあって敬虔に生きたい者は全員迫害を受けます。  
<span class="verse-number">13&ensp;</span>しかし、悪い者たちと詐欺師たちはますます悪質になり、騙し、騙されます。  
<span class="verse-number">14&ensp;</span>しかし、誰から学んだかを忘れず、学び、そして、確信したことに、あなたはとどまりなさい。  
<span class="verse-number">15&ensp;</span>そして、キリスト・イエスによる信仰を通して、救いのための、あなたを智者にする聖書を、あなたは小さい時から知っていることを忘れてはいけません。  
<span class="verse-number">16&ensp;</span>聖書はすべて**神**の息吹[^1]であり、教理のため、間違いを正すため、更正のため、義の指導のために有益です。  
<span class="verse-number">17&ensp;</span>それは**神**の男が完成し、すべての善い業に向け準備されるためです。  

## 第4章

したがって、**神**の御前と、ご来臨、そして王国の折、生きている者たちと死人たちを裁く**主**イエス・キリストの御前に、私はあなたを厳命します。  
<span class="verse-number">2&ensp;</span>御言葉を説きなさい。時が良くても悪くても備えなさい。すべての寛容と教えで、納得させ、叱責し、勧めなさい。  
<span class="verse-number">3&ensp;</span>なぜなら、人々は健全な教理に耐えられず、自らの欲望により、耳障りのいいことを聞きたがり、己自身のため、教師の数を増し、  
<span class="verse-number">4&ensp;</span>自らの耳を真理から遠ざけ、たわごとの方に向きを変えます。  
<span class="verse-number">5&ensp;</span>しかし、すべてに気をゆるめず、苦しみを耐え、巡回伝道師の働きをし、奉仕を全うしなさい。  
<span class="verse-number">6&ensp;</span>さて、私はすでに注ぎの供え物として注がれていて、私の去る時はすでに来ています。  
<span class="verse-number">7&ensp;</span>私は立派に戦い、完走し、信仰を守ってきました。  
<span class="verse-number">8&ensp;</span>最後に、正しい裁判官である**主**が、かの日に私に与えてくださる義の冠は、私のために保管されています。そして、私のためにだけではなく、**主**の来臨を慕うすべての者のためです。  
<span class="verse-number">9&ensp;</span>私の所に急いで来るように努力しなさい。  
<span class="verse-number">10&ensp;</span>なぜなら、デマスは今の世を愛し、私を捨て、テサロニケの方へ去りました。また、クレスケンスはガラテヤに、テトスはダルマテヤに去り、  
<span class="verse-number">11&ensp;</span>ルカのみが私と共にいます。マルコの所に行き、連れて来なさい。マルコは奉仕のため、私には有用な人物だからです。  
<span class="verse-number">12&ensp;</span>そして、私はテキコをエペソに遣わしました。  
<span class="verse-number">13&ensp;</span>トロアスでカルポの所に置いてきた外套、それと書物、特に羊皮紙の書物を持って来なさい。  
<span class="verse-number">14&ensp;</span>銅細工人のアレキサンデルは、私に一度ならず大変な悪をしました。**主**が彼の行為に応じて彼に報われますように。  
<span class="verse-number">15&ensp;</span>アレキサンデルは強く私たちの言葉に抵抗したから、あなたも彼に気をつけなさい。  
<span class="verse-number">16&ensp;</span>私の最初の弁明の時、全員私を見捨て、誰も共に立ちませんでした。このことで彼らが罪と定められませんように。  
<span class="verse-number">17&ensp;</span>しかし、**主**は私と共に立ち、私を強固にしてくださったので、御教えは私を通して完全に理解され、また全異邦人が聞くことができるためです。そして、私はライオンの口から助け出されました。  
<span class="verse-number">18&ensp;</span>そして、**主**はすべての悪い行為から私を救い出し、ご自身の天の王国のために私をお守りくださいます。栄光は永遠から永遠まで**主**にありますように。アーメン。  
<span class="verse-number">19&ensp;</span>プリスカとアクラにご挨拶をしてください。また、オネシポロの家人にご挨拶をしてください。  
<span class="verse-number">20&ensp;</span>エラストはコリントに留まったが、私は病気のトロピモをミレトに残して来ました。  
<span class="verse-number">21&ensp;</span>冬になる前に、何としてでも早く来なさい。ユブロ、プデス、リノス、クラウデヤ、またすべての兄弟たちが、あなたによろしくと言っています。  
<span class="verse-number">22&ensp;</span>**主**イエス・キリストがあなたの霊と共におられますように。恵みがあなたと共にありますように。アーメン。  


[^1]: 普通は「霊感」と訳すが、ギリシャ語から考えて、これは不適切である。また、「霊感」とは、元々仏教語である。
