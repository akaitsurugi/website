None of the content on this website is masked behind a paywall. Additionally, one of my objectives was to embrace the open source model by publicly releasing the files used to create these web pages. These source files are now available on [GitHub](https://github.com/akaitsurugi/website?target=_blank&rel=noopener&rel=noreferrer).

===

Putting the website in a git repo has additional benefits, such as making changes more intentional since they are tracked over time. There are some files that haven't been uploaded yet (notably, the config files), as I still need to check these for sensitive information. Nevertheless, the bulk of the data is in the `pages` folder.

Media files won't be included, as these are not useful to put into a git repo; but they are of course directly accessible on the website.
