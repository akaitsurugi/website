# ヤコブ

### ヤコブの手紙

## 第1章

謹啓。**神**と**主**イエス・キリストの僕であるヤコブより、ディアスポラ[^1]の十二部族へ。  
<span class="verse-number">2&ensp;</span>私の兄弟たちよ、様々な試練にあっても、その一つ一つは、すべて喜びであると承知しなさい。  
<span class="verse-number">3&ensp;</span>信仰の試練が忍耐を生むのは、知っての通りです。  
<span class="verse-number">4&ensp;</span>しかし、忍耐を完全に働かせなさい。あなたがたが、何も欠けたところのない、完全無欠な者になるためです。  
<span class="verse-number">5&ensp;</span>あなたがたの中で、誰であれ、知恵に欠ける人がいれば、**神**に願わせなさい。**神**は叱責せず、誰にでも惜しげなく、知恵を与えてくださいます。  
<span class="verse-number">6&ensp;</span>しかし、その人は疑うことなく、信仰によって願わせなさい。なぜなら、疑う人は、風に追い立てられ、荒れ狂う海の波のようです。  
<span class="verse-number">7&ensp;</span>ですから、そんな人には、**主**から何かを受け取るであろうなどと、思わせてはいけません。  
<span class="verse-number">8&ensp;</span>そういう人は二心があり、すべての道で不安定です。  
<span class="verse-number">9&ensp;</span>身分の低い兄弟は、高められたことを、喜びなさい。  
<span class="verse-number">10&ensp;</span>そして、金持ちは、低くされたことを喜びなさい。なぜなら、野の花のように、金持ちは枯れていきます。  
<span class="verse-number">11&ensp;</span>太陽は焼け付く熱を伴い、昇るやいなや、草を枯らし、その花々は落ち、その美しい姿は消滅します。金持ちも同様に、その道半ばで消え失せます。  
<span class="verse-number">12&ensp;</span>誘惑を堪え忍ぶ人は祝福されます。なぜなら、その者は認められると、**主**はご自分を愛する者に与えてくださると、約束された命の冠を、その人は受けるからです。  
<span class="verse-number">13&ensp;</span>人が誘惑された時、「私は**神**に誘惑された。」と言わせてはいけません。なぜなら、**神**が悪に誘惑されることはあり得ないし、まただれも、**神**ご自身は誘惑なさいません。  
<span class="verse-number">14&ensp;</span>しかし、人は各自、自らの欲望に引き寄せられ、おびき寄せられ、誘惑されるのです。  
<span class="verse-number">15&ensp;</span>次に、欲望がはらめば、罪を生みます。そして罪は時がみつると、死を生みます。  
<span class="verse-number">16&ensp;</span>私の愛する兄弟たちよ、騙されてはいけません。  
<span class="verse-number">17&ensp;</span>良い賜物、完璧な贈り物は、すべて上から、色々な光を持っておられる**御父**から降りて来ます。**御父**は何の変化も、向きを表す影もないかたであられます。  
<span class="verse-number">18&ensp;</span>私たちが**神**の造られたものからの、最初の収穫物の一つになるように定められ、真理の御言葉によって**神**は私たちを生み出されました。  
<span class="verse-number">19&ensp;</span>ですから、私の愛する兄弟たちよ、各自それぞれ、聞くにすばやく、語るに遅く、怒りに遅くなりなさい。  
<span class="verse-number">20&ensp;</span>人の怒りは**神**の義を生まないからです。  
<span class="verse-number">21&ensp;</span>従って、すべてのおぞましいもの、溢れる悪を捨て去り、あなたがたの中に植え付けらていて、あなたがたの魂を救うことができる御言葉を、へりくだって受け入れなさい。  
<span class="verse-number">22&ensp;</span>しかし、自らを欺いて、御言葉を聞くだけの者ではなく、実行する人になりなさい。  
<span class="verse-number">23&ensp;</span>なぜなら、人は誰であれ、御言葉を聞くだけで実行しなければ、鏡の中の、生まれたままの自分の顔を、調べている人のようです。  
<span class="verse-number">24&ensp;</span>なぜなら、その者は自分を調べはしたが、離れ去ると、自分がどんな姿だったか、たちどころに忘れます。  
<span class="verse-number">25&ensp;</span>しかし、完全な、すなわち自由の律法を細部まで見、律法に留まり、聞いて忘れる人ではなく、働きを実行するなら、その人は行ないにより祝福されます。  
<span class="verse-number">26&ensp;</span>もしあなたがたの中で、自分を宗教心のある人であると思っているのに、口を慎まず、自分の心を欺けば、その人の宗教心は無益です。  
<span class="verse-number">27&ensp;</span>**神**であられ、**御父**であられる方の御前での、清く、汚れない宗教心とはこうです。つまり、困窮時に、孤児や未亡人を訪問し、そして、この世からの汚れに染まらず、自らを汚れない者として維持することです。  

## 第2章

私の兄弟たちよ、栄光の**主**であられる私たちの**主**イエス・キリストの信仰を、差別を持って保持してはいけません。  
<span class="verse-number">2&ensp;</span>なぜなら、黄金の指輪の男が、豪華な服を着て、あなたがたの集会に来たとします。そして、不潔な服を着ている貧しい男も来たとします。  
<span class="verse-number">3&ensp;</span>あなたがたは豪華な服を着ている人に注目し、その男に言うでしょう。「この上席にお座りください。」そして、貧しい人に、「向こうに立つか、それとも私の足台の下に座りなさい。」と言うでしょう。  
<span class="verse-number">4&ensp;</span>これでは、あなたがた自身の間で差別をし、あなたがたが、悪意ある裁判官になったのではないですか。  
<span class="verse-number">5&ensp;</span>私の愛する兄弟たちよ、聞きなさい。**神**はこの世の貧しい人たちを選び、信仰に富ませ、そして、**神**を愛する人たちを、約束された王国の相続人にされたではありませんか。  
<span class="verse-number">6&ensp;</span>しかし、あなたがたはこの貧しい人を見下しました。金持ちたちはあなたがたを虐げ、裁判所に引きずり出す人々ではないですか。  
<span class="verse-number">7&ensp;</span>あなたがたの呼ばれている尊い御名[^2]を、金持ちたちは冒涜していませんか。  
<span class="verse-number">8&ensp;</span>「あなたの隣人をあなた自身のように愛しなさい。」と、聖書による王家の律法を、あなたがたが全うしているならば、あなたがたは良くやっています。  
<span class="verse-number">9&ensp;</span>しかし、差別をすることは罪を犯すことなのですから、律法により、差別したあなたがたは、犯罪者とみなされます。  
<span class="verse-number">10&ensp;</span>なぜなら、誰であれ、律法をすべて守っても、一つにでもつまずけば、全律法の有罪者となるからです。  
<span class="verse-number">11&ensp;</span>なぜなら、「姦淫をしてはいけない。」と言われている方は、「殺人をしてはいけない。」とも言われています。それで、あなたは姦淫はしなくても殺人をすれば、あなたは、律法の犯罪者となってしまいます。  
<span class="verse-number">12&ensp;</span>自由の律法を通し、裁かれる者として話し、かつ、行動しなさい。  
<span class="verse-number">13&ensp;</span>哀れみを示さなかった者には、哀れみのない裁きがあります。哀れみは裁きに打ち勝ちます。  
<span class="verse-number">14&ensp;</span>我が兄弟たちよ、誰かが、あの人は信仰があると言ったところで、行動が伴わなければ、何の得があるでしょうか。信仰はその人を救うことができますか。  
<span class="verse-number">15&ensp;</span>もし、兄弟や姉妹が裸で、日々の糧に欠いていて、  
<span class="verse-number">16&ensp;</span>そして、あなたがたの一人が、「安らかに行きなさい。暖を取り、十分に食べなさい。」と言っても、体に必要な物を与えなければ、何の益があるでしょうか。  
<span class="verse-number">17&ensp;</span>このように、信仰も同様です。信仰だけでは、行動が伴わなければ、死んでいます。  
<span class="verse-number">18&ensp;</span>しかし、ある人は言います。「あなたは信仰があるが、私には行動がある。行動のない、あなたの信仰を見せなさい。そうすれば、私は行動によって私の信仰を見せます。」  
<span class="verse-number">19&ensp;</span>**神**がお一人であられるとあなたは信じています。それはよいことです。悪霊どもでさえ信じて、震えています。  
<span class="verse-number">20&ensp;</span>しかし、愚かな人よ、行動を伴うことはない信仰は死んでいるとでも、理解したいのか。  
<span class="verse-number">21&ensp;</span>自らの子イサクを、祭壇の上に捧げた時、行動によって義と認められたのは、我々の父、アブラハムではなかったのですか。  
<span class="verse-number">22&ensp;</span>信仰にアブラハムの行動が伴い、行動により、信仰が全うされたことが、分からないのですか。  
<span class="verse-number">23&ensp;</span>そして、「アブラハムは**神**を信じた。そして、そのことを彼にとって義と計算された。」という聖書は成就されました。そして、アブラハムは**神**の友と呼ばれました。  
<span class="verse-number">24&ensp;</span>人は信仰だけではなく、行動によって義と認められていることは、あなたがたは分かっています。  
<span class="verse-number">25&ensp;</span>また同様に、あの使者たちを受け入れ、他の道から彼らを送り出した時、行動によって義と認められたのは、遊女、ラハブではなかったのですか。  
<span class="verse-number">26&ensp;</span>なぜなら、霊のない肉体は死んでいると同様に、行動のない信仰も死んでいます。  

## 第3章

私の兄弟たちよ、私たち教師は、人一倍厳しい裁きを受けることは知っているのですから、あなたがたの内、多くの人は、教師になってはいけません。  
<span class="verse-number">2&ensp;</span>私たちは皆、多くのことで失敗するからです。誰であれ言葉で失敗しないなら、その人は完全な男で、体全体に、はみをつけることもできます。  
<span class="verse-number">3&ensp;</span>見よ。私たちに従わすため、馬にはみをくわえさせ、そして馬の全身を引き回します。  
<span class="verse-number">4&ensp;</span>そして、船も御覧なさい。巨大ですが、激しい風で押されても、ほんの小さい舵一つで、操舵手の思うまま、向きを変えます。  
<span class="verse-number">5&ensp;</span>そして、同様に、舌は小さい一つの器官ですが、巨大なことを誇ります。小さい火一つで、どんなに大きな森も燃えるかを考えなさい。  
<span class="verse-number">6&ensp;</span>ですから、舌は火です。そして、不義の世界です。体全体を汚すため、そして、自然の周期に放火し、地獄により燃やされているように、私たちの器官の一つに組み入れられています。  
<span class="verse-number">7&ensp;</span>なぜなら、獣と鳥、地を這うもの、海の生き物、すべての種類は、人により制御されている、また制御されたことがあります。  
<span class="verse-number">8&ensp;</span>しかし、舌を制御することができる人は、誰もいません。舌は手に負えない悪であり、死に至る毒で満たされています。  
<span class="verse-number">9&ensp;</span>私たちの**父**なる**神**を、私たちは舌で祝福し、そして**神**の形に似せてつくられた、人間を呪います。  
<span class="verse-number">10&ensp;</span>祝福も呪いも同じ口から出ます。私の兄弟たちよ、こんなことがあってはいけません。  
<span class="verse-number">11&ensp;</span>泉は同じ穴から、清水と苦い水を同時に出すことがありますか。  
<span class="verse-number">12&ensp;</span>私の兄弟たちよ、イチジクの木がオリーブを実らせ、また、ぶどうのつるがイチジクを実らせることができますか。同様に、どんな泉も塩水と清水の両方を出しません。  
<span class="verse-number">13&ensp;</span>あなたがたの間で、賢く、そして理解力のある人は誰でしょうか。その人はよい生き様で、自らのわざは、知恵のあるへりくだりによって、行なわれていると示しなさい。  
<span class="verse-number">14&ensp;</span>しかし、もしあなたがたが心の中に、突き刺さるねたみと利己主義を持っているなら、自慢してはいけないし、そして、真理に逆う、うそを言ってはいけません。  
<span class="verse-number">15&ensp;</span>この知恵は上から下ったのではなく、地上のもので、肉に属するものであり、悪霊からのものです。  
<span class="verse-number">16&ensp;</span>なぜなら、ねたみと利己主義のあるところに、混乱とすべての悪が存在します。  
<span class="verse-number">17&ensp;</span>しかし、上からの知恵はまず清いものであり、それに平和的であり、柔和であり、逆らわず、哀れみと良い実で満たされ、差別なく、そして、偽善はありません。  
<span class="verse-number">18&ensp;</span>そして、義の実は、平和をつくりだす人々により、平和裏に蒔かれます。  

## 第4章

戦争や争いは、あなたがたの中のどこから来るのでしょうか。あなたがたの体の中、快楽を求めて戦う欲望から来るのではないですか。  
<span class="verse-number">2&ensp;</span>そして、あなたがたは渇望しても持てません。人を殺し、また不当に望めば手に入れられません。あなたがたは戦をし、争っています。それでも、あなたがたは所持していません。お願いしないからです。  
<span class="verse-number">3&ensp;</span>また、あなたがたは願っても受けとれません。自らの快楽に費やすために、見当はずれな願いをしているからです。  
<span class="verse-number">4&ensp;</span>姦淫する男たち、また、姦淫する女たちよ。世との友情は、**神**に敵対することと、分かっていないのですか。世の友人になろうとする者は、自らを**神**の敵にしているのです。  
<span class="verse-number">5&ensp;</span>または、「私たちの中に住んでいる**御霊**は、ねたむほど求めておられる。」と聖書は、意味もなく言っているとでも思うのですか。  
<span class="verse-number">6&ensp;</span>しかし、**神**は御恵みをさらに下さいます。従って、言われます。「**神**は傲慢な者たちには逆らうが、謙虚な者たちには御恵みを与えられる。」  
<span class="verse-number">7&ensp;</span>ですから、**神**には服従しなさい。悪魔に敵対しなさい。そうすると、悪魔はあなたがたから逃げます。  
<span class="verse-number">8&ensp;</span>**神**のそばに近づきなさい。そうすれば、**神**はあなたがたに近づいてくださいます。罪人たちよ、あなたがたの手を清潔にしなさい。二心の者たちよ、あなたがたの心を清めなさい。  
<span class="verse-number">9&ensp;</span>苦しみなさい。そして、悲しみなさい。そして、泣きなさい。あなたがたの笑いを悲しみに、あなたがたの喜びを悲嘆に変えなさい。  
<span class="verse-number">10&ensp;</span>自ら、**主**の御前にへりくだりなさい。そうすると、**主**はあなたがたを高くしてくださいます。  
<span class="verse-number">11&ensp;</span>兄弟たちよ、互いに悪口を言ってはいけません。兄弟の悪口を言って、兄弟を裁く者は、律法の悪口を言い、それを裁いています。しかし、もし律法を裁けば、あなたは律法を守る人ではなく、裁判官です。  
<span class="verse-number">12&ensp;</span>律法を定める方は一人です。この方は、救いも消滅もできる方です。互いに裁くあなたがたは、誰なのですか。  
<span class="verse-number">13&ensp;</span>「今日か明日、私たちはある町に行き、そこで売買をして、一年間過ごし、儲けよう。」と言っているあなたがたは、待ちなさい。  
<span class="verse-number">14&ensp;</span>しかし、明日は何が起こるか、あなたがたは分かりません。なぜなら、あなたがたの命とは何ですか。命は少しの間現れ、消えて行くかすみに過ぎません。  
<span class="verse-number">15&ensp;</span>その代わりに、あなたがたは言うべきです。つまり、「**主**がお望みになれば、私たちは生きて、あれもしよう、これもしよう。」  
<span class="verse-number">16&ensp;</span>しかし今、あなたがたは傲慢、そのものを誇っています。そういう誇りはすべて悪です。  
<span class="verse-number">17&ensp;</span>従って、よいことをすべきと分かっていながら、しない人には、それは罪になります。  

## 第5章

金持ちたちよ、来なさい。あなたがたの上に来る、あなたがたの悲惨のために、泣き、かつ、うめきなさい。  
<span class="verse-number">2&ensp;</span>あなたがたの富は腐りはて、あなたがたの服は虫に食われてしまいました。  
<span class="verse-number">3&ensp;</span>あなたがたの黄金も銀も腐食し、そのさびはあなたがたに敵対する証拠となり、あなたがたの肉を火のように食い尽くします。あなたがたは最後の日々に宝を積み上げたのです。  
<span class="verse-number">4&ensp;</span>見よ。あなたがたの土地で刈り入れをした、労働者たちから騙し取った賃金が叫んでいます。そして、その刈り入れをした人たちの叫び声も叫び、安息日の**主**の耳にとどいています。  
<span class="verse-number">5&ensp;</span>あなたがたは快楽と贅沢の中で、地上に暮らしてきています。虐殺の日に、あなたがたは心を太らせたのです。  
<span class="verse-number">6&ensp;</span>あなたがたに抵抗しない義人を、あなたがたは有罪にし、殺害しました。  
<span class="verse-number">7&ensp;</span>従って、兄弟たちよ、**主**の来臨まで忍耐しなさい。地の尊い実りを農夫はどのようにして待つか、考えなさい。早い雨と後の雨を受け取るまで、農夫は忍耐して待つのです。  
<span class="verse-number">8&ensp;</span>あなたがたも忍耐して持ちなさい。**主**の来臨は近づいているから、あなたがたの心を揺るがないものにしなさい。  
<span class="verse-number">9&ensp;</span>兄弟たちよ、罪に定められないように、互いに不平を言い合ってはいけません。見よ。裁判官は戸口に立っておられます。  
<span class="verse-number">10&ensp;</span>私の兄弟たちよ。**主**の御名で説いた預言者たちを、苦難を耐え忍ぶことと、忍耐の模範として受け入れなさい。  
<span class="verse-number">11&ensp;</span>見よ。耐え忍ぶ人たちを、私たちは祝福された人たちと認識しています。あなたがたはヨブの忍耐を聞きました。そして、**主**により意図された結果も見ました。つまり、**主**はとても同情深く、哀れみ深い方であられます。  
<span class="verse-number">12&ensp;</span>しかし、私の兄弟たちよ。何より、天により、また、地により、また他の誓い方でもいけません。偽善に陥らないように、「肯定」は「肯定」とし、「否定」は「否定」としなさい。  
<span class="verse-number">13&ensp;</span>あなたがたの中で、苦しんでいる人がいますか。その人は祈らせなさい。喜んでいる人はいますか。その人には、詩篇の歌を歌わせなさい。  
<span class="verse-number">14&ensp;</span>あなたがたの中で、病気の人はいますか。その人に、教会の長老たちを呼びなさい。そして、**主**の御名によって、長老たちは病人に油を注ぎ、病人のために祈ります。  
<span class="verse-number">15&ensp;</span>そして、信仰の祈りは病人を救い、**主**はその人を立ち上がらせてくださいます。そして、その人が罪を犯したなら、赦されます。  
<span class="verse-number">16&ensp;</span>あなたがたの悪行を互いに言い表し、治されるため、互いに祈りなさい。正しい人の忠実で、そして熱心な祈りは力があります。  
<span class="verse-number">17&ensp;</span>エリヤは、私たちと同じような心の人であったが、雨が降らないように熱心に祈りました。すると、三年と六ヶ月の間、地に雨は降りませんでした。  
<span class="verse-number">18&ensp;</span>そして、エリヤがまた祈ると、空は雨を降らせ、地はその実を実らせました。  
<span class="verse-number">19&ensp;</span>兄弟たちよ、あなたがたの中の誰かが、真理から離れ迷い出し、その人を連れ戻したとしたら、  
<span class="verse-number">20&ensp;</span>一人の罪人を、その離れ迷い出した道から、連れ戻す人は、死から一つのたましいを救い出し、無数の罪を遮断する人であると、その人に教えなさい。
