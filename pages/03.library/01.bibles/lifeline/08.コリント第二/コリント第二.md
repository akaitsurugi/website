# コリント第二

### コリント人への手紙　第二

## 第1章

**神**のご意志により、イエス・キリストの使徒であるパウロと、私たちの兄弟であるテモテより、コリントにある**神**の教会へ、そして、アカヤ全土の聖徒たちへ。  
<span class="verse-number">2&ensp;</span>私たちの**父**であられる**神**より、そして、**主**イエス・キリストより、あなたがたに恵みと平安がありますように。  
<span class="verse-number">3&ensp;</span>私たちの**主**イエス・キリストの**父**なる**神**、哀れみの**父**、すべての慰めの**神**へ、祝福がありますように。  
<span class="verse-number">4&ensp;</span>私たちが、**神**に慰められたあの慰めによって、あらゆる苦しみにいる人々を私たちが助けることができるように、**神**は、あらゆる苦しみの中にいる私たちを慰めてくださいます。  
<span class="verse-number">5&ensp;</span>キリストの苦難が私たちの中に、溢れんばかりにあるように、私たちの慰めもキリストを通して、溢れんばかりにあるからです。  
<span class="verse-number">6&ensp;</span>それで、もし私たちが苦しめられているなら、それはあなたがたの慰めと救いのためです。それは、私たちが苦しんでいる同じ苦しみを耐えるために、有益です。また、私たちが慰められるなら、それもあなたがたの慰めと救いのためです。  
<span class="verse-number">7&ensp;</span>そして、あなたがたはこの苦しみを共有するように、あなたがたは、この慰めの共有者になることを私たちは知っています。ですから、あなたがたに対する私たちは強く期待しています。  
<span class="verse-number">8&ensp;</span>兄弟たちよ、アジアで出会った私たちの苦しみについて知らぬままでいてほしくありません。それは、計測できないほど、私たちの耐えれる力以上で、命さえ望みなしとなりました。  
<span class="verse-number">9&ensp;</span>しかし、私たち自身で自らに死刑の判決を下しました。それは私たち自身を信頼するからでなく、死人たちを復活させてくださる**神**を信頼したからです。  
<span class="verse-number">10&ensp;</span>**神**は私たちをこのようなひどい死から救い出してくださいました。そして、**神**は私たちを救ってくださるのです。これからもなお、**神**は私たちを救ってくださることを、私たちは信頼しています。  
<span class="verse-number">11&ensp;</span>あなたがたも祈りで私たちを共に助けてくださっています。これは、大勢の人々を通して、私たちに与えられた賜物に、大勢の人々による感謝が捧げられるためです。  
<span class="verse-number">12&ensp;</span>なぜなら、私たちの誇りは私たちの良心の証です。つまり、この世で、またあなたがたに対してますます豊かに、肉の知恵によらず、**神**の恵みによって、私たち自身が真っ直ぐに、**神**の誠実さで、ますます豊かに生活をしてきたことです。  
<span class="verse-number">13&ensp;</span>あなたがたが読み取り、または理解できること以外、私たちはあなたがたに他の何も書いていないからです。それで、あなたがたが最後まで理解するように、私は望んでいます。  
<span class="verse-number">14&ensp;</span>同様に、あなたがたが部分的に私たちの言葉を理解したように、**主**イエスの日に、あなたがたは私たちの誇りであり、私たちもあなたがたの誇りなのです。  
<span class="verse-number">15&ensp;</span>それで、このことを確信して、あなたがたが二度目の恵みがあるように、私はもっと前にあなたがたの所へ行くつもりでした。  
<span class="verse-number">16&ensp;</span>そして、あなたがたの所を通ってマケドニアに行き、再びマケドニアからあなたがたの所に行き、そして、あなたがたに見送られてユダヤに行くつもりでした。  
<span class="verse-number">17&ensp;</span>従って、これを計画している時、私はこの計画をいいかげんに立てたのでしょうか。それとも、私が計画する事は、肉に基づくから、そこには私の身に付いた、ハイ、ハイ、イイエ、イイエ、があるのでしょうか。  
<span class="verse-number">18&ensp;</span>しかし、**神**は忠実であられますから、私たちのあなたがたへの言葉は、ハイ、イイエ、にはなりませんでした。  
<span class="verse-number">19&ensp;</span>なぜなら、私たち、つまり私とシルワノとテモテを通して、あなたがたの間に語られた**神の御子息**であられるイエス・キリストは、「はい」でもあり、「いいえ」でもある方にはならず、「はい」となられました。  
<span class="verse-number">20&ensp;</span>**神**の約束と言う約束はことごとく、その方により「はい」となり、その方により「アーメン」となり、私たちを通して**神**の栄光となったのです。  
<span class="verse-number">21&ensp;</span>そして、キリストにあって、あなたがたと共に、私たちを、確かな者としてくださり、そして、私たちに油を注いでくださる方は**神**です。  
<span class="verse-number">22&ensp;</span>**神**は私たちに判を押し、私たちの心の中で、**御霊**を保証金としてすでに与えてくださっています。  
<span class="verse-number">23&ensp;</span>そして、私が二度とコリントへ行かなかったのは、あなたがたを煩わせないためであったことを、私の魂にかけ、**神**を証人とて要求します。  
<span class="verse-number">24&ensp;</span>と言うのは、私たちはあなたがたの信仰を支配したことはありません。あなたがたの喜びのために、ともに働く仲間です。あなたがたは信仰により立ってきたからです。  

## 第2章

しかし、私はあなたがたの所に二度と、悲しみのままでは行かないと、自ら決めました。  
<span class="verse-number">2&ensp;</span>なぜなら、もし私があなたがたを悲しませれば、私が悲しくさせた人こそ、私を喜ばすのではないでしょうか。  
<span class="verse-number">3&ensp;</span>そして、私は以前これと全く同じことを書きました。それは、私の喜びは、皆さん全員の喜びであると、全員に関して確信があるからです。私が行った時、私が喜ばすべき人たちから、悲しみをもらうことのないためです。  
<span class="verse-number">4&ensp;</span>あなたがたを悲しませるためではなく、あなたがたに対する私の豊かな愛を、あなたがたが知るために、私は多くの困難と心の苦しみの中から、あふれる涙を通して、あなたがたにこれを書いたからです。  
<span class="verse-number">5&ensp;</span>しかし、もし、誰かが悲しませたとしても、私を悲しませたことにはなりません。ある程度までは、あなたがた全員を悲しませました。全員に重荷を負わせたくないからです。  
<span class="verse-number">6&ensp;</span>そんな人には、大多数の人による罰で十分です。  
<span class="verse-number">7&ensp;</span>ですから、逆にその者を赦し慰めるべきです。そうしなければ、多分その者は、溢れんばかりの悲しみに、飲み込まれてしまいます。  
<span class="verse-number">8&ensp;</span>従って、その者にあなたがたの愛を確認させるよう、私はあなたがたに勧めます。  
<span class="verse-number">9&ensp;</span>あなたがたがすべてのことにおいて、すなおに従うかどうかを試すためにも、私はこれを書いたからです。  
<span class="verse-number">10&ensp;</span>また、あなたがたが誰であれ赦すことは何でも、私も赦します。私が何であれ赦したなら、キリストの御前で、あなたがたのために、その人を赦したのです。  
<span class="verse-number">11&ensp;</span>それはサタンにつけ込まれないためです。私たちはサタンの策略を知らないわけではないからです。  
<span class="verse-number">12&ensp;</span>そして、私がキリストの福音を語るために、トロアスに行った時、**主**によって門が私に開かれた時、  
<span class="verse-number">13&ensp;</span>私の兄弟テトスが見つからず、私の霊は不安でした。しかし、私は彼らに別れを告げ、マケドニヤに行きました。  
<span class="verse-number">14&ensp;</span>キリストにあって常に私たちを勝利に導いてくださり、また私たちを通して、どこででも、ご自分の知識の香りをあらわしてくださる**神**に感謝します。  
<span class="verse-number">15&ensp;</span>救われる者たちの間でも、滅ぶ者たちの間でも、私たちは**神**にとって、キリストのかぐわしい香りだからです。  
<span class="verse-number">16&ensp;</span>ある者たちには、私たちは死に至る死の匂いですが、他の者たちには命に至る命の香りです。そして、この二つに、誰か叶うでしょう。  
<span class="verse-number">17&ensp;</span>**神**の御言葉を売る多くのインチキ商人と私たちは違います。真摯で、**神**からの者です。**神**の御前でキリストによって私たちは話します。  

## 第3章

私たちはまた、自らを推薦し始めているのですか。それとも、ある人々のように、あなたがたからの推薦の手紙、またはあなたがたへの推薦の手紙は必要なのですか。  
<span class="verse-number">2&ensp;</span>あなたがたが、私たちの心の中に書かれた、私たちの推薦の手紙なのです。すべての人に読まれ、知られています。  
<span class="verse-number">3&ensp;</span>あなたがたは、私たちの奉仕による、キリストの明確な手紙です。インクでなく、生ける**神**の**御霊**により、石の板の上でなく、肉の板の上、つまり心の板の上に書かれています。  
<span class="verse-number">4&ensp;</span>そして、キリストを通し**神**に対し、私たちはそう確信しています。  
<span class="verse-number">5&ensp;</span>このままでは、私たちは自ら何事も考える力はありません。私たちの十分な資格は**神**からです。  
<span class="verse-number">6&ensp;</span>**神**は、私たちを新しい契約に叶う奉仕者とされました。文字の者でなく、**御霊**による者です。文字は殺すが、**御霊**は命を与えられるからです。  
<span class="verse-number">7&ensp;</span>もし石に書かれ、刻まれた文字の死の務めが栄光の内に来たので、そのため、モーセの顔の栄光は過ぎ去って行く栄光のゆえに、イスラエルの子らがしっかり見れなかったとしたら、  
<span class="verse-number">8&ensp;</span>どうして**御霊**の務めはそれ以上の栄光がないのでしょうか。  
<span class="verse-number">9&ensp;</span>もし有罪の裁きをする務めに栄光があるなら、義の務めははるかにもっと栄光があります。  
<span class="verse-number">10&ensp;</span>なぜなら、また、このことに関して、かつて栄光とされたものは、すぐれたこの栄光のせいで、栄光ではなくなりました。  
<span class="verse-number">11&ensp;</span>なぜなら、過ぎ去るものが栄光であったなら、残ったものはさらにもっと栄光に包まれています。  
<span class="verse-number">12&ensp;</span>従って、このような希望があるから、私たちはとても勇敢に話します。  
<span class="verse-number">13&ensp;</span>そして、イスラエルの子らが、過ぎ去りつつあるものの最後をはっきり見ることができないように、自分の顔に覆いをかけたモーセとは違います。  
<span class="verse-number">14&ensp;</span>しかし、彼らの心は閉ざされました。なぜなら、今日まで、旧約聖書を読む時に、その覆いは取り除かれず、残っています。なぜなら、この覆いはキリストによって取り除かれるからです。  
<span class="verse-number">15&ensp;</span>しかし、今日まで、モーゼの書が読まれる時、彼らの心に別の覆いがあります。  
<span class="verse-number">16&ensp;</span>しかし、人が**主**に向く時、その覆いは取り除かれます。  
<span class="verse-number">17&ensp;</span>そして、**主**は**御霊**であられ、**主**の**御霊**がおられる所に、自由があります。  
<span class="verse-number">18&ensp;</span>しかし、私たちは皆、覆いのない顔で、鏡にうつる**主**の栄光を見ながら、栄光から栄光へと、**主**と同じ姿に変えられます。正に、**主**の**御霊**によるものです。  

## 第4章

従って、哀れみを受けたので、私たちはこの役目があるのですから、私たちは落胆していません。  
<span class="verse-number">2&ensp;</span>しかし、恥ずべき隠れたことは放棄しました。狡猾に歩まず、**神**の御言葉を悪用せず、真理を明確に示すことにより、**神**の御前ですべての人の良心に、私たち自身を勧めています。  
<span class="verse-number">3&ensp;</span>しかし、もし私たちの福音が隠されているなら、滅び行く者たちの中に隠されているのです。  
<span class="verse-number">4&ensp;</span>今の時代の神は、信じないこの者たちの心を盲目にし、**神**の御姿であられるキリストの栄光の福音の光を、この者たちの上には、輝かせないためです。  
<span class="verse-number">5&ensp;</span>私たちは、私たち自身のことを説いていず、**主**であるキリスト・イエス、そしてイエスのため、あなたがたの奴隷である、私たちのことを、説いているのです。  
<span class="verse-number">6&ensp;</span>暗闇から光に輝けと、命じられたあの**神**は、イエス・キリストの顔にある、**神**の栄光の知識に光を与えるため、私たちの心を照らされたからです。  
<span class="verse-number">7&ensp;</span>しかし、私たちはこの宝を土の器に入れて、持っています。卓越したこの力は、**神**からであり、私たちから出たのではありません。  
<span class="verse-number">8&ensp;</span>私たちはあらゆる側から圧迫されているが、窮地には立ってはいません。私たちは途方に暮れてはいますが、絶望はしていません。  
<span class="verse-number">9&ensp;</span>迫害されていますが、見捨てられてはいません。倒されましたが、全滅はしていません。  
<span class="verse-number">10&ensp;</span>私たちはいつも**主**イエスの死を身に負っています。それはまた、イエスの命が私たちの体に明らかになるためでもあります。  
<span class="verse-number">11&ensp;</span>生きている私たちが、絶えずイエスのために、死に渡されるのは、イエスの命もまた、私たちの死んでいく肉体に明らかになるためです。  
<span class="verse-number">12&ensp;</span>ですから、死は私たちの中で稼動し、命はあなたがたの中で稼動しています。  
<span class="verse-number">13&ensp;</span>そして、「私は信じたゆえに、話した。」と書いてあることによって、私たちは信仰の同じ霊を持っているから、私たちも信じるゆえに、話しています。  
<span class="verse-number">14&ensp;</span>それは、**主**イエスを復活させられた方が、私たちをもイエスと共に復活させ、そして、あなたがたと共に御前に立たせてくださることを、私たちは知っているからです。  
<span class="verse-number">15&ensp;</span>なぜなら、多くの人の中に広がった恵みが、**神**の栄光への感謝を豊かにするため、すべてはあなたがたのためです。  
<span class="verse-number">16&ensp;</span>ですから、私たちは落ち込んでいません。たとえ、私たちの外なる者は腐っても、内なる者は日ごとに更新されます。  
<span class="verse-number">17&ensp;</span>私たちのこの瞬間、軽い苦しみが私たちのために稼動しているのは、永遠の栄光の、卓越した重さのためです。  
<span class="verse-number">18&ensp;</span>私たちは見えるものではなく、見えないものに注目しているのです。見えるものは一時的ですが、見えないものは永遠だからです。  

## 第5章

なぜなら、もし私たちの地上の家、この仮小屋が破壊されれば、人手によらない、天国に、**神**による永遠の家があることを、私たちは知っています。  
<span class="verse-number">2&ensp;</span>ですから、天国の家を身にまといたいと、熱望し、私たちはこの家にいてうめいています。  
<span class="verse-number">3&ensp;</span>これを身にまとってしまえば、裸で発見されることはないからです。  
<span class="verse-number">4&ensp;</span>なぜなら、この仮小屋にいる私たちは重荷を負い、うめいています。身につけている物を脱がされたくないからでなく、当然死ぬものが命に飲み込まれようと、さらにもっと身にまとうのです。  
<span class="verse-number">5&ensp;</span>飲み込まれることこそのため、我々を整えてくださったのは**神**なのです。**御霊**の保証として、私たちに与えられたのです。  
<span class="verse-number">6&ensp;</span>したがって、私たちは常に確信があります。私たちは肉体に滞在している間は、**主**から離れていることを知っています。  
<span class="verse-number">7&ensp;</span>なぜなら、私たちは見ることによらず、信仰で歩みます。  
<span class="verse-number">8&ensp;</span>そして、私たちは確信があります。肉体から離れ、**主**と共にいれるのは、むしろうれしいのです。  
<span class="verse-number">9&ensp;</span>したがって、それに、ここにいても、いなくても、**主**に喜ばれるように狙いをつけています。  
<span class="verse-number">10&ensp;</span>善であろうが、悪であろうが、肉体で行なったすべてのことにしたがい、報いを受けるため、私たち一人一人は、キリストの裁きの座の前に出頭しなくてはならないからです。  
<span class="verse-number">11&ensp;</span>したがって、私たちは**主**の恐ろしさを知っているので、人々を説得しています。しかし、私たちは**神**に知られています。またあなたがたの良心にも知られていると、私は待ち望んでいます。  
<span class="verse-number">12&ensp;</span>ですから、改めてあなたがたに私たち自身を薦めません。心ではなく、外見を誇る人たちに対して、あなたがたが応じることができるように、私たちがあなたがたに代わって誇る機会を与えています。  
<span class="verse-number">13&ensp;</span>なぜなら、もし私たちが発狂したとすれば、**神**のためです。または、もし私たちが正常であれば、あなたがたのためです。  
<span class="verse-number">14&ensp;</span>なぜなら、一人のあの方がすべての人のために死んでくださったのだから、すべての人は死んだ、と私たちはこう判断しているから、キリストの愛が私たちを駆り立てています。  
<span class="verse-number">15&ensp;</span>そして、その方がすべての人のため死なれたのは、生きる者たちは、もはや自らのため生きるのではなく、みんなのため死なれ、復活された方のため生きるためです。  
<span class="verse-number">16&ensp;</span>したがって、これから私たちは肉により人を知ることをしません。しかし、私たちはキリストを肉によって知りましたが、今後、このようにキリストを知ることはしません。  
<span class="verse-number">17&ensp;</span>それで、もし人がキリストの中にいれば、その人は新しく造られた人です。古いものは過ぎ去りました。見よ、すべてのものは新しくなったのです。  
<span class="verse-number">18&ensp;</span>そして、すべてのものは**神**からのものです。イエス・キリストを通して、私たちを**神**と和解させ、私たちに和解の仕事を与えてくださったのです。  
<span class="verse-number">19&ensp;</span>それはすなわち、**神**はキリストにあって、世をご自分に和解させ、人々の悪行の責任を彼らに転嫁せず、そして私たちに和解の御言葉を委ねられました。  
<span class="verse-number">20&ensp;</span>それで、**神**は私たちを通して懇願しておられるように、私たちはキリストの大使です。私たちはキリストに代わって願います。「**神**と和解しなさい。」  
<span class="verse-number">21&ensp;</span>なぜなら、私たちがキリストにあって**神**の義となるために、**神**は罪を知られない方を、私たちのために罪となさったのです。  

## 第6章

それで私たちも、キリストと共に働く者として、あなたがたが**神**の恵みを空しく受け入れないように、勧めます。  
<span class="verse-number">2&ensp;</span>なぜなら、**神**は言われます。「あなたがたを受け入れるべき時、私はあなたがたを聞いた。そして、救いの日に、あなたがたを救った。」見よ、今は受け入れるべき時です。見よ、今は救いの日なのです。  
<span class="verse-number">3&ensp;</span>私たちの奉仕が咎められることはないよう、どんなつまずきになるものも与えません。  
<span class="verse-number">4&ensp;</span>しかし、**神**に仕える者として、すべての面で私たち自身を推薦します。多大な忍耐に、苦難に、まさかの時に、困難に、  
<span class="verse-number">5&ensp;</span>むちの傷に、入獄に、暴動に、労苦に、不眠に、断食に、  
<span class="verse-number">6&ensp;</span>清廉に、知識に、長期の忍耐に、親切に、**聖霊**に、真摯の愛に、  
<span class="verse-number">7&ensp;</span>真理の御言葉に、**神**の力に、左右の手の義の武器によって、  
<span class="verse-number">8&ensp;</span>光栄と屈辱を通して、悪口と賞賛を通して、たぶらかす者たちのようだが、真実な者たちであり、  
<span class="verse-number">9&ensp;</span>知られてないようで、良く知られていて、死にそうだが、見よ、私たちは生きています。懲らしめの罰を受けたようだが、まだ殺されていません。  
<span class="verse-number">10&ensp;</span>悲しんでいるようだが、いつも喜んでいます。貧しいようだが、多くの人を富ませます。何も持っていないようだが、何でも持っています。  
<span class="verse-number">11&ensp;</span>コリント人たちよ。私たちは何も隠さず話しています。私たちの口はあなたがたのために開かれています。  
<span class="verse-number">12&ensp;</span>私たちが、あなたがたを狭いところに閉じ込めているのではなく、あなたがたの感情そのものが、あなたがたを狭いところに閉じ込めています。  
<span class="verse-number">13&ensp;</span>それで、その返礼として（私は子どもたちに話すように話しています）、あなたがたも開かれたものになりなさい。  
<span class="verse-number">14&ensp;</span>信者ではない者たちと、不釣り合いなくびきにつながれてはいけません。なぜなら、義と無法は、何か仲間関係がありますか。また、光と闇は、何か合い通じるものがありますか。  
<span class="verse-number">15&ensp;</span>また、キリストはベリアル[^1]と、何か一致する所がありますか。また、信者は信者でない者とは、何か共通する所でもありますか。  
<span class="verse-number">16&ensp;</span>そして、**神**の神殿は、偶像と何か一致するものがありますか。なぜなら、あなたがたは生ける**神**の神殿なのだから、**神**がすでに言われています。「わたしは彼らの中に住み、彼らの間を歩く。そして、わたしは彼らの**神**になり、彼らはわたしの民になる。」  
<span class="verse-number">17&ensp;</span>したがって、「彼らの中から出て、別れなさい。」と**主**は言われます。「そして、汚れたものに触れてはいけない。そうすれば、わたしはあなたがたを受け入れる。」  
<span class="verse-number">18&ensp;</span>「そして、わたしはあなたがたに一人の**父**になる。そして、あなたがたはわたしの息子たちと娘たちになる。」と**全能の主**が言われます。  

## 第7章

したがって、愛する人たちよ、これらの約束を得たのだから、肉と霊のすべての汚れから、自らを清め、**神**の恐れで、完璧な聖になるよう、やり遂げましょう。  
<span class="verse-number">2&ensp;</span>心を私たちに開いてください。私たちは誰にも不正をしたことはありませんし、誰をも堕落させたこともなく、誰からも物を騙し取ったことはありません。  
<span class="verse-number">3&ensp;</span>私は非難するためにこれを言っているのではありません。以前、私はあなたがたに言ったからです。あなたがたは、私たちの心の中にいて、死ぬも生きるも一緒だからです。  
<span class="verse-number">4&ensp;</span>あなたがたに対して、私は勇敢に話し、大いにあなたがたを誇りにしています。私は慰めで満たされています。私たちのすべての苦しみの中で、私は非常に喜んでいます。  
<span class="verse-number">5&ensp;</span>なぜなら、私たちはマケドニアに来た時、私たちの体は休息は何もなく、そして、周りすべてと対立があり、外には争い、内には恐れがありました。  
<span class="verse-number">6&ensp;</span>しかし、意気消沈した者たちを慰める**神**は、テトスの来ることにより私たちを慰めてくださいました。  
<span class="verse-number">7&ensp;</span>テトスの来訪だけではありません。あなたがたの中にいて彼が受けた慰めにも慰められました。テトスがあなたがたの熱い望みと、あなたがたの悲しみ、私への熱心な思いを私たちに伝えてくれたことで、さらに私は喜びました。  
<span class="verse-number">8&ensp;</span>たとえ、私の手紙があなたがたを悲しませたとしても、以前は後悔したが、今は後悔していません。あの手紙があなたがたをわずかな時間、悲しませたのだ、と分かっているからです。  
<span class="verse-number">9&ensp;</span>私は今喜んでいます。あなたがたが、悲しい目にあったからでなく、あなたがたの悲しみが、悔い改めに至ったからです。何も私たちから損は受けないため、あなたがたは**神**によって悲しい目にあったからです。  
<span class="verse-number">10&ensp;</span>**神**による悲しみは、後悔しない救いへの悔い改めを生み出すが、この世界の悲しみは死を生み出すからです。  
<span class="verse-number">11&ensp;</span>あなたがたが**神**により悲しんだそのものを見よ。そのものこそが、あなたがたの中に、いかに勤勉を生み、いかにあなたがた自身の弁明を、義憤を、恐怖を、渇望を、熱心を、潔白を生んだことか。あなたがた自身、このことには自ら無罪であると証明したのです。  
<span class="verse-number">12&ensp;</span>したがって、私はあなたがたに書いたのは、それは不正を犯した人のためではなく、また、その不正に苦しむ人のためでもなく、**神**の御前に、あなたがたに対する、私たちの心づかいが、あなたがたに表れるためです。  
<span class="verse-number">13&ensp;</span>したがって、私たちはあなたがたの慰めによって慰められました。そして、テトスの喜びのため、さらに喜びました。あなたがた全員により、テトスの霊が元気を取り戻したからです。  
<span class="verse-number">14&ensp;</span>もしテトスにあなたがたのことを何か誇ったとしても、私は面目を失ってはいません。しかし、私たちはあなたがたに、すべて真実のことを話したと同様に、私たちのテトスに対する誇りは、真実と証明されました。  
<span class="verse-number">15&ensp;</span>そして、テトスはあなたがた全員の従順を忘れず、恐れとおののきであなたがたが彼を受け入れたことを忘れず、あなたがたに対する愛情はますます豊かになっています。  
<span class="verse-number">16&ensp;</span>それで、私は喜んでいます。あなたがたのことはすべて、自信を持っているからです。  

## 第8章

また、兄弟たちよ、マケドニアの諸教会に与えられた**神**の御恵みを、私たちはあなたがたに知らせます。  
<span class="verse-number">2&ensp;</span>すなわち、大試練の苦しみの中で、彼らの喜びの溢れと、底を突く貧困は、彼らの惜しみなく与える富となりました。  
<span class="verse-number">3&ensp;</span>なぜなら、私は証します。能力に応じて、いや、能力以上に、彼らは自ら進んで、  
<span class="verse-number">4&ensp;</span>その賜物と聖徒たちへの奉仕の交流を、私たちが受け入れるように、彼らは私たちに非常に熱心に願っていました。  
<span class="verse-number">5&ensp;</span>そして、私たちが望んだようにではなく、彼らはまず自らを**主**に捧げ、そして**神**のご意志を通し、私たちにもマケドニア人は自らを捧げました。  
<span class="verse-number">6&ensp;</span>これに関しては、テトスが前から始めていたので、この恵みをあなたがたコリント人の中で全うするよう、私たちはテトスに勧めました。  
<span class="verse-number">7&ensp;</span>しかし、あなたがたはすべてのこと、つまり、信仰に、言葉に、知識に、そしてすべての不断の努力で豊かになっています。同様に、私たちに対するあなたがたの愛で、この恵みを豊かにしなさい。  
<span class="verse-number">8&ensp;</span>私が話すのは命令ではなく、他者への不断の熱心で、あなたがたの愛の誠実を確認しているのです。  
<span class="verse-number">9&ensp;</span>なぜなら、あなたがたは私たちの**主**イエス・キリストの御恵みを知っています。すなわち、キリストは富んでおられたが、あなたがたのため、貧しくなられました。キリストの貧しさを通して、あなたがたが豊かになるためです。  
<span class="verse-number">10&ensp;</span>それから、このことに関して、提言します。あなたがたの益になることです。一年前から、あなたがたが始め、そしてこれはあなたがたが自ら願っただけではなく、実行していたことです。  
<span class="verse-number">11&ensp;</span>しかし、あなたがたは今していることを完了すべきです。それは、それを願う気持ちがあったと同様に、それを全うする気持ちを持ちなさい。  
<span class="verse-number">12&ensp;</span>もし、先ず、進んで行なう意志があるなら、人は持っているもので受け入れられます。持っていないものではありません。  
<span class="verse-number">13&ensp;</span>なぜなら、あなたがたが重荷を負い、他の人が楽をする、と言う意味ではありません。  
<span class="verse-number">14&ensp;</span>ただ、平等に、あなたがたの豊かさが、今、彼らの欠乏を満たし、彼らの豊かさがあなたがたの欠乏を満たし、平等になるためです。  
<span class="verse-number">15&ensp;</span>書いてあるとおりです。「多く集めた者は、あまった物がなく、またわずかしか集めた者でも、不足はなかった。」  
<span class="verse-number">16&ensp;</span>しかし、あなたがたのため、同じ心くばりをテトスの心の中に供えている**神**に、感謝をしています。  
<span class="verse-number">17&ensp;</span>なぜなら、テトスはその勧めを受け入れただけではなく、さらに熱心に、自由意志で、あなたがたの所に行きました。  
<span class="verse-number">18&ensp;</span>そして、私たちは彼と共に兄弟を遣わしました。あの男の福音に関しての賞賛はどの教会でもありました。  
<span class="verse-number">19&ensp;</span>それだけではなく、私たちと共にこの賜物を持って旅をするため、この兄弟は諸教会により、投票で選ばれました。**主**ご自身の栄光のため、またあなたがたの熱心さを示すため、この賜物は私たちにより管理されています。  
<span class="verse-number">20&ensp;</span>私たちが管理しているこの豊かな賜物のことで、私たちを咎める者が出ることを避けるためです。  
<span class="verse-number">21&ensp;</span>**主**の御前だけではなく、人の前でも、気をつけて正しい管理をしています。  
<span class="verse-number">22&ensp;</span>また、私たちの兄弟を彼らと共に遣わしました。様々な場面で真剣であると、私たちは試して認めた男です。私たちはあなたがたを大いに信頼しているので、この兄弟は今、さらに熱心になっています。  
<span class="verse-number">23&ensp;</span>もし誰かテトスのことを問うなら、彼はあなたがたに関して言えば私の仲間であり、共に働く者です。または、私たちの兄弟たちのことを尋ねるのなら、諸教会の使徒たちであり、キリストのご栄光なのです。  
<span class="verse-number">24&ensp;</span>ですから、兄弟たちに、そして諸教会の前で、あなたがたの愛とあなたがたについての私たちの誇りの証拠を示しなさい。  

## 第9章

さて、聖徒たちへの奉仕についてですが、私があなたがたに書く必要はありません。  
<span class="verse-number">2&ensp;</span>なぜなら、私は進んでしようとするあなたがたの心を知っているからです。そして、マケドニアの人々へ、アカイアでは一年前、もう準備は完了していると私は自慢しました。あなたがたの熱心は、大部分の人を奮起させました。  
<span class="verse-number">3&ensp;</span>それで、私たちのあなたがたについて、このことで自慢したことが、嘘にならないように、私は兄弟たちを遣わしました。それは、私が言ったように、あなたがたが、準備することができるためです。  
<span class="verse-number">4&ensp;</span>もし、私と共に行った、あのマケドニア人たちは、あなたがたが、準備ができていないと分かれば、この自信満々の自慢が、当然あなたがたを含めて言うのですが、私には恥になってしまいます。  
<span class="verse-number">5&ensp;</span>したがって、兄弟たちはあなたがたの所に先に行くように、また私があなたがたのs以前に約束した贈り物を、前もって準備してもらうように、私は勧める必要があると思いました。そうすると、贈り物は貪欲ではなく、祝福です。  
<span class="verse-number">6&ensp;</span>しかし、このことを私は言います。惜しんで蒔く者は、収穫はわずかですが、豊かに蒔く者は、収穫は豊かです。  
<span class="verse-number">7&ensp;</span>ですから、各自、惜しまずに、嫌々ながらでなく、決まりだからと言うのでなく、心の中で、目的を決めて与えなさい。喜びを持って与える人を**神**は愛してくださるからです。  
<span class="verse-number">8&ensp;</span>そして、**神**はあなたがたへの、すべての恵みを豊かにできる方であられます。それは、あなたがたを何事をも満足させ、すべてのよい働きのために、豊かなものを持たせるためです。  
<span class="verse-number">9&ensp;</span>それは書かれている通りです。「その方は広く散らし、貧しい人々に与えられた。その方の義は永遠にとどまる。」  
<span class="verse-number">10&ensp;</span>そして、蒔く者に種を、そして、食べ物にパンをくださる方は、あなたがたにも、蒔く種をくださり、そして増やし、またあなたがたの義の実を増やしてくださいます。  
<span class="verse-number">11&ensp;</span>一方、あなたがたは、すべての面に豊かで、何事も惜しまないので、そのことが、私たちを通し、**神**へ感謝をすることになります。  
<span class="verse-number">12&ensp;</span>なぜなら、この奉仕の働きは、聖徒たちの必要を満たすだけではなく、**神**への幾多の感謝を通して、豊かになるからです。  
<span class="verse-number">13&ensp;</span>この務めの証拠を通して、あなたがたの、キリストの福音に従うことを表明すること、またあなたがたが、彼らとすべての人々に惜しみなく分かち合うことが、**神**へ栄光を捧げています。  
<span class="verse-number">14&ensp;</span>また、あなたがたの中に、非常に豊かな**神**の恵みがあるので、あなたがたを慕っている彼らは、あなたがたのための祈りにより、**神**に栄光を捧げています。  
<span class="verse-number">15&ensp;</span>筆舌につくせない**神**の賜物に、**神**に感謝します。  

## 第10章

さて、私、パウロは、キリストの優しさと思いやりで、あなたがたに勧めます。あなたがたと顔を合わせる時は、私はへりくだっているが、離れた時はあなたがたに対して確信をもって接しています。  
<span class="verse-number">2&ensp;</span>しかし、私たちは、まるで肉によって歩んでいると思っている、あなたがたの中のどある人々に対して、私は確信をもってするつもりですが、皆さんといっしょの時、そんな考えで、私が確信を持ちすぎないように願っています。  
<span class="verse-number">3&ensp;</span>私たちは肉で歩んでいますが、私たちは肉では戦いません。  
<span class="verse-number">4&ensp;</span>なぜなら、私たちの戦いは肉によるものではなく、砦を引きずり倒す、**神**の強力な力だからです。  
<span class="verse-number">5&ensp;</span>議論に、また**神**の知識に敵対し、自らを高めるすべての高きものを投げ倒し、すべての考えを捕虜とし、捕らえ、キリストの従順に連行します。  
<span class="verse-number">6&ensp;</span>そして、あなたがたの従順が全うされると、すべての不従順を罰するので覚悟しています。  
<span class="verse-number">7&ensp;</span>あなたがたは、外見でものを見るのですか。もし人は自ら自分がキリストの者であると確信しているなら、その人はキリストの者であると同様に、私たちもキリストの者です。自らその人に考えさせなさい。  
<span class="verse-number">8&ensp;</span>なぜなら、我々を引きずり倒すためでなく、造り上げるため、**主**が我々に与えてくださった我々の権威を自慢することさえ、私は恥ずかしくありません。  
<span class="verse-number">9&ensp;</span>それは、手紙であなたがたに恐怖をいだかせないためです。  
<span class="verse-number">10&ensp;</span>「彼の手紙は重く、力強いのですが、彼の体格は貧弱で、彼の言葉は唾棄すべきです。」と人々は言っています。  
<span class="verse-number">11&ensp;</span>私たちがいない時の手紙では言葉ですが、行った時はそのような行動を起こします。そんな人は、そう思わせておきなさい。  
<span class="verse-number">12&ensp;</span>なぜなら、私たちは私たち自身を、あえて等級分けしませんし、お互いを賞賛しあったりしません。しかし、あの人たちは彼ら自身で互いを測り、互いを比較します。利口ではありません。  
<span class="verse-number">13&ensp;</span>しかし、私たちは無制限に自慢しません。一定の領分、すなわち、私たちを任命された、あなたがたに及ぶ、一定の領分の中でしか自慢しません。  
<span class="verse-number">14&ensp;</span>我々の権威があなたがたへ及ばない訳はありません。したがって、私たちは行き過ぎたことはしていません。なぜなら、あなたがたのところを目指し、私たちはキリストの福音をたずさえて来たのです。  
<span class="verse-number">15&ensp;</span>無制限に他人の働きを自慢したりしません。ただ、あなたがたの信仰が豊かになるのにつれ、あなたがたにより、ますます私たちの領分が大きくなる望みを持っています。  
<span class="verse-number">16&ensp;</span>それは、他の人の地にある、既に完成したものを自慢するのではなく、あなたがたの背後の地で、福音を宣べ伝えるためです。  
<span class="verse-number">17&ensp;</span>しかし、「自慢する者は、**主**にあって自慢しなさい。」  
<span class="verse-number">18&ensp;</span>自らを推挙する人は認められず、認められるのは、**主**が推挙する人だからです。  

## 第11章

私のわずかな愚かしさに耐えてくればいいと望んではいますが、私をもう耐えているでしょう。  
<span class="verse-number">2&ensp;</span>**神**の妬みで、私はあなたがたを妬んでいます。あなたがたを無垢な処女として、キリストに捧げるため、私はあなたがたを一人の夫、キリストに、婚約させたからです。  
<span class="verse-number">3&ensp;</span>しかし、あの蛇はその予測できないずるさで、エバを騙したように、キリストの持つ素直さが、あなたがたの知力も、腐敗するのではないかと、私は心配しています。  
<span class="verse-number">4&ensp;</span>なぜなら、誰かが来て、私たちが語ったことのない別のイエスを語り、また、あなたがたが受け入れたことのない、別の霊を受け入れ、また、受け入れたことがない別の福音を受け入れるなど、それをあなたがたが我慢するかも知れません。  
<span class="verse-number">5&ensp;</span>なぜなら、最も優れた使徒たちに比べても、私は少しも劣っていないと、思っています。  
<span class="verse-number">6&ensp;</span>私は演説の訓練を受けていませんが、知識に関してはそうではありません。しかし、あなたがたの間ですべての点を、私たちは完全に明らかにしました。  
<span class="verse-number">7&ensp;</span>あなたがたを高くするため、自らへりくだった私は、罪を犯したのですか。なぜなら、私は無報酬であなたがたに**神**の福音を説いたからです。  
<span class="verse-number">8&ensp;</span>私は、あなたがたへ奉仕をするため、他の諸教会を強引に、他の諸教会に押し入り、彼らから賃金を得ました。  
<span class="verse-number">9&ensp;</span>そして、あなたがたと共にいて、私は困った時、誰の重荷にもなりませんでした。マケドニアの兄弟たちが、足りない必要品を補ってくれたからです。あなたがたの重荷になることは、何事も避けましたし、今後もそうします。  
<span class="verse-number">10&ensp;</span>私にはキリストの真理があるので、アカヤの地方で、この誇りを誇る私を誰も止められません。  
<span class="verse-number">11&ensp;</span>なぜですか。私はあなたがたを愛していないからですか。**神**はご存じです。  
<span class="verse-number">12&ensp;</span>しかし、今、私がしている事は続けます。それは、私たちと同じように、彼らが誇れる立場を得ようと伺う機会を、切断するためです。  
<span class="verse-number">13&ensp;</span>なぜなら、そういう者たちは偽使徒であり、狡猾な働き人で、自らをキリストの使徒たちに姿形を変えています。  
<span class="verse-number">14&ensp;</span>そして、驚くことではありません。サタンさえも自らを光の御使いに姿形を変えるのです。  
<span class="verse-number">15&ensp;</span>したがって、サタンの奉仕者たちも自らを義の奉仕者に姿形を変えることは、驚くことではありません。彼らの最後は彼らのわざと一致します。  
<span class="verse-number">16&ensp;</span>私はもう一度言います。私は愚か者であると、誰にも思われたくありません。もし、そうでなければ、ほんの少し自慢しようとする、私を愚か者として受け入れてください。  
<span class="verse-number">17&ensp;</span>私が話すことは、**主**によって話しているのではく、愚か者でもあるかのように、この自慢話を自信をもって、話しています。  
<span class="verse-number">18&ensp;</span>多くの者は肉で自慢しているので、私も自慢しましょう。  
<span class="verse-number">19&ensp;</span>愚か者を喜んで耐えてくれるのは、あなたがた自身が賢いからです。  
<span class="verse-number">20&ensp;</span>人があなたがたを奴隷にしても、人があなたがたをむさぼり食べても、人があなたがたから収奪しても、その人自身が高い地位につけば、その人があなたがたの顔面を殴っても、あなたがたはその人を我慢するでしょう。  
<span class="verse-number">21&ensp;</span>私たちはこのことに弱すぎたよう、私は恥ずかしいことを言います。ですがしかし、人は自信があるなら、愚かなことをでも言います。私も自信があります。  
<span class="verse-number">22&ensp;</span>彼らはヘブライ人ですか。私もそうです。彼らはイスラエル人ですか。私もそうです。彼らはアブラハムの種ですか。私もそうです。  
<span class="verse-number">23&ensp;</span>彼らはキリストの奉仕者ですか。（愚か者として私は言います。）私はそれ以上の者です。労働はより多く、むち打たれの刑はより多く、牢獄に入れられたのもより多く、死はたびたびでした。  
<span class="verse-number">24&ensp;</span>四十から一を引いたむち打ちを、ユダヤ人たちから、五度、私は受けました。  
<span class="verse-number">25&ensp;</span>棒で三度殴られ、一度、投石の刑を受け、三度難船し、一昼夜海上にいたことがあります。  
<span class="verse-number">26&ensp;</span>私は旅行が多く、水の危険、強盗の危険、同胞の危険、異邦人の危険、町の危険、荒れ果てた所の危険、海の危険、偽兄弟の危険にも遇いました。  
<span class="verse-number">27&ensp;</span>疲労と重労働の苦しみ、たびたび眠れず、飢えと渇き、たびたび食事できず、寒く、裸でした。  
<span class="verse-number">28&ensp;</span>おまけに、他のこと、つまり毎日私に乗りかかってくる、全教会への重い心労です。  
<span class="verse-number">29&ensp;</span>弱いのは誰ですか。私は弱くないのですか。誰かがつまずかされる、私は義憤で燃えないのですか。  
<span class="verse-number">30&ensp;</span>私が自慢しなければならないのなら、自らの弱さについて自慢します。  
<span class="verse-number">31&ensp;</span>永遠に祝福されている、私たちの**主**イエス・キリストの**神**と**父**は、私がうそを言っていないことをご存じです。  
<span class="verse-number">32&ensp;</span>ダマスカスでは、アレタ王の配下の総督は、私を逮捕しようと、ダマスカス人の町を軍隊で監視していました。  
<span class="verse-number">33&ensp;</span>しかし、私はかごで城壁の窓から降ろされ、総督の手から逃げました。  

## 第12章

私が自慢してもそれは無益なことは明らかです。それで、**主**の幻と啓示に話を進めます。  
<span class="verse-number">2&ensp;</span>私はキリストの中にいる男を知っていました。十四年前、（**神**はご存じですが、肉体と共にであったか、また、肉体から分離してか、私は分かりません。）この人は第三の天に引き上げられました。  
<span class="verse-number">3&ensp;</span>そして、私はそういう人を知っています。**神**はご存じですが、肉体と共か、肉体から離れてか、私は分かりません。  
<span class="verse-number">4&ensp;</span>というのは、その人はパラダイスの中へと引き上げられ、口にしがたい言葉、つまり人が口にしてはいけない言葉を彼は聞きました。  
<span class="verse-number">5&ensp;</span>私がこういう人なら、自慢もするが、自分自身の弱さ以外、私は自分のことは自慢しません。  
<span class="verse-number">6&ensp;</span>なぜなら、私は愚か者ではないので、自慢したければ、真実を言います。しかし、見た目以上に、また、聞いた以上に、人々が私を評価しないため、私は自慢するのを差し控えます。  
<span class="verse-number">7&ensp;</span>そして、あの啓示の桁外れの偉大さのため、傲慢にならないため、つまり、サタンの使いが私を痛めつけるため、一本のとげが、私の体内に与えられたのです。  
<span class="verse-number">8&ensp;</span>これについては、抜いてくださるよう、私は三度**主**に願いました。  
<span class="verse-number">9&ensp;</span>すると、**主**は私に言われました。「私の恵みはあなたにとって十分です。私の力は弱さの中で、完璧となるからです」。したがって、キリストの御力が私に宿るように、私はなおさら自らの弱さの中にいて自慢します。  
<span class="verse-number">10&ensp;</span>それで、キリストのために、私は弱さ、侮辱、困窮、迫害、艱難を喜びます。弱い時に、私は強いからです。  
<span class="verse-number">11&ensp;</span>あなたがたに強いられ、私は自慢する愚か者になりました。なぜなら、私はあなたがたに誉めてもらうはずでした。たとえ、私が取るに足らない者でも、最高の使徒の後は行きません。  
<span class="verse-number">12&ensp;</span>まことに、使徒の数々のしるしは、あらゆる忍耐を伴い、あなたがたの間で行なわれました。それはしるしに、不可思議に、力ある業にです。  
<span class="verse-number">13&ensp;</span>なぜなら、他の教会に比べ、何かあなたがたに、劣っているものがあるのですか。私自身が、あなたがたの重荷にならなかったこと以外はないのではないですか。私のこの間違いを許してください。  
<span class="verse-number">14&ensp;</span>さて、三度目のあなたがたへ行く用意はできています。私はあなたがたの重荷にはなりません。あなたがたの物を求めるのではなく、あなたがたそのものを求めているからです。子どもが、親のため蓄えるべきではなく、親が子どもたちのため、蓄えるべきなのです。  
<span class="verse-number">15&ensp;</span>そして、あなたがたの魂のため、私は大いに喜んで費やしますし、費やされもします。あなたがたを満ち溢れるほど、愛すれば愛するほど、より少なく愛されてもです。  
<span class="verse-number">16&ensp;</span>私はあなたがたに重荷を負わせなかったにしても、ずる賢く、策略で、あなたがたを捕まえました。  
<span class="verse-number">17&ensp;</span>あなたがたのところへ遣わした人たちを利用して、私はあなたがたから何かを騙し取りましたか。  
<span class="verse-number">18&ensp;</span>私はテトスに頼み、テトスと共に私たちの兄弟を遣わしました。テトスはあなたがたから何かを騙し取りましたか。私たちは同じ霊で歩んだのではないのですか。同じ足跡を踏んだのではないのですか。  
<span class="verse-number">19&ensp;</span>繰り返します。あなたがたの前で私たちは自己弁護をしていると思いますか。キリストにあって**神**の御前で私たちは言っています。しかし、愛する人たちよ、私たちがするすべてのことは、あなたがたの徳育のためです。  
<span class="verse-number">20&ensp;</span>なぜなら、恐れていることがあります。私が行くと、私の望んでいないあなたがたを目にしないか、そして、あなたがたも望んでいない私を目にしないかです。また、争い、ねたみ、怒りの爆発、自己的な野望、悪口、ゴシップ、自惚れ、大騒ぎがあるのではと恐れています。  
<span class="verse-number">21&ensp;</span>今度また私が二度目に行く時、あなたがたの前で、私の**神**が私をはずかしめ、また以前犯した、不潔、性的な罪、好色を悔い改めていない大勢の人たちのために、私は嘆き悲しむのではと恐れています。  

## 第13章

今回行けば、あなたがたのところへは、三度目になります。「二人か三人の証人の口をそろえて、すべての言葉は確認される。」  
<span class="verse-number">2&ensp;</span>私は以前にもあなたがたに言ったが、二度目の時のように、前もって言います。今、そこにいない私は前に、罪を犯した人たちと、残り全員に、これを書きます。今度もし行けば、容赦しません。  
<span class="verse-number">3&ensp;</span>私の中にいるキリストが話していると言う証拠を、あなたがたは求めているからです。キリストはあなたがたに対して弱者ではなく、あなたがたの中にいるキリストは強い方なのです。  
<span class="verse-number">4&ensp;</span>キリストは弱さによって十字架につけられたのに、**神**の力によって生きておられます。私たちもキリストにあって弱いのですが、**神**のあなたがたに対する力により、私たちはキリストと共に生きます。  
<span class="verse-number">5&ensp;</span>あなたがたは信仰の中にいるかどうか、自らを調べなさい。自分たちを検査しなさい。イエス・キリストはあなたがたの中におられることが、自分自身でわからないのですか。ただし、あなたがたが失格していなければですが。  
<span class="verse-number">6&ensp;</span>しかし、私たちは失格していないと、あなたがたが知ることを、私は望んでいます。  
<span class="verse-number">7&ensp;</span>それで、あなたがたは悪を何も犯さないように、私は**神**に祈っています。それは、私たちは条件を満たしたように見えるためでなく、失格したように見えても、あなたがたが善を行なうためにです。  
<span class="verse-number">8&ensp;</span>私たちは真理に敵対しては何もできないが、真理のためにはできるからです。  
<span class="verse-number">9&ensp;</span>私たちは弱く、あなたがたが強い時、私たちはうれしいのです。そして、あなたがたが完璧になるのを祈っています。  
<span class="verse-number">10&ensp;</span>私がこれらのことを書き送るのは、留守にしているからです。もし、そこにいたら、破壊するためでなく、徳育のため、**主**が私に与えられた権威を、厳しく適応しないためだからです。  
<span class="verse-number">11&ensp;</span>最後に、兄弟たちよ、喜びなさい。完璧になり、励まされなさい。心を一つにし、平和になりなさい。そして、愛と平安の**神**があなたがたと共におられますように。  
<span class="verse-number">12&ensp;</span>聖なる口づけで、互いに挨拶をしなさい。  
<span class="verse-number">13&ensp;</span>すべての聖徒たちは、あなたがたに挨拶をおくります。  
<span class="verse-number">14&ensp;</span>**主**イエス・キリストの恵み、**神**の愛、**聖霊**の交流があなたがた全員と共にありますように。アーメン。  


[^1]: 悪魔の名前で、価値なしという意味である。
