# ヨハネ第三

### ヨハネの手紙　第三

長老より、私が真理によって愛している、愛しいガイオへ。  
<span class="verse-number">2&ensp;</span>愛する者よ、あなたのたましいが栄えていると同様に、あらゆることにあなたが栄え、かつ健康であることをお祈りします。  
<span class="verse-number">3&ensp;</span>なぜなら、あなたの中の真理、そして、その真理の道をあなたが歩んでいると、兄弟たちが来て証した時、私は非常に喜びました。  
<span class="verse-number">4&ensp;</span>私の子どもたちが、真理によって歩んでいると、聞くこと以上の喜びは、私にはありません。  
<span class="verse-number">5&ensp;</span>愛する者よ、あなたが兄弟たちに、また外国人たちにも、行なっていることは信頼に値します。  
<span class="verse-number">6&ensp;</span>この人々は、あなたの愛を教会の前で証しました。この人々を**神**にふさわしい者として送り出すなら、あなたは賞賛されることをしたことになります。[^1]  
<span class="verse-number">7&ensp;</span>なぜなら、この人々は、異邦人から何も取らずに、御名のために出かけたのです。  
<span class="verse-number">8&ensp;</span>従って、真理のため、共に働く者になれるよう、私たちは、このような人々を受け入れる義務があります。  
<span class="verse-number">9&ensp;</span>私は教会に手紙を書きました。しかし、彼らの間で、頭になりたがるデオテレペスは、私たちを歓迎はしないでしょう。  
<span class="verse-number">10&ensp;</span>従って、デオテレペスが行なっている行為を、また悪意ある言葉で私たちの悪口を、もし私が行けば、デオテレペスに思い出させてやります。そして、このことだけで、満足しないデオテレペスは、自ら進んで兄弟たちを歓迎せず、歓迎を求める人々を拒否し、教会の外へ追い出してしまいます。  
<span class="verse-number">11&ensp;</span>愛する者よ、悪に従わず、善に従いなさい。善を行なう人は、**神**からの人であるが、悪を行なう人は、**神**を見たことのない者です。  
<span class="verse-number">12&ensp;</span>すべての人から、そして真理それ自体からも、デメテリオは証されています。また、私たちも証をしています。そして、あなたがたは私たちの証が真実であると知っています。  
<span class="verse-number">13&ensp;</span>書くことが数多くありましたが、しかし、今は筆とインクで書くつもりはありません。  
<span class="verse-number">14&ensp;</span>しかし、私はあなたにすぐに会い、面と向き合い、話すことを望んでいます。あなたに平安がありますように。我が友人たちから挨拶を贈ります。名指しで挨拶してください。  


[^1]: つまり、集まった人の前で。
