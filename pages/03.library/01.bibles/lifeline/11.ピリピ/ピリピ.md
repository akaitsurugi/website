# ピリピ

### ピリピ人への手紙

## 第1章

イエス・キリストの僕であるパウロとテモテ、ピリピにいるキリスト・イエスにあって、監督、執事たちと共にいる、すべての聖徒たちへ。  
<span class="verse-number">2&ensp;</span>私たちの**父**なる**神**と**主**イエス・キリストより、あなたがたに恵みと平安がありますように。  
<span class="verse-number">3&ensp;</span>あなたがたとのあらゆる思い出に、私は、私の**神**に感謝し、  
<span class="verse-number">4&ensp;</span>いつも、あなたがた全員のことを祈るたびに、私は喜びをもって祈っています。  
<span class="verse-number">5&ensp;</span>それは、最初の日から今に至るまで、福音に関してあなたがたと、親交があるからです。  
<span class="verse-number">6&ensp;</span>あなたがたの内に、よい働きを始められた御方が、キリスト・イエスの日に至るまで、このよい働きを果たされると私は確信しています。  
<span class="verse-number">7&ensp;</span>それと同様、あなたがたについて、こう考えるのは正しいのです。なぜなら、私はあなたがたを心の中に抱いているのですから、私の鎖につながる時、そして、この福音を守り、確認する時、あなたがた皆は、私の恵みを分かち合っているからです。  
<span class="verse-number">8&ensp;</span>私は、イエス・キリストの心によって、あなたがた全員をいかに熱心に求めているかを、私の証人は**神**であられます。  
<span class="verse-number">9&ensp;</span>そして、私はこう祈ります。すなわち、あなたがたの愛は、知識、そしてすべてを見分ける力によって、さらに、ますますあふれ出ますように。  
<span class="verse-number">10&ensp;</span>それは、キリストの日まで、あなたがたは真摯で、非のない者になれるよう、優れているものを見分けることができるよう、あなたがた自身で試すためです。  
<span class="verse-number">11&ensp;</span>**神**の栄光と誉れのために、イエス・キリストを通しての義の実であなたがたが満たされたからです。  
<span class="verse-number">12&ensp;</span>さて、兄弟たちよ、私に起こったことは、かえって福音の推進力になったことを知ってほしい。  
<span class="verse-number">13&ensp;</span>官邸全体に、そして他のすべての人にも、私の鎖は、キリストにあることと知れ渡りました。  
<span class="verse-number">14&ensp;</span>そして、**主**にある兄弟たちのほとんどは、私の鎖によって自信をもって、さらに堂々と、恐れず御言葉を説いています。  
<span class="verse-number">15&ensp;</span>ある者たちは、妬みと反目からキリストを説いています。また、ある者たちは善意から説いています。  
<span class="verse-number">16&ensp;</span>前者は、己の野心からキリストを説いており、真の心なく、私の鎖の苦痛を増やそうと考えています。  
<span class="verse-number">17&ensp;</span>後者は、私が福音の防御を任命されたと知り、愛からキリストを説いています。  
<span class="verse-number">18&ensp;</span>しかし、それが何でしょう。どちらの道も、口実であろうが、真実であろうが、キリストは説かれています。そして、私はこのことを喜び、またこれからも喜びます。  
<span class="verse-number">19&ensp;</span>なぜなら、あなたがたの祈りとイエス・キリストの**御霊**が下さるものを通して、これが、救い出されることになると、私は知っているからです。  
<span class="verse-number">20&ensp;</span>これは、切なる期待と希望により、すなわち、なんら恥じることなく、勇敢に、常に、今も、キリストが私の体の中で、偉大になられるためです。私の生死に関わりありません。  
<span class="verse-number">21&ensp;</span>私にとって、生きることはキリストであり、死ぬこととは利益だからです。  
<span class="verse-number">22&ensp;</span>しかし、もし私が、肉体により生きるなら、それは私の労働が実となります。それでは、どちらを選ぶべきか、私には分かりません。  
<span class="verse-number">23&ensp;</span>しかし、私はこの二つ、生と死に押し潰されそうです。私が望むことは、世を去り、キリストと共になることです。その方がはるかによいのです。  
<span class="verse-number">24&ensp;</span>しかし、私が肉体に留まる方が、あなたがたにはより必要です。  
<span class="verse-number">25&ensp;</span>そして、私は留まって、引き続き、あなたがたの信仰の推進と喜びを共にすることは分かっています。私はこのことを確信しています。  
<span class="verse-number">26&ensp;</span>これは、あなたがたのところへ、私が今度現れることで、私の中におられるキリスト・イエスにより、あなたがたの誇りが満ち溢れるためです。  
<span class="verse-number">27&ensp;</span>ただ、キリストの福音にふさわしく市民として生活しなさい。そうすれば、私が行って、あなたがたに会っても、行かなくても、一つの霊にしっかり立ち、心を一つにして福音の信仰のため共に戦う、あなたがたのことを聞くでしょう。  
<span class="verse-number">28&ensp;</span>（そして、敵対する者たちのいかなるものにも、恐れてはいけません。）それは[^1]敵対する者にとっては破滅の証拠であるが、あなたがたにとっては救い、つまり、**神**からの救いの証拠です。  
<span class="verse-number">29&ensp;</span>なぜなら、キリストのためにあなたがたに与えられていることは、キリストを信じるだけではなく、キリストのために苦しむことです。  
<span class="verse-number">30&ensp;</span>あなたがたが私の中に見て、また今、聞いている苦闘が同じようにあなたがたに与えられているのです。  

## 第2章

従って、キリストに少しでも励みがあるなら、少しでも愛の慰めがあるなら、少しでも**御霊**の交流、少しでも心と哀れみがあるなら、  
<span class="verse-number">2&ensp;</span>あなたがたは同じ考え方と同じ愛をもって、同じ気持ちと、一致した心で、私の喜びで満たされなさい。  
<span class="verse-number">3&ensp;</span>何事も利己的な野望や、尊大によらず、かえって謙虚に、他の人は自分よりも、優れていると思いなさい。  
<span class="verse-number">4&ensp;</span>各自、自分のことだけでなく、他の人のことにも心を配りなさい。  
<span class="verse-number">5&ensp;</span>ですから、キリスト・イエスも持たれるこの考え方を、あなたがたも持ちなさい。  
<span class="verse-number">6&ensp;</span>イエスは、**神**の本性であられるのに、**神**と平等であられることを固守すべきとは思われず、  
<span class="verse-number">7&ensp;</span>イエスはご自分を空しくされ、僕の本性を受け入れられ、人間の姿になられました。  
<span class="verse-number">8&ensp;</span>そして、外観は人間として現れ、自らへりくだり、死に至るまで、つまり十字架の死に至るまでも、従順になられました。  
<span class="verse-number">9&ensp;</span>ゆえに、**神**もまたイエスを非常に高い地位につけられ、イエスにすべての名の上の御名をお与えになりました。  
<span class="verse-number">10&ensp;</span>それは、イエスの御名において、天上にいる者、地上にいる者、地下にいる者[^2]、すべての者がひざまずくためです。  
<span class="verse-number">11&ensp;</span>また、すべての舌は、**父**なる**神**の栄光に至り、イエス・キリストは**主**であられると明言するためです。  
<span class="verse-number">12&ensp;</span>ですから、私の愛する人たちよ、私がいる時だけではなく、今はなおさら、私の不在の時こそ、あなたがたはいつも聞き従ってきたように、最大限に、恐れおののき、あなた自身の救いを実行しなさい。  
<span class="verse-number">13&ensp;</span>なぜなら、**神**の善なる喜びを望ませ、実行させるように、働きかける方は、あなたがたの中にいる、**神**であられるからです。  
<span class="verse-number">14&ensp;</span>すべて、不平不満や言い争いをせずに行ないなさい。  
<span class="verse-number">15&ensp;</span>それは、歪曲し、邪悪なこの世代の中で、あなたがたは傷がなく、非の打ち所のない、**神**の純粋な子どもとなるためです。この邪悪な者たちの中で、あなたがたは世の光として輝いているのです。  
<span class="verse-number">16&ensp;</span>私の走りは無駄でなかったし、また無駄な働きもしなかったと、キリストの日に私が誇れるように、命の御言葉をしっかり堅持しなさい。  
<span class="verse-number">17&ensp;</span>万が一、私があなたがたの信仰の供え物と奉仕のために捧げられたとしても、私は幸せであり、あなたがた全員と共に喜びます。  
<span class="verse-number">18&ensp;</span>この同じ理由で、あなたがたも幸せになり、そして、私と共に喜びなさい。  
<span class="verse-number">19&ensp;</span>それで、あなたがたの近況を知れば、私も勇気づけられるから、私はすぐあなたがたのところに、テモテを遣わします。そうなるよう、**主**イエスにあって望んでいます。  
<span class="verse-number">20&ensp;</span>なぜなら、あなたがたの近況を、心底心配してくれる、彼と同じ心を持つ人は、私にはいません。  
<span class="verse-number">21&ensp;</span>なぜなら、人は皆、自分自身のことを求め、キリスト・イエスのことは求めていないからです。  
<span class="verse-number">22&ensp;</span>しかし、あなたがたは証明されたテモテの性格を知っています。つまり、テモテは父に仕える息子として、私と共に福音に仕えました。  
<span class="verse-number">23&ensp;</span>ゆえに、私の身辺の成り行きを見極めしだい、すぐにテモテを遣わそうと望んでいます。  
<span class="verse-number">24&ensp;</span>それで、私自身も近いうちに行けるものと、**主**を信頼しています。  
<span class="verse-number">25&ensp;</span>しかし、私の兄弟であり、共に働く者であり、戦友であり、また、あなたがたの使徒であり、私の不足を補ってくれたエパフロデトを、あなたがたのところに遣わすのが必要があると思いました。  
<span class="verse-number">26&ensp;</span>なぜなら、エパフロデトはあなたがた全員を深く慕っていたからです。また、あなたがたに病気だと伝わったので、エパフロデトがとても悩んでいました。  
<span class="verse-number">27&ensp;</span>というのは、エパフロデトは、実に死ぬほどの病気になりましたが、**神**は彼を哀れんでくださいました。さらに、**神**はエパフロデトだけではなく、私にも悲しみの上に悲しみを重ねないよう、私をも哀れんでくださいました。  
<span class="verse-number">28&ensp;</span>したがって、私はエパフロデトをとても切にあなたがたに遣わしました。あなたがたは再会を喜び、そして私の悲しみを和らげるためです。  
<span class="verse-number">29&ensp;</span>ですから、**主**にあって溢れる喜びを持って、エパフロデトを受け入れなさい。そして、こういう人々を尊敬しなさい。  
<span class="verse-number">30&ensp;</span>なぜなら、私への、あなたがたの奉仕が足りないところを補うため、エパフロデトは己の命を考えず、キリストの働きのため、死の直前に至ったのです。  

## 第3章

最後に、私の兄弟たちよ、**主**にあって喜びなさい。同じことを書きますが、私には煩わしくなく、あなたがたにとっては安全のためです。  
<span class="verse-number">2&ensp;</span>犬どもには気をつけなさい。悪をなす者たちにも気をつけなさい。切り離しをする者たちにも気をつけなさい。  
<span class="verse-number">3&ensp;</span>なぜなら、**御霊**によって**神**を礼拝し、キリスト・イエスのことを誇りとし、肉に信頼を置かない私たちこそが、割礼者なのです。  
<span class="verse-number">4&ensp;</span>にも関わらず、私も肉に信頼を置くことができます。誰か、肉に信頼を置くと考える人が他にいるなら、私はなおさらのことです。  
<span class="verse-number">5&ensp;</span>私は八日目に割礼を受け、イスラエルの国民であり、ベンヤミン族であり、ヘブル人らしいヘブル人であり、律法に関してはパリサイ人であり、  
<span class="verse-number">6&ensp;</span>熱心なことと言えば、教会を迫害し、律法にある義について言えば、非の打ち所のない者でした。  
<span class="verse-number">7&ensp;</span>しかし、私にとって利益であったものが、キリストのためには、損失であると私は思うに至りました。  
<span class="verse-number">8&ensp;</span>しかし、それゆえ、私の**主**キリスト・イエスを知るということは、比べようがないすばらしさのゆえに、すべてのものは損失であると、私は思うに至っています。私はキリストのためにはすべてのものを損失とし、キリストを得るためには、それらを廃棄物であると思うに至っています。  
<span class="verse-number">9&ensp;</span>そして、律法による自己本位の義を持たず、キリストへの信仰に基づき、信仰を通しての**神**からの義を持ち、キリストの内に私が見いだされるのです。  
<span class="verse-number">10&ensp;</span>それは、私がキリストの死に同化され、その方の復活の力と苦しみを共有することを理解するためです。  
<span class="verse-number">11&ensp;</span>それは、私がどうしても、死者たちからの復活を達成したいのです。  
<span class="verse-number">12&ensp;</span>私はもうそれを達成しているとか、または、完璧になったということではないのです。かえって、私はキリストに捕らえられた者なので、その捕らえられた目的となるものを得るため、私は追い続けています。  
<span class="verse-number">13&ensp;</span>兄弟たちよ、すでに追いついたとは、自ら結論を出していません。しかし、私がしようとしていることは一つです。つまり、後にあるものを忘れ、前にあるものに手を伸ばし、  
<span class="verse-number">14&ensp;</span>キリスト・イエスにある**神**の高い召命の賞のため、目的地を目指し、前進しています。  
<span class="verse-number">15&ensp;</span>ゆえに、成熟している私たちは、このように考えましょう。そして、もし何か違う考えを、あなたがたがもつなら、**神**はそれを明らかにしてくださいます。  
<span class="verse-number">16&ensp;</span>ともかく、この同じ原則により、すでに達した所まで、同じことを考えて歩みましょう。  
<span class="verse-number">17&ensp;</span>兄弟たちよ、共に私のことを模範とする人になりなさい。そして、私たちを模範として、同じように歩む人たちに心を留めなさい。  
<span class="verse-number">18&ensp;</span>なぜなら、私が幾度も話した人々のことを、今も涙して書いているのですが、大勢の人々が、キリストの十字架の敵として歩んでいるのです。  
<span class="verse-number">19&ensp;</span>この者たちの行き着くところは破滅であり、彼らの**神**は彼らの腹[^3]であり、彼らの栄光は恥の中にあり、彼らは世俗的に考えます。  
<span class="verse-number">20&ensp;</span>しかし、私たちの国籍は天国にあります。そこから私たちは**救い主**、すなわち**主**イエス・キリストを渇望しているのです。  
<span class="verse-number">21&ensp;</span>**主**は何事もご自分に従わせることができる働きにより、ご自分の栄光の体の形になるように、私たちの粗末な体を変身させてくださいます。  

## 第4章

ですから、私の敬愛する心を慕う兄弟たち、私の喜びであり、冠でもある兄弟たちよ。このように**主**にあってしっかり立ちなさい。愛しい人たちよ。  
<span class="verse-number">2&ensp;</span>**主**にあって同じ考えを、私はユウオデヤに勧め、またスントケにも勧めます。  
<span class="verse-number">3&ensp;</span>そして、真の仲間よ、この婦人たち、ユウオデヤとスントケを助けてくれるよう、あなたにもお願いします。この婦人たちは、命の書に名前が記載されている、クレメンスや、他の私の同労者たちと福音のため、私と共に、働いたのです。  
<span class="verse-number">4&ensp;</span>いつも**主**にあって喜びなさい。もう一度言います。喜びなさい。  
<span class="verse-number">5&ensp;</span>あなたがたの寛容をすべての人に知らせなさい。**主**はすぐ側におられます。  
<span class="verse-number">6&ensp;</span>何も心配せず、感謝をもって、すべてのことについて祈りと願いで、あなたがたの求めていることを、**神**に知っていただきなさい。  
<span class="verse-number">7&ensp;</span>そして、人智を超える**神**の平安が、キリスト・イエスにあって、あなたがたの心と思いを守ります。  
<span class="verse-number">8&ensp;</span>最後に、兄弟たちよ、すべての真実なこと、すべての威厳のあること、すべての正しいこと、すべての清いこと、すべての愛すべきこと、すべての誉められるべきこと、何であれ徳があったら、何らかの賞賛できるなら、これらのことを深く考えなさい。  
<span class="verse-number">9&ensp;</span>私をとおして学んだこと、受け入れたこと、聞いたこと、見たことを実行しなさい。そして、平安の**神**はあなたがたと共におられます。  
<span class="verse-number">10&ensp;</span>しかし、私が**主**にあって大いに喜んだのは、今ついにあなたがたの私への心配が、再び、今すでに実ったからです。心配してくれたのですが、あなたがたには機会がありませんでした。  
<span class="verse-number">11&ensp;</span>私は困窮しているから、言っているわけではありません。私はいかなる状態であっても、満足することを学んだからです。  
<span class="verse-number">12&ensp;</span>私は辱めも豊かさも知っています。どんな所でも、またどんな事にも、満腹と空腹、繁栄と衰退を身につけました。  
<span class="verse-number">13&ensp;</span>私を強めてくださるキリストにより、私は何でもできます。  
<span class="verse-number">14&ensp;</span>にも関わらず、あなたがたは私の苦難を良くぞ共にしてくれました。  
<span class="verse-number">15&ensp;</span>さて、福音の宣教の開始、私がマケドニアから出かけた時、あなたがた以外、どの教会も、私と物のやり取りに関しては協力してくれなかったことを、あなたがたピリピ人も知っています。  
<span class="verse-number">16&ensp;</span>というのは、テサロニケにいた私にさえ、一再ならず、私の窮乏のため援助を送ってくれました。  
<span class="verse-number">17&ensp;</span>私が求めているのは贈り物ではなく、あなたがたの帳簿の溢れる実りです。  
<span class="verse-number">18&ensp;</span>それで、あなたがたからのものを、エパフロデトから受け取り、私は、何でもあり余るほど持っていて、満ちています。これらは**神**には、香ばしい香りで、ふさわしい捧げ物であり、好ましいものです。  
<span class="verse-number">19&ensp;</span>そして、キリスト・イエスにより、栄光にある**神**の豊かさをもって、私の**神**は、あなたがたの必要をすべて満たしてくださいます。  
<span class="verse-number">20&ensp;</span>それでは、私たちの**父**なる**神**に、栄光が世々永遠にありますように。アーメン。  
<span class="verse-number">21&ensp;</span>キリスト・イエスにある、すべての聖徒たちに挨拶してください。私と共にいる兄弟たちも、あなたがたに挨拶を送ります。  
<span class="verse-number">22&ensp;</span>すべての聖徒たちはあなたがたに挨拶を送ります。特に、カイザル家所属の人たちから、挨拶を送ります。  
<span class="verse-number">23&ensp;</span>私たちの**主**イエス・キリストの御恵みがあなたがた全員と共にありますように、アーメン。  


[^1]: 恐れないこと。  
[^2]: 「Underworld」、かかわりを持った悪霊と一緒に地下にいる者。  
[^3]: 当時の人には、腹は食欲、欲望の象徴であった。
