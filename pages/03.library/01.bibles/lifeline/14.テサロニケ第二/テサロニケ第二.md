# テサロニケ第二

### テサロニケ人への手紙　第二

## 第1章

パウロ、シルワノ、テモテより、私たちの**父**なる**神**と**主**イエス・キリストにあるテサロニケの人たちの教会へ。  
<span class="verse-number">2&ensp;</span>私たちの**父**なる**神**と**主**イエス・キリストより、あなたがたへ恵みと平安がありますように。  
<span class="verse-number">3&ensp;</span>兄弟たちよ、私たちが、常にあなたがたのことで**神**に感謝をするのは当然のことです。なぜなら、あなたがたの信仰は大いに成長しており、そして、あなたがた一人一人、互いの愛は豊かになっているからです。  
<span class="verse-number">4&ensp;</span>ですから、すべての迫害と苦しみにあって、あなたがたの示す忍耐と信仰を、**神**の諸教会の間で、私たちは自慢しています。  
<span class="verse-number">5&ensp;</span>**神**の王国にふさわしいと認められるため、**神**の裁きは正しいという証拠です。あなたがたもその王国のため、苦しんでいるのです。  
<span class="verse-number">6&ensp;</span>当然、あなたがたを苦しませる者たちに、**神**は苦で報います。それは**神**にとって正義だからです。  
<span class="verse-number">7&ensp;</span>また、**主**イエスがご自分の力ある御使いたちと共に天から現れ、苦しんでいるあなたがたに、私たちと共に、休息を下さることも、**神**の正義です。  
<span class="verse-number">8&ensp;</span>**主**はその時、**神**を知らない者たちと、私たちの**主**イエス・キリストの福音に従わない者たちに、燃えさかる復讐の火を与えられます。  
<span class="verse-number">9&ensp;</span>**主**の御前から、また**主**の御力の栄光から、この者たちは永遠の破滅に至る罰に処せられます。  
<span class="verse-number">10&ensp;</span>それは、あなたがたの間で、私たちの証は信じられたので、**主**がおいでになるその日、ご自分の聖徒たちの間で栄光を受けられるため、信じる者たちの間で賞賛されるために来られます。  
<span class="verse-number">11&ensp;</span>したがって、**神**があなたがたをこの召命に記帳するのにふさわしいと認め、あらゆるよきことの善なる喜び、そして力が備わった信仰の働きで満たしてくださるように、私たちはいつも祈っています。  
<span class="verse-number">12&ensp;</span>そして、私たちの**神**と**主**イエス・キリストの御恵みにより、あなたがたの中で、私たちの**主**イエス・キリストの御名は、栄光をお受けになり、そしてあなたがたは**主**の中で栄光を与えられるように、私たちはいつも祈っています。  

## 第2章

さて、兄弟たちよ、**主**イエス・キリストの来臨と、イエスのところに集う、私たちの集まりのことで、皆さんにお願いします。  
<span class="verse-number">2&ensp;</span>つまり、霊からであろうが、言葉からであろうが、私たちから出たと言われるような手紙からであろうが、キリストの日がもうすでに来たと聞いても、簡単に動揺しないよう、お願いします。  
<span class="verse-number">3&ensp;</span>誰からも、いかなる方法でも、騙されないようにしなさい。背教がまずきて、そして、罪の者、つまり破滅の子が現れない限り、その日は来ません。  
<span class="verse-number">4&ensp;</span>破滅の子は、**神**と呼ばれ、礼拝されるすべてのものに敵対し、自らをその上の高きにつけます。それは、**神**として、**神**の神殿の中に座り、自らを**神**であると示すためです。  
<span class="verse-number">5&ensp;</span>私がまだ皆さんと共にいた時、これらのことを教えたのを、忘れたのですか。  
<span class="verse-number">6&ensp;</span>そして、時がきてその者が現れるのを、阻止しているものは何か、皆さんは今、知っています。  
<span class="verse-number">7&ensp;</span>なぜなら、不法の奥義はもうすでに働いています。ただ、今、阻止している方が取り除かれるまで抑えられています。  
<span class="verse-number">8&ensp;</span>そして、それから、不法の者が現れます。**主**はこの者をご自分の口からの息でせん滅させ、ご自分の来臨の光によって滅ぼされます。  
<span class="verse-number">9&ensp;</span>この不法の者が現れるのは、サタンの働きによるもので、あらゆる力、しるし、偽りの不思議な業によるものです。  
<span class="verse-number">10&ensp;</span>またそれは、救われるため、真理の愛を受け入れることなく、滅んでいく者たちの間に、あらゆる不義の欺瞞を使って現れます。  
<span class="verse-number">11&ensp;</span>そしてこのために、**神**を受け入れなかった者たちが、その嘘を信じるように、**神**はその者たちに強い思い込みを送られます。  
<span class="verse-number">12&ensp;</span>また、それは真理を信じないで、不義を楽しんだすべての者が裁かれるためです。  
<span class="verse-number">13&ensp;</span>しかし、**主**に愛されている兄弟たちよ、皆さんのことを、私たちがいつも**神**に感謝するのは当然です。なぜなら、**御霊**の聖別と真理を信じることにより、救いのため、**神**は最初から皆さんをお選びになったからです。  
<span class="verse-number">14&ensp;</span>私たちの**主**イエス・キリストの栄光を獲得させるため、**神**は私たちの福音によって、この真理に皆さんを召されたのです。  
<span class="verse-number">15&ensp;</span>従って、兄弟たちよ、しっかり立ち、言葉や、また私たちの手紙によってあなたがたに教えられた伝統を握りしめなさい。  
<span class="verse-number">16&ensp;</span>では、私たちを愛し、御恵みにより永遠の慰めと、すばらしい希望を与えてくださった、私たちの**主**イエス・キリストご自身と、私たちの**神**であられ、**御父**であられる方が、  
<span class="verse-number">17&ensp;</span>皆さんの心を慰め、皆さんが使うすべての言葉と行ないが、善きものとなるよう、**主**が立たせてくださいますように。  

## 第3章

最後に、兄弟たちよ、あなたがたの所と同様に、**主**の御言葉がすばやく走り、そして栄光を受けるように、私たちのために祈ってください。  
<span class="verse-number">2&ensp;</span>また、私たちが無法な者たちから、そして悪者たちから救い出されるように祈ってください。なぜなら、すべての人が信仰を持っている訳ではないからです。  
<span class="verse-number">3&ensp;</span>しかし、皆さんをしっかり立たせ、悪魔から守ってくださる**主**は、真実な方です。  
<span class="verse-number">4&ensp;</span>そして、あなたがたは私たちが命じることを実行し、また今後も実行すると、**主**にあって私たちは確信しています。  
<span class="verse-number">5&ensp;</span>そして**主**が、皆さんの心を**神**の愛の中へ、そしてキリストの忍耐の中へ導いてくださいますように。  
<span class="verse-number">6&ensp;</span>兄弟たちよ、私たちの**主**イエス・キリストの御名により、私たちはあなたがたに命じます。私たちから受け入れた教理によってではなく、無秩序に歩むすべての兄弟たちから離れなさい。  
<span class="verse-number">7&ensp;</span>なぜなら、私たちは皆さんの間で無秩序でなかったから、皆さんたち自ら、模範として私たちにどう従うべきかを、以前から理解しています。  
<span class="verse-number">8&ensp;</span>そして、私たちは他人のパンを只で食べませんでした。かえって、皆さんの誰の重荷にもならないよう、夜も昼も疲労困憊するまで働き、努めを果たしました。  
<span class="verse-number">9&ensp;</span>それは私たちに権威がない訳ではなく、あなたがたがどのように、私たちの模範に従うべきか、私たち自身が模範を示すためでした。  
<span class="verse-number">10&ensp;</span>なぜなら、私たちが皆さんと共にいた時でさえ、私たちは皆さんにこう命じました。「誰であれ働かないのなら、その者は食べてはいけない。」  
<span class="verse-number">11&ensp;</span>あなたがたの間を無秩序に歩み、まったく働かず、人のことに口を出す者がいると、私たちは聞いているからです。  
<span class="verse-number">12&ensp;</span>それで、そういう者たちも静かに働き、己のパンを食べるように、私たちの**主**イエス・キリストを通して私たちは命じ、かつ勧めます。  
<span class="verse-number">13&ensp;</span>しかし、皆さん、兄弟たちよ、良い行ないをするのに、たゆまず良い行ないに励みなさい。  
<span class="verse-number">14&ensp;</span>誰であれ、この手紙にある言葉に従わない者がいれば、その者が恥じ入るよう、交際してはいけません。  
<span class="verse-number">15&ensp;</span>しかし、その人を敵と判断せず、兄弟として諭しなさい。  
<span class="verse-number">16&ensp;</span>では、平和の**主**ご自身が、どんな状況下でも、あらゆる方法で、あなたがたに平和を与えられますように。**主**があなたがた全員と共におられますように。  
<span class="verse-number">17&ensp;</span>この挨拶は、私パウロ自らの手によるものです。どの手紙にもこう私の手で書いたのは、私パウロが書いたというしるしです。  
<span class="verse-number">18&ensp;</span>私たちの**主**イエス・キリストの御恵みが、皆さん全員と共にありますように。アーメン。
