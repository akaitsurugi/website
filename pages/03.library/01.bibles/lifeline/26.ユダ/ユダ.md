# ユダ

### ユダの手紙

イエス・キリストの僕、ヤコブの兄弟であるユダより、**父**なる**神**によって聖別され、イエス・キリストによって、庇護の下にあり、招かれた人たちへ。  
<span class="verse-number">2&ensp;</span>哀れみと平和と愛が、幾重にもあなたがたにありますように。  
<span class="verse-number">3&ensp;</span>愛する人々よ、私たちが共有する同じ救いについて、あなたがたに、真剣に書いていると、信仰のため、熱心に戦うことを勧める要があると気が付きました。その信仰とは、その昔、聖徒たちに、たった一度の与えられた信仰です。  
<span class="verse-number">4&ensp;</span>なぜなら、ある者たちが、密かに忍び込んでいるからです。昔、この裁きにつけるべき者たちとして、書き留められた不敬虔な者たちです。**神**の御恵みを際限のない欲望に変え、唯一の**主**であられる**神**と、私たちの**主**イエス・キリストを否定している者たちです。  
<span class="verse-number">5&ensp;</span>しかし、**主**がエジプトの地から民を救ってから、後で信じない者たちを破滅させられたことを、あなたがたは、以前は知っていました。このことを改めて思い出してもらおうと私は決めました。  
<span class="verse-number">6&ensp;</span>また、自らの義務を守らず、己の居住地を離れた御使いたちを、**主**は大いなる日の裁きにそなえ、永遠の鎖で繋ぎ、暗やみの下に拘留されています。  
<span class="verse-number">7&ensp;</span>ソドムとゴモラ，そしてその周辺の町々もそれらと同様に、性的な罪に自らを渡してしまい、他の肉体を求めたので、見せしめとされ、永遠の火の刑罰に苦しんでいます。  
<span class="verse-number">8&ensp;</span>同様に、夢を見るこの者たちは肉体を汚し、権威を拒み、栄光ある者たちを侮辱しています。  
<span class="verse-number">9&ensp;</span>しかし、御使いの頂点にいるミカエルは、悪魔と対決し、モーセの亡きがらについて論争した時、悪魔に対して、あえて侮辱の攻撃の言葉を口にせず、「**主**がお前をたしなめるように。」と言いました。  
<span class="verse-number">10&ensp;</span>しかし、この者たちは、己の知らないことは何でも悪く言います。また、知性のない動物のように、本能のままで知ることの内に、自ら腐り果てています。  
<span class="verse-number">11&ensp;</span>この者たちに災いあれ！なぜなら、彼らはカインの道を歩み、貪欲に利益のためバラムの轍を踏み、コラの反乱で消滅ました。  
<span class="verse-number">12&ensp;</span>この者たちは、あなたがたの愛の宴の危険な暗礁です。臆することなく、あなたがたと共に宴の席につき、己たちのみに装います。この者たちは、風に運ばれる、水分のない雲であり、実のない晩秋の木で、二度死んでいるので、根から引き抜かれています。  
<span class="verse-number">13&ensp;</span>荒れ狂う海の波であり、自らの恥を泡にして出す、暗黒の闇が永遠に備えられた、行く所のない星たちです。  
<span class="verse-number">14&ensp;</span>ところで、アダムから七代目のエノクも、この者たちについて預言して、言いました。「見よ、**主**は一万人のご自分の聖徒たちと共に来られる。  
<span class="verse-number">15&ensp;</span>すべての者の裁きを行ない、そして、その者どもの中の、不敬虔の者どもすべて、不敬虔な方法で犯した、不敬虔な行為を、不敬虔な罪人どもが、キリストに対し、口にした赦しがたい言葉の数々を、吐いた者どもを彼らの罪に定めるためである。」  
<span class="verse-number">16&ensp;</span>この者どもは不平を言う者どもであり、不満を言う者どもであり、己が欲望のまま歩んでいます。彼らは大げさな言葉を吐き、利を得るためには人々にへつらいます。  
<span class="verse-number">17&ensp;</span>しかし、あなたがた、愛する人々よ、私たちの**主**イエス・キリストの使徒たちによって、以前話された言葉の数々を忘れてはいけません。  
<span class="verse-number">18&ensp;</span>それは、終わりの時代に、自らの不敬虔な欲望に従って歩む、あざける者たちが来ると、使徒たちがあなたがたに言ったからです。  
<span class="verse-number">19&ensp;</span>この者たちは、**御霊**を持たず、分裂を引き起こす、みだらな者たちです。  
<span class="verse-number">20&ensp;</span>しかし、あなたがた、愛する人々よ、あなたがたのもっとも聖なる信仰で自らを築き、**聖霊**により祈り、  
<span class="verse-number">21&ensp;</span>永遠の命に至る私たちの**主**イエス・キリストの哀れみを求めながら、**神**の愛の中であなたがた自身を守りなさい。  
<span class="verse-number">22&ensp;</span>そして、人を見極め、ある者は哀れみなさい。  
<span class="verse-number">23&ensp;</span>そして、他の人々は炎から引き上げ、恐怖心を与えて助け出し、肉により汚されたものは、衣服さえもいといなさい。  
<span class="verse-number">24&ensp;</span>さて、あなたがたをつまずきから守り、傷のない者として、大いなる喜びを持って、あの方の栄光の御前に、あなたがたを立たせることができる方、  
<span class="verse-number">25&ensp;</span>この知恵ある唯一の方、つまり、私たちの**救い主**であられる**神**に、栄光と威厳、支配力と権威が、今も永遠もありますように。アーメン。
