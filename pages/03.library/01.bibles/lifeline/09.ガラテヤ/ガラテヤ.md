# ガラテヤ

### ガラテヤ人への手紙

## 第1章

人間からでなく、また人間を通してではなく、イエス・キリストを通し、またイエスを死人たちの中から復活させられた**父**なる**神**を通し、使徒であるパウロより、  
<span class="verse-number">2&ensp;</span>また、私と共にいる兄弟たちより、ガラテヤ地方の諸教会へ。  
<span class="verse-number">3&ensp;</span>**父**なる**神**と私たちの**主**イエス・キリストより、あなたがたに恵みと平安がありますように。  
<span class="verse-number">4&ensp;</span>私たちの**神**であり、**父**であられる方の御意志により、この邪悪な時代から我々を救い出すため、**主**イエスは、ご自身を私たちの罪のために与えてくださいました。  
<span class="verse-number">5&ensp;</span>**父**なる**神**に、栄光が世々永遠にありますように。アーメン。  
<span class="verse-number">6&ensp;</span>キリストの御恵みにより、あなたがたを召された方から、あなたがたがかくも早く離れ、他の福音に向かったのを私は驚いています。  
<span class="verse-number">7&ensp;</span>これは福音と別のもので、単にある者たちがあなたがたを撹乱して、キリストの福音を歪曲させようと願っているのです。  
<span class="verse-number">8&ensp;</span>しかし、もし私たちにせよ、また天国からの御使いにせよ、私たちがすでに、あなたがたに説いた福音と違うどんな福音を説いても、その者を呪われ者とせよ。  
<span class="verse-number">9&ensp;</span>私たちは前に言ったように、私は今もう一度言います。誰であれ、あなたがたが受け入れた福音と違う福音を、あなたがたに説くなら、その者を呪われ者とせよ。  
<span class="verse-number">10&ensp;</span>なぜなら、今私は人間を確信させようとしているのか、それとも**神**をなのですか。はたまた、人間を喜ばそうとしているのですか。なぜなら、もし私が人間をまだ喜ばせているのなら、私はキリストの僕ではありません。  
<span class="verse-number">11&ensp;</span>しかし、兄弟たちよ、私が説いた福音は、人間によるものではないことを、私はあなたがたにはっきりさせておきます。  
<span class="verse-number">12&ensp;</span>なぜなら、私はこの福音を人間から受けたのでも、教えられたのでもなく、イエス・キリストの啓示を通して受けたものだからです。  
<span class="verse-number">13&ensp;</span>それに、以前、ユダヤ教であった私の行動を、あなたがたはすでに聞いています。私は**神**の教会を際限なく迫害し、荒らすところでした。  
<span class="verse-number">14&ensp;</span>そして、同胞中、多数の同年輩の者より、私はユダヤ教の中で上位にいた、父祖たちの伝統には誰よりも熱心でした。  
<span class="verse-number">15&ensp;</span>しかし、私の母の胎内から私を分かち、ご自分の御恵みを通して私を召された**神**が、  
<span class="verse-number">16&ensp;</span>異邦人の間でイエスの福音を説くため、私の内に**御子息**を啓示されたことを**神**がお喜びになった時、私は肉と血にはすぐに相談しませんでした。  
<span class="verse-number">17&ensp;</span>また、私より先に使徒であった人たちの所、エルサレムに上らず、私はアラビアに行き、ダマスカスに戻りました。  
<span class="verse-number">18&ensp;</span>それから三年後、私はペテロに会って確かめるため、エルサレムに上りました。そして、彼と共に十五日間滞在しました。  
<span class="verse-number">19&ensp;</span>しかし、**主**の弟であるヤコブ以外、他の使徒には会いませんでした。  
<span class="verse-number">20&ensp;</span>それで、私があなたがたに書くことに関しては、**神**の御前で、見よ、私はうそをつきません。  
<span class="verse-number">21&ensp;</span>その後、私はシリア、キリキヤの地方に行きました。  
<span class="verse-number">22&ensp;</span>それで、キリストにある、ユダヤの諸教会では、私は顔では知られていませんでしたが、  
<span class="verse-number">23&ensp;</span>彼らは、「私たちを前に迫害していた人が、前に荒らそうとした信仰を今、説いています。」と聞いていただけでした。  
<span class="verse-number">24&ensp;</span>彼らは私のことで**神**に栄光を捧げていました。  

## 第2章

それから十四年後、私はバルナバと共に、テトスも連れ、再びエルサレムに上りました。  
<span class="verse-number">2&ensp;</span>そして、私は啓示によって上りました。それで、私が異邦人の間で説く福音をエルサレムの人たちに説明しました。これはどうしても私が走るため、また走ったことが無駄にならないよう、人望のある人たちに密かに伝えました。  
<span class="verse-number">3&ensp;</span>しかし、私と共にいたテトスでさえ、ギリシャ人であっても、割礼を強制されませんでした。  
<span class="verse-number">4&ensp;</span>しかし、密かに連れてこられた偽兄弟たちが現れました。私たちを奴隷にするため、彼らはキリスト・イエスによりもたらされた私たちの自由を探るため、忍び込みましたが、  
<span class="verse-number">5&ensp;</span>福音の真実があなたがたと共にとどまるよう、私たちは一時も彼らに従わず、一歩も譲りませんでした。  
<span class="verse-number">6&ensp;</span>しかし、ひとかどの人物に見えた者たちから――以前の現実は、私には何も関係はありません。**神**は人を外観で受け入れたりされません。――それで、ひとかどの人物に見えた者たちは、何も新しいことを私に付け加えませんでした。  
<span class="verse-number">7&ensp;</span>しかし、それどころか、割礼者への福音がペテロに委ねられたと同様に、非割礼者への福音は私に委ねられたことを、彼らがわかった時、  
<span class="verse-number">8&ensp;</span>（なぜなら、割礼者の使徒の役のために、ペテロを通して働いてくださった方は、また私を通して異邦人のために働いてくださったのです。）  
<span class="verse-number">9&ensp;</span>そして、柱と思われていたヤコブとケパとヨハネが、私に与えられた恵みを知った時、交流の右手を私とバルナバに差し出しました。これは、私たちが異邦人たちに行き、彼らは割礼者たちに行くためでした。  
<span class="verse-number">10&ensp;</span>貧しい人々を私たちが心に留めることだけをその三人が求めました。このことこそ、私が熱心に努力したことです。  
<span class="verse-number">11&ensp;</span>ところが、ペテロがアンテオケに来た時、ペテロは非難されるべきでしたから、私は面と向かって彼に反対しました。  
<span class="verse-number">12&ensp;</span>なぜなら、ある人々がヤコブのところから来る以前、ペテロは異邦人たちと共によく食事をしていました。しかし、その人々が来ると、ペテロは割礼の人々を恐れ、退き、自ら去りました。  
<span class="verse-number">13&ensp;</span>そして、残りのユダヤ人たちもペテロと共に偽善を行なったから、バルナバでさえ彼らの偽善に魅了されました。  
<span class="verse-number">14&ensp;</span>しかし、彼らは福音の真理に関して、正しく歩まないのを見た時、私は全員の前でペテロに言いました。「ユダヤ人でありながら、ユダヤ人の暮らしをせず、異邦人の暮らしをしているのなら、なぜ異邦人にユダヤ人の生活を強制しますか。」  
<span class="verse-number">15&ensp;</span>私たち生まれながらのユダヤ人であり、異邦人なる罪人ではなく、  
<span class="verse-number">16&ensp;</span>律法の働きによってでなく、キリスト・イエスに対する信仰を通して人は義と認められることを知り、私たちもキリスト・イエスを信じました。それは律法の働きによらずに、キリストに対する信仰によって、私たちは義と認められるためです。律法の働きで、肉である者は誰も義と認められないからです。  
<span class="verse-number">17&ensp;</span>しかし、もしキリストによって義と認められようと求めながら、私たちでさえ罪人と分かったとしたら、キリストは罪の助力者でしょうか。断じてそうではありません。  
<span class="verse-number">18&ensp;</span>なぜなら、もし私が破壊したものを再建するなら、私は自らを違反者であると自己宣伝をしています。  
<span class="verse-number">19&ensp;</span>それで、**神**に対して生きるため、私は律法を通し律法に死にました。  
<span class="verse-number">20&ensp;</span>キリストと共に私は十字架につけられました。そして、生きているのは、もう私自身ではなく、私の中に生きておられるキリストです。そして、今、肉に生きている私の命は、私を愛し、ご自分を私のために与えてくださった、**神の御子息**に対する信仰によって生きています。  
<span class="verse-number">21&ensp;</span>私は**神**の御恵みを空しくしていません。なぜなら、もし義が律法を通してくるとすれば、それではキリストは空しく死なれたことになります。  

## 第3章

あぁ、愚かなガラテヤ人たちよ。イエス・キリストがあなたがたの目の前で十字架につけられたのを、あなたがたの間で描いたのに、誰があなたがたに魔法をかけ、真理に従うことのないようにしたのですか。  
<span class="verse-number">2&ensp;</span>このことだけはあなたがたから、私は知りたい。あなたがたは、律法による行為から**御霊**を受けたのですか。それとも、信仰で聞くことからですか。  
<span class="verse-number">3&ensp;</span>あなたがたはそんなに愚かなのですか。**御霊**で始めたのに、今、肉によって完成されたと言うのですか。  
<span class="verse-number">4&ensp;</span>あなたがたは、かくも多くのことを、無意味に苦しんだのですか。もし本当に無意味であったとすれば…。  
<span class="verse-number">5&ensp;</span>それでは、あなたがたに**御霊**を与えて下さり、あなたがたの間で力ある御業を行なわれる方は、それを律法を通して行なわれるのですか、信仰で聞くことを通して行なわれるのですか。  
<span class="verse-number">6&ensp;</span>それと同じように、「アブラハムは**神**を信じた。そして、**神**はそれを彼にとって義と計算された。」  
<span class="verse-number">7&ensp;</span>従って、信仰の者たちのみ、その者たちがアブラハムの子たちであることを知りなさい。  
<span class="verse-number">8&ensp;</span>そして、**神**は異邦人たちを信仰によって義と認められると聖書は、予知し、前もってアブラハムに福音を説きました。「あなたにおいてすべての国民は祝福される。」  
<span class="verse-number">9&ensp;</span>それで、信仰の人々は、忠実なアブラハムと共に祝福されます。  
<span class="verse-number">10&ensp;</span>なぜなら、一人一人律法の働きの者たちは、呪いの下にいます。なぜなら、「律法の書にすべしと書いてあるすべてのことを継続し、それらを実行しない者は、呪われている。」と書いてあるからです。  
<span class="verse-number">11&ensp;</span>しかし、**神**の御前には、律法によって義と認められる者は誰もいない、ということは明らかです。なぜなら、「義人は、信仰によって生きる」からです。  
<span class="verse-number">12&ensp;</span>しかし、律法は、信仰からではありません。かえって、「律法を行なう者は、律法によって生きるべし。」  
<span class="verse-number">13&ensp;</span>キリストは、私たちの身代わりに、呪われた者となり、律法の呪いから私たちを買い戻されました。なぜなら、「すべて木に掛かる者は、呪われている。」と書いてあるからです。  
<span class="verse-number">14&ensp;</span>それは、イエス・キリストによって、アブラハムの祝福が異邦人に行くためであり、私たちは信仰を通して**御霊**の約束を受けるためでした。  
<span class="verse-number">15&ensp;</span>兄弟たちよ、私は人間の作法で話します。人間の契約でさえ、一旦結ばれると誰もそれを無効にし、またはそれに付け加えることはありません。  
<span class="verse-number">16&ensp;</span>さて、アブラハムと彼の種にこの約束はされました。**神**は、「そして、種たちに」、と、大勢を指すのではなく、一人を指すように言われました。「そして、あなたの種に」、つまりその種はキリストであられます。  
<span class="verse-number">17&ensp;</span>それで、私が言いたいのはこうです。**神**によりキリストに関して、先に確定された契約を、四百三十年後つくられた律法が、その約束を無とするため、契約を破棄することはありません。  
<span class="verse-number">18&ensp;</span>なぜなら、もし相続が律法からくるものなら、すでに約束からのものではありません。しかし、**神**は約束によりアブラハムに相続を与えられました。  
<span class="verse-number">19&ensp;</span>では、律法はなぜありますか。違反のため付け加えられたのであり、約束されたその種がくるまで、御使いたちを通して、仲介者の手によって定められました。  
<span class="verse-number">20&ensp;</span>しかし、仲介者は一方のためだけではないが、**神**は一方の方です。  
<span class="verse-number">21&ensp;</span>では、律法は**神**の約束に反しているでしょうか。断じてそうではありません。なぜなら、もし命を与えることができる律法が与えられたとすれば、間違いなく義は律法から出たでしょう。  
<span class="verse-number">22&ensp;</span>しかし、イエス・キリストに対する信仰によって、約束は信じる人々に与えられるように、聖書はすべての人々を罪の下に囲い込んだのです。  
<span class="verse-number">23&ensp;</span>しかし、信仰がくる前は、私たちは律法により守られ、後に明らかになる信仰がくるまで囲い込まれていました。  
<span class="verse-number">24&ensp;</span>それで、私たちが信仰によって義と認められるため、律法が我々をキリストに導く教育係です。  
<span class="verse-number">25&ensp;</span>しかし、信仰は来たので、私たちはもう教育係の下にいません。  
<span class="verse-number">26&ensp;</span>なぜなら、あなたがた全員、キリスト・イエスに対する信仰を通して、**神**の子どもだからです。  
<span class="verse-number">27&ensp;</span>それで、キリストの中へ浸礼された一人一人は、キリストを着たからです。  
<span class="verse-number">28&ensp;</span>ユダヤ人もギリシャ人もなく、奴隷も自由人もなく、男も女もありません。なぜなら、あなたがた全員は、キリスト・イエスにあって一つだからです。  
<span class="verse-number">29&ensp;</span>そして、あなたがたがキリストのものであるなら、アブラハムの種であり、そして約束により、相続人です。  

## 第4章

では、私は言います。相続人は子どもである間は、すべてのものの所有者であるが、奴隷との相違はなく、  
<span class="verse-number">2&ensp;</span>父が決めた日まで、相続人は家令や執事の下にいるに過ぎません。  
<span class="verse-number">3&ensp;</span>同様に、私たちも子どもであった時、世の基本の考えの下では、奴隷状態でした。  
<span class="verse-number">4&ensp;</span>しかし、時の満ちる時期が来て、**神**はご自分の**御子息**を遣わされ、女から生まれさせ、律法の下に生まれさせられました。  
<span class="verse-number">5&ensp;</span>それは、養子として我らが身分を受け入れるため、律法の下にいる者たちを**神**は買い戻し、  
<span class="verse-number">6&ensp;</span>そして、あなたがたは子であるがゆえに、「アバ[^1]、**父**よ」と叫ぶご自身の**御子息**の**御霊**を、あなたがたの心の中へ**神**は遣わされたのです。  
<span class="verse-number">7&ensp;</span>そのように、あなたがたはもう、奴隷ではなく子です。そして、子なら、キリストを通して**神**の相続人です。  
<span class="verse-number">8&ensp;</span>しかし、あなたがたは**神**を本当に知らなかった時、あなたがたは、本来神々ではないものに仕えました。  
<span class="verse-number">9&ensp;</span>しかし今、**神**を知ったので、いや、**神**により知られたので、どうしてもう一度向きを変え、再び弱く貧しい世の基本に改めて奴隷になりたいのですか。  
<span class="verse-number">10&ensp;</span>あなたがたは、日々、月々、各季節、各年を守っています。  
<span class="verse-number">11&ensp;</span>あなたがたのため私が労働したことが、無駄であったかと、あなたがたのことを、私は心配しています。  
<span class="verse-number">12&ensp;</span>兄弟たちよ、私もあなたがたのようになったのであるから、私のようにあなたがたもなることを私は求めます。あなたがたは全く私を損ねることをしなかったが、  
<span class="verse-number">13&ensp;</span>最初、私は肉の弱さのため、あなたがたに福音を説いたことをあなたがたは知っています。  
<span class="verse-number">14&ensp;</span>そして、私の肉にある私の試練を、あなたがたは軽蔑せず、また拒否もせず、それどころか、あなたがたは私を**神**の使いとして、またキリスト・イエスとして受け入れてくれました。  
<span class="verse-number">15&ensp;</span>では、あなたがたのあの祝福はどこへ行ったのですか。私はあなたがたについて証します。もしできることなら、あなたがたが目をえぐり出し、私に差し出すと…。  
<span class="verse-number">16&ensp;</span>それでは、あなたがたに真理を言うので、私はあなたがたの敵になったのですか。  
<span class="verse-number">17&ensp;</span>彼らはあなたがたを熱心に求めているが、善意からではありません。むしろ、あなたがたが彼らのために熱心になるよう、あなたがたを排除したいのです。  
<span class="verse-number">18&ensp;</span>しかし、私があなたがたと共にいる時だけでなく、善意からいつも熱心に求められるのはよいことです。  
<span class="verse-number">19&ensp;</span>私の子どもたちよ、キリストがあなたがたの中で形づくられるまで、私はまた産みの苦しみをしています。  
<span class="verse-number">20&ensp;</span>しかし、私は今あなたがたの側にいて、私の声の調子を変えたいです。なぜなら、私はあなたがたのことで困っているからです。  
<span class="verse-number">21&ensp;</span>私に言いなさい。律法の下にいたい者たちよ、律法を聞かないのですか。  
<span class="verse-number">22&ensp;</span>なぜなら、アブラハムに二人の息子がいました。一人は女奴隷から、一人は自由の女から生まれた、と書いてあります。  
<span class="verse-number">23&ensp;</span>しかし、女奴隷からの子は肉によって生まれたが、自由の女から生まれた子は、約束によって生まれました。  
<span class="verse-number">24&ensp;</span>これらのことは、アレゴリー[^2]です。なぜなら、この女たちは二つの契約です。一つはシナイ山から出て、奴隷の子を産みました。それはハガルです。  
<span class="verse-number">25&ensp;</span>なぜなら、ハガルは、アラビアにあるシナイ山であり、今のエルサレムに相当します。エルサレムは母子共に奴隷です。  
<span class="verse-number">26&ensp;</span>しかし、上にあるエルサレムは自由であり、私たち全員の母です。  
<span class="verse-number">27&ensp;</span>なぜなら、書いてあります。「おう、不妊の女よ、子を産めない女よ、喜べ。産みの苦しみのない女よ、さあ、喜び叫べ。見放された女は、夫のいる女より、子が多くいるからである。」  
<span class="verse-number">28&ensp;</span>それで兄弟たちよ、イサクのように、私たちも約束の子どもです。  
<span class="verse-number">29&ensp;</span>しかし、その時、肉によって生まれた人が、**御霊**によって生まれた人を迫害したように、今も同様です。  
<span class="verse-number">30&ensp;</span>しかし、聖書は何と言っていますか。「女奴隷とその子を追放せよ。なぜなら、女奴隷の子は決して、自由の女の子とともに相続できない。」とあります。  
<span class="verse-number">31&ensp;</span>それで兄弟たちよ、私たちは女奴隷の子どもではなく、自由の女の子どもです。  

## 第5章

したがって、キリストが私たちを自由にしてくださいました。その自由の中にしっかり立ち、再び奴隷の首輪に繋がってはいけません。  
<span class="verse-number">2&ensp;</span>見よ。私、パウロはあなたがたに言います。もしあなたがたは割礼を受ければ、キリストはあなたがたにとって無益となります。  
<span class="verse-number">3&ensp;</span>そして、すべて割礼を受ける者に、私は再度証します。その者は律法全体を守る債務があります。  
<span class="verse-number">4&ensp;</span>律法によって義と認められようとするあなたがたは、キリストからの繋がりが切れ、御恵みから離れ落ちたのです。  
<span class="verse-number">5&ensp;</span>なぜなら、私たちは**御霊**を通し、信仰により、義の希望を熱望しています。  
<span class="verse-number">6&ensp;</span>キリスト・イエスにあって、愛を通して働く信仰以外、割礼であろうが、不割礼であろうが、役に立たないからです。  
<span class="verse-number">7&ensp;</span>あなたがたはよく走って来ました。誰があなたがたを妨害したか、すなわち、真理に従うなと。  
<span class="verse-number">8&ensp;</span>この誘いは、あなたがたを召してくださる方からのものではありません。  
<span class="verse-number">9&ensp;</span>わずかなパン種が、パン生地全体を発酵させます。  
<span class="verse-number">10&ensp;</span>あなたがたが二心でないことを、**主**にあって私は確信しています。しかし、誰であれ、あなたがたをかき乱す者は、そのさばきを負います。  
<span class="verse-number">11&ensp;</span>そして兄弟たちよ、もし私はまだ割礼を説いていたとしたら、なぜ私はまだ迫害にあうのですか。そうしていたなら、十字架のつまずきが取り除かれているのです。  
<span class="verse-number">12&ensp;</span>あなたがたを騒がす者たちが、彼ら自身が自らを斬り捨てることさえ、望んでいます。  
<span class="verse-number">13&ensp;</span>なぜなら、兄弟たちよ、あなたがたは自由に召されたと言って、肉のための機会として自由を使うためでなく、愛を通して互いに仕えなさい。  
<span class="verse-number">14&ensp;</span>なぜなら、律法全体は、一言で十分です。つまり、「あなたの隣人を、自分自身のように愛しなさい。」  
<span class="verse-number">15&ensp;</span>しかし、もしあなたがたが互いに噛み付き合い、また、互いにむさぼれば、互いに滅ぼされないように気をつけなさい。  
<span class="verse-number">16&ensp;</span>では、私は言います。**御霊**によって歩みなさい。そうすると、あなたがたは肉の欲望を満たすことは決してありません。  
<span class="verse-number">17&ensp;</span>なぜなら、肉は**御霊**に敵対することを欲します。そして、**御霊**は肉に敵対することを欲します。この二つは互いに敵対しています。ですから、あなたがたはしたいことをすることができないのです。  
<span class="verse-number">18&ensp;</span>しかし、もしあなたがたが**御霊**によって導かれているなら、あなたがたは律法の下にいません。  
<span class="verse-number">19&ensp;</span>では、肉の行為は明白であり、これらです――姦淫、性的な罪、汚れ、好色、  
<span class="verse-number">20&ensp;</span>偶像崇拝、魔術、敵意、争い、ねたみ、怒り、不正競争、分裂、分派を起こすこと、  
<span class="verse-number">21&ensp;</span>羨望、殺人、酩酊、酒宴、そのようなことです。私は以前に言ったように、先に言います。そのようなことを行なっている者たちは、**神**の王国を相続することはありません。  
<span class="verse-number">22&ensp;</span>しかし、**御霊**の実は、愛、喜び、平安、寛容、やさしさ、善良、忠実、  
<span class="verse-number">23&ensp;</span>柔和、自制心です。これらに敵対する律法はありません。  
<span class="verse-number">24&ensp;</span>そして、キリストの者たちは、肉とその情熱と欲望を共に十字架につけました。  
<span class="verse-number">25&ensp;</span>私たちは、**御霊**によって生きているなら、**御霊**によって歩もうではありませんか。  
<span class="verse-number">26&ensp;</span>私たちは、思い上がる者にならず、互いに争いを求めたり、互いに妬んだりしてはいけません。  

## 第6章

兄弟たちよ、もし人がどんな悪行であれ捕えられたなら、霊的であるあなたがたは、あなた自身もまた、誘惑されないように気をつけながら、そういう人を柔和な霊で正しなさい。  
<span class="verse-number">2&ensp;</span>互いの重荷を負い、そのようにしてキリストの律法を全うしなさい。  
<span class="verse-number">3&ensp;</span>なぜなら、何でもない只の人でありながら、自分を偉いと思うなら、その人は自分自身をだましています。  
<span class="verse-number">4&ensp;</span>しかし、各自、自らの働きを調べなさい。そうすれば、自慢することは他人に対してではなく、自らのことにだけです。  
<span class="verse-number">5&ensp;</span>なぜなら、各自、自らの責任を負うからです。  
<span class="verse-number">6&ensp;</span>また、御言葉を教えられる者は、すべてのよいものを、教えてくれる人と分かち合いなさい。  
<span class="verse-number">7&ensp;</span>迷わされてはいけません。**神**は侮られることはありません。なぜなら、蒔けば、その者がそれを刈り入れます。  
<span class="verse-number">8&ensp;</span>なぜなら、自らの肉のために蒔く者は、肉から腐敗物を刈り入れます。しかし、**御霊**のために蒔く者は、**御霊**から永遠の命を刈り入れます。  
<span class="verse-number">9&ensp;</span>そして、よいことをするのに、絶望してはいけません。なぜなら、失望しなければ、時期が来ると我々は刈り入れるからです。  
<span class="verse-number">10&ensp;</span>したがって機会があれば、すべての者たちに、特に信仰の家の者たちには、善を行ないましょう。  
<span class="verse-number">11&ensp;</span>私が自らの手で、どんな大きな字を書いたかをご覧なさい。  
<span class="verse-number">12&ensp;</span>肉にあって外見を立派に見せたい者たちは、キリストの十字架のため、ただ迫害されないように、あなたがたに割礼を受けるのを強制しています。  
<span class="verse-number">13&ensp;</span>なぜですか。割礼派の者たちでさえ、律法を守らないのに、あなたがたに割礼を受けさせようと求めるのは、彼らがあなたがたの肉を自慢したいためです。  
<span class="verse-number">14&ensp;</span>しかし、私たちの**主**イエス・キリストの十字架以外、私は断じて自慢してはいけません。これにより、世は私に対して十字架に架け、私は世に対して十字架に架けられたのです。  
<span class="verse-number">15&ensp;</span>なぜなら、キリスト・イエスにあっては、割礼も不割礼もなんら益はないが、益は新しい創造です。  
<span class="verse-number">16&ensp;</span>そして、この原則によって歩むすべての人々の上に、また**神**のイスラエルの上に、平安と哀れみがありますように。  
<span class="verse-number">17&ensp;</span>今後、誰も私を煩わせないでください。私は**主**イエスのしるしを身に負っているからです。  
<span class="verse-number">18&ensp;</span>兄弟たちよ、私たちの**主**イエス・キリストの御恵みが、あなたがたの霊と共にありますように。アーメン。  


[^1]: 当時のユダヤ人の言語であったアラム語で、「お父ちゃん」というような意味。  
[^2]: 意味が隠された話。寓話でも、比喩でもなく、象徴的な長い話。
