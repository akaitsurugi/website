# エペソ

### エペソ人への手紙

## 第1章

**神**のご意志を通して、イエス・キリストの使徒であるパウロより。エペソにいる聖徒たちと、キリスト・イエスにあって忠実な人たちへ。  
<span class="verse-number">2&ensp;</span>私たちの**父**なる**神**と**主**イエス・キリストより、あなたがたに恵みと平安がありますように。  
<span class="verse-number">3&ensp;</span>**神**、すなわち私たちの**主**イエス・キリストの**父**に、祝福がありますように。天の処所で、私たちをすべての霊的な祝福で、キリストによって祝福して下さり、  
<span class="verse-number">4&ensp;</span>私たちがキリストの御前に、愛により神聖で、非のうちどころのない者であるようにと、世の始まりの前に、**神**はキリストによって私たちをそのように選ばれました。  
<span class="verse-number">5&ensp;</span>それは、**神**のご意志の喜びに従い、私たちをイエス・キリストによってご自身の子とすることを、あらかじめ定められ、  
<span class="verse-number">6&ensp;</span>**主**は私たちを、愛する方にあって私たちを恵まれた者とあらかじめ定められたのは、**主**の恵みの栄光の賞賛のためです。  
<span class="verse-number">7&ensp;</span>**主**により、**主**の恵みの豊かさに応じ、**主**の血を通して私たちは贖いを、すなわち悪行の赦しを得ているのです。  
<span class="verse-number">8&ensp;</span>**主**はすべての知恵と知力によって、その恵みを私たちのために溢れさせ、  
<span class="verse-number">9&ensp;</span>自ら意図された喜びにより、ご自身のご意志の奥義を我々に明かになさいました。  
<span class="verse-number">10&ensp;</span>それは、時が満ちたことによる摂理の時代に、キリストにあるすべてのもの、すなわち天にあるものを、また地にあるものを、**主**はキリストのうちにまとめて集められるためです。  
<span class="verse-number">11&ensp;</span>すべてのことを、ご自身の定めにより行なう方のご計画のご意思により、あらかじめ定められた私たちも、相続の一つを得ました。  
<span class="verse-number">12&ensp;</span>それは、最初にキリストを信頼した私たちが、キリストの栄光の誉れとなるためです。  
<span class="verse-number">13&ensp;</span>真理の御言葉、すなわちあなたがたの救いの福音を聞き、あなたがたもキリストを信頼しました。あなたがたもキリストを信じたので、約束の**聖霊**によって証印を押されました。  
<span class="verse-number">14&ensp;</span>購入されたものが買い戻されるまで、**聖霊**は私たちの相続の保証です。それは、**主**の栄光の誉れです。  
<span class="verse-number">15&ensp;</span>したがって、**主**イエスに対するあなたがたの信仰と、すべての聖徒たちに対してあなたがたの愛を聞いた私も、  
<span class="verse-number">16&ensp;</span>祈るたび、あなたがたに触れ、あなたがたのために感謝することを、止めたりしません。  
<span class="verse-number">17&ensp;</span>それは、私たちの**主**イエス・キリストの**神**、すなわち栄光の**父**はあなたがたに、**神**の知識により、知恵と啓示の心を与えて下さり、  
<span class="verse-number">18&ensp;</span>**神**の召されることの希望は何かと知るため、また聖徒たちにある**神**の相続の栄光の富は何であるかを知るため、あなたがたの心の目は明るくされ、  
<span class="verse-number">19&ensp;</span>また、私たち信じる者に対して、いかに**神**の力が卓越であるかを、**神**の偉大な力ある働きにより、あなたがたが知るようになるためです。  
<span class="verse-number">20&ensp;</span>**神**はキリストを死人たちの中から復活させた時、その力をキリストのうちに行なわれ、そしてキリストを天の処所でご自分の右に座らせられました。  
<span class="verse-number">21&ensp;</span>すべての君主、権力者、実力者、支配者、および、この時代だけではなく、これから来る時代に、名前をつけて呼ばれるすべての名の、はるか上にキリストの御名を置かれました。  
<span class="verse-number">22&ensp;</span>そして、**神**はすべてのものをキリストの足の下に置かれ、またすべてのものの頭であるキリストを、教会に与えられました。  
<span class="verse-number">23&ensp;</span>教会とは、キリストの体であり、すべてを満たす方の満ちているそのものです。  

## 第2章

そして、悪行と罪によって死んでいた[^1]が、**神**によって生かされたあなたがたよ。  
<span class="verse-number">2&ensp;</span>かつて、この世の流れに従い、虚の権威の支配者、すなわち、不従順の子たちの中で今も働く霊に従い、あなたがたは歩んでいました。  
<span class="verse-number">3&ensp;</span>彼らの中で私たちも以前は、私たちの肉の欲望によって行ない、肉と知力の欲を満たし、他の者たちと同様、当然、怒りの子どもたちでした。  
<span class="verse-number">4&ensp;</span>しかし、哀れみに富む**神**は、私たちを愛してくださったご自身の偉大な愛ゆえに、  
<span class="verse-number">5&ensp;</span>私たちは悪行の中で死んでいた時にも、**神**は私たちをキリストと共に生かし（あなたがたが救われたのは、恵みによる）、  
<span class="verse-number">6&ensp;</span>私たちを共に復活させ、キリスト・イエスにあって天の諸処に座らせてくださいました。  
<span class="verse-number">7&ensp;</span>これは、キリスト・イエスにおいて、私たちに向けた、**神**の思いやりにより、恵みの卓越した豊かさを、これから来る各時代に、示すためでした。  
<span class="verse-number">8&ensp;</span>なぜなら、恵みにより、信仰を通してあなたがたは救われたのです。そして、それはあなたがた自身から出たのではありません。これは**神**の賜物です。  
<span class="verse-number">9&ensp;</span>行ないから出たのではありません。誰も自慢しないためです。  
<span class="verse-number">10&ensp;</span>私たちは**神**の作品なのですから、**神**が事前に用意なさった、よい業の中を歩むよう、キリスト・イエスの中に創造されたのです。  
<span class="verse-number">11&ensp;</span>従って、このことを忘れてはいけません。すなわち、肉にあって、かつては異邦人であったあなたがたは、手で肉に割礼をした者に無割礼の者と呼ばれています。  
<span class="verse-number">12&ensp;</span>当時のあなたがたは、キリストなしでイスラエルの国家から遠ざけられ、約束の契約には無縁であり、この世では希望なく、そして**神**もない人たちでした。  
<span class="verse-number">13&ensp;</span>しかし、かつて、キリスト・イエスにあって遠く離れていたあなたがたは、キリストの血によって、今は近い者とされました。  
<span class="verse-number">14&ensp;</span>二つを一体にし、中央の分離壁を破壊したキリストこそが、私たちの平和だからです。  
<span class="verse-number">15&ensp;</span>キリストはご自分の肉により、敵対するもの、すなわち、いろいろな法令を含む戒めの律法を破棄されたのは、この二つから一人の新しい人を彼ご自身で創り出し、平和を創り出すためです。  
<span class="verse-number">16&ensp;</span>そして、十字架を通して一体となった両者を、**神**に対して和解をさせ、それによって敵対するものを死滅させるためでした。  
<span class="verse-number">17&ensp;</span>そして、キリストは来られ、遠ざけられていたあなたがたに、そして近くにいた人たちに平和を説かれました。  
<span class="verse-number">18&ensp;</span>そのゆえ、キリストを通して私たち両者とも、一人の**御霊**によって、**父**のみもとに行くという到達があります。  
<span class="verse-number">19&ensp;</span>それで、今あなたがたはもう見知らぬ者たち、また外国人ではなく、聖徒たちと同国人であり、**神**の家の者です。  
<span class="verse-number">20&ensp;</span>その家とは、使徒たちと預言者たちの土台の上に建てられ、イエス・キリストご自身はその基準となるかなめ石です。  
<span class="verse-number">21&ensp;</span>キリストにあって合体させられた建物全体は、**主**にあって聖なる神殿に成長します。  
<span class="verse-number">22&ensp;</span>このキリストを通してあなたがたも**神**の住まわれる所として**御霊**にあって共に建てられています。  

## 第3章

そのため、あなたがた異邦人のため、キリスト・イエスの囚人である、私パウロは、  
<span class="verse-number">2&ensp;</span>私に与えられた、あなたがたのための、**神**の恵みの摂理について、あなたがたが、聞いたことがきっとあると思いますが、  
<span class="verse-number">3&ensp;</span>つまり、**神**は啓示によって私にその奥義を教えられました。（私は以前、手短に書いた通り、  
<span class="verse-number">4&ensp;</span>もしあなたがたはそれを読めば、キリストの奥義に関して私の知識のほどが分かるはずです。）  
<span class="verse-number">5&ensp;</span>この奥義は、今、**御霊**によってご自身の聖なる使徒たちと預言者たちに啓示されたようには、他の時代では人の子たちに啓示されませんでした。  
<span class="verse-number">6&ensp;</span>これは、異邦人たちが福音を通して、同様に相続者となり、一体となり、キリストにより、**神**の約束を共に受ける者たちとなるためです。  
<span class="verse-number">7&ensp;</span>私がその福音の奉仕者になったのは、**神**の力の働きに従い、私に与えられた**神**の恵みの賜物によります。  
<span class="verse-number">8&ensp;</span>これは、異邦人たちの間では、探し出すことのできないキリストの富を説くように、全聖徒の中で最も小さい者より、さらに小さい私に、この恵みは与えられました。  
<span class="verse-number">9&ensp;</span>また、イエス・キリストを通して万物を創造された**神**の内に、各時代から隠された奥義の共有、すべての人々に明らかにするためです。  
<span class="verse-number">10&ensp;</span>それは、**神**の変幻自在な智恵を、教会を通して天の諸所にいる権威ある者たちにと権力者たちに、今、分からせるためです。  
<span class="verse-number">11&ensp;</span>これは、私たちの**主**キリスト・イエスにあって、**神**が各時代に関するご意志により行われました。  
<span class="verse-number">12&ensp;</span>**主**に対する信仰を通し、私たちは**主**にあって勇敢、そして確信による到達があります。  
<span class="verse-number">13&ensp;</span>したがって、あなたがたのための、私の試練に絶望しないことを願います。それはあなたがたの栄光なのです。  
<span class="verse-number">14&ensp;</span>それがため、私たちの**主**イエス・キリストの**父**に私はひざまずきます。  
<span class="verse-number">15&ensp;</span>（全家族は、天にいる者も地にいる者も、キリストから名を付けられています。）  
<span class="verse-number">16&ensp;</span>これは、ご自分の栄光の豊かさによって、ご自身の**御霊**を通して、内にいる自分を、強靱さを、あなたがたに**神**が与えられる願いです。  
<span class="verse-number">17&ensp;</span>それは、信仰を通して、キリストがあなたがたの心にお住みになるためです。そして、あなたがたは愛に根ざして、しっかりと立たせられ、  
<span class="verse-number">18&ensp;</span>聖徒たち全員と共に、その愛の幅、長さ、深さ、高さを知るためです。  
<span class="verse-number">19&ensp;</span>また、**神**の満たしのすべてで満たされるため、人知を超えるキリストの愛を知るためです。  
<span class="verse-number">20&ensp;</span>ついで、私たちの内で働いている力によって、私たちの願うこと、また考えることをはるかに超えて、すべてができるその方に、  
<span class="verse-number">21&ensp;</span>教会の内に、キリスト・イエスにより、全世代にわたり、永遠から永遠に、**神**に栄光がありますように。アーメン。  

## 第4章

従って、**主**の囚人である私は、あなたがたが召された召命にふさわしく歩むことを懇願します。  
<span class="verse-number">2&ensp;</span>すべてに謙虚さと無私の心と、忍耐をもち、互いに愛をもって堪え忍び、  
<span class="verse-number">3&ensp;</span>平和の絆によって**御霊**の一致を守ることを懇願します。  
<span class="verse-number">4&ensp;</span>体は一つであり、**御霊**はお一人であられます。あなたがたの召命の唯一の希望で召されたのと同様です。  
<span class="verse-number">5&ensp;</span>**主**はお一人、信仰は一つ、浸礼は一つ、  
<span class="verse-number">6&ensp;</span>すべての者の**父**であられる**神**は唯一であられ、すべての者の上におられ、そして全員を貫き、またあなたがた全員の内におられます。  
<span class="verse-number">7&ensp;</span>しかし、キリストの賜物の秤により、御恵みは私たち一人一人に与えられました。  
<span class="verse-number">8&ensp;</span>そのため、**神**は言われます。「彼は上に高く昇られた時、捕囚を捕囚人とし、人々に賜物を与えられました。」  
<span class="verse-number">9&ensp;</span>それで、「彼は昇られた。」という意味は、キリストはまず地の低い所に降りられた、という意味ではありませんか。  
<span class="verse-number">10&ensp;</span>降りて来られたこの方こそ、すべてを満たすために、すべての天のはるか上に上られた方であられます。  
<span class="verse-number">11&ensp;</span>そして、キリストは自ら、ある者たちは使徒、[^2]ある者たちは預言者、ある者たちは伝道師、[^3]ある者たちは牧師、あるいは教師としてくださいました。  
<span class="verse-number">12&ensp;</span>それは聖徒たちを、奉仕のために完全装備して、キリストの体を築き上げるため、  
<span class="verse-number">13&ensp;</span>**神の御子息**の信仰と、**御子息**に関する知識に一体となり、完全な者に、キリストの満ちている秤の水準に私たち全員が到達するまでです。  
<span class="verse-number">14&ensp;</span>それは人々のずるさにより、偽りのごまかしに至るまで、風波に吹き飛ばされ、どんな教理の風にもと連れ去られる子どもでは、私たちはもうありません。  
<span class="verse-number">15&ensp;</span>それどころか、愛によって真理を話し、すべてのことにおいて私たちは成長し、頭であられるキリストに達することができます。  
<span class="verse-number">16&ensp;</span>キリストよりのこの体全体は、各関節の働きにより、結合され、編み込まれ、各部はそれぞれ受け持つ働きを能率的に行なうことにより、体を成長させ、自ら愛により築き建てられます。  
<span class="verse-number">17&ensp;</span>従って、私はこれを言い、**主**によって証します。あなたがたは、他の異邦人が彼らの価値のない知力によって歩むように、もう歩んではいけません。  
<span class="verse-number">18&ensp;</span>彼らの理解力は暗黒で、彼らの内にある無知のため、また、盲目の心のため、**神**の命から遠ざけられ、  
<span class="verse-number">19&ensp;</span>彼らは貪欲に、あらゆる不潔なことを行なうため、感覚が麻痺し、彼ら自らを肉欲に引き渡しました。  
<span class="verse-number">20&ensp;</span>しかし、あなたがたはそうキリストを学んだわけではありません。  
<span class="verse-number">21&ensp;</span>もし、あなたがたが、本当にイエスの話を聞き、イエスに教わったのなら、真理はイエスにあるのであるから、  
<span class="verse-number">22&ensp;</span>あなたがたは、以前のあなたがたの行為に関することであるが、偽りの肉欲のために腐りはてる、あの古い男を脱ぎ捨てなさい。  
<span class="verse-number">23&ensp;</span>あなたがたは自らの知力の霊で更新され、  
<span class="verse-number">24&ensp;</span>真の義と聖さによって、**神**からつくられた新しい人をあなたがたは着るためです。  
<span class="verse-number">25&ensp;</span>従って、虚偽を脱ぎ捨て、私たちは互いの一部分であるから、「互いの隣人に真理を言いなさい。」  
<span class="verse-number">26&ensp;</span>「怒っても、罪を犯してはいけない。」怒りをそのままにして、太陽を沈ませてはいけません。  
<span class="verse-number">27&ensp;</span>また、悪魔に場所を与えてはいけません。  
<span class="verse-number">28&ensp;</span>盗みをしていた者はもう盗みをやめ、むしろ労働をし、必要とする人に何か与えることができるように、手を使い正しい働きをしなさい。  
<span class="verse-number">29&ensp;</span>あなたがたの口から腐った言葉を一言も出さないようにしなさい。むしろ、聞き手に恵みを上げられるように、必要な時、徳を高めるよい言葉だけ言いなさい。  
<span class="verse-number">30&ensp;</span>また、贖いの日まで封印を押された方であられる、**神**の**聖霊**を悲しませてはいけません。  
<span class="verse-number">31&ensp;</span>すべての苦々しさ、憤り、怒り、大騒ぎ、悪口は、すべての悪意と共に、あなたがたから捨てなさい。  
<span class="verse-number">32&ensp;</span>そして、あなたがたも互いに親切な者、あわれみ深い者になり、そして、**神**がキリストにより、あなたがたを赦してくださったように、互いに赦し合いなさい。  

## 第5章

したがって、愛されている子どもとして、**神**をなぞる者になりなさい。  
<span class="verse-number">2&ensp;</span>そして、キリストは私たちを愛され、私たちのため、**神**への良い香の捧げ物とし、いけにえとし、ご自身を捧げられたように、愛の内を歩みなさい。  
<span class="verse-number">3&ensp;</span>しかし、聖徒たちとしてふさわしいように、性的な罪、すべての汚い行為、むさぼりは、あなたがたの間で口にすることさえ、いけません。  
<span class="verse-number">4&ensp;</span>また、卑わいなこと、馬鹿話し、汚い冗談、これらはそぐわないことは避け、むしろ感謝をしなさい。  
<span class="verse-number">5&ensp;</span>なぜなら、性的な罪を犯した者、汚い者、偶像崇拝者であるむさぼる者は、誰もキリストと**神**の王国に相続するものはないと、あなたがたは知っているからです。  
<span class="verse-number">6&ensp;</span>あなたがたは空虚な言葉で、何者にも騙されてはいけません。**神**の怒りは不従順な子たちの上に、これらのことが原因で来るからです。  
<span class="verse-number">7&ensp;</span>それで、彼らと共に受ける者となってはいけません。  
<span class="verse-number">8&ensp;</span>あなたがたは以前、暗やみであったが、今は**主**の内にある光なのだからです。光の子たちとして歩みなさい。  
<span class="verse-number">9&ensp;</span>（**御霊**の実はすべての善いもの、正義、真理の内にあるからです。）  
<span class="verse-number">10&ensp;</span>**主**にとって、受け入れることは、何であるか見つけ出しなしさい。  
<span class="verse-number">11&ensp;</span>そして、暗やみの実を結ばない行為を共にせず、むしろ、その行為を弾劾しなさい。  
<span class="verse-number">12&ensp;</span>彼らが密かに犯すことは、口にすることさえ恥ずかしいことだからです。  
<span class="verse-number">13&ensp;</span>しかし、すべて弾劾されるものは光により明確にされます。なぜなら、明確にするものは何であれ光だからです。  
<span class="verse-number">14&ensp;</span>したがって、**神**は言われます。「起きよ、寝ている者。そして、死人たちの中から目覚め立て。すると、キリストはあなたを照らされる。」  
<span class="verse-number">15&ensp;</span>それで、あなたがたは愚者としてではなく、賢者として、正確に歩み、  
<span class="verse-number">16&ensp;</span>時間を正しく使いなさい。今は邪悪の日々だからです。  
<span class="verse-number">17&ensp;</span>それで、愚者でなく、**主**のご意志は何であるかを理解しなさい。  
<span class="verse-number">18&ensp;</span>そして、葡萄液に酔ってはいけません。放蕩はその中にあります。むしろ、**御霊**で満たされなさい。  
<span class="verse-number">19&ensp;</span>そして、詩と賛美と霊的な歌で互いに語り合い、心の中で**主**に向かい歌い、誉め歌い、  
<span class="verse-number">20&ensp;</span>いつも、私たちの**主**イエス・キリストの御名によって、すべてのものを、**父**なる**神**に感謝しなさい。  
<span class="verse-number">21&ensp;</span>キリストに対する恐れの内にいて、互いに従いなさい。  
<span class="verse-number">22&ensp;</span>妻たちよ、**主**に従うように、あなたがた自身の夫に従いなさい。  
<span class="verse-number">23&ensp;</span>なぜなら、キリストは教会の頭であられ、自らはその体の救い主でもあられるように、夫は妻の頭だからです。  
<span class="verse-number">24&ensp;</span>それで、教会がキリストに従うように、あらゆることにおいて、妻たちも自らの夫に従いなさい。  
<span class="verse-number">25&ensp;</span>夫たちよ、キリストが教会を愛し、教会のためにご自分を捧げられたように、あなたがたも、妻たちを愛しなさい。  
<span class="verse-number">26&ensp;</span>それは御言葉により、沐浴することによって、キリストが教会を聖別し、清められるためであり、  
<span class="verse-number">27&ensp;</span>栄光ある教会として、ご自分に贈るためです。しみ、しわなどの類がなく、かえって聖なる、無欠な教会になるためです。  
<span class="verse-number">28&ensp;</span>そのように、夫たちは自らの身を愛するように、自らの妻を愛すべきです。自分の妻を愛する者は、自らを愛しています。  
<span class="verse-number">29&ensp;</span>なぜなら、自らの身を憎む者は未だかつてなく、それどころか、これを育み、大切にします。正に**主**が教会に対してお行ないになるのと同じです。  
<span class="verse-number">30&ensp;</span>なぜなら、私たちはキリストの体の一部であり、あの方の肉と骨からできているのです。  
<span class="verse-number">31&ensp;</span>「それがため、男はその父と母を離れ、その妻と結ばれ、その二人は一つの体となる。」  
<span class="verse-number">32&ensp;</span>この奥義は大きいです。しかし、私はキリストと教会のことを指して言っています。  
<span class="verse-number">33&ensp;</span>ともかく、あなたがたも一人一人、自らの妻を自分自身のように愛し、妻も自分自身の夫に畏敬の念を持ちなさい。  

## 第6章

子どもたちよ、**主**にあって自分の親たちに従いなさい。それは正しいことだからです。  
<span class="verse-number">2&ensp;</span>「あなたの父と母に敬意をはらいなさい。」とは、約束を伴う最初の戒めです。  
<span class="verse-number">3&ensp;</span>「それは、あなたが幸福であり、地上で長生きするためである。」と。  
<span class="verse-number">4&ensp;</span>そして、父親であるあなたがたよ、あなたがたの子どもを怒らせてはいけません。むしろ、**主**のしつけと諭しで育てなさい。  
<span class="verse-number">5&ensp;</span>僕たちよ、恐れおののきをもって、心から真摯に、キリストに対するように、あなたがたの肉からなる主人たちに従いなさい。  
<span class="verse-number">6&ensp;</span>見た目だけの奉仕ではなく、人を喜ばすようではなく、キリストの僕たちとして、心底より**神**のご意志を行ない、  
<span class="verse-number">7&ensp;</span>人間にするようにではなく、善意をもって進んで**主**に奉仕をしなさい。  
<span class="verse-number">8&ensp;</span>それは、僕であれ、自由人であれ、人はどんなよい行為をしても、**主**から同じものを受けるとは知ってのとおりです。  
<span class="verse-number">9&ensp;</span>そして、主人たちよ。僕たちに同じことをし、脅すのを止めなさい。あなたがた自身の**主**も天におられ、偏見のない方と、知ってのとおりです。  
<span class="verse-number">10&ensp;</span>最後に、私の兄弟たちよ、**主**と**主**の強い力により強められなさい。  
<span class="verse-number">11&ensp;</span>悪魔のずる賢さに立ち向かうことができるように、**神**の全武具を身につけなさい。  
<span class="verse-number">12&ensp;</span>なぜなら、私たちの格闘は血と肉に対してではなく、数々の統治者、数々の権威、この時代の暗やみの支配者たち、天の所々にいるいろいろな悪霊たちに対してです。  
<span class="verse-number">13&ensp;</span>そのため、**神**の全武具を手に取りなさい。悪の日に立ち向かうことができるように、またすべてのことを実行し、立つことができるためです。  
<span class="verse-number">14&ensp;</span>したがって、立って、腰に真理の革帯を締め、義の胸当てを着け、  
<span class="verse-number">15&ensp;</span>足に平和の福音の備えを履き、  
<span class="verse-number">16&ensp;</span>すべての上、あの悪い者が放つ火矢を消すことができる信仰の大盾を手に取りなさい。  
<span class="verse-number">17&ensp;</span>そして、救いのかぶと、そして**神**の御言葉である**御霊**の剣を受け取りなさい。  
<span class="verse-number">18&ensp;</span>常に、**御霊**にあって、全祈り、全願いで祈りなさい。それが目的で、すべての聖徒たちのために、すべての忍耐力と願いによって目を覚ましていなさい。  
<span class="verse-number">19&ensp;</span>また、福音の奥義を伝えるように、私は勇敢に口を開くのに、言葉が私に与えられるように、私のために祈りなさい。  
<span class="verse-number">20&ensp;</span>そして、福音を勇敢に説くことができるため、祈ってください。私はそう説くべきです。（福音のために、私は鎖で繋がれている大使です。）  
<span class="verse-number">21&ensp;</span>さて、私に関すること、そして私がどうしているかを、あなたがたも知ることができるように、愛する兄弟、**主**にあって忠実な奉仕者であるテキコが、あなたがたにすべてを明らかにします。  
<span class="verse-number">22&ensp;</span>あなたがたは私たちのことを知り、そして彼はあなたがたの心を安心させることができるよう、それが彼をあなたがたに遣わした目的です。  
<span class="verse-number">23&ensp;</span>**父**なる**神**、そして**主**イエス・キリストから、兄弟たちに平安、また信仰を伴う愛がありますように。  
<span class="verse-number">24&ensp;</span>私たちの**主**イエス・キリストを、朽ちない愛するすべての人の上に、恵みがありますように。アーメン。
