# 使徒

### 使徒の働き

## 第1章

テオピロよ、すべてイエスの行ない始め、及び教え始められたことを、私は以前に記録した。  
<span class="verse-number">2&ensp;</span>イエスが**聖霊**を通して、ご自分が選ばれた使徒たちに命令を与えた後、上げられる日までであった。  
<span class="verse-number">3&ensp;</span>イエスは、苦しみの後、多くの確かな証拠によって、生きていることを示されてご自分を使徒たちに現された。イエスは、四十日間使徒たちに見られ、**神**の王国のことについて説かれた。  
<span class="verse-number">4&ensp;</span>そして、使徒たちと集まり、エルサレムから離れず、**父**の約束を待てと命じられた。「約束とは、あなたがたがわたしから聞いたあの約束です。」と、イエスは言われた。  
<span class="verse-number">5&ensp;</span>「なぜなら、ヨハネは間違いなく水で浸礼[^1]を授けたが、あなたがたはこれからそれほど日数の経たないうちに、**聖霊**によって浸されます。」  
<span class="verse-number">6&ensp;</span>そこで、使徒たちは共に集まった時、イエスに聞いて言った。「**主**よ、この時、イスラエル王国を昔に戻してくださいますか。」  
<span class="verse-number">7&ensp;</span>しかし、イエスは彼らに言われた。「**父**は時、季節を自らの権威においています。あなたがたの知るところではありません。  
<span class="verse-number">8&ensp;</span>しかし、**聖霊**があなたがたの上に来られた時、あなたがたは力を受けます。そして、エルサレムにも、ユダヤとサマリヤのすべてに、地球の果てまで、あなたがたはわたしの証人となります。」  
<span class="verse-number">9&ensp;</span>さて、これらのことを言われた後、使徒たちが見ている間に、イエスは上げられ、雲はイエスを受け入れ、彼らの視界から消えられた。  
<span class="verse-number">10&ensp;</span>そして、イエスが昇天されると、彼らは空をしっかり見上げている時、見よ、白い服の二人の男が、使徒たちの脇に立っていた。  
<span class="verse-number">11&ensp;</span>そして、二人は言った。「ガリラヤ人の男たち、あなたがたはなぜ立って天を見つめているのか。天に上げられたこの同じイエスは、あなたがたが見た、上げられたと同様の方法で、また来られる。」  
<span class="verse-number">12&ensp;</span>それから、使徒たちは、エルサレムの近くにあるオリーブと呼ばれる山からエルサレムに帰った。これは安息日の旅程である。  
<span class="verse-number">13&ensp;</span>そして、使徒たちは中に入り、上の部屋にのぼった。そこには、ペテロとヤコブとヨハネとアンデレ、ピリポとトマス、バルトロマイとマタイ、アルパヨの息子ヤコブと熱心党員シモンとヤコブの息子ユダが泊まっていた。  
<span class="verse-number">14&ensp;</span>この人たちはみんな、女たち、イエスの母マリヤ、イエスの兄弟たちと共に、心を一つにして、祈りと願いを続けていた。  
<span class="verse-number">15&ensp;</span>そして、その日々、ペテロはその弟子たち（名前から想像すると、合わせて百二十ぐらい）の間に立って、言った。  
<span class="verse-number">16&ensp;</span>「男たち、兄弟たちよ。イエスを逮捕した者たちの手引となったユダについて、**聖霊**が、前もってダビデの口を通して言われた、この御言葉は、成就されなければならなかったのです。  
<span class="verse-number">17&ensp;</span>ユダは私たちと共に数えられ、この奉仕の役わりを得たからです。  
<span class="verse-number">18&ensp;</span>それで、この男は不正の報酬で畑を買い、真っ逆さまに堕ち、真中で砕け、内臓はすべて飛び出しました。  
<span class="verse-number">19&ensp;</span>そして、これはエルサレムに住むみんなに知られました。それで、その畑は彼らの言葉で、アケレダマ、つまり、血の畑と呼ばれています。  
<span class="verse-number">20&ensp;</span>なぜなら、『彼の住まいは荒れ果てた所にさせ、誰もそこに住まないようにさせよ。』そして、『彼の職務は、別の者に取らさせよ。』と詩篇の書に書いてあります。  
<span class="verse-number">21&ensp;</span>したがって、**主**イエスが行き来された間、私たちといつも共にいた男たちの間から、  
<span class="verse-number">22&ensp;</span>つまり、ヨハネのバプテスマから、イエスが私たちから上げられたその日まで、この人たちの一人は、私たちと共にイエスの復活の証人になるべきです。」  
<span class="verse-number">23&ensp;</span>そこて、ユストとも言われる、バルサバと呼ばれるヨセフと、マッテヤという二人を彼らは立てた。  
<span class="verse-number">24&ensp;</span>そして、みんなは祈り、言いました。「すべての人の心をご存じのあなた、**主**よ、この二人のどちらを選ばれたかを、お示しください。  
<span class="verse-number">25&ensp;</span>ユダが自分の場所に行けるように、不正を犯して放棄したこの奉仕の務めと使徒の職務を、その人はユダの役わりを受けるためです。」  
<span class="verse-number">26&ensp;</span>それで、彼らはくじを投げると、くじはマッテヤに落ちました。それで、マッテヤは十一人の使徒と共に承認された。  

## 第2章

さて、五旬節の日が満ちた時、弟子たちは心を一つにして、同じ場所にいた。  
<span class="verse-number">2&ensp;</span>すると、突然天から、恐ろしく吹く風のような音がして、それは、みんなが座っている家全体を満たした。  
<span class="verse-number">3&ensp;</span>それから、炎のようなものが分裂した舌のように現れ、一人一人の上に座った。  
<span class="verse-number">4&ensp;</span>すると、全員、**聖霊**に満たされ、話できるように、**御霊**が与えた言語に応じて、他の諸々の言語で話しだした。  
<span class="verse-number">5&ensp;</span>そして、天の下のすべての国々から、ユダヤの信仰心厚い男たちが来て、エルサレムに住んでいた。  
<span class="verse-number">6&ensp;</span>この音がして、群衆が集まって来た。みんなは怪しみ、戸惑った。弟子たちが各自の言葉で話すのを聞いたからである。  
<span class="verse-number">7&ensp;</span>それで、人々はみんな驚き不思議に思い、互いに言った。「見よ。話しているこの人たちはみんな、ガリラヤ人ではないのですか。  
<span class="verse-number">8&ensp;</span>それで、私たちが生まれた所の言葉で、私たちは聞いているのはなぜですか。  
<span class="verse-number">9&ensp;</span>パルテヤ人とメジヤ人とエラム人、またメソポタミヤとユダヤとカパドキヤとポントとアジア[^2]に住んでいる人たち、  
<span class="verse-number">10&ensp;</span>フルギヤとパンフリヤ、エジプトとクレネに近いリビヤ地方などに住んでいる人たち、また滞在中のローマからのユダヤ人も改宗者も、  
<span class="verse-number">11&ensp;</span>クレテ人もアラビヤ人も、私たちみんな、**神**のすばらしい働きを私たち自身の言語で彼らが語っているのを聞いています。」  
<span class="verse-number">12&ensp;</span>それで、人々はみな驚き、訳が分からず、「これはいったいどうなっているんだ。」と互いに言い合っていた。  
<span class="verse-number">13&ensp;</span>他の人たちは馬鹿にして言った。「あの者たちは葡萄ジュースで満たされたのだ。」  
<span class="verse-number">14&ensp;</span>しかし、十一人と共に立っていたペテロは、声を高くし、彼らに宣言した。「ユダヤの男たち、そして、エルサレムに住んでいるすべての男たちよ、あなたがたに知って欲しいのです。私の言葉を聞いてください。  
<span class="verse-number">15&ensp;</span>あなたがたが想像するように、この人たちは酔ってはいません。今は一日の第三時[^3]だからです。  
<span class="verse-number">16&ensp;</span>しかし、これは預言者ヨエルを通して説かれたことです。  
<span class="verse-number">17&ensp;</span>**神**は言われます。『最後の日々にこうなる。わたしはすべての肉の上に私の霊から注ぐ。そして、あなたがたの息子たちと娘たちは預言し、あなたがたの若い男たちは幻を見、あなたがたの老人たちは夢を見る。  
<span class="verse-number">18&ensp;</span>そして、その日々にわたしは自らの霊をわたしの男どれいと女どれいの上に注ぐ。すると彼らは預言する。  
<span class="verse-number">19&ensp;</span>そして、わたしは上の天で不思議なわざを与え、下にある地球でしるし、つまり、血と火と煙の霧を与える。  
<span class="verse-number">20&ensp;</span>**主**の偉大な、輝かしい日が来る前に、太陽は暗やみに変えられ、月は血に変えられる。  
<span class="verse-number">21&ensp;</span>そして、**主**の御名を呼び求める者は誰でも救われる、ということになる。』  
<span class="verse-number">22&ensp;</span>イスラエルの男たちよ、この言葉を聞いてください。あなたがたも知っているとおり、イエスを通して、**神**があなたがたの間で行なわれた奇跡と不思議なわざやしるしにより、**神**によりあなたがたに、ナザレのイエスは示されました。  
<span class="verse-number">23&ensp;</span>**神**の定められたご意志と予知によって、渡されたこのイエスを、あなたがたは不法の手で捕らえ、十字架につけ、殺してしまいました。  
<span class="verse-number">24&ensp;</span>イエスが死に拘束されることは不可能であったから、**神**はイエスの死の苦しみを解かれ、彼を復活させられました。  
<span class="verse-number">25&ensp;</span>なぜなら、ダビデはイエスについて言っています。『私が動揺することのないように、**主**は私の右の手のところにおられるから、私の目の前に、私はいつも**主**を見た。  
<span class="verse-number">26&ensp;</span>このため、私の心はうれしくなり、私の舌はとても喜んだ。それに、私の肉体は希望の内に止まる。  
<span class="verse-number">27&ensp;</span>なぜなら、あなたは私の魂を黄泉[^4]に置き去りにしない。また、あなたは、あなたの**聖なる者**が、朽ち果てるのを見るのを許されない。  
<span class="verse-number">28&ensp;</span>あなたは私に命の道を教えてくださった。あなたの御前で、あなたは私を喜びで満たしてくださる。』  
<span class="verse-number">29&ensp;</span>男たちよ、兄弟たちよ。先祖ダビデのことを、私に自由にあなたがたに話させてください。彼は死に、そして葬られました。そして、彼の墓は今日まで私たちと共にあります。  
<span class="verse-number">30&ensp;</span>それで、ダビデは預言者であり、**神**が誓いをたて、ダビデの体の実から、肉によって、ご自分の王座につけるため、キリストを復活させると、誓われたことをダビデは知っていました。  
<span class="verse-number">31&ensp;</span>ダビデはこれを前もって見て、キリストの魂はハデスに置き去りにせず、その肉体が朽ち果てたのを見なかったと、キリストの復活について語りました。  
<span class="verse-number">32&ensp;</span>このイエスを、**神**は復活させてくださいました。私たちはみんなこのことの証人です。  
<span class="verse-number">33&ensp;</span>したがって、**神**の右の手は上げられ、**父**から**聖霊**の約束を受けたイエスは、今あなたがたが見たり聞いたりしているものにそれを注がれました。  
<span class="verse-number">34&ensp;</span>ダビデは天に上らなかったが、彼は自ら言ったからです。『**主**は私の主に言われた。「わたしの右手に座りなさい。  
<span class="verse-number">35&ensp;</span>わたしがあなたの敵をあなたの足台にするまでである。」』と。  
<span class="verse-number">36&ensp;</span>それで、あなたがたが十字架刑にしたこのイエスを、**神**は**主**にも、またキリストにもされたことを、イスラエルのすべての家に、確実に伝えてください。」  
<span class="verse-number">37&ensp;</span>すると、彼らはこれを聞き、心を突き刺され、ペテロと他の使徒たちに言った。「男たちよ、兄弟たちよ。私たちはどうしたらよいのでしょうか。」  
<span class="verse-number">38&ensp;</span>そこで、ペテロは彼らに言った。「悔い改めなさい。そして、あなたがた一人一人イエスの御名により、赦されている罪のゆえに、浸礼を受けなさい。そうすれば、あなたがたは**聖霊**の賜物を受けます。  
<span class="verse-number">39&ensp;</span>なぜなら、この約束はあなたがたと、あなたがたの子どもに、また遠くにいるすべての人たちに、私たちの**神**である**主**が、お呼びになる人たち全員に、与えられています。」  
<span class="verse-number">40&ensp;</span>そして、他にも数多くの言葉でペテロは彼らに証をし、彼らに勧めて言った。「この邪悪な世代から救われなさい。」  
<span class="verse-number">41&ensp;</span>それから、彼の言葉を喜びを持って受け入れた人たちに、浸礼を授けた。そしてその日、およそ三千の魂が彼らに加えられた。  
<span class="verse-number">42&ensp;</span>そして、使徒たちの教え、交流、パンを裂くこと、それに祈りを、彼らは確実に続けていた。  
<span class="verse-number">43&ensp;</span>すると、すべての魂の上に恐れが来た。そして、使徒たちを通して多くの不思議なことと、しるしが行なわれた。  
<span class="verse-number">44&ensp;</span>さて、信じた人たちはみんな同じところにいて、すべての物を共有していた。  
<span class="verse-number">45&ensp;</span>そして、彼らは資産と金目の物を売り、必要とする人、誰とでも分かち合った。  
<span class="verse-number">46&ensp;</span>毎日休むことなく、神殿の敷地で一心に、家から家へとパンを裂き、彼らは大喜びで、純な心で食事をしていた。  
<span class="verse-number">47&ensp;</span>彼らは**神**を誉めたたえ、すべての国民に好意を持たれた。そして、**主**は毎日救われる人たちを教会に加えてくださっていた。  

## 第3章

さて、祈りの時である第九時に、ペテロとヨハネは共に神殿の敷地に上った。  
<span class="verse-number">2&ensp;</span>そして、神殿の敷地の「美しき」と呼ばれる門に、母の胎内にいた時から、足の不自由な男が毎日、人たちに運ばれ、置かれた。それは、男が神殿の敷地に入る人たちから施しを求めるためであった。  
<span class="verse-number">3&ensp;</span>ペテロとヨハネが神殿の敷地に入ろうとするのを見て、足が不自由な男は施しを願った。  
<span class="verse-number">4&ensp;</span>それで、ヨハネと共に、ペテロはその男に目をとめ、言った。「私たちの方を見なさい。」  
<span class="verse-number">5&ensp;</span>そこで、彼らが何かくれるかと思い、男は彼らに注意をはらった。  
<span class="verse-number">6&ensp;</span>そして、ペテロは言った。「私は銀も金も持っていないが、持っているものをあなたにあげる。ナザレのイエス・キリストの御名によって、立って歩きなさい。」  
<span class="verse-number">7&ensp;</span>そして、ペテロは男の右手をつかみ、立たせた。すると、男の足と足首の骨はたちどころに力を受けた。  
<span class="verse-number">8&ensp;</span>そして、男は飛び跳ねながら、立って、歩きながら、使徒たちと共に神殿の敷地に入った。歩き、飛び跳ね、そして**神**を賛美しながらである。  
<span class="verse-number">9&ensp;</span>そこで、神殿の敷地で歩いたり**神**を賛美したりするのを、すべての民は見た。  
<span class="verse-number">10&ensp;</span>ほどなく、神殿の敷地の美しきの門で施しを求めていたのが、この男であったことを人たちは理解した。それで、彼の身に起こったことに、人たちは驚きと不可解に満たされた。  
<span class="verse-number">11&ensp;</span>そして、不自由な足がいやされたこの男が、ペテロとヨハネを離さずにいると、すべての民はとても驚き、「ソロモンの柱廊」と呼ばれる柱廊にいた彼らの所へ、共に走って行った。  
<span class="verse-number">12&ensp;</span>それで、ペテロはそれを見て、民に反応して答えた。「イスラエルの男たちよ、あなたがたはなぜこれに驚いていますか。また、私たちは私たち自身の力か、敬虔さでこの男を歩かせたかのように、私たちをなぜ見つめるのですか。  
<span class="verse-number">13&ensp;</span>アブラハムとイサクとヤコブの**神**、すなわち私たちの父たちの**神**は、彼の**御子息**、イエスに栄光を与えられました。そのイエスを、ピラトが釈放することを決めていたのに、あなたがたは渡して、ピラトの前で否定してしまいました。  
<span class="verse-number">14&ensp;</span>しかし、あなたがたは**聖なる**、**正しい方**を拒否し、人殺しを渡せと要求し、  
<span class="verse-number">15&ensp;</span>命の**王子**を殺しました。**神**はその方を死者たちの中から復活させました。私たちはそのことの証人です。  
<span class="verse-number">16&ensp;</span>そして、その方の御名の信仰によって、その方の御名は、あなたがたが見知っているこの男を、強くしました。そして、その方を通してくる信仰は、あなたがた全員の前で、この人にこの健やかさを与えました。  
<span class="verse-number">17&ensp;</span>そして、兄弟たちよ、無知ゆえに、あなたがたがそうした事を私は知っています。あなたがたの支配者たちもそうです。  
<span class="verse-number">18&ensp;</span>しかし、**神**の全預言者の口を通して、**神**が前もって言われたこと、すなわちキリストは苦しまれることを、**神**はこうして成就されたのです。  
<span class="verse-number">19&ensp;</span>だから、**主**の御前から、休息の時が来るように、あなたがたの罪が残らず消されるために、悔い改め、改心しなさい。  
<span class="verse-number">20&ensp;</span>それは、以前、あなたがたに教えられたイエス・キリストを、**神**が遣わしてくださるためです。  
<span class="verse-number">21&ensp;</span>すべてが復興する時まで、天国はこのイエスを受け入れなければなりません。世界が始まってから、**神**はご自分の全預言者の口を通してこのことを語られました。  
<span class="verse-number">22&ensp;</span>なぜなら、モーセは本当に父たちに言いました。『あなたがたのために、あなたがたの兄弟の中から、**主**であるあなたがたの**神**は、私のような預言者を上げられる。その人が言うことは何でも、あなたがたはすべてのことにおいて聞かねばならない。  
<span class="verse-number">23&ensp;</span>そして、その預言者の話を聞こうとしないすべてのたましいは、国民の中から完全に抹殺される。』  
<span class="verse-number">24&ensp;</span>サムエルとその後の預言者たち、すべての預言者たち、すなわち語ったことのある預言者たちは、みなこれらの日を預言しました。  
<span class="verse-number">25&ensp;</span>あなたがたは預言者たちの息子たち、また**神**が私たちの父たちと結んだその契約の息子たちです。**神**はアブラハムに言われました。『そして、あなたの種によって、地球のすべての家族は祝福される。』  
<span class="verse-number">26&ensp;</span>ご自身の**御子息**、イエスを復活させられ、**神**は最初にイエスをあなたがたに遣わされました。一人一人が、悪から向きを変えるあなたがたを、祝福するためです。」  

## 第4章

さて、ペテロたちが民に話していた間、祭司たちと神殿の守衛長とサドカイ派の人たちが彼らの所にやって来た。  
<span class="verse-number">2&ensp;</span>ペテロとヨハネは民を教え、イエスについては死人たちの中からの復活を説いたので、その人たちは動転して、  
<span class="verse-number">3&ensp;</span>ペテロとヨハネを手にかけ、すでに夕方であったので、次の日まで彼らを拘置した。  
<span class="verse-number">4&ensp;</span>しかし、御言葉を聞いた多くの人たちは信じ、男の人数は約五千に達した。  
<span class="verse-number">5&ensp;</span>さて、ことの次第はこうであった。日は過ぎ、翌日になり、民の支配者たち、長老たち、律法学者たち、  
<span class="verse-number">6&ensp;</span>また、大祭司のアンナスとカヤパとヨハネとアレキサンデレ、そしてその大祭司の家族の者全員、共にエルサレムにいた。  
<span class="verse-number">7&ensp;</span>それで、ペテロとヨハネを彼らの真ん中に立たせ、質問してきた。「お前たちは、どんな力によって、または何の名によってこれをやったのか。」  
<span class="verse-number">8&ensp;</span>すると、**聖霊**に満たされていたペテロは彼らに言った。「民の支配者たちとイスラエルの長老たちよ、  
<span class="verse-number">9&ensp;</span>もし、弱者にしたよい行ないのため、すなわち、その人がそれによって健全な体になったことで、今日この日に私たちが裁かれるなら、  
<span class="verse-number">10&ensp;</span>あなたがたみんなにも、イスラエルの民みんなにも、知ってもらいます。この男が完全な体で、あなたがたの前に立っているのは、あなたがたが十字架につけ、**神**が死人たちの中から復活させた、ナザレのイエス・キリストの御名によるのです。  
<span class="verse-number">11&ensp;</span>これは、『家を建てるあなたがたに拒否された石で、隅の礎となった』その石です。  
<span class="verse-number">12&ensp;</span>しかし、他にはいかなる救いもありません。なぜなら、私たちが救われる必要がある御名は他には、天の下、人間の間には与えられていないからです。」  
<span class="verse-number">13&ensp;</span>そこで、彼らはペテロとヨハネの大胆さを見、彼らは教育もなく、平民であるとわかって、驚いた。そして、二人は以前イエスと共にいたことに、彼らは気がついた。  
<span class="verse-number">14&ensp;</span>そして、治された人が、二人と立っているのを見ると、彼らはそのことに反対しては何も言えなかったのである。  
<span class="verse-number">15&ensp;</span>しかし、その三人を議会から出て行くように命じ、彼らは各自相談して、  
<span class="verse-number">16&ensp;</span>言った。「あの者たちを、どうしようか。なぜなら、彼らを通して明白なしるしが行なわれたことは、エルサレムに住んでいる者全員に明らかで、私たちはそれを否定することができないからだ。  
<span class="verse-number">17&ensp;</span>しかし、民の間にこれ以上広がらないように、この名で彼らが話さぬよう、厳しく脅かそう。」  
<span class="verse-number">18&ensp;</span>それで、彼らはペテロとヨハネを呼び、イエスの御名により、一言も口にしたり、教えたりしてはならないと、命じた。  
<span class="verse-number">19&ensp;</span>すると、ペテロとヨハネは答えて、彼らに言った。「**神**に聞くより、あなたがたに聞く方が、**神**の前で正しいのか、あなたがたが裁きなさい。  
<span class="verse-number">20&ensp;</span>私たちは見たこと、聞いたこと以外は話せないからです。」  
<span class="verse-number">21&ensp;</span>それで、民のことでは、ペテロとヨハネを罰する方法を見つけれず、さらに脅かしてから、二人を放免した。民はみな、行なわれたことに、**神**に栄光を捧げたからである。  
<span class="verse-number">22&ensp;</span>四十歳を超えていた男に、この治しのしるしが行なわれたからである。  
<span class="verse-number">23&ensp;</span>それで、放免されると、二人は仲間の所に行き、大祭司たちと長老たちが、彼らに言ったことをすべて伝えた。  
<span class="verse-number">24&ensp;</span>それで、仲間たちはそれを聞き、心を合わせて**神**に向かって声をあげ、言った。「**主**よ、あなたは天と地と海と、その中にあるすべてのものを造られた**神**です。  
<span class="verse-number">25&ensp;</span>あなたはあなたの僕、ダビデの口を通して言われた**神**です。『国々はなぜ、騒ぎ立て、民はなぜむなしいことを企んだのか。  
<span class="verse-number">26&ensp;</span>**主**と**主**のキリストに反対して、地球の王たちは立ち上がり、支配者たちは共に集まった。』  
<span class="verse-number">27&ensp;</span>なぜなら、ヘロデもポンテオ・ピラトも、異邦人たちも、そしてイスラエルの民も、集められ、あなたが油を注がれた、あなたの聖なる子イエスに実際に敵対しました。  
<span class="verse-number">28&ensp;</span>それは、あなたの御手とあなたのご意志が、前もって定められたことを、それらが何であれ、彼らが行なうためでした。  
<span class="verse-number">29&ensp;</span>それで**主**よ、今彼らの脅かしをご覧になり、そして、あなたの僕たちに、あなたの御言葉をすべて話せる大胆さを与えてください。  
<span class="verse-number">30&ensp;</span>御手を差し出すことにより治し、また、あなたの聖なる子イエスの御名を通して、しるしと不思議なわざが行なわれるためです。」  
<span class="verse-number">31&ensp;</span>そして、祈り終わった時、彼らが集まっていた所が揺れ動かされた。そして彼らは全員、**聖霊**に満たされ、**神**の御言葉を大胆に語った。  
<span class="verse-number">32&ensp;</span>さて、信じたその群衆は、心もたましいも一つにして、自分の持ち物は自分の物であると言う者はなく、すべての物を共有した。  
<span class="verse-number">33&ensp;</span>そして、大きな力で、使徒たちは**主**イエスの復活を証した。そして、大きな恵みも全員にあった。  
<span class="verse-number">34&ensp;</span>それは、彼らの間に誰も乏しい人はいなかったからである。なぜなら、土地か家を持つ人は全員、それを売り、売れた物の代価を持って来て、  
<span class="verse-number">35&ensp;</span>使徒たちの足もとに置いたからである。使徒たちは誰であれ、各自の必要に応じて配った。  
<span class="verse-number">36&ensp;</span>そして、キプロス出身のレビ人で、使徒たちにバルナバ（訳すると、慰めの息子）とも呼ばれた、ジョセスは、  
<span class="verse-number">37&ensp;</span>土地を持っていたが、それを売り、その代価を使徒たちの足もとに置いた。  

## 第5章

しかし、アナニヤという一人の男が、妻サッピラと共に資産を売り、  
<span class="verse-number">2&ensp;</span>妻もそれを知った上で、代価の一部を残し、ある部分は持って行き、使徒たちの足もとに置いた。  
<span class="verse-number">3&ensp;</span>すると、ペテロは言った。「アナニヤよ、**聖霊**にうそをつき、土地の代価の一部を自分のため、残しておくように、サタンはなぜ、お前の心に満ちてしまったのか。  
<span class="verse-number">4&ensp;</span>そのままにしておけば、お前の物のままでは、なかったのか。そして、土地は売られてからも、お前の手中ではなかったのか。お前はなぜこれをお前の心の中に抱いたのか。お前は人にではなく、**神**にうそをついたのだ。」  
<span class="verse-number">5&ensp;</span>すると、この言葉を聞くやいなや、アナニヤは倒れ、最後の息をした。そして、これらのことを聞いた全員に、大きな恐れが生じた。  
<span class="verse-number">6&ensp;</span>そして、若い男たちが立ち上がり、彼を包み、運び出し、埋葬した。  
<span class="verse-number">7&ensp;</span>さて、三時間ぐらい後、何が起こったか知らず、彼の妻が入って来た。  
<span class="verse-number">8&ensp;</span>そこで、ペテロは彼女に答えた。「私に言いなさい。土地はこのぐらいで売ったのか。」彼女は言った。「はい、そのぐらいです。」  
<span class="verse-number">9&ensp;</span>それで、ペテロは彼女に言った。「お前たちはどうして、**主**の**霊**を試そうと二人で決めてしまったのか。見よ。お前の夫を埋葬した男たちの足は、戸口のところに来ている。そして、彼らはお前を運び出す。」  
<span class="verse-number">10&ensp;</span>それで、彼女はたちどころに、ペテロの足もとに倒れ、最後の息をした。そして、例の若い男たちが入ってきて、死んでいる彼女を見つけ、彼女を運び出し、夫のとなりに埋葬した。  
<span class="verse-number">11&ensp;</span>そして、全教会員と、これらのことを聞いた者全員に、大きな恐れが生じた。  
<span class="verse-number">12&ensp;</span>また、使徒たちの手を通して、民の間で多くのしるしと不思議なわざが行なわれた。そして、全員、心を一つにして、ソロモンの柱廊にいた。  
<span class="verse-number">13&ensp;</span>しかし、残りの人たちは誰も勇気を出して彼らに加わろうとはしなかったが、民は彼らを高く評価していた。  
<span class="verse-number">14&ensp;</span>そして、さらに男も女も、信者は多く**主**に加えられた。  
<span class="verse-number">15&ensp;</span>それに続き、人たちは病人たちを通りに運び出し、せめて通り過ぎるペテロの影が彼らの上に落ちるように、床や布団の上に寝かした。  
<span class="verse-number">16&ensp;</span>また、周辺の町から、群衆はエルサレムに集まった。彼らは病人や汚れた霊で苦しんでいる人たちを運んできた。そして彼らは全員、治された。  
<span class="verse-number">17&ensp;</span>それで、大祭司と彼と共にいた人たち（それはサドカイ派の人たちである）は全員立ち上がり、彼らは憤りに満ち、  
<span class="verse-number">18&ensp;</span>使徒たちに自ら手を出して、彼らを公のさらし者にした。  
<span class="verse-number">19&ensp;</span>しかし、夜になると、**主**の御使いが来て、牢獄の扉を開け、使徒たちを連れ出し、言った。  
<span class="verse-number">20&ensp;</span>「行って、神殿の敷地に立ち、民全員に、この命の言葉を伝えなさい。」  
<span class="verse-number">21&ensp;</span>彼らはこれを聞き、朝早く神殿の敷地に入り、教えていた。しかし、大祭司と共にいた人たちが来て、議会の人たちも、イスラエルの子らの長老議会の全員も集め、使徒たちを連れに来るよう、人たちを牢獄に派遣した。  
<span class="verse-number">22&ensp;</span>しかし、役人たちは行ったが、牢獄に使徒たちを見つけれず、戻って報告して、  
<span class="verse-number">23&ensp;</span>言った。「間違いなく、牢獄はすべて確実に閉じられたままであり、また、扉の前には番兵たちは立っていましたが、扉を開けると、中には誰もいませんでした。」  
<span class="verse-number">24&ensp;</span>それで、祭司たちと神殿警護隊長と、大祭司たちはこれらの言葉を聞き、この結果はどうなるかと、途方に暮れた。  
<span class="verse-number">25&ensp;</span>すると、一人の人が来て、伝えた。「御覧なさい。あなたがたが牢獄に入れた男たちが、神殿の敷地に立って、民を教えています。  
<span class="verse-number">26&ensp;</span>それで、警護隊長は部下たちと共に行き、手荒なことをせず、使徒たちを連れてきた。それは、石投げで殺されるのでないかと、民を恐れたからである。  
<span class="verse-number">27&ensp;</span>そして、使徒たちを連れてきて、議会員たちの前に立たせ、大祭司は彼らを問いただして、  
<span class="verse-number">28&ensp;</span>言った。「お前たちはこの名によって教えないようにと、私たちは厳しく命じたのではなかったか。それで、見なさい。お前たちはお前たちの教理でエルサレムを満たして、あの男の血を私たちに返そうとしている。」  
<span class="verse-number">29&ensp;</span>しかし、ペテロと他の使徒たちは答えて言った。「私たちは人間よりも、**神**に従わなければなりません。  
<span class="verse-number">30&ensp;</span>あなたがたが、木にぶら下げて殺したイエスを、我らの父たちの**神**は復活させてくださいました。  
<span class="verse-number">31&ensp;</span>悔い改めと罪の赦しをイスラエルに与えるために、**君主**と**救い主**になるように、**神**はその方をご自分の右手に上げられました。  
<span class="verse-number">32&ensp;</span>そして、私たちは、これらのことの、イエスの証人です。また、イエスに従う人たちに、**神**が与えられた**聖霊**の証人でもあります。」  
<span class="verse-number">33&ensp;</span>これを聞いて、彼らは激怒し、使徒たちを殺そうと企んだ。  
<span class="verse-number">34&ensp;</span>そこで、律法の教師であり、民みんなに尊敬されていたガマリエルという、議会員の一人が立ち上がり、しばらく使徒たちを外に出すように命じた。  
<span class="verse-number">35&ensp;</span>そして、彼は議員たちに言った。「イスラエルの男たちよ。この男たちに関して行なおうとしていることには、十分気をつけなさい。  
<span class="verse-number">36&ensp;</span>なぜなら、以前、チゥダは立ち上がって、自分が偉いと言いました。それで、人数四百人ぐらい、彼の仲間になりました。彼は殺され、彼に説得された人たちは散らされ、無になりました。  
<span class="verse-number">37&ensp;</span>この者の後、人口登録の当時、ガリラヤのユダは立ち上がり、多くの民を引きつけました。彼も死に、従った者全員、散らされました。  
<span class="verse-number">38&ensp;</span>それで今、私はあなたがたに言います。この人たちから離れて、ほっておきましょう。なぜなら、もしこの計画、あるいはこの働きが人間から来たのなら、無になります。  
<span class="verse-number">39&ensp;</span>しかし、もし、これが**神**からのものならば、覆すことはできません。それどころか、**神**ご自身と戦っていた、などとならないようにしなさい。  
<span class="verse-number">40&ensp;</span>彼らはガマリエルに賛成し、使徒たちを呼び、彼らをむち打ってから、もうイエスの名によって説かないように命じて、彼らを放免した。  
<span class="verse-number">41&ensp;</span>それで、イエスの名のために、恥を受けるようにふさわしくされたことを喜びながら、使徒たちは議会から去って行った。  
<span class="verse-number">42&ensp;</span>そして、毎日神殿の敷地、またすべての家で、イエスをキリストとして説くのを、彼らは止めなかった。  

## 第6章

さて、そのころ、弟子たちの人数は増えていて、ギリシャ語を使うユダヤ人たち[^5]から、ヘブライ語を使うユダヤ人たちへ非難があがった。なぜなら、彼らの未亡人たちは、日常の配給で、ないがしろにされていたからである。  
<span class="verse-number">2&ensp;</span>それで、例の十二人は弟子たち全員を呼び、言った。「私たちが**神**の御言葉を離れ、食事の世話をするのは好ましくありません。  
<span class="verse-number">3&ensp;</span>それで、兄弟たちよ、この用務に当たるため、評判のよい、**聖霊**と知恵で満たされた七人の男を、あなたがたの中から見つけ出しなさい。  
<span class="verse-number">4&ensp;</span>私たちは、祈りと御言葉の奉仕だけをたゆまず続きます。」  
<span class="verse-number">5&ensp;</span>そして、その言葉は群衆全員を喜ばせた。それで彼らは、信仰と**聖霊**とに満ちた人であるステパノ、そしてピリポとプロコロとニカノルとテモンとパルメナと、アンテオケの改宗者ニコラオを選び、  
<span class="verse-number">6&ensp;</span>彼らは使徒たちの前にその七人を立てた。そして、使徒たちは祈って、七人の上に手を置いた。  
<span class="verse-number">7&ensp;</span>それで、**神**の御言葉は成長し、また、エルサレムで弟子の人数が大いに増加し、大勢の祭司も信仰に従った。  
<span class="verse-number">8&ensp;</span>そして、信仰と力で満たされたステパノは、民の間で不思議なわざとしるしを行なっていた。  
<span class="verse-number">9&ensp;</span>それで、「リベルテン[^6]のシナゴーグ」の中から、クレネ人とアレキサンドリヤ人とキリキヤやアジアの人たちは、立ち上がり、ステパノと論争した。  
<span class="verse-number">10&ensp;</span>しかし、彼らはステパノが話した知恵と**御霊**の抵抗できなかった。  
<span class="verse-number">11&ensp;</span>それで、彼らは男たちを誘導し、言わせた。「ステパノがモーセと**神**に対して、冒涜の言葉を言うのを、私たちは聞いたことがあります。」  
<span class="verse-number">12&ensp;</span>そして、彼らは民、長老たち、そして律法学者たちをたきつけ、ステパノの所に行き、彼を捕まえ、議会[^7]に連れて行った。  
<span class="verse-number">13&ensp;</span>また、彼らは偽の証人を立て、言った。「この者はこの聖なる場所と律法に対して、冒涜の言葉を言うのを止めません。  
<span class="verse-number">14&ensp;</span>このナザレのイエスは、この場所を破壊し、モーセが私たちにもたらした習慣を変える、と私たちはこの人が言うのを聞いたからです。」  
<span class="verse-number">15&ensp;</span>それで、議会に座っていた者たちが全員ステパノをよく見ると、彼の顔が御使いの顔のように見えた。  

## 第7章

それで、大祭司は言った。「今のことは、本当にそうか。」  
<span class="verse-number">2&ensp;</span>ステパノは言った。「男たち、兄弟たち、それに父たちよ、聞いてください。私たちの父アブラハムはカランに住む前、まだメソポタミヤにいた時、栄光の**神**がアブラハムに現れ、  
<span class="verse-number">3&ensp;</span>彼に言われました。『あなたの土地から、そして親族から出て、わたしが示す土地に入りなさい。』  
<span class="verse-number">4&ensp;</span>それで、アブラハムはカレデヤ人の土地を出て、ハランに住みました。彼の父が死んだ時、あなたがたが今住んでいるこの土地に、**神**は彼を移されました。  
<span class="verse-number">5&ensp;</span>しかし、片足を置くほどの土地も、**神**は彼に相続させず、また、アブラハムに子どもがいなかった時も、**神**はこの土地を彼に所有物として与え、また彼の後の子孫に与えると約束してくださいました。  
<span class="verse-number">6&ensp;</span>しかし、**神**はこのように言われました。すなわちアブラハムの子孫は外国に住み、外国人は彼らをどれいにして、四百年虐待します。  
<span class="verse-number">7&ensp;</span>そして、『彼らがどれいとして仕える国を、わたしは裁く。』と**神**は言われました。『そしてその後、彼らは出て来て、この場所でわたしに仕える。』  
<span class="verse-number">8&ensp;</span>それから、**神**はアブラハムに割礼の契約を与えられました。それから、アブラハムはイサクをもうけ、第八日[^8]に彼に割礼をしました。イサクはヤコブをもうけ、ヤコブは十二人の父祖をもうけました。  
<span class="verse-number">9&ensp;</span>そして父祖たちはねたみ、ヨセフをエジプトへ売りました。しかし、**神**はヨセフと共におられました。  
<span class="verse-number">10&ensp;</span>それで、**神**は、ヨセフのすべての苦しみから、彼を救い出され、エジプトの王である、ファラオの前で恵みと知恵を与えられました。そして、彼をエジプト全土とファラオの家全体の司にしました。  
<span class="verse-number">11&ensp;</span>さて、ききんと大きい艱難が、エジプトとカナンの全土に来ました。そして私たちの父祖たちは、食べ物は見いだせませんでした。  
<span class="verse-number">12&ensp;</span>しかし、エジプトに穀物があると聞いて、ヤコブはまず私たちの父祖たちを送り出しました。  
<span class="verse-number">13&ensp;</span>そして、二回目に、ヨセフは自分の兄弟たちに身の上を話し、ヨセフの家族はファラオに知られることになりました。  
<span class="verse-number">14&ensp;</span>彼の父ヤコブとそのすべての親族、七十五人をヨセフは人を遣わして呼び寄せました。  
<span class="verse-number">15&ensp;</span>それで、ヤコブはエジプトに下って行き、彼と私たちの父祖たちはそこで死にました。  
<span class="verse-number">16&ensp;</span>そして、彼らはシケムに運ばれ、アブラハムはシケムの父であるハモルの息子たちから、いくらかの銀で買った墓に葬られました。  
<span class="verse-number">17&ensp;</span>しかし、**神**がアブラハムに誓われた約束の時が近づくにしたがい、民はエジプトで成長して、人数は増えました。  
<span class="verse-number">18&ensp;</span>このことは、ヨセフを知らない別の王が立つまででした。  
<span class="verse-number">19&ensp;</span>この男は我々一族を、先祖たちを緻密な悪巧みを持って扱い、父祖たちの乳児は生存できないよう、遺棄させました。  
<span class="verse-number">20&ensp;</span>そのころ、モーセは生まれました。そして、彼は**神**にとってうるわしく、三ヶ月間父親の家で育てられました。  
<span class="verse-number">21&ensp;</span>しかし、モーセは遺棄され、ファラオの娘は彼を取り上げ、自らの息子として育てました。  
<span class="verse-number">22&ensp;</span>そして、モーセはエジプト人たちのすべての英知で教え込まれ、言葉にも行為にも力がありました。  
<span class="verse-number">23&ensp;</span>さて、彼が四十歳の時、イスラエルの子らである、彼の兄弟たちを訪ねて見ようかと、彼の心に浮かんできました。  
<span class="verse-number">24&ensp;</span>そして、その兄弟の一人がエジプト人に不当に扱われるのを見、モーセは彼を守り、虐待された兄弟の敵に復讐し、そのエジプト人を殴り倒しました。  
<span class="verse-number">25&ensp;</span>なぜなら、**神**が彼の手により、彼らに救いを与えられることを、兄弟たちは理解してくれると思っていましたが、理解してくれませんでした。  
<span class="verse-number">26&ensp;</span>そして次の日、モーセはけんかをしている二人の前に現れ、仲直りをさせようとして、言いました。『男たちよ、お前たちは兄弟だ。なぜ互いに害を与えているのか。』  
<span class="verse-number">27&ensp;</span>しかし、この隣人に害を与える方の者は、モーセを押しのけ、言いました。『我々の上に、誰がお前を支配者と裁判官にしたのか。  
<span class="verse-number">28&ensp;</span>昨日殺したエジプト人と同じように、俺を殺す気か。』  
<span class="verse-number">29&ensp;</span>それで、この言葉で、モーセは逃げ、ミデアンの地の住民になりました。そこで、息子二人をもうけました。  
<span class="verse-number">30&ensp;</span>さて、四十年経ってから、シナイ山の荒野で、藪の火の炎の中で彼に**主**の御使いは現れました。  
<span class="verse-number">31&ensp;</span>そして、モーセはそれを見て、その光景に驚きました。そこで、モーセはよく見ようと近づいた時、**主**の声が彼に聞こえました。  
<span class="verse-number">32&ensp;</span>**主**は言われました。『わたしはお前の父祖の**神**である、すなわち、アブラハムの**神**、イサクの**神**、ヤコブの**神**である。』それでモーセは震え、眺める勇気がありませんでした。  
<span class="verse-number">33&ensp;</span>すると、**主**は彼に言われました。『サンダルを足から脱ぎなさい。あなたの立っている所は聖なる地だからである。  
<span class="verse-number">34&ensp;</span>エジプトにいるわたしの民の虐待を、わたしは見た。そして、わたしは彼らの嘆きを聞き、彼らを連れ出すため下って来た。さあ今だ、来なさい。わたしはあなたをエジプトへ遣わす。』  
<span class="verse-number">35&ensp;</span>『誰があなたを、支配者と裁判官にしたのか。』と言って、彼らが拒否したあのモーセを、藪の中に現れた御使いの手により、**神**が支配者と救出者にするため、遣わされたあのモーセです。  
<span class="verse-number">36&ensp;</span>エジプトの地で、紅海で、また四十年間、荒野で、不思議なわざとしるしを行なった後、モーセは彼らを連れ出しました。  
<span class="verse-number">37&ensp;</span>『あなたの**主**である**神**は、あなたがたの兄弟たちの中から、あなたがたのために私のような預言者を立たせます。あなたがたは彼の言葉を聞きます。』とイスラエルの子らに言ったのは、このモーセです。  
<span class="verse-number">38&ensp;</span>シナイ山で話しかけた御使いと共に、荒野での会合にいたのは、この人でした。また私たちの父祖たちと共に、私たちに与えるため、生きている啓示を受け取ったのは、この人でした。  
<span class="verse-number">39&ensp;</span>私たちの父祖たちはこの人に従わず、この人を拒否しました。そして、彼らは心の中で、エジプトに心を向け、  
<span class="verse-number">40&ensp;</span>アロンに言いました。『私たちの前を行く神々をつくってください。エジプトの地から私たちを連れ出したあのモーセは、今、どうなっているのか私たちは知らないからです。』  
<span class="verse-number">41&ensp;</span>そしてその頃、彼らは子牛を作り、偶像に供え物を捧げ、自らの手の技を楽しんだのです。  
<span class="verse-number">42&ensp;</span>それで、**神**は背を向け、天の軍を拝むよう、彼らを渡されました。預言者の巻物に書いてあるとおりです。『荒野にいた四十年の間、屠殺した動物と捧げ物をお前たちはわたしに捧げなかったのか。イスラエルの家よ。  
<span class="verse-number">43&ensp;</span>また、ひれ伏すために作った偶像、モロクの天幕とお前たちの神ロンパの星を、お前たちはかついだ。それで、わたしはバビロンの向こうまで、お前たちを移す。』  
<span class="verse-number">44&ensp;</span>私たちの父祖たちは、荒野に証の天幕を持っていました。それは**神**がモーセに、彼の見たとおりの形に従い作れ、と指示したものです。  
<span class="verse-number">45&ensp;</span>また私たちの父祖たちは、その天幕を何代に渡り受け継ぎ、異邦人所有の地へヨシュアと共に持って入り、ダビデの時代まで、父祖たちの目の前から、**神**が異邦人を追い出されました。  
<span class="verse-number">46&ensp;</span>ダビデは**神**の御前に恵みを見いだし、ヤコブの**神**のために住む所を探し出したいと願いました。  
<span class="verse-number">47&ensp;</span>しかし、**神**に家を建てたのは、ソロモンでした。  
<span class="verse-number">48&ensp;</span>しかし、最も高い方は、手で造られた神殿にはお住みになりません。それは、預言者が言うとおりです。  
<span class="verse-number">49&ensp;</span>『天はわたしの王座、また地球はわたしの足台である。わたしのために、どんな家を建てるか。』と**主**は言われます。『わたしの休息の場は何か。  
<span class="verse-number">50&ensp;</span>わたしの手は、これらのものをすべて造らなかったのか。』  
<span class="verse-number">51&ensp;</span>がんこで、心と耳に割礼を受けていない者たちよ。あなたたちの父祖たちと同様に、いつも**聖霊**に逆らっています。  
<span class="verse-number">52&ensp;</span>あなたがたの父祖たちが、迫害しなかったのはどの預言者ですか。そして、義なる方のおいでになるのを、前もってはっきりと語った預言者たちを、殺しました。あなたたちはその義なる方の裏切り者と、そして殺す者になりました。  
<span class="verse-number">53&ensp;</span>御使いたちの配慮によってそのあなたがたは律法を受けたが、守りませんでした。」  
<span class="verse-number">54&ensp;</span>彼らはこれを聞いて、心の中まで切り込まれ、ステパノに対して歯をかみならしていた。  
<span class="verse-number">55&ensp;</span>しかし、**聖霊**に満たされていたステパノは、天を見つめ、**神**の栄光と、**神**の右に立っておられるイエスを見た。  
<span class="verse-number">56&ensp;</span>そして言った。「見よ。天が開かれ、神の右に立っておられる、私は人の**子**が見えます。」  
<span class="verse-number">57&ensp;</span>すると、彼らは大声で叫び、耳をおおい、いっせいにステパノをめがけて駆け出した。  
<span class="verse-number">58&ensp;</span>そこで、ステパノを彼らは町から追い出し、彼に石投げで殺した。そして、この石投げの証人たちは自分たちの服をサウロと呼ばれている、若い男の足下に置いた。  
<span class="verse-number">59&ensp;</span>そして、「**主**イエスよ、私の霊を受け入れてください。」と**神**を呼び続けているステパノに、彼らは石を投げ続けていた。  
<span class="verse-number">60&ensp;</span>すると、ステパノはひざまずき、大声で叫んだ。「**主**よ、彼らにこの罪を負わせないでください。」と言い終わり、彼は眠りに落ちた。  

## 第8章

さて、そのサウロはステパノの殺人に賛成していた。その日、エルサレムの教会に対して、激しい迫害が起こり、使徒たち以外、全員は、ユダヤとサマリヤの各地方に散らされた。  
<span class="verse-number">2&ensp;</span>それで、信仰心厚い男たちは、埋葬のためステパノを運び出し、ステパノのことで悲嘆にくれた。  
<span class="verse-number">3&ensp;</span>しかし、サウロは教会に大打撃を与え、一軒残らず入り込み、男も女も引きずり出し、投獄した。  
<span class="verse-number">4&ensp;</span>人たちは散らされていたので、よき知らせの御言葉を説きながら、あらゆる所に行った。  
<span class="verse-number">5&ensp;</span>そうして、ピリポはサマリヤの町に下り、彼らにキリストを説いた。  
<span class="verse-number">6&ensp;</span>そして、群衆はピリポが話すことを、心を一つにして、注意深く聞き、彼の行なうしるしを見た。  
<span class="verse-number">7&ensp;</span>悪霊に占領されていた大勢の人から、汚れた霊は大声で叫びながら、出て来、また、中風の人や足の不自由な人が大勢治されたからである。  
<span class="verse-number">8&ensp;</span>そして、その町に大きな喜びがあった。  
<span class="verse-number">9&ensp;</span>しかし、その町に、シモンという男がいた。その町で以前、妖術を使い、サマリヤの人たちを驚かして、自らを偉大なる者であると、言っていた。  
<span class="verse-number">10&ensp;</span>地位のない者から、最高の地位の者まで、全員シモンに注目して、言っていた。「この人こそが**神**の偉大な力です。」  
<span class="verse-number">11&ensp;</span>そして、長い間、妖術で人を驚かしていたから、人はみな、彼に注目していた。  
<span class="verse-number">12&ensp;</span>しかし、**神**の王国に関すること、またイエス・キリストの御名に関してピリポが良き知らせを説いた時、彼らはピリポを信じ、男も女も浸礼を授けられた。  
<span class="verse-number">13&ensp;</span>それでシモン自身も信じ、そして、浸礼を授けられた時、彼は引き続きピリポと共にいて、力ある大きな業と、しるしが行なわれるのを見てひどく驚いた。  
<span class="verse-number">14&ensp;</span>さて、エルサレムにいた使徒たちが、サマリヤが**神**の御言葉を受け入れたと聞き、ペテロとヨハネを彼らに遣わした。  
<span class="verse-number">15&ensp;</span>二人は下って来て、彼らが**聖霊**を受け入れるように祈った。  
<span class="verse-number">16&ensp;</span>**聖霊**はまだ誰にも降っておらず、ただ**主**イエスの御名によって彼らは、浸礼を授けられただけであったからである。  
<span class="verse-number">17&ensp;</span>使徒たちが、彼らの頭に手を置いたその時、彼らは**聖霊**を受けた。  
<span class="verse-number">18&ensp;</span>すると、使徒たちの手を置くことを通し、**聖霊**が与えられるのを見たシモンは、お金を差し出し、  
<span class="verse-number">19&ensp;</span>言った。「私が手を置く人は誰であれ、**聖霊**を受けられるように、その力を私にもください。」  
<span class="verse-number">20&ensp;</span>しかし、ペテロはシモンに言った。「**神**の賜物はお金で買えるのだと思ったあなたの銀貨も、あなたも共に破滅に至るように。  
<span class="verse-number">21&ensp;</span>このことではあなたには、役目もなければ、場所もない。あなたの心は**神**の御前で正しくないからだ。  
<span class="verse-number">22&ensp;</span>だから、あなたのこの悪事を悔い改めよ。それから、あなたの心の思いはもしかして赦してくだされるかと、**神**に祈れ。  
<span class="verse-number">23&ensp;</span>なぜなら、あなたは胆汁で苦み、不義で縛られているのが私には見える。」  
<span class="verse-number">24&ensp;</span>するとシモンは答えて言った。「あなたが言ったことが何も、私に起こらないように、私のために**主**に祈ってください。」  
<span class="verse-number">25&ensp;</span>さて、使徒たちは証をし、**主**の御言葉を語り、サマリヤ人の多くの村で福音を伝え、エルサレムに帰った。  
<span class="verse-number">26&ensp;</span>それで、**主**の一人の御使いがピリポに話して、言った。「立ち上がり、南の方へ向かいなさい。エルサレムからガザまで下っていく道です。」そこは砂漠である。  
<span class="verse-number">27&ensp;</span>それで彼は立ち上がり、行った。すると見よ、エチオピヤの女王カンダケの宦官であり、女王の宝すべてを管理する、エチオピアの男が、礼拝するためにエルサレムに上って行き、  
<span class="verse-number">28&ensp;</span>帰る途中であった。そして、彼の馬車の中に座り、預言者イザヤの書を読んでいた。  
<span class="verse-number">29&ensp;</span>すると、**御霊**はピリポに言われた。「馬車に近づき、追いつき、その馬車と同行せよ。」  
<span class="verse-number">30&ensp;</span>そこで、ピリポは走って行き、宦官が預言者イザヤの書を読んでいるのを聞き、言った。「失礼ですが、読んでいることが理解できますか。」  
<span class="verse-number">31&ensp;</span>すると、彼は言った。「誰かが、私を導いてくれなければ、どうして理解できるでしょう。」そして、上がって共に座ることをピリポに願った。  
<span class="verse-number">32&ensp;</span>彼が読んでいた聖書の箇所はこれである。「彼はほふれるために連れて来られた羊のように、また、毛を刈る人の前に言葉のない子羊のように、その方は口を開けなかったのである。  
<span class="verse-number">33&ensp;</span>その方は謙虚で、彼の正義は取り上げられ、命は地球から取り去られたので、誰が彼の世代を語るのか。」  
<span class="verse-number">34&ensp;</span>それで、宦官はピリポに答えて言った。「あなたにお聞きします。この預言者は誰のことを言っているのですか。預言者自身ですか、それとも別の人ですか。」  
<span class="verse-number">35&ensp;</span>そこで、ピリポは自らの口を開き、この聖書の箇所をはじめ、宦官にイエスの福音を伝えた。  
<span class="verse-number">36&ensp;</span>さて、彼らが道を行くと、水場に来た。すると、宦官は言った。「見てください。水です。私が浸礼を授かるのに、妨げるものが何かあるでしょうか。」  
<span class="verse-number">37&ensp;</span>するとピリポは言った。「もしあなたが心を尽くして信じるのなら、可能です。」すると、「**神の御子息**はイエス・キリストであることを信じます。」と宦官は答えて言い、  
<span class="verse-number">38&ensp;</span>馬車を止めよと命じた。そして、ピリポも宦官も水に入り、ピリポは彼に浸礼を授けた。  
<span class="verse-number">39&ensp;</span>さて、二人は水から出ると、**主**の**御霊**はピリポを取り去られた。宦官はもうピリポを見ることなく、喜びながら自らの道を進んで行った。  
<span class="verse-number">40&ensp;</span>しかし、ピリポはアゾトで見つかり、カイザリヤに着くまで、すべての町を通り、福音を伝えていた。  

## 第9章

さて、**主**の弟子たちに対してまだ、脅しと殺人の息を口から吐き、サウロは大祭司の所に行き、  
<span class="verse-number">2&ensp;</span>ダマスコの各シナゴーグへ、大祭司からの手紙を願った。男であろうと女であろうと、この道の者を見つけたら、その者を縛り、エルサレムへ連れて行くためである。  
<span class="verse-number">3&ensp;</span>そして、サウロは旅をし、ダマスコに近づいた時、突然天からの光が彼の回りを照らした。  
<span class="verse-number">4&ensp;</span>すると、サウロは地面に倒れ、自分に言う声を聞いた。「サウロ、サウロ。なぜわたしを迫害しているのか。」  
<span class="verse-number">5&ensp;</span>そしてサウロは言った。「あなたはどなたですか、**主**よ。」すると**主**は言われた。「あなたが迫害しているイエスである。突棒[^9]に逆らい蹴るのは、あなたには難しい。」  
<span class="verse-number">6&ensp;</span>すると、震えかつ驚いているサウロは言った。「**主**よ、あなたは私が何をすることを望んでおられるのですか。」すると、**主**は彼に言われた。「立ち上がり、町に入れ。そうすると、あなたは何をすべきかが、教えられる。」  
<span class="verse-number">7&ensp;</span>サウロと共に旅をしていた男たちは、声を聞いたが、誰も見えず、何も言えず立っていた。  
<span class="verse-number">8&ensp;</span>それで、サウロは地面から立ち上がり、目を開けると、誰も見えなかった。そして、彼らはサウロの手を取り、彼を導き、ダマスコに入った。  
<span class="verse-number">9&ensp;</span>そして、サウロは三日間、目が見えず、食べたり飲んだりしなかった。  
<span class="verse-number">10&ensp;</span>しかし、ダマスコにアナニヤという一人の弟子がいた。そして**主**は幻を通して彼に話された。「アナニヤよ。」そして彼は言った。「ご覧ください、**主**よ。私です。」  
<span class="verse-number">11&ensp;</span>**主**は彼に言われた。「立ち上がって、『まっすぐ』と呼ばれている通りに行き、ユダの家でタルソスのサウロと言う名の人を探しなさい。なぜなら、見よ、彼は祈っているからである。  
<span class="verse-number">12&ensp;</span>そして、アナニヤという名の人が入って来て、目が以前のように見えるようにと、片手を彼に置くのを幻を通して見た。」  
<span class="verse-number">13&ensp;</span>すると、アナニヤは答えた。「**主**よ、エルサレムでこの男は、あなたの聖徒たちにどんなにひどいことをしたか、多くの人からこの男のことを私は聞いています。  
<span class="verse-number">14&ensp;</span>そして、ここであなたのお名前を呼ぶ人たちを、全員縛る権威を大祭司から受けています。」  
<span class="verse-number">15&ensp;</span>しかし、**主**は彼に言われた。「行きなさい。異邦人、王たち、それとイスラエルの子らの前に、サウロはわたしの名を持って行くための、わたしの選んだ器だからである。  
<span class="verse-number">16&ensp;</span>それは、わたしの名のため、彼がどんなに多く苦しまなければならないかを、わたしは彼に示すためである。」  
<span class="verse-number">17&ensp;</span>それで、アナニヤは行き、その家に入り、サウロに両手を置いて言った。「サウロ兄弟よ。あなたが来る途中で、あなたに現れた**主**イエスが、あなたが以前のように見えるようになり、そして**聖霊**で満たされるようにと、私を遣わされました。」  
<span class="verse-number">18&ensp;</span>すると、たちどころにサウロの目からうろこのような物が落ち、彼の目は以前のように見えるようになった。そして、彼は立ち上がり、浸礼を授けられた。  
<span class="verse-number">19&ensp;</span>それで、食事をし、元気になり、それから、幾日かを弟子たちと共にサウロはダマスコで過ごした。  
<span class="verse-number">20&ensp;</span>そこで、彼はすぐシナゴーグでキリストのこと、すなわち**神の御子息**であると説いた。  
<span class="verse-number">21&ensp;</span>すると、聞いたすべての人たちは驚いて、言った。「エルサレムでこの名を呼び求めた人たちを打ち倒し、またその人たちを縛ったままで、大祭司たちの所に連れて行けるよう、ここに来たのは、この男ではありませんか。」  
<span class="verse-number">22&ensp;</span>サウロはますます力を増し、イエスはキリストであることを証明し、ダマスコに住んでいるユダヤ人を戸惑わせた。  
<span class="verse-number">23&ensp;</span>それで、多くの日が過ぎてから、ユダヤ人たちはサウロを殺そうと企んだ。  
<span class="verse-number">24&ensp;</span>しかし、彼らの企みはサウロに知られた。彼らは昼も夜もそれぞれの門を見張っていた。サウロを殺すためである。  
<span class="verse-number">25&ensp;</span>それで、弟子たちは夜中サウロを連れて、大きなかごで、壁を通して彼を降ろした。  
<span class="verse-number">26&ensp;</span>そして、サウロがエルサレムに来て、弟子たちの仲間入りをはかったが、弟子たちはみんなサウロを恐れ、彼が弟子であることを信じなかった。  
<span class="verse-number">27&ensp;</span>しかし、バルナバは彼を受け入れ、使徒たちの所へ連れて行った。それで、サウロが道で**主**を見て、**主**がサウロに話され、サウロがダマスコでイエスの御名によって大胆に説いたことを使徒たちに伝えた。  
<span class="verse-number">28&ensp;</span>それで、サウロは彼らと共にエルサレムに入ったり出たりしていた。  
<span class="verse-number">29&ensp;</span>そして彼は**主**イエスの御名によって大胆に話し、ギリシャ語を話すユダヤ人たちに反論していた。それで彼らはサウロを殺そうとした。  
<span class="verse-number">30&ensp;</span>それを知った兄弟たちは、彼をカイザリヤに連れて下り、タルソへ送った。  
<span class="verse-number">31&ensp;</span>それで、ユダヤとガリラヤとサマリヤの地方の諸教会には平安があり、基礎ができあがった。そして、**主**の恐れと**聖霊**の励ましによって歩み、人の数は増えた。  
<span class="verse-number">32&ensp;</span>さて、ことの次第はこうであった。ペテロは全土をまわり、ルダに住む聖徒たちの所にも下った折に、  
<span class="verse-number">33&ensp;</span>そこで、中風で、八年間寝たきりのアイネヤという名の人を見いだし、  
<span class="verse-number">34&ensp;</span>そして、ペテロは彼に言った。「アイネヤよ、キリストであるイエスはあなたを治してくださいます。立ち上がり、床をかたづけなさい。」すると、彼はすぐ立ち上がった。  
<span class="verse-number">35&ensp;</span>それで、ルダとサロンに住んでいた者はみな、彼を見て、**主**に向かい、**主**を信頼した。  
<span class="verse-number">36&ensp;</span>ヨッパにタビタという弟子がいた。訳すとドルカスである。この女性は自らの善行と施しのわざで、満たされていた。  
<span class="verse-number">37&ensp;</span>しかし、ことの次第はこうであった。そのころ、タビタは病気になり死んだ。人たちは彼女を洗って、上の部屋に横たえた。  
<span class="verse-number">38&ensp;</span>ルダはヨッパに近いので、弟子たちは、ペテロがそこにいると聞いていたので、二人の男をペテロに遣わし、彼らを待たすことのないように、来てくれるよう願った。  
<span class="verse-number">39&ensp;</span>それで、ペテロは立ち上がり、二人と共に行った。着いた時、人たちは上の部屋に彼を連れて行った。そして、未亡人たちはみな、彼のそばに立ち、泣きながらドルカスが彼女らと共にいた時に作った衣と、上着のすべてを見せていた。  
<span class="verse-number">40&ensp;</span>しかし、ペテロは全員外に出して、ひざまずいて祈った。そして、その体の方に向きを変え、ペテロは言った。「タビタよ、立ち上がれ。」すると彼女は目を開けた。そして、ペテロを見て、彼女は起きて座った。  
<span class="verse-number">41&ensp;</span>それで、ペテロは手をかし、彼女を立たせた。そして、聖徒たちと未亡人たちを呼び、生きているタビタを示した。  
<span class="verse-number">42&ensp;</span>これはヨッパ中に知られるようになり、多くの人たちは**主**を信じた。  
<span class="verse-number">43&ensp;</span>このようにして、ペテロはヨッパのなめし屋、シモンの所に多くの日々、滞在した。  

## 第10章

さて、カイザリヤにコルネリオと言う男がいた。彼は「イタリア」と呼ばれていた六百人部隊の百人隊長で、  
<span class="verse-number">2&ensp;</span>敬虔な人で、家の者全員と共に**神**を恐れ、人々に気前よく施し、常に**神**に祈っていた。  
<span class="verse-number">3&ensp;</span>その日の第九時間目ごろ[^10]、彼は幻で、**神**の御使いが入って来て、彼に「コルネリオよ」、と言っているのを、はっきり見た。  
<span class="verse-number">4&ensp;</span>そして、御使いをよく見ると、彼は怖くなり、言った。「**主**よ、何事ですか。」すると御使いは言った。「お前の祈りと施しは、記念のため**神**の前に上がって来た。  
<span class="verse-number">5&ensp;</span>それで今すぐ、ヨッパに男たちを遣わし、ペテロと名付けれらたシモンを迎えに行け。  
<span class="verse-number">6&ensp;</span>その人は、海辺のなめし屋、シモンの家に滞在している。お前は何をすべきかを、その人は教えてくれる。」  
<span class="verse-number">7&ensp;</span>それで、コルネリオに話しかけた御使いが去って行くと、彼は自分の家の僕二人と、いつも彼に仕えている敬虔な兵隊の一人を呼び、  
<span class="verse-number">8&ensp;</span>そして、これらすべてを彼らに説明してから、彼は三人をヨッパに遣わした。  
<span class="verse-number">9&ensp;</span>次の日、彼らは旅を進め、町に近づくと、ペテロは祈るため、屋上に上って行った。第六時間目のころ[^11]である。  
<span class="verse-number">10&ensp;</span>それで、彼は空腹になり、食を求めた。しかし、彼らが準備をしている間に、彼の上に恍惚状態に落ちた。  
<span class="verse-number">11&ensp;</span>彼は見た。開かれて天、巨大なシーツのような物体、四隅が結ばれ、ペトロに下り、地上に達した。  
<span class="verse-number">12&ensp;</span>その中に、地球の、四足動物全種と野生の獣、這う物、空の鳥がいた。  
<span class="verse-number">13&ensp;</span>そして、声がペテロに響いた。「立ち上がれ、ペテロよ、ほふり、食べよ。」  
<span class="verse-number">14&ensp;</span>しかし、ペテロは言った。「それはだめです、**主**よ。私は決して清くない物や汚れた物を食べたことがないからです。」  
<span class="verse-number">15&ensp;</span>すると、また声が彼に響いた。二度目である。「**神**が清められた物を、お前は『清くないもの』と呼んではいけない。」  
<span class="verse-number">16&ensp;</span>このことは三度行なわれた。そして、その物体は再び空中へと取り上げられた。  
<span class="verse-number">17&ensp;</span>さて、ペテロは彼自身の中で見たこの幻が、何を意味するのかと迷っている間に、見よ、コルネリオから遣わされた男たちは、シモンの家を尋ね、門前に立っていた。  
<span class="verse-number">18&ensp;</span>そして、彼らは声を出し、ペテロと名付けられているシモンはここに滞在しているかと尋ねた。  
<span class="verse-number">19&ensp;</span>しかし、ペテロは幻について熟考していると、**御霊**が彼に言われた。「見よ。三人の男があなたを探しています。  
<span class="verse-number">20&ensp;</span>立ち上がり、降りて行き、何も疑わずに彼らと共に行きなさい。わたしが彼らを遣わしたのだからです。」  
<span class="verse-number">21&ensp;</span>それで、コルネリオから遣わされた男たちの所へ、ペテロは降りて行き、言った。「見よ。この私が、あなたがたの探している者です。あなたがたは、何の理由で来たのですか。」  
<span class="verse-number">22&ensp;</span>すると、彼らは言った。「百人隊長コルネリオは正しい男で、**神**を恐れ、ユダヤ人の国中で評判がよく、聖なる御使いによりあなたを彼の家に迎えに行き、あなたからの言葉を聞くように指示されました。」  
<span class="verse-number">23&ensp;</span>それで、ペテロは彼らを呼び入れ、滞在させた。そして翌日、ペテロは彼らと共に出かけた。また、ヨッパからの兄弟たち何人かが同行した。  
<span class="verse-number">24&ensp;</span>そしてその翌日、彼らはカイザリヤに入った。そこでコルネリオはすでに親戚や親友を呼び集めて、彼らを待っていた。  
<span class="verse-number">25&ensp;</span>そして、ペテロは入って行くと、コルネリオは彼に会うとすぐ、足もとにひれ伏して、彼を拝んだ。  
<span class="verse-number">26&ensp;</span>しかし、ペテロは彼を立たせて、言った。「立ちなさい。私自身も人間です。」  
<span class="verse-number">27&ensp;</span>そして、彼と話しながら、ペテロは中に入り、集まった大勢の人たちがいるのを見た。  
<span class="verse-number">28&ensp;</span>ペテロは彼らに言った。「ユダヤ人の男が他国の人と接したり、共に行くことは、どれほど違法であるかを、あなたがたはご存じです。しかし、どの人が清いとか、汚れているとか、言ってはいけない、と**神**は私に示してくださいました。  
<span class="verse-number">29&ensp;</span>それで、迎えを受けるとすぐ、私は一言も異議を唱えず来ました。それでお聞きします。何のことで私を迎えに来たのですか。」  
<span class="verse-number">30&ensp;</span>それで、コルネリオは言った。「四日前から、私はこの時刻まで断食をしていました。そして、第九時間目[^12]、私は私の家で祈っていました。すると、見よ、輝く服を着ている男が私の前に立っていました。  
<span class="verse-number">31&ensp;</span>そして、彼は言いました。『コルネリオよ、あなたの祈りは聞かれて、あなたの施しは**神**の御前に覚えられています。  
<span class="verse-number">32&ensp;</span>ですから、ヨッパに遣わし、そこでペテロと名付けられているシモンを招きなさい。この男は、海辺に家がある、なめし屋のシモンの所に滞在しています。彼は来ると、あなたに話しかけます。』  
<span class="verse-number">33&ensp;</span>ですから、私はすぐにあなたに人たちを遣わしました。よくぞ来てくださいました。さあ、私たち全員、**神**の御前に今います。**神**により命じられたすべてのことを聞くためです。」  
<span class="verse-number">34&ensp;</span>それから、ペテロは口を開き、言った。「まことに、**神**は差別を示されない方であられることに、気付きました。  
<span class="verse-number">35&ensp;</span>それどころか、いかなる国の人でも、**神**を恐れ、義の働きをする人は**神**に受け入れられることもです。  
<span class="verse-number">36&ensp;</span>**神**がイスラエルの子らにお送りになった御言葉、すなわち、イエス・キリストを通して、平安の福音を伝えることです。この方はすべての**主**です。  
<span class="verse-number">37&ensp;</span>ガリラヤでヨハネが説いた浸礼の後、ガリラヤから始まり、ユダヤの全土で説かれた、あなたがたの知っているあの御言葉です。  
<span class="verse-number">38&ensp;</span>**神**がナザレのイエスになさったのは、**聖霊**と力で油を注がれました。そのイエスは、**神**が共におられたから、悪魔に抑圧されているすべての人を治し、よいことをしながら、巡回しておられました。  
<span class="verse-number">39&ensp;</span>そしてユダヤ人の地、エルサレム、両方の地で、その方が行なわれたすべてのことの、私たちは証人です。そして、彼らはその方を木にかけて殺してしまいました。  
<span class="verse-number">40&ensp;</span>**神**はそのイエスを三日目に復活させ、公に彼を示されました。  
<span class="verse-number">41&ensp;</span>全国民でなく、**神**の御前に前もって選ばれた証人たちに、すなわちキリストが死人の中から復活された後、その方と共に飲食した私たちに示されました。  
<span class="verse-number">42&ensp;</span>そして、キリストは生きている人たちと、死んだ人たちの裁き人になるよう、**神**により任命された方であることを、私たちに国民に説くよう、**神**は命じられました。  
<span class="verse-number">43&ensp;</span>彼を信じる人は誰でも、キリストの御名を通し、罪の赦しを受けると、全預言者がこの方のことを証しています。」  
<span class="verse-number">44&ensp;</span>ペテロがこれらの言葉をまだ話していた間、聞いている人々すべての上に**聖霊**が降られた。  
<span class="verse-number">45&ensp;</span>すると、ペテロと共に来ていた、割礼の信者、全員は驚いた。**聖霊**の賜物は異邦人にも注がれていたからである。  
<span class="verse-number">46&ensp;</span>その人たちは違う言語で話し、かつ**神**を賛美するのを聞いたからである。それで、ペテロは答えた。  
<span class="verse-number">47&ensp;</span>「**聖霊**を私たちと同様に受け入れたこの人たちが、浸礼されないように、水を拒否できる人はいますか。」  
<span class="verse-number">48&ensp;</span>それで、彼らが**主**の御名によって浸礼を授けられるよう、ペテロは彼らに命じた。その後、彼らは数日滞在してくれるように彼に願った。  

## 第11章

さて、異邦人も**神**の御言葉を受けたことを、使徒たちとユダヤにいる兄弟たちも聞いた。  
<span class="verse-number">2&ensp;</span>そして、ペテロはエルサレムに上った時、割礼の人たちは彼と論争して、  
<span class="verse-number">3&ensp;</span>言った。「あなたは、割礼を受けていない男たちの所に入り込み、彼らと共に食事をしました。」  
<span class="verse-number">4&ensp;</span>しかし、ペテロは最初から順序だてて彼らに説明し、言った。  
<span class="verse-number">5&ensp;</span>「私はヨッパの町で祈っていると、恍惚状態で私は幻を見ました。巨大なシーツのような物体が四隅が結ばれ、下ってきて、私の所に来ました。  
<span class="verse-number">6&ensp;</span>それをじっと見ながら考えていると、私は四つ足の動物と野生の獣と這うものと空の鳥を見ました。  
<span class="verse-number">7&ensp;</span>また、私に言う声を聞きました。『ペテロよ、立ち上がって、ほふって、食べよ。』  
<span class="verse-number">8&ensp;</span>しかし、私は言った。『だめです、**主**よ。私は決して清くない物や汚れた物を口に入れたことがないからです。』  
<span class="verse-number">9&ensp;</span>しかし、その声は再び天から私に答えました。『**神**が清められた物を、あなたは清くないと呼んではいけない。』  
<span class="verse-number">10&ensp;</span>それで、これは三度行なわれ、すべて再び天に引き上げられました。  
<span class="verse-number">11&ensp;</span>正に、ちょうどその時、カイザリヤから遣わされた三人の男が、私がいた家の前に立ったのです。  
<span class="verse-number">12&ensp;</span>次に、**御霊**は、何も疑うことなく、彼らと共に行け、と私に言われました。それで、この六人の兄弟は私と共に行って、みんなでその男の家に入りました。  
<span class="verse-number">13&ensp;</span>そして、家の中に立っている御使いを見、その御使いが、『ヨッパに男たちを遣わし、ペテロと名付けられているシモンを迎えに行け。』と言った事の次第を話しました。  
<span class="verse-number">14&ensp;</span>『その人は、あなたとあなたの家の者全員が救われる言葉を、教えてくれる。』と言いました。  
<span class="verse-number">15&ensp;</span>そして、私は話しかけると、最初の時に我々の上に降られたと同じように、**聖霊**は彼らの上に降りて来られました。  
<span class="verse-number">16&ensp;</span>それで、私は**主**の言葉を思い出しました。**主**はこのように言われました。『ヨハネは水で浸礼を授けたが、あなたがたは**聖霊**で浸礼されます。』  
<span class="verse-number">17&ensp;</span>従って、私たちが**主**イエス・キリストを信じた時と同様に、**神**は彼らに全く同じ賜物を与えられたのだから、私が**神**に逆らえるのなら、私は何者でしょう。」  
<span class="verse-number">18&ensp;</span>これらの事を聞いて、彼らは黙った。そして、彼らは**神**に栄光を捧げ、言った。「では、**神**は異邦人にも、命に至る悔い改めを与えられたのです。」  
<span class="verse-number">19&ensp;</span>さて、ステパノのことで起きた迫害で、散らされた人たちは、フェニキヤ、キプロス、アンテオケまで行ったので、ユダヤ人にしか御言葉を語っていなかった。  
<span class="verse-number">20&ensp;</span>しかし、彼らの中に、数人のキプロス人とクレネ人がいた。そして、その人たちがアンテオケに着いた時、ギリシャ派のユダヤ人たちに話しかけ、そして**主**イエスの福音を伝えていた。  
<span class="verse-number">21&ensp;</span>そして、**主**の手は彼らと共にあり、大勢の人たちは信じ、**主**の方へと向きを変えた。  
<span class="verse-number">22&ensp;</span>それで、この出来事に関するニュースは、エルサレムの教会の耳に達し、そして、彼らはアンテオケにバルナバを遣わした。  
<span class="verse-number">23&ensp;</span>彼は行き、**神**の恵みをすでに見たので、喜び、そして、心に定め、**主**にしっかりついて行くよう、彼ら全員に勧めた。  
<span class="verse-number">24&ensp;</span>彼はよい男で、**聖霊**と信仰で満たされていたからである。それで、かなり多くの人たちが**主**に加えられた。  
<span class="verse-number">25&ensp;</span>しかし、バルナバはサウロを捜しにタルソに出発した。  
<span class="verse-number">26&ensp;</span>そして彼を見つけ、アンテオケに連れて行った。それでことの次第はこうであった。彼らは教会と共に集い、多くの人たちを教え、丸一年間過ごした。そして、弟子たちはアンテオケで初めてクリスチャンと呼ばれた。  
<span class="verse-number">27&ensp;</span>さて、そのころエルサレムから預言者たちはアンテオケに下って来た。  
<span class="verse-number">28&ensp;</span>その中の一人、アガボと言う名の者が立ち上がり、世界中に大飢饉が起こる、と**御霊**によって示した。それはクラウデオ・カイザルの治世に起きた。  
<span class="verse-number">29&ensp;</span>それで、弟子たちは一人一人の能力に応じ、ユダヤに住んでいる兄弟たち援助を送ることを決めた。  
<span class="verse-number">30&ensp;</span>彼らはこれを実行し、バルナバとサウロの手によって長老たちに送った。  

## 第12章

さて、この時、ヘロデ王は教会の一部の人たちを危害を加えようと、手を伸ばし、  
<span class="verse-number">2&ensp;</span>ヨハネの兄弟、ヤコブを剣で殺した。  
<span class="verse-number">3&ensp;</span>すると、このことがユダヤ人を喜ばしたとわかったヘロデは、ペテロも逮捕した。種なしパンの期間中のことであった。  
<span class="verse-number">4&ensp;</span>それで、ヘロデはペテロを逮捕し、投獄し、四人からなる一組の兵士四組に引き渡し、彼を守らせた。過越祭の後、国民の前に、彼を引き出すつもりであった。  
<span class="verse-number">5&ensp;</span>従って、ペテロは牢獄に閉じ込められていたが、彼のため、教会によって祈りは中断することなく**神**に捧げられた。  
<span class="verse-number">6&ensp;</span>そして、ヘロデは彼を引き出そうとした時の、その夜、ペテロは二本の鎖につながれて二人の兵士の間で眠っていた。そして、扉の前にいた番兵たちは牢獄を守っていた。  
<span class="verse-number">7&ensp;</span>すると、見よ。**主**の御使いの一人が、彼の脇に立ち、一筋の光が牢獄を照らした。そして、彼はペテロのわき腹を打ち、彼を立たせ、言った。「すぐ立ちなさい。」すると、彼の鎖は彼の手から離れ落ちた。  
<span class="verse-number">8&ensp;</span>御使いは彼に言った。「帯をしめて、サンダルを結びなさい。」ペテロはそうした。そして彼はペテロに言った。「上衣を着て、私について来なさい。」  
<span class="verse-number">9&ensp;</span>それでペテロは外に出て、御使いについて行ったが、御使いを通して行なわれた事は、現実のこととは思わず、幻を見ていると思った。  
<span class="verse-number">10&ensp;</span>二人は第一、そして第二の番兵の所を通過し、町に通じる鉄の門に着くと、門はひとりでに開いたので、彼らは出て行き、ある通りを下って行くと、御使いはペテロから急に離れた。  
<span class="verse-number">11&ensp;</span>それで、ペテロは我に返り言った。「今、私は確信しました。**主**が御使いをお遣わしになり、ヘロデの手、また、すべてのユダヤ国民の期待から私を救い出してくださいました。」  
<span class="verse-number">12&ensp;</span>それで彼はこのことを理解し、多くの人が集まって祈っていた所、すなわち、マルコと言う名のヨハネの母の家に行った。  
<span class="verse-number">13&ensp;</span>それで、ペテロは門の扉を叩くと、ロダと言う名の少女が応対に出た。  
<span class="verse-number">14&ensp;</span>ペテロの声であるとわかると、彼女は喜びのあまり門を開けずに駆け込み、ペテロが門の前に立っていると伝えた。  
<span class="verse-number">15&ensp;</span>であるが彼らは少女に言った。「お前は狂っている。」しかしなお、少女が、そうなのだと、言い張ったので、彼らは、「それは彼の使いの者だ。」と言った。  
<span class="verse-number">16&ensp;</span>しかし、ペテロは叩き続けると、彼らは扉を開け、ペテロを見て驚いた。  
<span class="verse-number">17&ensp;</span>しかし、ペテロは何も言うなと手で合図し、**主**が彼を牢獄からどのようにして出してくださったかを、彼らに語った。そして彼は言った。「行って、ヤコブと兄弟たちにこれらのことを伝えなさい。」そして彼は去って行き、他の所に行った。  
<span class="verse-number">18&ensp;</span>さて、明るくなるやいなや、ペテロに起きたことに関して、小さくはない騒ぎが兵士たちの間にあった。  
<span class="verse-number">19&ensp;</span>しかし、ヘロデはペテロを捜しても、見つけられず、番兵たちを取り調べ、死刑にすべしと命じた。それから、ユダヤからカイザリヤまで下って行き、そこに滞在した。  
<span class="verse-number">20&ensp;</span>さて、ヘロデはツロとシドンの市民に対して大変怒っていたが、市民たちは心を合わせて彼の所に行き、王の侍従であるブラストに取り入り、平和を求めた。彼らの国はこの王の国から食料を供給されていたからである。  
<span class="verse-number">21&ensp;</span>それで、設定された日に、王の盛装したヘロデは、王座に座り、彼らに向かい演説をした。  
<span class="verse-number">22&ensp;</span>すると、市民たちは叫び続けた。「神の声だ！人間の声ではない！」  
<span class="verse-number">23&ensp;</span>すると即座に、**主**の一人の御使いが彼を打った。**神**に栄光を捧げなかったからである。そして、彼は這う虫たちに食べられ、息が絶えた。  
<span class="verse-number">24&ensp;</span>しかし、**神**の御言葉は成長し、数をました。  
<span class="verse-number">25&ensp;</span>さて、使命を果たして、バルナバとサウロは、マルコと言う名のヨハネも連れ、エルサレムから帰ってきた。  

## 第13章

さて、アンテオケの教会に、預言者や教師がいた。バルナバとニゲルと呼ばれるシメオンとクレネ人ルキオと領主ヘロデと共に育ったマナエンとサウロである。  
<span class="verse-number">2&ensp;</span>彼らは**主**に仕えて、断食をしていると、**聖霊**が言われた。「バルナバとサウロをわたしのために分けよ。わたしがさせる働きのため、彼らを召し出したのだから。  
<span class="verse-number">3&ensp;</span>それから、彼らは断食し、祈り、二人に手を置き、送り出した。  
<span class="verse-number">4&ensp;</span>**聖霊**に遣わされた二人は、セルキヤに下り、そこから船でキプロスに行った。  
<span class="verse-number">5&ensp;</span>そして、サラミスに到着すると、ユダヤ人のシナゴーグで**神**の言葉を教えた。そして、彼らはヨハネを助手とした。  
<span class="verse-number">6&ensp;</span>さて、島を通ってパポスに行くと、バルイエスというユダヤ人で、一人のにせ預言者の魔術師に出会った。  
<span class="verse-number">7&ensp;</span>魔術師は地方総督セルギオ・パウロと共にいた。総督は聡明な人であった。この人はバルナバとサウロを迎えにやり、**神**の御言葉を聞くことを願った。  
<span class="verse-number">8&ensp;</span>しかし、魔術師のエルマ（彼の名前はそう訳すべきである）は、使徒たちに反対して、地方総督を信仰から遠ざけようとした。  
<span class="verse-number">9&ensp;</span>それで、パウロとも呼ばれているサウロは、**聖霊**に満たされて、エルマをしっかり見つめ、  
<span class="verse-number">10&ensp;</span>言った。「あらゆる悪巧み、あらゆる不正で満ちている、あなた、悪魔の息子よ、あらゆる義の敵よ、**神**のまっすぐな道を曲げることを止めないか。  
<span class="verse-number">11&ensp;</span>そして今、見よ。**主**の御手は今、あなたに置かれている。そしてあなたは盲目になり、しばらく太陽を見ない。」すると、突然、彼の回りに黒い霧が下り、彼は手を引いて導いてくれる人を捜し回った。  
<span class="verse-number">12&ensp;</span>すると、何が起きたかを目にしたこの地方総督は、**主**の教えに驚愕し、信じた。  
<span class="verse-number">13&ensp;</span>さて、パウロと仲間たちはパポスから出航し、パンフリヤのペルガに行った。そして、ヨハネは彼らから別れ、エルサレムに戻った。  
<span class="verse-number">14&ensp;</span>そして、彼らはペルガを離れ、ピシデヤのアンテオケに行った。彼らは安息日にシナゴーグに入り、座った。  
<span class="verse-number">15&ensp;</span>律法と預言書の講読の後、シナゴーグの支配者たちは彼らに人を送って、言った。「男たちよ、兄弟たちよ、民のため、励みの言葉があったら、言ってください。」  
<span class="verse-number">16&ensp;</span>するとパウロは立ち上がり、手で合図し、言った。「イスラエルの男たち、**神**を恐れる人たちよ、お聞きなさい。  
<span class="verse-number">17&ensp;</span>このイスラエルの民の**神**は、私たちの先祖を選び、エジプトに、外国人として住んでいた彼らを、高められました。そして、御腕を高く上げ、民をエジプトから連れ出してくださいました。  
<span class="verse-number">18&ensp;</span>そこで、およそ四十年間、荒野での民の行ないを堪え忍ばれました。  
<span class="verse-number">19&ensp;</span>そして、カナンの地方で、**神**は七つの国を滅ぼし、イスラエルの土地を彼らにくじで分配されました。  
<span class="verse-number">20&ensp;</span>その後、約四百五十年の間、預言書サムエルの時まで、**神**は士師を与えられました。  
<span class="verse-number">21&ensp;</span>その後、彼らは王を願い、**神**はベニヤミン族の男、キスの息子サウロを四十年間与えられました。  
<span class="verse-number">22&ensp;</span>そして、サウロを退けてから、**神**はダビデを王として彼らのため、立たせられました。**神**はダビデに証して言われました。『わたしの心にかなう男、わたしの意志をすべて行なう男、エッサイの息子ダビデをわたしは見つけだした。』  
<span class="verse-number">23&ensp;</span>この男の種から、約束により、**神**はイスラエルのため、**救い主**、イエスを立たせられました。  
<span class="verse-number">24&ensp;</span>ヨハネがイスラエルのすべての民に、悔い改めの浸礼を初めて教えた後、すなわちイエスが来られる前でした。  
<span class="verse-number">25&ensp;</span>そして、ヨハネは道程を終える頃に、言った。『あなたがたは、私を誰だと思っているのですか。私はあの方じゃありません。しかし、見よ。私の後から来られる方がおられます。私はその方の足のサンダルのひもを解く価値もありません。』  
<span class="verse-number">26&ensp;</span>男たち、兄弟たちよ、またアブラハムの家の息子たちよ、またあなたがたの中で**神**を恐れる人たちよ、この救いの言葉は、あなたがたに送られたのです。  
<span class="verse-number">27&ensp;</span>なぜなら、エルサレムに住んでいる者たちとその支配者たちは、イエスを知らず、また安息日毎に読まれる預言者たちの声さえ知らなかったので、その方を裁くことにより、これらの声を成就してしまいました。  
<span class="verse-number">28&ensp;</span>そして、その方の中に、死に価する理由を見いだせなかったのに、彼を死刑にすべしと、ピラトに願いました。  
<span class="verse-number">29&ensp;</span>その方について書かれたすべてのことが成就して、彼を木から降ろして、墓の中に置きました。  
<span class="verse-number">30&ensp;</span>しかし、**神**はその方を死人の中から復活させてくださいました。  
<span class="verse-number">31&ensp;</span>イエスと共にガリラヤからエルサレムに上った人たちにより、何日もイエスは目撃され、そして彼らはこの民への証人です。  
<span class="verse-number">32&ensp;</span>そして、この福音、私たちの先祖たちにされた約束を、私たちはあなたがたに伝えます。  
<span class="verse-number">33&ensp;</span>イエスを復活させ、**神**はこのことを我らのため、彼らの子どものため成就されました。詩篇第二篇にも書いてあるように、『あなたはわたしの**息子**である。わたしは今日、あなたを生んだ。』  
<span class="verse-number">34&ensp;</span>そして、もう朽ちるものに戻ることのないよう、**神**は死人たちの中から、イエスを復活させてくださったことについて、こう言われました。『ダビデの確かな聖なるものをわたしはあなたに与える。』  
<span class="verse-number">35&ensp;</span>従って、詩篇の別の所では、**神**はこう言っておられます。『あなたはあなたの**聖なる者**に、その者が腐りはてることを見せることは許さない。』  
<span class="verse-number">36&ensp;</span>それで、**神**のご意志により、彼の世代に仕えてから、ダビデは眠りに落ち、彼の先祖と共にうめられ、彼は腐りはてることを見ました。  
<span class="verse-number">37&ensp;</span>しかし、**神**が復活させた方は、腐りはてることをご覧にならなかったのです。  
<span class="verse-number">38&ensp;</span>従って、この方を通して、あなたがたに語られている、罪の赦しを、知っていただきたいです。  
<span class="verse-number">39&ensp;</span>そして、モーセの律法によって義と認められなかったすべてのことも、信じているすべての人は、この方によって、義と認められています。  
<span class="verse-number">40&ensp;</span>従って、預言者たちによって語られたことが、あなたの上に来ないように気をつけなさい。  
<span class="verse-number">41&ensp;</span>『見よ、侮辱する者たちよ。驚け、そして滅びよ。わたしはお前たちの日に、誰がお前たちに伝えても、お前たちが決して信じないわざを、わたしは行なうからだ。』」  
<span class="verse-number">42&ensp;</span>それで、ユダヤ人はシナゴーグから出て行く時、この言葉を次の安息日に彼らに語ってくれるように願った。  
<span class="verse-number">43&ensp;</span>そこで、シナゴーグの集会は終わったので、ユダヤ人と敬虔な改宗者は大勢パウロとバルナバに従った。二人は、彼らに話し、**神**の恵みにとどまるよう説得した。  
<span class="verse-number">44&ensp;</span>次の安息日に、町のほぼ全員、**神**の御言葉を聞きに集まった。  
<span class="verse-number">45&ensp;</span>しかし、ユダヤ人は群衆を見て、妬みで満たされ、反対し、冒涜し、パウロの話した言葉に敵対した。  
<span class="verse-number">46&ensp;</span>しかし、パウロとバルナバは大胆に言った。「**神**の御言葉は、まずあなたがたに語るべきだったが、あなたがたはそれを拒み、自らを永遠の命にふさわしくないと判断しているから、ほら、私たちは異邦人の方を向いています。  
<span class="verse-number">47&ensp;</span>なぜなら、**主**はそのように私たちに命令されました。『わたしはあなたがたを異邦人への光として置いた。地球の果てまでの救いにするためである。』」  
<span class="verse-number">48&ensp;</span>すると、異邦人たちはこれを聞き、喜び、**主**の御言葉に栄光をささげた。そして、永遠の命に指名されたみなの人は信じた。  
<span class="verse-number">49&ensp;</span>そして、**主**の御言葉は、この地方全体に広げられた。  
<span class="verse-number">50&ensp;</span>しかし、ユダヤ人たちは敬虔で優れた女たちと、町のおもだった男たちを扇動し、パウロとバルナバに対して迫害を加え、このユダヤ人たちの地から追い出した。  
<span class="verse-number">51&ensp;</span>しかし、二人は彼らに対し足のほこりを払い落とし、イコニオムに行った。  
<span class="verse-number">52&ensp;</span>そして弟子たちは喜びと**聖霊**で満たされていた。  

## 第14章

さて、ことの次第はこうであった。イコニオムでユダヤ人のシナゴーグに彼らは共に入り、多数のユダヤ人、ギリシャ人も共に信じるように彼らは説いた。  
<span class="verse-number">2&ensp;</span>しかし、信じないユダヤ人たちは、異邦人たちを動揺させ、兄弟たちに反対し、彼らの心に毒を入れた。  
<span class="verse-number">3&ensp;</span>それで、彼らは長い間そこに滞在し、**主**にあって大胆に語った。ご自分の恵みの言葉を証する**主**は、彼らの手を通して、しるしと不思議なわざを行なうことを許された。  
<span class="verse-number">4&ensp;</span>しかし、町の大多数は分かれた。ユダヤ人側と、使徒たち側であった。  
<span class="verse-number">5&ensp;</span>それで、異邦人たち、ユダヤ人たちは、彼らの支配者たちと共に押し寄せ、彼ら二人を不当に扱い、石で殺そうとしたので、  
<span class="verse-number">6&ensp;</span>二人はそれに気づき、ルカオニヤの町であるルステラとデルベ、そしてその付近の地方にのがれた。  
<span class="verse-number">7&ensp;</span>そして、彼らはそこで福音を伝えていた。  
<span class="verse-number">8&ensp;</span>そして、ルステラに、足に力のない男が座っていた。母の胎内から足が不自由で、歩いたことは決してなかった。  
<span class="verse-number">9&ensp;</span>この男はパウロが話すのを聞き、パウロは彼を見つめ、治される信仰があると分かり、  
<span class="verse-number">10&ensp;</span>大声で言った。「まっすぐ、足をつかって立ち上がりなさい。」すると、彼は飛びはね、歩いた。  
<span class="verse-number">11&ensp;</span>すると、群衆はパウロが行なったことを見、声をはりあげ、ルカオニヤ語で言った。「神々が人間の形で、私たちのところに降りて来られました。」  
<span class="verse-number">12&ensp;</span>そして、彼らはバルナバをゼウスと呼び、パウロをヘルメスと呼んだ。パウロは話し手の長であったからである。  
<span class="verse-number">13&ensp;</span>その時、街の前にある、ゼウスの宮の祭司が、雄牛と花輪を門の所に持って来た。群衆と共に犠牲を捧げるつもりであつた。  
<span class="verse-number">14&ensp;</span>しかし、使徒たち、バルナバとパウロはこれを聞き、自らの服を裂き、群衆の中に駆け込み、叫び、  
<span class="verse-number">15&ensp;</span>言った。「男たちよ、なぜこんな事をするのですか。私たちもあなたがたと同じ性質を持つ男であり、これら役に立たない物から、天と地と海と、その中にあるすべてのものを造った、生ける**神**に立ち返るように、あなたがたに福音を伝えているのです。  
<span class="verse-number">16&ensp;</span>過ぎ去った世代では、すべての国々は自らの道を歩むことを、この**神**は許されました。  
<span class="verse-number">17&ensp;</span>しかし、**神**はご自分を、証されなかったわけではありません。**神**がなされたよいことの内には、天から我々に雨を与え、実りの季節を与え、私たちの心を食べ物と喜びで満たされました。」  
<span class="verse-number">18&ensp;</span>このように言って、群衆が二人に犠牲をささげることを、彼らはかろうじて阻止した。  
<span class="verse-number">19&ensp;</span>それで、アンテオケとイコニオムから、ユダヤ人たちはそこに行った。そして、群衆を説得し、パウロを投石の刑にした。そして、パウロが死んだと思い、彼らはパウロを町から引きずり出した。  
<span class="verse-number">20&ensp;</span>しかし、弟子たちは彼の回りに集まると、彼は立ち上がり、町に入った。そして、翌日バルナバと共にデルベに出発した。  
<span class="verse-number">21&ensp;</span>そして、その町に福音を伝え、多くの弟子を作り、彼らはルステラとイコニオムとアンテオケに戻り、  
<span class="verse-number">22&ensp;</span>弟子たちのたましいを強め、信仰にとどまるように強く勧め、言った。「私たちは多くの苦しみを通過し、**神**の王国に入らなければなりません。」  
<span class="verse-number">23&ensp;</span>それで、二人はすべての教会で長老を任命し、断食をもって祈り、彼らが信じていた**主**に弟子たちを任せた。  
<span class="verse-number">24&ensp;</span>そして、ピシデヤを通り、彼らはパンフリヤに行った。  
<span class="verse-number">25&ensp;</span>さて、ペルガで御言葉を話し、二人はアタリヤに下った。  
<span class="verse-number">26&ensp;</span>そこから、アンテオケへ出航した。そこは彼らが今完遂した**神**の働きのために、**神**の恵みへ渡された所であった。  
<span class="verse-number">27&ensp;</span>二人は到着し、教会を集めて、**神**が二人と共になされたすべてのことを、また**神**が異邦人たちに信仰の扉を開いてくださったことを報告した。  
<span class="verse-number">28&ensp;</span>二人は長い間、そこに弟子たちと滞在した。  

## 第15章

さて、ある男たちが、ユダヤから下って来て、兄弟たちに教えた。「モーセの慣習に従って割礼を受けない限り、あなたがたは救われることはできない。」  
<span class="verse-number">2&ensp;</span>それで、パウロとバルナバは、彼らと大衝突と大論争が起こり、パウロとバルナバ、それに、参加した数人が、エルサレムに上り、この件に関して、使徒たちと長老たちに問うべきであると、彼らは決めた。  
<span class="verse-number">3&ensp;</span>それで、教会によって送り出された彼らは、フェニキヤとサマリヤを通過し、異邦人の改宗を詳しく伝え、全兄弟たちを大いに喜ばせた。  
<span class="verse-number">4&ensp;</span>そして、彼らはエルサレムに着くと、教会、使徒たち、長老たちに歓迎された。そして**神**が彼らと共になされたことを、一部始終報告した。  
<span class="verse-number">5&ensp;</span>しかし、パリサイ派の中で、何人かの信じる者たちが立ち上がり、言った。「彼らを割礼し、モーセの律法を守るよう、命じることが必要です。」  
<span class="verse-number">6&ensp;</span>それで、使徒たちと長老たちはこの件をよく相談しようと、共に集まった。  
<span class="verse-number">7&ensp;</span>多くの論争の後、ペテロは立って彼らに言った。「男たち、兄弟たちよ。かなり以前、あなたがたの知っているように、**神**は私たちの中から私をお選びになりました。それは、私の口を通して、異邦人が福音の言葉を聞き、信じるためでした。  
<span class="verse-number">8&ensp;</span>それで、人の心を知っておられる**神**は、私たちにくださったと同じように、**聖霊**を彼らに与え、異邦人たちに証され、  
<span class="verse-number">9&ensp;</span>**神**は、私たちと彼らの間に区別をもうけず、彼らの心を信仰によって清められました。  
<span class="verse-number">10&ensp;</span>それなのに、私たちも私たちの先祖たちも、負うことができなかったくびきを、弟子たちの首にかけ、あなたがたは今なぜ**神**を試すのですか。  
<span class="verse-number">11&ensp;</span>しかし、彼らと同様に、**主**イエス・キリストの恵みにより、彼らが救われると同様、私たちも救われると、私たちは信じています。」  
<span class="verse-number">12&ensp;</span>すると、全会衆は静まり、異邦人の間で、**神**はどれほど多くの不思議なわざとしるしを、パウロとバルナバを通して行なわれたかを、彼らが伝えるのを聞いた。  
<span class="verse-number">13&ensp;</span>すると、彼らは黙ると、ヤコブは答えて言った。「男たちよ、兄弟たちよ、私の話を聞いてください。  
<span class="verse-number">14&ensp;</span>異邦人の中から、御名のため、民を取り出すため、最初に**神**は異邦人を訪れられたことを、シモンは伝えました。  
<span class="verse-number">15&ensp;</span>預言者たちの言葉はこれと一致します。書いてあるとおりです。  
<span class="verse-number">16&ensp;</span>『この後、わたしは戻り、ダビデの倒された天幕を建て直す。わたしはその廃墟を再建し、それを立ち上げる。  
<span class="verse-number">17&ensp;</span>これらをすべて行なっている**主**は言われる。人類の残りが、そして、わたしの名で呼ばれる全異邦人たちが**主**を求めるようになるためである。』  
<span class="verse-number">18&ensp;</span>**神**のすべてのわざは、永遠から**神**はご存知です。  
<span class="verse-number">19&ensp;</span>従って、異邦人の中から、**神**に向き合っている人たちを、惑わせるべきではない、と私は判断します。  
<span class="verse-number">20&ensp;</span>しかし、偶像によるけがれ、性的な罪、絞め殺されたもの、血は避けよ、と私たちは彼らに書きます。  
<span class="verse-number">21&ensp;</span>なぜなら、多くの世代を通じて、どの町でも、安息日ごとにシナゴーグでモーセは読まれ、説く人たちがいます。」  
<span class="verse-number">22&ensp;</span>それで、使徒たち、長老たちは、全教会員と共に、彼ら自身の仲間の内から選んだ男たちを、パウロとバルナバと共に、アンテオケに遣わすことを、よしとした。選ばれたのは、兄弟たちの指導者たちで、バルサバと呼ばれるユダ、とシラスであった。  
<span class="verse-number">23&ensp;</span>長老たちは彼らの手を使い、この手紙を書いた。「使徒たち、長老たち、そして兄弟たちから、アンテオケとシリアとキリキヤにいる異邦人の兄弟たちへ。  
<span class="verse-number">24&ensp;</span>私たちはこう聞きました。私たちから行ったある人たちは、言葉によってあなたがたを悩まし、あなたがたのたましいを不安にし、言いました。『あなたがたは割礼を受け、律法を守らなければなりません。』しかし、私たちはそんな命令を彼らに与えていません。  
<span class="verse-number">25&ensp;</span>私たちは心を一つにして集まり、選ばれた男たちを、私たちの愛するバルナバとパウロと共に遣わすことをよしとしました。  
<span class="verse-number">26&ensp;</span>彼らは、私たちの**主**イエス・キリストの御名のために、自らの命をかけた人たちです。  
<span class="verse-number">27&ensp;</span>ですから、私たちはユダとシラスを遣わしました。私たちと同じ事を、彼らは口頭で伝えます。  
<span class="verse-number">28&ensp;</span>次の不可避な事がら以上に、大きい重荷をあなたがたに与えないことは、**聖霊**にも私たちにも、よいと思いました。  
<span class="verse-number">29&ensp;</span>すなわち、偶像に捧げられた物から、血から、絞め殺された物から、性的な罪から離れることです。これらのことから自らを守れば、あなたがたは繁栄します。健やかでありますように。」  
<span class="verse-number">30&ensp;</span>それで、彼らは送り出され、アンテオケに着いた。そして、会衆を集め、手紙を渡した。  
<span class="verse-number">31&ensp;</span>全員は手紙を読み、その励ましを喜んだ。  
<span class="verse-number">32&ensp;</span>それで、ユダとシラス自身も預言者であったので、多くの言葉を通して兄弟たちを励まし、そして強めた。  
<span class="verse-number">33&ensp;</span>そして、しばらくそこに滞在し、兄弟たちから使徒たちへの平和の言葉を添え、送り返された。  
<span class="verse-number">34&ensp;</span>しかし、シラスは、そこに残るのをよしとした。  
<span class="verse-number">35&ensp;</span>パウロもバルナバも、アンテオケに残り、大勢の他の人たちと共に、**主**の御言葉を教えたり伝えたりした。  
<span class="verse-number">36&ensp;</span>それから数日後、パウロはバルナバに言った。「私たちが**主**の御言葉を語ったすべての町に戻り、私たちの兄弟たちを尋ね、彼らがいかに過ごしているかを見ましょう。」  
<span class="verse-number">37&ensp;</span>それで、バルナバはマルコと呼ばれるヨハネを連れて行くことを決めた。  
<span class="verse-number">38&ensp;</span>しかし、パンフリヤで彼らから離れ、働きに行かなかったマルコを連れて行くべきでないと、パウロは思っていた。  
<span class="verse-number">39&ensp;</span>すると、二人の間の論争はあまりにも激しくなったので、互いに分かれた。それで、バルナバはマルコを連れて、キプロスへ出航した。  
<span class="verse-number">40&ensp;</span>しかし、パウロはシラスを選び、兄弟たちから**神**の恵みへ任され、  
<span class="verse-number">41&ensp;</span>そして、彼は諸教会を強固にしながら、シリア、キリキヤを通過した。  

## 第16章

それから彼、パウロはデルベ、そしてルステラに行った。すると、見よ、そこにテモテという一人の弟子がいた。信じている一人のユダヤ婦人の息子で、父はギリシャ人であった。  
<span class="verse-number">2&ensp;</span>彼はルステラとイコニオムにいた兄弟たちに、評判がよかった。  
<span class="verse-number">3&ensp;</span>パウロはテモテに同行して欲しかった。それで、その地方のユダヤ人たちのため、テモテを連れて行き割礼をした。彼の父はギリシャ人であることを、彼らは皆、知っていたからである。  
<span class="verse-number">4&ensp;</span>それで、町々を通って行くと、エルサレムの使徒たちと長老たちで決められた規定を、彼らはみんなに伝えていた。  
<span class="verse-number">5&ensp;</span>それで、諸教会は信仰を強められ、毎日に人数が増えた。  
<span class="verse-number">6&ensp;</span>さて、フルギヤやガラテヤの地方を通過した時、彼らはアジアで御言葉を語るのを**聖霊**によって禁じられた。  
<span class="verse-number">7&ensp;</span>ムシヤの方へ行ってから、ビテニヤに入ろうとしたが、**御霊**は許されなかった。  
<span class="verse-number">8&ensp;</span>それで、ムシヤを迂回し、トロアスに下って行った。  
<span class="verse-number">9&ensp;</span>すると、夜通し幻がパウロに現れた。一人のマケドニヤ人が立っていて、パウロに願って、言った。「マケドニヤまで来て、私たちを助けてください。」  
<span class="verse-number">10&ensp;</span>パウロはこの幻を見て、マケドニヤの人たちに福音を伝えるように、**主**が私たちを召されたのであると分かり、私たちはただちに、マケドニヤに行こうとした。  
<span class="verse-number">11&ensp;</span>それで、トロアスを出航し、我々はサモトラケへ直航し、翌日、ネアポリス港に着いた。  
<span class="verse-number">12&ensp;</span>そこからマケドニア地方の第一の町で、植民地のポリスであるピリピに行き、そして、数日間その町に滞在した。  
<span class="verse-number">13&ensp;</span>安息日に、私たちは町を出て、普段、祈りが行なわれている川辺に行った。そこで、集まっていた女たちと話をしていた。  
<span class="verse-number">14&ensp;</span>さて、テアテラ町の出身で、**神**を礼拝し、紫色の布を売る、ルデヤという一人の女が私たちの話を聞いていた。パウロによって話されたことに注意せよと、**主**は彼女の心を開かれた。  
<span class="verse-number">15&ensp;</span>そして、ルデヤとその家の者全員が浸礼を受け、彼女は私たちに願って言った。「私が**主**に忠実であると判断したなら、私の家に来て滞在してください。」と彼女は懇願した。  
<span class="verse-number">16&ensp;</span>さて、ことの次第はこうであった。私たちが祈りに行くと、占いの霊を持ち、占いで彼女の主人たちに利益をもたらしていた、奴隷の少女がいた。彼女は私たちと出会った。  
<span class="verse-number">17&ensp;</span>この少女はパウロと私たちについて来て、叫び、言った。「救いの道を私たちに説いているこの人たちは、いと高き**神**の僕たちです。」  
<span class="verse-number">18&ensp;</span>少女は幾日もこれを行なった。しかしパウロはとても困惑し、向きを変え、少女に言った。「イエス・キリストの御名によって、私はお前に命じる。少女から出よ。」すると、霊はその時出た。  
<span class="verse-number">19&ensp;</span>しかし、彼女の主人たちは利益の望みがなくなったとわかった時、パウロとシラスを捕まえ、市場にいた支配者たちの所に引っ張って行った。  
<span class="verse-number">20&ensp;</span>そして、行政官たちの所に連れて行き、言った。「この者たちはユダヤ人で、私たちの町を大混乱させています。  
<span class="verse-number">21&ensp;</span>そして、ローマ人である私たちが、受け入れても守っても不法になる習慣を教えています。」  
<span class="verse-number">22&ensp;</span>それで、群衆は彼らに反対して、立ち上がり、行政官たちは彼らの服をむしり取り、杖で打たれよ、と命じた。  
<span class="verse-number">23&ensp;</span>彼らに多くのムチを当て、牢獄に投げ入れ、看守に彼らを厳重に見張るよう命じた。  
<span class="verse-number">24&ensp;</span>こう命令を受け、看守は彼らを奥の牢に入れ、足かせに足を固定した。  
<span class="verse-number">25&ensp;</span>しかし、パウロとシラスは真夜中、**神**に祈り、そして賛美の歌を歌い、囚人たちは聞いていた。  
<span class="verse-number">26&ensp;</span>すると突然、大地震があり、牢獄の土台が揺れた。すると、そのとたん、すべての扉が開き、全員の鎖は外れた。  
<span class="verse-number">27&ensp;</span>そして、看守は眠りから覚め、牢獄の扉が開いているのを見、囚人たちはすでに逃げたと思い、剣を抜き、正に自らを殺そうとするところで、  
<span class="verse-number">28&ensp;</span>パウロは大声で叫び言った。「自らを傷つけるな。我々全員はここにいる。」  
<span class="verse-number">29&ensp;</span>それで、看守は灯りを求め、駆け込み、震えながらパウロとシラスの前にひれ伏し、  
<span class="verse-number">30&ensp;</span>そして、二人を外に連れ出し言った。「先生たち、救われるために、私は何をすべきですか。」  
<span class="verse-number">31&ensp;</span>そこで、彼らは言った。「**主**イエス・キリストを信じなさい。そうすれば、あなたも、あなたの家の者全員をも救われます。」  
<span class="verse-number">32&ensp;</span>それから、看守と彼の家にいた全員に、二人は**主**の御言葉を話した。  
<span class="verse-number">33&ensp;</span>そして、夜のその同じ時間、看守は二人を連れて行き、彼らのムチの傷を洗った。そしてすぐに彼と、彼に属するすべての人たちは、浸礼された。  
<span class="verse-number">34&ensp;</span>そして、二人を自分の家の中に連れて行き、彼らの前に食べ物を置いた。そして、**神**を信じたことを家の者と全員で、看守は喜んだ。  
<span class="verse-number">35&ensp;</span>明るくなり、行政官たちは代理の役人を遣わし、言った、「その男たちを放免せよ。」  
<span class="verse-number">36&ensp;</span>それで、看守はこの言葉をパウロに報告して、言った。「行政官たちはあなたがたを釈放するため、代理人たちを遣わしました。ですから、すぐ出発してください。安らかに行ってください。」  
<span class="verse-number">37&ensp;</span>しかし、パウロは彼らに言った。「無実のローマ人を公衆の面前でムチで打たせ、牢獄に入れ、そして今、秘密裏に追い出すのか。それは違う。彼ら自らが来て、私たちを連れ出せ。」  
<span class="verse-number">38&ensp;</span>代理の役人たちはこの言葉を行政官たちに伝えた。するとパウロたちはローマ人であると聞き、行政官たちは恐怖を感じ、  
<span class="verse-number">39&ensp;</span>そして、やって来て、二人に願い、連れ出し、町から出てくれるように頼んだ。  
<span class="verse-number">40&ensp;</span>それで二人は牢獄を出、ルデヤの家に入った。そして、兄弟たちに会い、彼らを励まし、そして出発した。  

## 第17章

さて、使徒たちはアムピポリスとアポロニヤを通り、テサロニケに着いた。そこに、ユダヤ人の一軒のシナゴーグがあった。  
<span class="verse-number">2&ensp;</span>それで、パウロは彼の習慣のとおり、ユダヤ人たちの中に入り、三安息日にわたり聖書に基づき彼らと論じ合い、  
<span class="verse-number">3&ensp;</span>キリストは苦しみ、死人の中から復活しなければならかったと解釈し、論証し、言った。「私があなたがたに説くこのイエスこそが、あのキリストなのです。」  
<span class="verse-number">4&ensp;</span>そして、その中のある人たちは信頼するにいたり、大勢の敬虔なギリシャ人はパウロとシラスの仲間になった。また、指導的立場にある女性も少なくなかった。  
<span class="verse-number">5&ensp;</span>しかし、納得にいたらなかったユダヤ人たちはねたみ、市場の無頼漢たちを連れ、群衆を集め、町中を混乱させ、二人を群衆の所へ連れ出そうとヤソンの家を襲ったが、  
<span class="verse-number">6&ensp;</span>二人を見つけ出せず、ヤソンと数人の兄弟たちを町の支配者たちの所へ、引きずって行き、大声で叫んだ。「世界を転覆したこの者たちはここにも来ている。  
<span class="verse-number">7&ensp;</span>ヤソンはその者たちを受け入れ、そして全員、カイザルの布告に反するふるまいをし、イエスと言う別の王がいると言っている。」  
<span class="verse-number">8&ensp;</span>それで、ユダヤ人たちは、これを聞いた群衆と町の支配者たちを動揺させた。  
<span class="verse-number">9&ensp;</span>それで、ヤソンとその他の人たちから保証を取り、支配者たちは彼らを放免した。  
<span class="verse-number">10&ensp;</span>それから、兄弟たちはただちに、夜、パウロとシラスをベレヤへ送り出した。到着した二人はユダヤ人たちのシナゴーグに入った。  
<span class="verse-number">11&ensp;</span>そこの人たちは、テサロニケの人たちより高貴であり、心をすべて整え、御言葉を受け入れ、それらの事はその通りなのか、毎日熱心に聖書を調べた。  
<span class="verse-number">12&ensp;</span>それで、彼らの多くの者は信じた。その中にギリシャ人、評判のよい女も男も少なからずいた。  
<span class="verse-number">13&ensp;</span>しかし、**神**の御言葉はパウロによってベレヤでも教えられていたと、テサロニケのユダヤ人たちは知り、そこにも行き、群衆を扇動した。  
<span class="verse-number">14&ensp;</span>それで、兄弟たちは、直ちにパウロを送り出し、海に行かせたが、シラスとテモテはベレヤに残った。  
<span class="verse-number">15&ensp;</span>それで、パウロを案内した人たちは彼をアテネに連れて行き、パウロの所に、シラスとテモテは大至急来るよう命を受け、彼らは去った。  
<span class="verse-number">16&ensp;</span>さて、パウロが二人をアテネで待つ間、アテネの町は偶像で満たされていたので、彼の霊はかき乱された。  
<span class="verse-number">17&ensp;</span>そこで、彼はシナゴーグで、ユダヤ人たちと、そして礼拝する異邦人たちと論じ合い、また毎日市場で、たまたまそこにいた人たちと論じ合っていた。  
<span class="verse-number">18&ensp;</span>そこで、エピクロス派やストア派のある哲学者たちはパウロに出会った。そして、中には、「この知ったかぶり屋は何を言いたいのか。」と言い、また、パウロはイエスと復活の福音を彼らに伝えていたから、「外国の神々を宣言しているのだ。」と言う者もいた。  
<span class="verse-number">19&ensp;</span>そして、彼らはパウロをアレオパゴスに連れて行き、言った。「あなたの話している新しい教理は、何なのか教えていただけますか。  
<span class="verse-number">20&ensp;</span>なぜなら、奇妙に響くことを、あなたは我々に聞かせています。だから、これらのことは、どういう意味なのかを知りたいです。」  
<span class="verse-number">21&ensp;</span>なぜなら、すべてのアテネ人とそこに滞在していた外国人は、新しいことを言ったり聞いたりするためだけに、時間を過ごしていた。  
<span class="verse-number">22&ensp;</span>それでパウロはアレオパゴスの中央に立って言った。「アテネの男たちよ、すべてのことにおいて、あなたがたは宗教心があると認めます。  
<span class="verse-number">23&ensp;</span>なぜなら、あなたがたの崇拝する物をよく観察しながら、街を通ると、『未知の**神**へ』、と刻んである碑文のある祭壇を見つけました。ですから、あなたがたが知らないで拝んでいる**神**、この方こそを私はあなたがたに伝えます。  
<span class="verse-number">24&ensp;</span>世とその中にあるすべてのものをお造りになった**神**は、天と地の**主**ですから、手で作られた神殿にはお住みになりません。  
<span class="verse-number">25&ensp;</span>また、この方はすべての者に、命と息と万物とを与えてくださるのだから、必要なものが、何かあるかのように、人の手による奉仕は不要です。  
<span class="verse-number">26&ensp;</span>そして、この**神**は一つの血から、人間たちのすべての国を造られ、全地上に住まわせ、また、人間の時を定められ、その居住境界を定められました。  
<span class="verse-number">27&ensp;</span>私たち一人一人から**主**は確かに遠くないです。彼らが**主**を求め、**主**を見いだすようにとの望みのためです。  
<span class="verse-number">28&ensp;</span>なぜなら、私たちは**主**にあって生き、動き、また存在します。あなたがたの仲間のある詩人たちも言ったように、『ゆえに、我らは**神**の子孫である。』と。  
<span class="verse-number">29&ensp;</span>従って、私たちは**神**の子孫ですから、**神**、すなわち絶対者は、何か人間の技だとか、考えで形作られた金や銀や石などに、似ているなどと考えてはいけません。  
<span class="verse-number">30&ensp;</span>まことに、この無知の時代を、**神**は大目に見てこられましたが、今、**神**はどこにいようと、すべての人たちに悔い改めるように命じておられます。  
<span class="verse-number">31&ensp;</span>それで、**神**はご自身で任命された男の方により、ある日を、世界を正しく裁く日と定められました。**神**はその男の方を死人たちの中から復活させられたことにより、すべての人にこの確証を与えられました。」  
<span class="verse-number">32&ensp;</span>それで、彼らは死者の復活を聞くと、ある者たちはあざ笑ったが、「このことについて、あなたにまたお聞きします。」と言う者もいた。  
<span class="verse-number">33&ensp;</span>それで、こうしてパウロは彼らの中から出て行った。  
<span class="verse-number">34&ensp;</span>しかし、その中のある男たちはパウロの仲間になり、信じた。その中に、アレオパゴス裁判官のデオヌシオが、またダマリスという名の女、また他にも彼らと共にいた。  

## 第18章

さて、その後、パウロはアテネを出、コリントに行き、  
<span class="verse-number">2&ensp;</span>そして、アクラという名前の一人のユダヤ人を見いだした。ポント生まれで、妻プリスキラと共に、最近イタリアから来ていた。なぜなら、クラウデオは全ユダヤ人に、ローマを出るように命じたからである。そしてパウロは彼らの所に行った。  
<span class="verse-number">3&ensp;</span>それで、彼は同じ職業であったから、パウロは彼らと共に滞在し、働いた。彼らの職業はテント作りであったからである。  
<span class="verse-number">4&ensp;</span>そこで、パウロは安息日毎にシナゴーグで論じ、ユダヤ人、  
<span class="verse-number">5&ensp;</span>そこで、シラスとテモテはマケドニアから下って来た時、パウロは霊に強く動かされ、イエスがあのキリストであると、ユダヤ人たちに証言していた。  
<span class="verse-number">6&ensp;</span>しかし、彼らはパウロに反対し、冒涜したので、パウロは着衣を振り払い、彼らに言った。「あなたがたの血は、あなたがたの頭上にある。私は清い。今後、私は異邦人の所に行く。」  
<span class="verse-number">7&ensp;</span>そして、彼はそこから離れ、ユストという、ある人の家に入った。彼は**神**を礼拝し、家はシナゴーグに隣接していた。  
<span class="verse-number">8&ensp;</span>そして、シナゴーグの支配者であったクリスポは、家の者全員と共に**主**を信じた。そして、コリント人の多くは聞き、信じ、そして浸礼された。  
<span class="verse-number">9&ensp;</span>さて、夜中に**主**は幻でパウロに言われた。「怖がらずに話しなさい。黙っていてはいけない。  
<span class="verse-number">10&ensp;</span>わたしはあなたと共にいるからである。この町にわたしの民が大勢いるから、あなたを襲い、けがをさす者は誰もいない。」  
<span class="verse-number">11&ensp;</span>それで、パウロはそこに一年六ヶ月滞在し、彼らの間に**神**の御言葉を教えていた。  
<span class="verse-number">12&ensp;</span>さて、ガリオがアカヤ地方の総督であった時、ユダヤ人たちは団結し、パウロに対して立ち上がり、裁判の席に連れて行き、  
<span class="verse-number">13&ensp;</span>言った。「この者は、人々に律法に反する方法で**神**を礼拝するように勧めています。」  
<span class="verse-number">14&ensp;</span>そして、パウロが口を開けようとした時、ガリオはユダヤ人たちに言った。「ユダヤ人たちよ、もし不正か邪悪な犯罪であれば、お前たちに我慢する理由にもなるだろうが、  
<span class="verse-number">15&ensp;</span>しかし、問題が言葉、名前、またお前たちの律法のことなら、お前たち自身でかたずけよ。なぜなら、私はそんなことの裁判官になりたくない。」  
<span class="verse-number">16&ensp;</span>そして彼はユダヤ人たちを裁判の席から追い出した。  
<span class="verse-number">17&ensp;</span>すると、ギリシャ人全員は、シナゴーグの支配者ソステネを捕まえ、裁判の席の前で彼を殴り続けた。しかし、ガリオはこれらのことを意に留めなかった。  
<span class="verse-number">18&ensp;</span>さて、パウロはなお多くの日々そこに滞在した。それから、兄弟たちに別れを告げ、シリアへ出航した。プリスキラとアクラはパウロの供をした。パウロは誓いを立てていたから、ケンクレヤで髪の毛を切った。  
<span class="verse-number">19&ensp;</span>そしてエペソに来て、二人をそこに残し、彼自身はそこのシナゴーグに入り、ユダヤ人たちと論じ合った。  
<span class="verse-number">20&ensp;</span>彼らはもっと長く共に滞在するようにと頼んだが、パウロは同意しなかった。  
<span class="verse-number">21&ensp;</span>しかし、別れを告げ、パウロは言った。「なんとしても、エルサレムの今度の晩餐は、私は守らなければなりません。しかし、**神**のご意志なら、私はまたあなたがたの所に戻ってきます。」そして、彼はエペソから出航した。  
<span class="verse-number">22&ensp;</span>そして、カイザリヤに上陸し、教会に挨拶に上がってから、彼はアンテオケに下って行った。  
<span class="verse-number">23&ensp;</span>そして、しばらくそこに滞在してから、パウロは出発し、すべての弟子たちを強めながら、ガラテヤとフルギヤの地方を順に回った。  
<span class="verse-number">24&ensp;</span>さて、アレキサンドリ生まれで、弁が立ち、聖書に精通している、アポロという名前のユダヤ人の男がエペソに来た。  
<span class="verse-number">25&ensp;</span>この男は**主**の道について、教えを受けたことがあり、霊にあって熱心で、ヨハネの浸礼だけ知っていたのに、**主**のことを正確に話し、教えた。  
<span class="verse-number">26&ensp;</span>それで、彼は大胆にシナゴーグで話し始めた。アクラとプリスキラは聞き、アポロをそばに招き、**神**の道をさらに正確に説明した。  
<span class="verse-number">27&ensp;</span>そして、アポロはアカヤへ渡ろうと決めた時、兄弟たちは弟子たちに手紙を書き、彼を受け入れてくれるように強く勧めた。そして、アポロは到着し、恵みを通して信じた人たちを大いに助けた。  
<span class="verse-number">28&ensp;</span>なぜなら、彼はみんなの前でユダヤ人たちを力強く論破し、イエスはあのキリストであられると、聖書を通して示した。  

## 第19章

それで、ことの次第はこうであった。アポロはコリントにいた間、パウロは北の地方を通り、エペソに着いた。そして、何人かの弟子を見いだし、  
<span class="verse-number">2&ensp;</span>彼らに言った。「信じた時、**聖霊**を受け入れましたか。」すると、彼らはパウロに言った。「いや、**聖霊**なるものさえ、私たちは聞いたこがありませんでした。」  
<span class="verse-number">3&ensp;</span>彼は彼らに言った。「では、何の根拠で浸礼を授けられましたか。」彼らは言った。「ヨハネの浸礼が根拠です。」  
<span class="verse-number">4&ensp;</span>するとパウロは言った。「ヨハネは本当に悔い改めの浸礼で浸礼を授け、人々はヨハネの後に来る方、つまりキリスト・イエスを信じるべき、と民に言いました。」  
<span class="verse-number">5&ensp;</span>彼らはこれを聞いた時、**主**イエスの御名を根拠で浸礼を授けられた。  
<span class="verse-number">6&ensp;</span>そして、パウロは彼らの上に手を置いた時、**聖霊**は彼らの上に下り、彼らは言語で話し、預言をした。  
<span class="verse-number">7&ensp;</span>男たちは全員でおよそ十二人であった。  
<span class="verse-number">8&ensp;</span>それで、パウロはシナゴーグに入り、三ヶ月間、臆することなく話し、**神**の王国のことについて論じ納得させた。  
<span class="verse-number">9&ensp;</span>しかし、ある者たちはかたくなに信ぜず、群衆の前で彼の語るこの道の悪口を言ったので、パウロは彼から去り、弟子たちを退かせ、毎日ツラノの学校[^13]で論じ合っていた。  
<span class="verse-number">10&ensp;</span>そして、これは二年間続き、ユダヤ人もギリシャ人も、アジアに住む者、全員が、**主**イエスの御言葉を聞いた。  
<span class="verse-number">11&ensp;</span>それで、**神**はパウロの手により、いつもとは違う奇跡を行っていた。  
<span class="verse-number">12&ensp;</span>それは、人々が手拭き、または前掛けを、パウロの体から病人のところへ持って行きさえすれば、病気は彼らを去り、悪霊は彼らから出て行った。  
<span class="verse-number">13&ensp;</span>それで、巡回する悪魔払い師の中のある者たちも、悪霊がついている者たちに対して、**主**イエスの御名を呼ぶように決めて、言った。「パウロが説いているイエスによってお前に厳命する。」  
<span class="verse-number">14&ensp;</span>このことをしている者の中に、ユダヤ人の大祭司、スケワの七人の息子たちがいた。  
<span class="verse-number">15&ensp;</span>しかし、悪霊は答えて言った。「俺はイエスを知っている。パウロを知っている。お前たちは誰か。」  
<span class="verse-number">16&ensp;</span>すると、悪霊が入っている男は、飛びかかり、打ち倒し、徹底的に七人をやっつけたので、彼らは裸にされ、傷を負い、その家から逃げ出した。  
<span class="verse-number">17&ensp;</span>この事件は、エペソに住んでいたすべてのユダヤ人にも、ギリシャ人にも知れ渡ったので、全員の上に恐れが降り、**主**イエスの御名は大いにあがめられた。  
<span class="verse-number">18&ensp;</span>そして、信じた多くの人たちは来て、行為を言い表し、表明した。  
<span class="verse-number">19&ensp;</span>また、魔術を行なっていた多くの者たちは、この種の本を持ってきて、みんなの前で焼き尽くした。本の価値を計算すると、合計は銀貨五万枚になった。  
<span class="verse-number">20&ensp;</span>従って、**主**の御言葉は力強く成長し、強くなった。  
<span class="verse-number">21&ensp;</span>これらのことが成就され、パウロはマケドニヤとアカヤを通って、エルサレムに行くことを**御霊**により決定し、言った。「エルサレムに行ってから、ローマも見なければなりません。」  
<span class="verse-number">22&ensp;</span>それで、彼に仕えている二人、テモテとエラストをマケドニヤに遣し、パウロ自身はしばらくアジアに滞在した。  
<span class="verse-number">23&ensp;</span>さて、その頃、この道について大騒動が起こった。  
<span class="verse-number">24&ensp;</span>銀細工師で、デメトリオと言うある男が、アルテミスの宮々[^14]を銀で作り、職人たちに、少なからずの利益を得させていたからである。  
<span class="verse-number">25&ensp;</span>この男は彼らと同種の職業の者たちを呼び集めて、言った。「男たちよ、この仕事から私たちの富はきているのを、あなたがたは知っています。  
<span class="verse-number">26&ensp;</span>そして、エペソでだけではなく、ほとんどアジア中に、手で作った神々は神ではない、とこのパウロは多くの人々を納得させ、遠ざけたことを、あなたがたは見聞きしています。  
<span class="verse-number">27&ensp;</span>ですから、私たちのこの商売は成り立たなくなる危険性があるだけではなく、全アジア、全世界もが拝む、大いなる女神アルテミス[^15]の宮は無くなったと思われ、彼女の偉大さは地に落とされそうです。」  
<span class="verse-number">28&ensp;</span>さて、これを聞いた職人たちは、怒りで満たされ、叫び、言った。「偉大なるエペソ人のアルテミスよ！」  
<span class="verse-number">29&ensp;</span>それで町全員は混乱で満たされ、パウロの同行者であったマケドニヤ人のガイオとアリスタルコを捕え、彼らはいっせいに劇場の中に突入した。  
<span class="verse-number">30&ensp;</span>そして、パウロがその町の人々の中に入ろうとした時、弟子たちは彼を許そうとはしなかった。  
<span class="verse-number">31&ensp;</span>それで、パウロの友人であった、アジア州の議員たちのある人々は、劇場に入らないようにと言う願いを、人を介してパウロに送った。  
<span class="verse-number">32&ensp;</span>そこで、集会は混乱したので、大方の者はなぜ集まったか分からず、あることを叫ぶ者、また別なことを叫ぶ者も中にいた。  
<span class="verse-number">33&ensp;</span>それで、彼らはアレキサンデルを群衆の中から引き出し、ユダヤ人たちは彼を前方に押し出した。そして、アレキサンデルは片手を振り、集まった人たちに自らを弁護したいと願った。  
<span class="verse-number">34&ensp;</span>しかし、彼がユダヤ人であると群衆が分かると、全員声を一つにし、二時間ほど叫んだ。「エペソの偉大なるアルテミスよ！」  
<span class="verse-number">35&ensp;</span>そして、町の書記は群衆を静め、言った。「エペソの男たちよ。エペソは、偉大なる女神アルテミスの宮と、またゼウスから降臨した像を守護している町であることを、知らない人はいるでしょうか。  
<span class="verse-number">36&ensp;</span>ですから、これらのことは否定できないのですから、沈黙しなさい。急いですることは何もないのです。  
<span class="verse-number">37&ensp;</span>なぜなら、あなたがたがここに連れて来た男たちは、宮の泥棒でもないし、あなたがたの女神を冒涜する者でもありません。  
<span class="verse-number">38&ensp;</span>ですから、デメトリオと彼の仲間の職人たちが誰かに対して訴えがあれば、法廷は開いているし、地方総督たちもいます。互いに訴えさせましょう。  
<span class="verse-number">39&ensp;</span>しかし、あなたがたが他のことで要求があるのなら、それは法律会議で決めましょう。  
<span class="verse-number">40&ensp;</span>私たちは今日の騒動について、問いただされる危険があります。なぜなら、この秩序のない集合の理由を説明できないからです。」  
<span class="verse-number">41&ensp;</span>これらのことを言って、彼は群衆を解散させた。  

## 第20章

騒動が治まった後、パウロは弟子たちを彼のそばに呼び、挨拶し、マケドニヤへ行くべく出発した。  
<span class="verse-number">2&ensp;</span>それで、多くの言葉で彼らを励ましながら、その地方を通り、ギリシャに入った。  
<span class="verse-number">3&ensp;</span>そこに三ヶ月滞在した。そして、彼がシリアへ出航しようとする時、ユダヤ人たちは彼に対して妨害を企んだので、マケドニヤを通って戻ろうと決めた。  
<span class="verse-number">4&ensp;</span>ベレヤ人ソパテロ、テサロニケ人アリスタルコとセクンド、そして、デルベ人ガイオとテモテ、アジア人テキコとトロピモがパウロに同行していた。  
<span class="verse-number">5&ensp;</span>この人々は先行し、トロアスで私たちを待っていた。  
<span class="verse-number">6&ensp;</span>しかし、種なしパンの日々の後、私たちはピリピから出航し、トロアスで五日後に合流し、彼らと七日間そこに滞在した。  
<span class="verse-number">7&ensp;</span>さて、その週の最初の日に、弟子たちはパンを裂くために集まっており、翌日出発する予定であったパウロは彼らに語り、真夜中まで話を続けた。  
<span class="verse-number">8&ensp;</span>多くの灯火が、彼らの集まっていた上の部屋にあった。  
<span class="verse-number">9&ensp;</span>パウロが長時間、論じていたので、ユテコという名の青年が窓に腰かけ、眠気に負け、深い眠りに沈み、三階から落ち、抱き起こすと死んでいた。  
<span class="verse-number">10&ensp;</span>しかし、パウロは下に降り、青年の上に身を伏せ、彼を抱擁し、言った。「諸君、騒いではいけない。彼の命はまだ彼の中にあります。」  
<span class="verse-number">11&ensp;</span>そして、パウロは上に行き、パンを裂き、食べ、夜明けまで長い間話をしてから、出発した。  
<span class="verse-number">12&ensp;</span>そして、彼らは生きている青年を連れて上がり、あり得ない慰めを得た。  
<span class="verse-number">13&ensp;</span>それから、私たちは先に船に乗り、アソスに出航した。そこでパウロを船に乗せるためであった。自らそう命じていたのは、パウロは徒歩で行くつもりである。  
<span class="verse-number">14&ensp;</span>パウロはアソスで私たちと会い、私たちは彼を乗船させ、ミテレネに行った。  
<span class="verse-number">15&ensp;</span>私たちはそこから出航し、翌日、キヨスの対岸に行った。次の日、サモスに到着し、トログニオムに滞在した。その翌日、ミレトに行った。  
<span class="verse-number">16&ensp;</span>なぜなら、アジアで時間を過ごさないよう、パウロはエペソに寄らないと、決めていたからである。パウロはできれば、五旬節の日にエルサレムにいたかったからである。  
<span class="verse-number">17&ensp;</span>パウロは、ミレトからエペソに人を送り、教会の長老たちを呼び集めた。  
<span class="verse-number">18&ensp;</span>長老たちがパウロの所に来ると、彼は長老たちに言った。「私がアジアに来た最初の日から、あなたがたの間で、私がいつもどういう生活をしていたか、あなたがたは知っています。  
<span class="verse-number">19&ensp;</span>ユダヤ人たちの企みによって私に起こった、試練を伴うことで、多くの涙と、謙虚を尽くし、私は**主**に仕えました。  
<span class="verse-number">20&ensp;</span>また、役に立つことは何も後ろに留めず、あなたがたに告げ、公でも、各家でも教え、  
<span class="verse-number">21&ensp;</span>ユダヤ人にもギリシャ人にも、**神**に対する悔い改めと、私たちの**主**に対する信仰を証しました。  
<span class="verse-number">22&ensp;</span>それで、見よ。私は今、霊に縛られ、エルサレムに向かっています。そこで私に何が起こるかわからないのですが、  
<span class="verse-number">23&ensp;</span>縛る物と苦しみはすべての町で私を待ち受けていると、**聖霊**は証して言っておられます。  
<span class="verse-number">24&ensp;</span>しかし、これらの中のもので、私を動かすものは何もありません。また、私は喜びを持って私の行程を終えることができるよう、また**主**イエスから頂いた責務、すなわち**神**の恵みの福音を完全に証することを終えるため、私の命は私自身にとって貴重なものとは思っていません。  
<span class="verse-number">25&ensp;</span>見よ、あなたがたの間で**神**の王国を説いた私の顔を、もうあなたがたは見ることはないと、今、私はわかっています。  
<span class="verse-number">26&ensp;</span>従って、私はすべての人の血に関して、無実であることを今日あなたがたに証をします。  
<span class="verse-number">27&ensp;</span>私はあなたがたに、**神**のすべてのご意志を伝えることから逃げなかったからです。  
<span class="verse-number">28&ensp;</span>だから、あなたがた自身に、そして群れの者全員に配慮しなさい。**神**がご自分の血によって買い入れられた教会を牧会するために、**聖霊**はあなたがたを監督者にされたのです。  
<span class="verse-number">29&ensp;</span>私は知っているのです。私の出立後、残酷な狼があなたがたの間に入り、群れに情け容赦しません。  
<span class="verse-number">30&ensp;</span>また、あなたがたの中からも男らが現れ、彼らの後ろに弟子たちを引いて行こうと、歪曲したことを言います。  
<span class="verse-number">31&ensp;</span>従って、用心しなさい。私が三年間夜も昼も休まずに、涙で、すべての人に警告したことを心に留めておきなさい。  
<span class="verse-number">32&ensp;</span>それで兄弟たちよ、私は今**神**と**神**の恵みの御言葉に、あなたがたを委ねます。その御言葉はあなたがたを打立て、聖別されたすべての人々の間で、あなたがたに相続権を与えることができます。  
<span class="verse-number">33&ensp;</span>私は他人の金や銀や衣服を、不当に望んだことはありません。  
<span class="verse-number">34&ensp;</span>かえって、私の必要とするもの、また、私と共にいた人たちの必要とするものを、この両手が満たしたことを、あなたがた自身が知っています。  
<span class="verse-number">35&ensp;</span>こう働くことにより、弱い人を助けなければならないと、私はあなたがたに、あらゆる方法で示しました。また、『受け取るより、与える方がもっと祝福されている。』と言われた、**主**イエスの御言葉を心に留めなさい。」  
<span class="verse-number">36&ensp;</span>パウロはこれらのことを言い終え、ひざまずき、全員と共に祈った。  
<span class="verse-number">37&ensp;</span>彼ら全員手放しで泣き、パウロの首にしがみ付き、口づけした。  
<span class="verse-number">38&ensp;</span>彼らが最も悲しんだのは、彼らは以後、パウロの顔は見ることはない、と言ったパウロの言葉であった。彼らは、船までパウロを見送りに行った。  

## 第21章

さて、ことの次第はこうであった。私たちは彼らから別れ、出航し、直線コースをとり、コスに着いた。翌日はロドスに、そこからパタラに着いた。  
<span class="verse-number">2&ensp;</span>そして、フェニキヤにまで行く船を見つけて、私たちは乗船し、出航した。  
<span class="verse-number">3&ensp;</span>キプロスを目にし、その左側を通過し、シリアへと向かい、ツロで船は荷を降ろすことになっていたので、そこで着岸した。  
<span class="verse-number">4&ensp;</span>弟子たちを探し出し、彼らと共にそこに七日間滞在した。エルサレムに上らないようにと、彼らは**御霊**を通して、パウロに言った。  
<span class="verse-number">5&ensp;</span>その滞在の日々が終わり、私たちは出て、道を進むと、彼らは妻たちと子どもたちと共に、私たちが町を出るまで、我らに同行した。そして私たちは砂浜でひざまずき、祈った。  
<span class="verse-number">6&ensp;</span>互いに挨拶を交わし、私たちは乗船し、彼らは家へ帰った。  
<span class="verse-number">7&ensp;</span>ツロからの行程を終え、私たちはトレマイに着き、兄弟たちに挨拶をして、一日彼らと滞在した。  
<span class="verse-number">8&ensp;</span>その翌日、パウロの仲間の私たちは出発し、カイザリヤに到着し、七人の中の一人であった伝道師、ピリポの家に入り、彼の所に滞在した。  
<span class="verse-number">9&ensp;</span>この人には、預言する処女の娘が四人いた。  
<span class="verse-number">10&ensp;</span>私たちは幾日か滞在していると、アガボという預言者がユダヤから下って来た。  
<span class="verse-number">11&ensp;</span>この男は私たちのところに来て、パウロの帯を取り、彼の手足を縛り、言った。「**聖霊**は言われます。『こうして、エルサレムのユダヤ人たちはこの帯の持ち主を縛り、異邦人の手に渡す。』」  
<span class="verse-number">12&ensp;</span>これらのことを聞くと、私たちも、そこの住民も、エルサレムに上らないよう、パウロに懇願した。  
<span class="verse-number">13&ensp;</span>するとパウロは答えた。「泣きながらあなたがたは私の心を砕くとは、どういうことですか。なぜなら、縛られるだけでなく、**主**イエスの御名のため、私はエルサレムで死ぬ覚悟はできているからです。」  
<span class="verse-number">14&ensp;</span>パウロは説得される様子がないので、私たちは中止し、言った。「**主**のご意志が行なわれますように。」  
<span class="verse-number">15&ensp;</span>さて、この数日の後、私たちは荷を整え、エルサレムへ上って行った。  
<span class="verse-number">16&ensp;</span>カイザリヤの弟子たち幾人かは、私たちと共に行き、私たちが泊まろうとする、最初から弟子の一人、キプロスのマナソンを連れて来た。  
<span class="verse-number">17&ensp;</span>私たちがエルサレムに着くと、兄弟たちは喜んで私たちを受け入れてくれた。  
<span class="verse-number">18&ensp;</span>翌日、パウロは私たちと共にヤコブのところに入ると、長老たちは全員すでにいた。  
<span class="verse-number">19&ensp;</span>パウロは彼らに挨拶し、異邦人の間で**神**がパウロの奉仕を通してなさったことを詳しくパウロは伝えた。  
<span class="verse-number">20&ensp;</span>彼らはこれを聞いて**主**に栄光を捧げた。そして、彼らはパウロに言った。「兄弟よ、幾万人ものユダヤ人が信じていることをご覧なさい。そして彼らは全員、律法に熱心です。  
<span class="verse-number">21&ensp;</span>異邦人の間にいるユダヤ人に、子どもたちに割礼はすべきでない、そして、慣習によって歩んではいけない、モーセを捨てよと、あなたは教えていると、ユダヤ人たちはそう知らされています。  
<span class="verse-number">22&ensp;</span>ではどうでしょう。あなたが着いたと聞けば、会議はもちろん聞かれます。  
<span class="verse-number">23&ensp;</span>したがって、私たちのいう通りにしてください。私たちの中に、誓願を立てている男が四人います。  
<span class="verse-number">24&ensp;</span>彼らを連れて行き、彼らと共に清めを受け、彼らが頭を剃れるよう、その四人の費用を払ってください。それは、あなたについて、教えられたことは事実無根であることで、あなたは秩序通り歩み、律法を守っている人であることを、すべての人が知るためです。  
<span class="verse-number">25&ensp;</span>しかし、信じている異邦人に関しては、彼らはそのようなことは、行なう必要はないと、私たちは決めて書き留めました。ただし、偶像に捧げられた物、血、絞め殺された物、性的な罪を、彼らは遠ざけるべきです。」  
<span class="verse-number">26&ensp;</span>それから、その男たちを連れて行ったパウロは、翌日、彼らと共に清めを受け、清めの日々が終了したと告げるべく、神殿の敷地に入った。その時が、一人一人の捧げ物をすべき日時であった。  
<span class="verse-number">27&ensp;</span>それで、その七日間がほとんど終わった時、アジア出身のユダヤ人が神殿の敷地でパウロを見て、全群衆を扇動し、彼を捕まえた。  
<span class="verse-number">28&ensp;</span>そして、叫んだ。「イスラエルの男たちよ、助けてください。この男は、いたる所で、国民、律法、この地、その他もろもろに反する教えをあらゆる人にしています。そしてまた、神殿敷地の中へ、ギリシャ人を入れ、この聖なる地を彼は汚してしまいました。」  
<span class="verse-number">29&ensp;</span>なぜなら、彼らは以前、エペソ人のトロピモが、パウロと共に町の中にいるのを見かけたので、パウロが神殿の敷地内に彼らを連れて来たのだと想像したのである。  
<span class="verse-number">30&ensp;</span>町全体も混乱し、町の人々は駆け集まり、パウロを捕まえ、神殿の敷地から引きずり出し、すぐ各扉は閉められた。  
<span class="verse-number">31&ensp;</span>彼らは皆パウロの殺害しようとすると、エルサレム中が騒いでいると言う情報が守備隊の千人隊長に届いた。  
<span class="verse-number">32&ensp;</span>隊長は即刻兵士たちと百人隊長たちを連れ、彼らのところへ駆けつけた。隊長と兵士たちを見ると、彼らはパウロを殴打するのを止めた。  
<span class="verse-number">33&ensp;</span>それで、隊長は近寄り、パウロを捕まえ、二本の鎖で縛るように命じ、そして、何者か、そして何をしたのか、と問いただしていた。  
<span class="verse-number">34&ensp;</span>すると、群衆の中のある者たちは、あることを叫び、またほかの者たちは別のことを叫んでいた。それで、大混乱のため、隊長は事実を確かめられず、パウロを城塞へ連れて行くように命じた。  
<span class="verse-number">35&ensp;</span>階段に着くと、群衆の暴力のせいで、パウロは兵士たちによって担ぎ込まれるほかはなかった。  
<span class="verse-number">36&ensp;</span>なぜなら、町の群衆は後を追い、「あの者を排除せよ」、と叫んでいたからである。  
<span class="verse-number">37&ensp;</span>それから、城塞の中へ引き入れられようとした時、パウロは隊長に言った。「話しかけてもいいですか。」隊長は答えた。「お前はギリシャ語が話せるのか。  
<span class="verse-number">38&ensp;</span>先ごろ、反乱を扇動し、四千人の暗殺者を荒野に導いた、あのエジプト人ではないのか。」  
<span class="verse-number">39&ensp;</span>パウロは言った。「私はキリキヤのタルソ出身のユダヤ人で、卑しくない町の市民です。お願いします。町の人々に話をする許可をください。」  
<span class="verse-number">40&ensp;</span>それで、隊長から許可を与えられ、パウロは階段に立ち、市民に手を振った。すると、大いなる沈黙あり、彼はヘブライ語で彼らに話しかけ、言った。  

## 第22章

「男たち、兄弟たち、父たちよ、今あなたがたへの、私の弁明を聞いてください。」  
<span class="verse-number">2&ensp;</span>パウロがヘブライ語で、彼らに話しかけているのを聞き、人々はさらに静かになった。彼は言った。  
<span class="verse-number">3&ensp;</span>「私はまことのユダヤ人の男で、キリキヤのタルソで生まれ、この町でガマリエルの足もとで育てられました。我々の父祖たちの律法によって厳しく教えられ、あなたがた皆さんも今日熱心であると同じように、**神**に対して熱心でした。  
<span class="verse-number">4&ensp;</span>死に至るまで、この道を迫害し、男も女も縛って、牢獄に引き渡しました。  
<span class="verse-number">5&ensp;</span>大祭司も長老議会の全員も証するように、私は彼らから兄弟たちへの手紙を受け取り、ダマスコに行き、そこにいる人たちをも鎖で縛り、罰を受けさせるため、エルサレムに連れて行く途中でした。  
<span class="verse-number">6&ensp;</span>さて、ことの次第はこうでした。旅行中、ダマスコに近づくと、天からのすごい光が、私の回りを突然照らしました。  
<span class="verse-number">7&ensp;</span>私は地に倒れ、私に言う声を聞きました。『サウロよ、サウロよ、なぜわたしを迫害しているのか。』  
<span class="verse-number">8&ensp;</span>それで私は答えました。『あなたはどなたですか、**主**よ。』**主**は私に言われました。『あなたが迫害しているナザレのイエスである。』  
<span class="verse-number">9&ensp;</span>私と共にいた者たちは当然この光を見て、恐れましたが、私に語った方の声は聞こえませんでした。  
<span class="verse-number">10&ensp;</span>それで私は言いました。『私は何をすべきですか、**主**よ。』**主**は私に言われました。『立ち上がり、ダマスコに行け。そこで、あなたがなするべきことがすべて告げられる。』  
<span class="verse-number">11&ensp;</span>そして、あの光の栄光で、私は見えなくなったので、共にいた者たちの手に導かれ、私はダマスコに入りました。  
<span class="verse-number">12&ensp;</span>そこで、律法に沿って生きる敬虔な人で、そこに住んでいた全ユダヤ人に評判のよい、アナニヤと言う人が、  
<span class="verse-number">13&ensp;</span>私のところに来ました。彼は立って、私に言いました。『サウロ兄弟よ、目が見えるようなりなさい。』と同時に、私は彼を見上げました。  
<span class="verse-number">14&ensp;</span>するとアナニヤは言いました。『我々の父祖たちの**神**は、あなたが**神**のご意志を知るために、あなたを選ばれました。また、義なる方を見、その方の口からの声を聞くためにです。  
<span class="verse-number">15&ensp;</span>あなたの見たこと、聞いたことは、あなたがすべての人への**神**の証人となるからです。  
<span class="verse-number">16&ensp;</span>さあ今、なぜじっとしているのですか。立ち上がり、浸礼を受けなさい。そして、**主**の御名を呼び、あなたの罪を洗い流しなさい。』  
<span class="verse-number">17&ensp;</span>それで、ことの次第はこうでした。私はエルサレムに戻り、神殿の敷地で祈り、忘我状態になりました。  
<span class="verse-number">18&ensp;</span>そして、あの方が私に話しておられるのを見ました。『急いで、はやくエルサレムを出なさい。わたしについてのあなたの証を、彼らは受け入れないからです。』  
<span class="verse-number">19&ensp;</span>それで私は言いました。『**主**よ、どこのシナゴーグでも私はあなたを信じる人々を牢獄に入れ、打ち据えていたことを、彼らは知っています。  
<span class="verse-number">20&ensp;</span>そして、あなたの殉教者ステパノの血が流された時、私もそこに立って、彼の殺害に同意し、彼を殺す者たちの外套を見張っていました。』  
<span class="verse-number">21&ensp;</span>すると、**主**は私に言われました。『行け。わたしはあなたをここから遠い異邦人たちに遣わす。』」  
<span class="verse-number">22&ensp;</span>彼らは聞いていて、この言葉にくると、声を上げて言った。「地上からこの者を取り除け。生存する価値はない者だから。」  
<span class="verse-number">23&ensp;</span>そして、彼らは叫び、上着を引きちぎり、土を空中に投げたので、  
<span class="verse-number">24&ensp;</span>彼らがどんな理由で、彼に対してそう叫んでいたかを知るため、ムチ打ちで取り調べるべきであると言い、千人隊長はパウロを城塞の中へ連れて行けと命令した。  
<span class="verse-number">25&ensp;</span>それで、彼らはパウロを皮ひもで縛ると、パウロはそばに立っていた百人隊長に言った。「有罪判決を受けていないローマ市民を、ムチ打つことは、合法なのか。」  
<span class="verse-number">26&ensp;</span>百人隊長はそれを聞き、千人隊長のところに行き、報告した。「この男は、ローマ市民ですから、これからなさることに気をつけてください。」  
<span class="verse-number">27&ensp;</span>それで、千人隊長は行き、パウロに言った。「我に告げよ。お前はローマ市民か。」彼は言った。「そうです。」  
<span class="verse-number">28&ensp;</span>千人隊長は答えた。「大金で、私はこの市民権を獲得した。」パウロは言った。「私は生まれながらの市民です。」  
<span class="verse-number">29&ensp;</span>それで、パウロを問いただそうとした者たちは、即刻彼から引き下がった。そして、千人隊長はパウロがローマ市民であったことが分かり、また彼を縛ったことを恐れた。  
<span class="verse-number">30&ensp;</span>翌日、パウロはなぜユダヤ人に訴えられたかと、確かなことを知りたくて、千人隊長はパウロの拘束を解き、大祭司たちと彼らの議会の全員に来るように命じ、パウロを連れて来て、彼らの前に立たせた。  

## 第23章

するとパウロは議会全員を見据えて言った。「男たち、兄弟たちよ。この日まで私は、**神**の御前に私の全良心に恥じることなく生きてきました。」  
<span class="verse-number">2&ensp;</span>すると、大祭司アナニヤはパウロの近くに立っていた者たちに、パウロの口を打てと命じた。  
<span class="verse-number">3&ensp;</span>すると、パウロは彼に言った。「白く塗られた壁よ。**神**があなたを打つ。なぜならあなたは律法により私を裁くために座っているが、律法に反し私を打てと命じるからだ。」  
<span class="verse-number">4&ensp;</span>すると、そばに立っていた者たちは言った。「お前は**神**の大祭司をののしるのか。」  
<span class="verse-number">5&ensp;</span>するとパウロは言った。「兄弟たちよ、彼が大祭司とは知りませんでした。『人は民の支配者を悪く言ってはいけない。』と書いてあるからです。」  
<span class="verse-number">6&ensp;</span>しかし、一部はサドカイ派の人たちで、他はパリサイ派の人たちであると気づき、パウロは議会に叫んだ。「男たち、兄弟たちよ。私はパリサイ人で、パリサイ人の子です。死人たちの希望と復活のことで、私は裁かれている最中です。」  
<span class="verse-number">7&ensp;</span>彼がこう言った時、パリサイ派の人たちとサドカイ派の人たちの間で、論争が起こり、集会は分裂した。  
<span class="verse-number">8&ensp;</span>なぜなら、サドカイ派の人たちは、復活なし、御使いなし、霊なし、と言い、パリサイ派の人たちはいずれもある、と認めるからである。  
<span class="verse-number">9&ensp;</span>それで大騒ぎになった。パリサイ派の律法学者たちは立ち上がり、反論して言った。「私たちはこの人に悪を見い出せない。しかし、霊または御使いが彼に話かけたのなら、**神**と戦うのはやめようでないか。」  
<span class="verse-number">10&ensp;</span>すると、大混乱になり、千人隊長はパウロが彼らに粉々に引き裂かれると恐れ、兵士たちに、下って行き、力づくでパウロを彼らの中から連れ出し、城塞の中へ連れて行けと命じた。  
<span class="verse-number">11&ensp;</span>夜になり、**主**はパウロのそばにお立ちになり、言われた。「パウロよ、勇気を出せ。わたしのためにあなたはエルサレムで証をしたように、ローマでもわたしのために証をしなければならないからである。」  
<span class="verse-number">12&ensp;</span>夜が明けると、ユダヤ人の中のある人たちは共に徒党を組み、互いを呪いの下に置き、パウロを殺すまでは、飲食をしないと言っていた。  
<span class="verse-number">13&ensp;</span>四十人以上の者が、この共同謀議を企てた。  
<span class="verse-number">14&ensp;</span>彼らは大祭司たちと長老たちのところに行き、そして言った。「パウロを殺すまでは何も食べないと、大いなる誓願を持って、私たちは互いを呪いの下に置きました。  
<span class="verse-number">15&ensp;</span>それで、今、パウロについて、あたかももっと取り調べることがあるかのように、明日、彼を連れてくるよう、千人隊長に議会と共に示してください。私たちは千人隊長が近くに来る前に、殺す手筈ができています。」  
<span class="verse-number">16&ensp;</span>パウロの姉妹の息子は、彼らの待ち伏せを聞き、行き、城塞に入り、パウロに知らせた。  
<span class="verse-number">17&ensp;</span>それで、パウロは百人隊長の一人を近くに呼び、言った。「この青年を千人隊長のところに連れて行ってください。千人隊長に申し上げたいことがあるそうです。」  
<span class="verse-number">18&ensp;</span>それで百人隊長は青年を千人隊長のところへ連れて行き、言った。「囚人のパウロが私を呼び、この青年をあなたに連れて行くように、私に頼みました。青年はあなたに言いたいことがあるそうです。」  
<span class="verse-number">19&ensp;</span>千人隊長は青年の手を取り、脇に連れて行き、密かに尋ねた。「私に言いたいこととは何か。」  
<span class="verse-number">20&ensp;</span>青年は言った。「ユダヤ人たちは、あたかもパウロをもっと尋問すると言って、あなたにパウロを議会に連れて来るように頼むことを決めました。  
<span class="verse-number">21&ensp;</span>しかし、信用しないでください。なぜなら、彼らの中の四十人以上が待ち伏せしているからです。そしてその四十人は、パウロを殺すまで、飲食をしないと、互いを呪いの下に置いています。そして今彼らは準備でき、あなたからの知らせを待っているのです。」  
<span class="verse-number">22&ensp;</span>それで千人隊長は青年を去らせ、「私に教えたこのことを、誰にももらしてはいけない。」と命じた。  
<span class="verse-number">23&ensp;</span>それから、彼は百人隊長を二人呼び、言った。「今夜の三時間目[^16]に、カイザリヤに行くため、兵士二百人、騎兵七十人、槍兵二百人を整え、  
<span class="verse-number">24&ensp;</span>そして、パウロが乗る馬を準備し、総督ペリクスのところへ彼を無事に連れて行け。」  
<span class="verse-number">25&ensp;</span>彼は次のように手紙を書いた。  
<span class="verse-number">26&ensp;</span>「クラウデオ・ルシヤより、偉大なる総督ペリクス閣下にご挨拶を申し上げます。  
<span class="verse-number">27&ensp;</span>この男はユダヤ人に捕まり、彼らに殺されるところでした。彼はローマ市民であるとわかって、私は兵士たちを率いて、彼は助け出しました。  
<span class="verse-number">28&ensp;</span>そして、私は彼らがこの男を訴える理由を知りたく、彼らの議会の前に連れて行きました。  
<span class="verse-number">29&ensp;</span>彼らの律法に関する件で訴えられていますが、死や鎖に値する刑に該当しないと分かりました。  
<span class="verse-number">30&ensp;</span>そして、ユダヤ人たちがその男に対して企んでいることが、私に伝えられ、私はすぐ彼をあなたのところに送り、また彼を訴える者たちはあなたの前で訴えよ、と命令しました。敬白。」  
<span class="verse-number">31&ensp;</span>そこで、命令された通り、兵士たちはパウロを受け取り、夜中にアンテパトリスに連れて行き、  
<span class="verse-number">32&ensp;</span>そして翌日、騎兵たちがパウロと共に行くので、兵士たちは、彼らを残し、城塞に帰った。  
<span class="verse-number">33&ensp;</span>兵士たちはカイザリヤに着き、総督に手紙を渡し、パウロをも彼の前に立たせた。  
<span class="verse-number">34&ensp;</span>総督は手紙を読み、パウロはどの州の出身かと尋ねると、キリキヤからであるとわかり、  
<span class="verse-number">35&ensp;</span>言った。「お前を訴える者たちも来たら、お前の話をよく聞く。」そして、ヘロデの官邸内で保護するよう命じた。  

## 第24章

さて、五日後、大祭司アナニヤは長老たちと、テルトロという弁護士と共に下ってきた。この者たちはパウロに反対する根拠を総督に出した。  
<span class="verse-number">2&ensp;</span>テルトロは呼び出されると、パウロを訴え出し、言った。「あなたを通じて我々は大いなる平和を得て、またあなたの先見で、この国に繁栄がもたらされていますので、  
<span class="verse-number">3&ensp;</span>最も高貴なるペリクスよ、私たちは常に、全地域で、大いなる感謝を持って、それを受けています。  
<span class="verse-number">4&ensp;</span>しかしながら、私はもうあなたをこれ以上煩わすことのないよう、あなたの寛容によって、私たちから一言、二言を聞いてくださるよう、お願い申し上げます。  
<span class="verse-number">5&ensp;</span>なぜなら、あの者は疫病であり、世界中のユダヤ人の間で、混乱を起こす者で、ナザレ人の異端の首謀者であると、私たちは調査してわかったからです。  
<span class="verse-number">6&ensp;</span>彼は神殿の敷地さえも汚そうとし、そして私たちの律法によって裁くつもりで、彼を捕らえました。  
<span class="verse-number">7&ensp;</span>千人隊長ルシヤがやってきて、乱暴にも力ずくで私たちの手から彼を連れて行き、  
<span class="verse-number">8&ensp;</span>彼を訴える者たちは、あなたのもとに行くように命じました。ご自分でパウロを取り調べになれば、私たちが彼を訴えているすべての件を、確かめることができます。」  
<span class="verse-number">9&ensp;</span>ユダヤ人も全員同意し、これらのことはすべて事実であると主張した。  
<span class="verse-number">10&ensp;</span>そこで、総督はパウロに話すよう、うなずくと、彼は答えた。「あなたは長年この国の裁判官であると私は知っておりますので、私はうれしく思い、自らのため弁明いたします。  
<span class="verse-number">11&ensp;</span>なぜなら、私が礼拝のため、エルサレムに上ったのは、わずか十二日前であることは、あなたは確認することができます。  
<span class="verse-number">12&ensp;</span>神殿の敷地で、誰か、人と論争している私を、また街であれ、シナゴーグであれ、群衆を扇動している私を、彼らは見かけたはずはありません。  
<span class="verse-number">13&ensp;</span>そして、私を今訴えた事犯を、彼らは証明することができません。  
<span class="verse-number">14&ensp;</span>しかし、あなたに、このことを言い表します。あの者たちが異端と呼んでいる道により、私は私の父祖たちの**神**を礼拝し、律法と預言者たちの書に書いてあるすべてのことを信じています。  
<span class="verse-number">15&ensp;</span>彼ら自身も受け入れている希望、すなわち、死人の中から、正しい者たちも、邪悪な者たちも、両者とも未来に復活があるという希望を、私は**神**に対して抱いています。  
<span class="verse-number">16&ensp;</span>したがって、いつも**神**と人間に対し、違反のない良心を持とうと、私はいつも努めています。  
<span class="verse-number">17&ensp;</span>それで、多くの年月を経て、施しを私の国に、また捧げ物を持ってくるため参りましたが、  
<span class="verse-number">18&ensp;</span>その最中、私は群衆といっしょでなく、騒ぎなどはなく、神殿の敷地に清められた私を、アジア出身のユダヤ人のある者たちが、見つけたのです。  
<span class="verse-number">19&ensp;</span>もし私に対して何かの訴えがあれば、その者たちはここ、あなたの前に出るべきです。  
<span class="verse-number">20&ensp;</span>そうでなければ、私が議会の前に立っている間に、私に何かの不正を見つけたなら、ここにいる者たちにそれを言わせてください。  
<span class="verse-number">21&ensp;</span>それとも、あの者たちの中に立ち、『死人たちの復活のことで、私は今日あなたがたに裁かれている最中です。』と私が叫んだ一言以外はありません。」  
<span class="verse-number">22&ensp;</span>ペリクスはこれらのことを聞き、彼はこの道に関してもっと正確に知っていたので、「千人隊長ルシヤが下って来たら、お前たちに関する件を裁可する。」と言って、彼らの件を延期した。  
<span class="verse-number">23&ensp;</span>そうして、パウロを保護し、自由を許し、また彼の関係者がパウロを訪ね、世話することを禁じてはいけないと、ペリクスは百人隊長に命じた。  
<span class="verse-number">24&ensp;</span>数日後、ペリクスはユダヤ人である妻ドルシラと共に来て、パウロを呼び、キリストに対する信仰について聞いた。  
<span class="verse-number">25&ensp;</span>そこで、パウロは義と、自制と、来たるべき裁きについて論じると、ペリクスは恐れ始め、答えた。「今のところは、去れ。時をみて、また呼ぶ。」  
<span class="verse-number">26&ensp;</span>それと同時に、パウロを釈放し、金銭を引き出そうと、さらに、何度もパウロを呼び出し、彼と話をした。  
<span class="verse-number">27&ensp;</span>しかし、二年経ってから、ポルキオ・フェストがペリクスの後任になったが、ペリクスはユダヤ人に迎合し、パウロをつないだままにした。  

## 第25章

さて、フェストは彼の州に来て三日後、カイザリヤからエルサレムに上った。  
<span class="verse-number">2&ensp;</span>すると、大祭司とユダヤ人たちの主だった者たちは、パウロに反対であることをフェストにはっきり告げた。そして願った。  
<span class="verse-number">3&ensp;</span>道中パウロを殺すため、待ち伏せしているので、パウロをエルサレムへ出頭させるよう願った。  
<span class="verse-number">4&ensp;</span>しかし、パウロはカイザリヤで拘置されるべきであり、彼自身も急ぎそこに行く、とフェストは答えた。  
<span class="verse-number">5&ensp;</span>それから、彼は言った。「ゆえに、お前たちの中で権威ある者たちは、私と共に下って行き、この男に関して何かあるなら、そこで訴えよ。」  
<span class="verse-number">6&ensp;</span>彼らと共に十日以上滞在し、カイザリヤに下って行った。次の日、彼は裁きの席に座り、パウロを連れて来るように命じた。  
<span class="verse-number">7&ensp;</span>そして、パウロが来ると、エルサレムから下って来たユダヤ人たちは、パウロを取り囲み、多くの重大な訴えをしたが、立証不能であった。  
<span class="verse-number">8&ensp;</span>パウロは自ら弁明した。「ユダヤ人の律法に対しても、神殿に対しても、カイザルに対しても、私は何の罪も犯していません。」  
<span class="verse-number">9&ensp;</span>しかし、フェストはユダヤ人たちに迎合しようと思い、パウロに答え、言った。「お前はエルサレムに上り、この事に関して私に裁かれることを望むか。」  
<span class="verse-number">10&ensp;</span>しかし、パウロは言った。「私は当然裁かれるべき場所である、カイザルの裁き座の前に立ちます。よくご存じのように、私はユダヤ人たちに何も不正なことをしていません。  
<span class="verse-number">11&ensp;</span>しかし、もし私が法を犯し、死に値する罪を犯したならば、死ぬことを拒みません。しかし、この者たちが訴えることが無であるなら、誰も私を彼らに渡すことができません。私はカイザルに上訴します。」  
<span class="verse-number">12&ensp;</span>それで、フェストは議会と話し合い、答えた。「カイザルにお前は上訴した。カイザルのところにお前は行く。」  
<span class="verse-number">13&ensp;</span>さて、数日経てから、アグリッパ王とベルニケは、フェストに挨拶のため、カイザリヤに来た。  
<span class="verse-number">14&ensp;</span>そして、彼らはそこで多くの日も過ごしている間、フェストはパウロの件を王に知らせて言った。「ペリクスが囚人として残した一人の男がいます。  
<span class="verse-number">15&ensp;</span>私がエルサレムにいた時、ユダヤ人の大祭司たちと長老たちがこの者のことを私に知らせて、裁きを願い出しました。  
<span class="verse-number">16&ensp;</span>私は彼らに答えました。『訴えられた者が訴える者たちに、顔と顔を合わせて、自分に関する訴えを自ら聞く前に、その者を破滅に引き渡すのは、ローマ人の習慣ではない。』  
<span class="verse-number">17&ensp;</span>したがって、ユダヤ人たちが集まった時、私は延期せず、翌日裁きの席に座り、その男を連れて来いと命じました。  
<span class="verse-number">18&ensp;</span>訴える者たちは立ち上がると、私が思っていたほどの訴えをせず、  
<span class="verse-number">19&ensp;</span>彼ら自身の宗教と、死んだイエスと言う者が、生きていると主張していたパウロに対して、いくつかのやり取りがありました。  
<span class="verse-number">20&ensp;</span>それで、私自身はそういう問に関して戸惑い、パウロに聞きました。『エルサレムに行き、そこでこれらの件に関して、裁きを受ける気があるか。』  
<span class="verse-number">21&ensp;</span>しかし、パウロはアウグスト[^17]の決断が下るまで、保護してくれるように上訴した時、彼をカイザルに送ることができる時まで、彼は保護されるべしと、命じました。」  
<span class="verse-number">22&ensp;</span>すると、アグリッパはフェストに言った。「私もその者の話しを聞くと思います。」彼は「明日、聞かせましょう。」と言った。  
<span class="verse-number">23&ensp;</span>次の日、アグリッパとベルニケは華麗に飾り立ててやって来て、千人隊長や町の主立った人たちと共に会堂に入ると、フェストは命じると、パウロは連れて来られた。  
<span class="verse-number">24&ensp;</span>フェストは言った。「アグリッパ王、そして私たちと共にいるすべての男たちよ。エルサレムでもここでも、ユダヤ人全会衆が、もはやこの男は、生存する価値のない者だと叫び、私に願い出ました。その男をあなたがたは目にしているのです。  
<span class="verse-number">25&ensp;</span>しかし、彼は死に価することを何も犯していず、そして、彼自身はすでにアウグストに上訴していたことがわかり、アウグストに送ることを私は決めました。  
<span class="verse-number">26&ensp;</span>彼に関し、私の主君に文書にする、確かなことは何もありません。したがって、取り調べの後、何か文書にすることがあるように、この者をあなたがた、特にあなた、アグリッパ王の前に連れて来ました。  
<span class="verse-number">27&ensp;</span>なぜなら、囚人を送るのに、その者に対する告訴状なしで送ることは、私にとって道理に合わないからです。」  

## 第26章

するとアグリッパはパウロに言った。「お前は、自らの弁明が許されている。」それでパウロは手を伸ばし、弁明した。  
<span class="verse-number">2&ensp;</span>「アグリッパ王よ、今日、あなたの前で全ユダヤ人たちが、私を訴えているすべてのことについて、私自身が弁明します。私は私自身、このことを幸いと思っています。  
<span class="verse-number">3&ensp;</span>特に、ユダヤ人たちに関するすべての習慣、問題について、あなたは専門家だからです。したがって、お願い致します。忍耐して、私の言葉をお聞きください。  
<span class="verse-number">4&ensp;</span>エルサレムで私の同国人と共に過ごした、青年時代の最初からの私の生活ぶりをユダヤ人全員が知っています。  
<span class="verse-number">5&ensp;</span>もし彼らが証言をする気があれば、私たちの宗教のもっとも厳しい教派であるパリサイ人として、生きてきた私を、彼らは最初から知っています。  
<span class="verse-number">6&ensp;</span>そして今、**神**が私たちの父たちに与えてくださった約束の希望のために、私は立ち、裁かれています。  
<span class="verse-number">7&ensp;</span>私たちの十二部族は、この約束の実現を望み、夜も昼も熱心に**神**に仕えています。アグリッパ王よ、この希望のゆえに私は、ユダヤ人たちに訴えられています。  
<span class="verse-number">8&ensp;</span>**神**が死人たちを復活させることは、あなたにとってなぜ信じ難いのですか。  
<span class="verse-number">9&ensp;</span>そうです、私はナザレのイエスの御名に逆らい、多くのことをしなければならないと思いました。  
<span class="verse-number">10&ensp;</span>その通りのことを、エルサレムでもしました。そこで、大祭司たちから権限を与えられ、聖徒たちを多く牢獄に閉じ込め、そして、彼らが殺された時、私もその聖徒たちに敵対の票を投じました。  
<span class="verse-number">11&ensp;</span>そして、どこのシナゴーグでも彼らをたびたび罰し、無理に冒涜をさせました。私は彼らに対して非常に激怒し、外国の町までも彼らを追いかけ、迫害しました。  
<span class="verse-number">12&ensp;</span>このような事をしながら、大祭司たちからの権限と使命を携え、ダマスカスへ旅立ちますと、  
<span class="verse-number">13&ensp;</span>真昼でした。王よ、道を行くと、私と同行者の回りを照らす、太陽より明るい天からの光を私は見ました。  
<span class="verse-number">14&ensp;</span>そして、私たち全員、地に倒れ、ヘブライ語で私に話かける声を耳にし、その声は言われました。『サウロ、サウロ、なぜわたしを迫害するのか。家畜棒[^18]を蹴るのは、あなたにはつらいことである。』  
<span class="verse-number">15&ensp;</span>私は言いました。『**主**よ、あなたはどなたですか。』**主**は言われました。『わたしはあなたが迫害しているイエスである。  
<span class="verse-number">16&ensp;</span>しかし、起き上がり、自らの足で立て。なぜなら、あなたが見たこと、またわたしがこれからあなたに啓示することの、奉仕者と証人にするために、わたしはあなたに現われたのである。  
<span class="verse-number">17&ensp;</span>わたしはあなたを国民から、また異邦人たちから救い出す。わたしはあなたをこの者たちに遣わす。  
<span class="verse-number">18&ensp;</span>暗やみから光へ、サタンの力から**神**の方へ彼らの向きを変えるため、彼らの目を開けるためである。それは、罪の赦しと、わたしに対する信仰によって聖別された者たちの間での相続を受けるためである。』  
<span class="verse-number">19&ensp;</span>それゆえに、アグリッパ王よ、私はその天からのビジョンに対して不従順ではありませんでした。  
<span class="verse-number">20&ensp;</span>しかし、まずダマスカスとエルサレムにいる人々に、またユダヤ全土、そして異邦人たちに、彼らは悔い改め、**神**の方を向き、悔い改めにふさわしい行為を行なうべきだと、私は説きました。  
<span class="verse-number">21&ensp;</span>こんなわけで、ユダヤ人たちは神殿で私を捕まえ、私を殺そうとしたのです。  
<span class="verse-number">22&ensp;</span>ですから、**神**から助けを頂き、私は今日までしっかり立ち、預言者たちとモーセが起こると言ったことのほか、何も言わず、地位の低い者にも高い者にも証をしてきました。  
<span class="verse-number">23&ensp;</span>それは、キリストは苦しみ、死人たちの中から復活する最初の人になり、国民と異邦人に光を伝える、ということです。」  
<span class="verse-number">24&ensp;</span>パウロがこう弁明していると、フェストは大声で言った。「パウロよ、お前は気が狂っている。学問のし過ぎがお前を狂気にした。」  
<span class="verse-number">25&ensp;</span>パウロは言った。「最も高貴なるフェストよ、私は狂っていず、真理と道理にかなう言葉を言っています。  
<span class="verse-number">26&ensp;</span>なぜなら、私が王の前で自由に話しているのは、王もこれらのことはご存じだからです。これらのことは片隅で行なわれなかったので、王はこれらのことは何一つ見逃さなかったのです。  
<span class="verse-number">27&ensp;</span>アグリッパ王よ、あなたは預言者たちを信じていますか。信じていると私は確信しております。」  
<span class="verse-number">28&ensp;</span>アグリッパはパウロに言った。「わずかな言葉で、私がクリスチャンになるよう、お前は私を納得させるのか。」  
<span class="verse-number">29&ensp;</span>パウロは言った。「**神**に願うのはあなただけではなく、今日私の話を聞いてくださる全員が、この鎖は別ですが、言葉が少なくても多くても、私のようになることです。」  
<span class="verse-number">30&ensp;</span>パウロはこう言うと、王も、同様に総督とベルニケ、そして彼らと共に座っていた人たちも立ち上がった。  
<span class="verse-number">31&ensp;</span>そして、彼らは退いて互いに言った。「この人は、死刑や鎖に値することは、何もしていない。」  
<span class="verse-number">32&ensp;</span>またアグリッパはフェストに言った。「この男はカイザルに上訴しなかったならば、釈放されたであろうに。」  

## 第27章

さて、私たちはイタリアへ出航すると決まった時、彼らはパウロと他の囚人を数人、アウグスト六百人部隊のユリアスという百人隊長に引き渡した。  
<span class="verse-number">2&ensp;</span>こうして、アジアの所々に立ち寄ろうと思い、アドラミテオからの船に私たちは乗った。テサロニケのマケドニヤ人アリスタルコも同行していた。  
<span class="verse-number">3&ensp;</span>翌日、私たちはシドンに上陸した。ユリアスはパウロを親切に接し、友人のところに行って歓待されることを許した。  
<span class="verse-number">4&ensp;</span>そこから出航してから、向かい風であったので、私たちは風の当たらないキプロスの岸を航行した。  
<span class="verse-number">5&ensp;</span>キリキヤとパンフリヤの沖を通り、私たちはルキヤ地方のミラに着いた。  
<span class="verse-number">6&ensp;</span>そこで、百人隊長はイタリアへ航行するアレキサンドリヤの船を探し出し、私たちを乗せた。  
<span class="verse-number">7&ensp;</span>数日間は遅々として進まず、苦労してクニドの沖に着き、風は我々の進行を許さず、サルモネの沖にある、風の当たらないクレテの側を航行した。  
<span class="verse-number">8&ensp;</span>苦労してそこを通り、ラサヤと言う町の近くにある「良き港」と言う所に着いた。  
<span class="verse-number">9&ensp;</span>多大な時が過ぎ、断食の時はすでに過ぎており、航行は危険になったので、パウロは彼らに助言し、  
<span class="verse-number">10&ensp;</span>言った。「男たちよ、この航海は、船と積荷だけではなく、我らの命まで失う災難で、大きな損失を伴うと私には分かります。」  
<span class="verse-number">11&ensp;</span>しかし、百人隊長はパウロの言ったことより、舵手と船主を信頼した。  
<span class="verse-number">12&ensp;</span>そして、その港は冬を越すのに適さないから、そこからも出航し、何とかして南西と北西の方にあるピニクスというクレテの港で、冬を過ごそうと大多数の人は進言した。  
<span class="verse-number">13&ensp;</span>南風が穏やかに吹き、望み通りになったと思い、錨を引き上げ、クレテのそばを航行した。  
<span class="verse-number">14&ensp;</span>しばらく経つと、ユーロクルドンという逆風の暴風が起き、  
<span class="verse-number">15&ensp;</span>船はつかまり、風に向かって前進できず、私たちは船を風のふくままにした。  
<span class="verse-number">16&ensp;</span>そして、クラウダという小島の風下を進み、辛うじて小舟は手入れのできる状態になった。  
<span class="verse-number">17&ensp;</span>小舟を船に引き上げ、ロープで船を補強し、スルテス流砂に乗り上げることを恐れ、帆を降ろし、流された。  
<span class="verse-number">18&ensp;</span>私たちは暴風に激しくもてあそばれたので、翌日、積荷を投棄した。  
<span class="verse-number">19&ensp;</span>そして、三日目、私たちは自らの手で船具を投げ捨てた。  
<span class="verse-number">20&ensp;</span>そして、太陽も星も、数日間見えず、大暴風に押しつけられ、ついに私たちは救われる希望はすべて取り去られていた。  
<span class="verse-number">21&ensp;</span>しかし、彼らは長期間、食べ物を断っていたので、パウロは彼らの中央に立ち、言った。「男たちよ、私の言うことを聞き、クレテから出航しなければ、この災害と損失を受けなかったはずです。  
<span class="verse-number">22&ensp;</span>そして、今、気を強く持つことを勧めます。失うのは船だけです。あなたがたの中で、命を失う人はいないからです。  
<span class="verse-number">23&ensp;</span>なぜなら、私が所属し、また仕えている**神**の御使いが、今夜私のそばに立ち、  
<span class="verse-number">24&ensp;</span>言いました。『恐れてはいけません、パウロよ。あなたはカイザルの前に、連行されて行かなければなりません。そして見よ、あなたと共にいる全員を、**神**はあなたに与えられました。』  
<span class="verse-number">25&ensp;</span>ですから、男たちよ、気を強く持ちない。**神**が私に言われたことはそうなると、私は信じているからです。  
<span class="verse-number">26&ensp;</span>ともかく、私たちはまずある島に漂着しなければなりません。」  
<span class="verse-number">27&ensp;</span>十四日目の夜になり、私たちはアドリア海を為す術もなく漂うことになり、真夜中ごろ、何か陸地に近づいていると船員たちは気がした。  
<span class="verse-number">28&ensp;</span>それで水深を測ると、二十オルギィア[^19]であるとわかり、もう少し行き、また水深を測ると、十五オルギィアであるとわかった。  
<span class="verse-number">29&ensp;</span>そこで、船が岩に乗り上げることを恐れ、船員たちは船尾から錨を四つ投げ下ろし、朝が来るように祈っていた。  
<span class="verse-number">30&ensp;</span>それで、船員たちは船から逃げようとして、船首から錨を下ろすふりをして、小舟を海に降ろすと、  
<span class="verse-number">31&ensp;</span>パウロは百人隊長と兵士たちに言った。「この人たちが船に残らなければ、あなたがたは救われることができません。」  
<span class="verse-number">32&ensp;</span>それで兵士たちは小舟の綱を切り落とした。  
<span class="verse-number">33&ensp;</span>夜明けになろうとする時、パウロは全員に食べ物を取るように勧め、言った。「何も食べ物を手にせず、何も口にせず、待ち望むこと、今日で十四日目です。  
<span class="verse-number">34&ensp;</span>ですから、食べ物を取るよう私は勧めます。これはあなたがたの生存のためであり、だれの頭からも髪の毛一本落ちないためです。」  
<span class="verse-number">35&ensp;</span>パウロはこう言って、パンを受け取り、全員の前で**神**に感謝を捧げ、それを裂き、食べ始めた。  
<span class="verse-number">36&ensp;</span>それで全員勇気付けられ、彼ら自身も食べた。  
<span class="verse-number">37&ensp;</span>我ら総員、二百七十六人乗船していた。  
<span class="verse-number">38&ensp;</span>さて、全員十分食べ、船を軽くし、小麦を海に投げ捨てた。  
<span class="verse-number">39&ensp;</span>そして、朝になり、どこの地か分からなかったが、砂浜がある入り江が見え、できればそこに船を乗り上げようと計画し、  
<span class="verse-number">40&ensp;</span>それで、錨を放し、海に捨て、一方、舵の綱を緩め、そして、風に帆を上げ、砂浜に向かった。  
<span class="verse-number">41&ensp;</span>しかし、二つの海流が出会う所にぶつかり、彼らは船を陸地に乗り上げ、船首は突き刺さり、動けなくなり、船尾は波の力で破壊された。  
<span class="verse-number">42&ensp;</span>囚人が一人と言え泳いで逃亡しないよう、彼らを殺すのが兵士たちの計画であった。  
<span class="verse-number">43&ensp;</span>しかし、百人隊長はパウロを守ろうとして、兵士たちのその意図を中止させ、泳げる者は先に海に飛び込み、陸地まで行けと命じた。  
<span class="verse-number">44&ensp;</span>そして後の者は、板や、船の一部で、全員、陸地に着いた。こうして、全員、無事に陸地に逃れた。  

## 第28章

さて、危機を脱し、その島はマルタと呼ばれていると分かった。  
<span class="verse-number">2&ensp;</span>そして、降る雨と寒さのため、その土着の人たちは火をたき、私たち全員を受け入れ、大いに親切にしてくれた。  
<span class="verse-number">3&ensp;</span>しかし、パウロは柴をたくさん集め、火にくべると、一匹のまむしが熱の中から出て来て、パウロの手にしっかり付いた。  
<span class="verse-number">4&ensp;</span>それで土着の人たちはその動物がパウロの手からぶら下がっているのを見て、互いに言った。「きっとこの男は人殺しで、海からは助かっても、正義は生きることを許さない。」  
<span class="verse-number">5&ensp;</span>しかし、パウロはその動物を火の中へ振り落とし、何の害も受けなかった。  
<span class="verse-number">6&ensp;</span>彼らは腫れ上がるか、突然倒れて死ぬかと思っていたが、長い間パウロを見ていても、何の害も彼に起きないと分かると、考えを変え、パウロは神であると言った。  
<span class="verse-number">7&ensp;</span>その地方に、島の首長のポプリオという男の所有地があった。彼は私たちを歓迎し、三日間礼を持って、私たちを歓待してくれた。  
<span class="verse-number">8&ensp;</span>すると、事件が起きた。すなわち、ポプリオの父は熱病と赤痢で寝ていたが、パウロは彼のところに入り、祈り、彼に手を置いて彼を治した。  
<span class="verse-number">9&ensp;</span>このことがあり、島の病気のある他の人たちも来て、治された。  
<span class="verse-number">10&ensp;</span>彼らはいろいろな方法で私たちに敬意を表した。そして、私たちが出航した時、必要な物を贈ってくれた。  
<span class="verse-number">11&ensp;</span>三ヶ月後、島で冬を過ごした「デオスクロ」[^20]の船首像がある、アレキサンドリヤの船に乗って出航した。  
<span class="verse-number">12&ensp;</span>そして、シラクサに寄港し、三日間停泊した。  
<span class="verse-number">13&ensp;</span>そこから私たちは巡航し、レギオンに着き、一日後南風が吹きだし、翌日私たちはポテオリに入港し、  
<span class="verse-number">14&ensp;</span>そこで、私たちは兄弟たちを探し出し、七日間共に滞在するように勧められた。こうして私たちはローマのほうに行った。  
<span class="verse-number">15&ensp;</span>そこで、兄弟たちは私たちのことを聞き、遠くアピオ・ポロとトレス・タベルネまで来て、私たちを迎えてくれた。パウロは彼らを見て、**神**に感謝し、勇気づけられた。  
<span class="verse-number">16&ensp;</span>さて、私たちはローマに着くと、百人隊長は囚人たちを司令官に渡したが、パウロは彼自身を見張る兵士と共に、単独での滞在を許された。  
<span class="verse-number">17&ensp;</span>そして、三日後のことであった。パウロはユダヤ人の指導者たちを呼び集めた。彼らが集まると、パウロは彼らに言った。「男たち、兄弟たちよ。我々の国民、そして父たちの習慣に反することは、何もしていないのに、私は囚人として、エルサレムよりローマ人たちの手に渡されました。  
<span class="verse-number">18&ensp;</span>私を取り調べた彼らは、死刑にする理由がないので、私を釈放したかったのです。  
<span class="verse-number">19&ensp;</span>しかし、ユダヤ人たちは反論し、私は止むをえず、カイザルに上訴しました。私は自国民を訴える理由は何もなかったのですが、  
<span class="verse-number">20&ensp;</span>会って話すため、あなたがたをお呼びしました。なぜなら、イスラエルの希望のため、私はこの鎖で縛られているからです。」  
<span class="verse-number">21&ensp;</span>すると彼らはパウロに言った。「ユダヤから、あなたについての手紙は受け取っていないし、またここに来た兄弟たちで、あなたの事を話した者、また、あなたを悪く言う者は誰もいませんでした。  
<span class="verse-number">22&ensp;</span>それで、あなたから、あなたが考えていることを聞きたいです。なぜなら、この宗派に関して、すべての所で反対されていると、私たちは知っているからです。」  
<span class="verse-number">23&ensp;</span>それで、ユダヤ人たちはパウロのために日を決めると、多くのユダヤ人は彼の宿舎に来た。パウロは彼らに**神**の王国を説明し、厳粛にそれを証し、またモーセの律法と預言者の書より、朝から晩までイエスについて彼らを納得させていた。  
<span class="verse-number">24&ensp;</span>そして、ある者たちは、話されたことで納得させられたが、不信のままの者もいた。  
<span class="verse-number">25&ensp;</span>それで、相互に同意のないまま、パウロが言った一言を聞き、別れた。「預言者イザヤを通して、私たちの父たちに**聖霊**は正に言われました。  
<span class="verse-number">26&ensp;</span>『この民の所に行き、言え。「聞くであろうが、お前たちは理解しない。見るであろうが、お前たちは認めない。」と。  
<span class="verse-number">27&ensp;</span>なぜなら、この民の心は鈍感になった。彼らの持つ耳は聞こえず、彼らが持つ目は閉じてしまったのは、目で見ず、耳で聞かず、心で理解せず、翻って立ち戻って、私が治すことがないためである。』  
<span class="verse-number">28&ensp;</span>したがって、**神**の救いはすでに異邦人に送られていて、そして異邦人はそれを聞くことを知りなさい。」  
<span class="verse-number">29&ensp;</span>パウロはこう言うと、ユダヤ人たちは出て行き、互いの間で大論争をした。  
<span class="verse-number">30&ensp;</span>そして、パウロはまる二年、自分自身の借家に住み、すべて彼のところに来る人々を受け入れ、  
<span class="verse-number">31&ensp;</span>**神**の王国を説き、**主**イエス・キリストに関することを実に大胆に、禁止されることなく教えていた。
