# コロサイ

### コロサイ人への手紙

## 第1章

**神**のご意志を通して、イエス・キリストの使徒であるパウロ、そして我らの兄弟テモテより、  
<span class="verse-number">2&ensp;</span>コロサイにいる聖徒たち、すなわち、キリストにあって忠実な兄弟たちへ。私たちの**父**なる**神**と**主**イエス・キリストより、あなたがたに恵みと平安がありますように。  
<span class="verse-number">3&ensp;</span>私たちの**主**イエス・キリストの**父**なる**神**に、私たちは感謝し、いつもあなたがたのために祈っています。  
<span class="verse-number">4&ensp;</span>それは、キリスト・イエスへの、あなたがたの信仰と、すべての聖徒たちへのあなたがたの愛を聞いた時からです。  
<span class="verse-number">5&ensp;</span>それは、あなたがたのために天に備えられている希望のためです。あなたがたは以前、福音の真理の御言葉の中にこの希望を聞いています。  
<span class="verse-number">6&ensp;</span>その福音は、全世界に達したように、あなたがたに達しており、あなたがたが真理によって**神**の御恵みを聞いて理解したあの日以来、あなたがたの間に、そして実が結ばれたように、全世界で福音は実を結びつつあります。  
<span class="verse-number">7&ensp;</span>それは、私たちの愛する僕仲間であり、あなたがたのための忠実な、キリストの奉仕者であるエパフラスからも、あなたがたも学んだとおりです。  
<span class="verse-number">8&ensp;</span>エパフラスもあなたがたの愛を、**御霊**によって私たちに伝えました。  
<span class="verse-number">9&ensp;</span>こういう理由で、このことを聞いた日以来、私たちもあなたがたのため、すべての知恵と霊的な理解により、あなたがたが**神**のご意志の考えに満たされるように、祈り、また願い続け、止めたりしません。  
<span class="verse-number">10&ensp;</span>また、完全に**主**に喜んで頂くため、あなたがたは**主**にふさわしく歩み、すべてのよい行ないで、あなたがたが実を結び、**神**の知識に関して、あなたがたが成長することを願っています。  
<span class="verse-number">11&ensp;</span>あなたがたは、**神**の栄光ある力により、喜びをもってすべての忍耐と耐えしのびのため、すべての力によって強められ、  
<span class="verse-number">12&ensp;</span>光の中にいる聖徒たちの相続を、共に受け入れる資格あるものと、私たちをしてくださった**御父**に、感謝するように祈っています。  
<span class="verse-number">13&ensp;</span>**御父**は私たちを暗黒の力から救い出し、ご自分の愛の**御子息**の王国に移してくださったのです。  
<span class="verse-number">14&ensp;</span>この**御子息**の血を通して私たちには贖いがあります。すなわち、罪の赦しです。  
<span class="verse-number">15&ensp;</span>**御子息**は目に見えない**神**の御姿であられ、全創造の長子であられます。  
<span class="verse-number">16&ensp;</span>なぜなら、天にあるものも、地にあるものも、見えるものも、目に見えないものも、王座も政府も、支配者も権力も、すべては**御子息**によって造られました。すべてのものは**御子息**を通し、また**御子息**のために造られました。  
<span class="verse-number">17&ensp;</span>そして、**御子息**はすべてのものに先立って存在され、すべてのものは**御子息**の手中に存在しています。  
<span class="verse-number">18&ensp;</span>また、始まりであり、死人たちの中から生まれた長子であられる**御子息**は、体の頭、すなわち教会の頭です。それは、**御子息**がすべてのことにおいて第一者となられるためでした。  
<span class="verse-number">19&ensp;</span>すべて満ちているものが、**御子息**の中に宿ることが、**御父**を喜ばせたからです。  
<span class="verse-number">20&ensp;</span>そして、**御子息**を通し、地上であろうと天であろうと、ご自分が万物を和解させるため、**御子息**の十字架の血を通して平和を造られました。  
<span class="verse-number">21&ensp;</span>そして、自らの悪行により、以前は遠ざけられ、心中、敵であったあなたがたを、**御子息**は今、和解されました。  
<span class="verse-number">22&ensp;</span>それは、ご自分の肉の体で死を通し、あなたがたを聖なる者、非難されるところのない、責めのない者として**御子息**の御前に立たせるためです。  
<span class="verse-number">23&ensp;</span>ただし、あなたがたが、この信仰に引き続き踏み留まり、しっかりと基礎に立ち、耳にした福音の希望から、もし移ることがなければです。その福音とは、天下のすべての者に説かれ、私パウロが奉仕者となっている福音です。  
<span class="verse-number">24&ensp;</span>今、あなたがたのため、私は自分の苦しみを喜び、キリストの体である教会のため、キリストの苦悩に欠けているものを、私の肉によって私は満たします。  
<span class="verse-number">25&ensp;</span>**神**の御言葉を成就するため、あなたがたのため、**神**から与えられた摂理により、私はその教会の奉仕者になりました。  
<span class="verse-number">26&ensp;</span>各時代から、何代にわたり隠されて来た奥義が、今、**神**の聖徒たちに啓示されたのです。  
<span class="verse-number">27&ensp;</span>異邦人の間にこの奥義の栄光の富が何であるかを聖徒たちに教えることを、**神**は定められました。それは、あなたがたの中におられるキリスト、つまり栄光の希望です。  
<span class="verse-number">28&ensp;</span>すべての人をキリスト・イエスにあって完全な人として立たせるために、私たちはキリストを説き、すべての人に警告し、あらゆる知恵を持ってすべての人に教えています。  
<span class="verse-number">29&ensp;</span>私の中で力強く働くキリストの働きにより、このために、私もまた懸命に努力しています。  

## 第2章

あなたがた、そしてラオデキヤの人たちのため、私の肉体の顔を見たことのない人たちのため、私はどんなに大きな争いをしているかを、知って欲しいからです。  
<span class="verse-number">2&ensp;</span>それは、その人たちの心が励まされ、愛で結ばれ、理解力の確信の全幅的な豊かさを得、そして**神**、すなわち**御父**とキリストの二人の奥義の知識に到達するためです。  
<span class="verse-number">3&ensp;</span>知恵と知識のすべての宝は、キリストの中に隠されています。  
<span class="verse-number">4&ensp;</span>今、私がこのことを言うのは、何人も甘言で、あなたがたを惑わすことがないためです。  
<span class="verse-number">5&ensp;</span>なぜなら、私の体はそこにいないが、私の霊は、あなたがたと共にいて、あなたがたの秩序正しさ、それにキリストに対する信仰の不変なことを知り、私は喜んでいます。  
<span class="verse-number">6&ensp;</span>したがって、あなたがたは、**主**であられるキリスト・イエスを受け入れたように、彼にあって歩み、  
<span class="verse-number">7&ensp;</span>教えられたとおり、キリストに根を張り、**主**によって建てられ、信仰にあっては動かず、感謝をもって信仰に溢れていなさい。  
<span class="verse-number">8&ensp;</span>キリストによらない、人間の伝統だとか、この世の基本原則だとかによる、虚ろである欺瞞と哲学[^1]であなたがたを獲物にする者がいないかと、気をつけなさい。  
<span class="verse-number">9&ensp;</span>キリストには、**神性**の全てが形をもって、宿っておられるからです。  
<span class="verse-number">10&ensp;</span>そして、すべての権力と権威の頭であられるキリストのうちにあなたがたはいて、完成されています。  
<span class="verse-number">11&ensp;</span>キリストにあって、あなたがたも人の手によらない割礼を受けました。つまり、キリストの割礼によって、肉体的な罪の体を脱ぎ捨てることにより、  
<span class="verse-number">12&ensp;</span>あなたがたはキリストにあって浸礼により、共に葬られ、また、その浸礼により、キリストを、死人たちの中から復活させられた**神**の働きに対する信仰を通し、あなたがたはキリストと共に復活しました。  
<span class="verse-number">13&ensp;</span>そして、**主**はあなたがたのすべての違法行為を赦し、あなたがたの悪行と肉の割礼のないままで、死んでいるあなたがたを、**主**はキリストと共に生き返らせてくださいました。  
<span class="verse-number">14&ensp;</span>それは、私たちをさいなむ、手書きの数々の勅令、つまり、私たちには正反対なものをぬぐい去りました。そして、キリストは道を塞ぐものを取り去り、十字架に釘付けされました。  
<span class="verse-number">15&ensp;</span>権威ある者と力ある者を武装解除され、公のさらし者にし、この者たちを十字架によって凱旋なさいました。  
<span class="verse-number">16&ensp;</span>ですから、食物や飲物に関して、また祭りや新月や安息日に関して、誰にもあなたがたを裁かせてはいけません。  
<span class="verse-number">17&ensp;</span>これらは来るべきものの影であるが、実体はキリストに関わりのあるものです。  
<span class="verse-number">18&ensp;</span>謙虚なふりと、御使いたちへの礼拝にふけり、見てもいないものに攻め入り、自らの肉の心で膨らんでいる者に、あなたがたの報奨を騙し取られてはいけません。  
<span class="verse-number">19&ensp;</span>そういう者は、頭にしっかり結びついていません。頭は体全体に、関節と靭帯を通し栄養を送ります。そして**神**に育てられて成長するためです。  
<span class="verse-number">20&ensp;</span>従って、もしあなたがたが、世の基本原則から離れて、キリストと共に死んだのなら、なぜこの世に生きているかのように、自らを世の規則に従うのですか。  
<span class="verse-number">21&ensp;</span>つまり、「触れるな、味見するな、掴むな。」  
<span class="verse-number">22&ensp;</span>これらはすべて、用いれば消滅するものです。それは、人々の命令や教理によるのです。  
<span class="verse-number">23&ensp;</span>これらはまったくのところ、自らつくった宗教とへりくだり、肉体の苦行、これら三つは知恵の様相を呈しているが、肉体の満足に関しては、逆効果です。  

## 第3章

それで、もしあなたがたはキリストと共に復活させられたなら、天にあるものを求めなさい。そこに、キリストが**神**の右手に座っておられます。  
<span class="verse-number">2&ensp;</span>地上にあるものではなく、天にあるものに心を留めなさい。  
<span class="verse-number">3&ensp;</span>あなたがたはすでに死に、あなたがたの命は、**神**におられるキリストと共に隠されているからです。  
<span class="verse-number">4&ensp;</span>私たちの命であられるキリストがお現われになると、あなたがたもキリストと共に、栄光のうちに現われます。  
<span class="verse-number">5&ensp;</span>従って、あなたがたの地上にある各部、つまり性的な罪、汚れ、情欲、悪欲、偶像崇拝である強欲は殺しなさい。  
<span class="verse-number">6&ensp;</span>これらのことが理由で、**神**の怒りは不従順の子たちの上に来ます。  
<span class="verse-number">7&ensp;</span>あなたがたも以前、これらのことによって生活していた時、これらのことによって歩んでいました。  
<span class="verse-number">8&ensp;</span>しかし今、あなたがた自身、これらすべてのこと、つまり怒り、憤り、悪意、そしり、あなたがたの口から出る、卑猥な言葉は脱ぎ捨てなさい。  
<span class="verse-number">9&ensp;</span>互いに嘘を言ってはいけません。なぜなら、あなたがたは古い人をその行為と共に脱ぎ、  
<span class="verse-number">10&ensp;</span>新しい人を着たからです。新しい人を創造された方の姿形に似せて、その方の知識により、新たにされたのです。  
<span class="verse-number">11&ensp;</span>そこには、  
<span class="verse-number">12&ensp;</span>従って、**神**に選ばれた聖なる、愛されている人たちとして、憐れみの心、思いやり、へりくだり、柔和、そして忍耐を身にまといなさい。  
<span class="verse-number">13&ensp;</span>互いに堪え忍び、互いに赦し合いなさい。もし誰であれ誰かに不満があるなら、キリストがあなたがたを赦してくださったように、あなたがたも互いに赦さなければなりません。  
<span class="verse-number">14&ensp;</span>しかし、これらすべてのことの上に、完全の結びである愛をまといなさい。  
<span class="verse-number">15&ensp;</span>そして、あなたがたの心は、**神**の平安の支配に任せなさい。そのためにも、あなたがた全員は、一体として召されたのです。そして、感謝をする人でいなさい。  
<span class="verse-number">16&ensp;</span>キリストの御言葉を、あなたがたの中に豊かに住まわせ、あらゆる知恵で、互いに教え諭し、詩と賛美と霊の歌を、心から恵みによって**主**に歌いなさい。  
<span class="verse-number">17&ensp;</span>そして、あなたがたのすることは、言葉によろうが行為によろうが、すべて、**主**イエスの御名によって行ない、イエスを通して**父**なる**神**に感謝しなさい。  
<span class="verse-number">18&ensp;</span>妻たちよ、**主**にある者として、ふさわしく、夫に従いなさい。  
<span class="verse-number">19&ensp;</span>夫たちよ、妻たちを愛し、そして、妻たちに対して苦々しい思いをもってはいけません。  
<span class="verse-number">20&ensp;</span>子どもたちよ、すべてのことについて両親に従いなさい。なぜなら、**主**はこのことを、お喜びになるからです。  
<span class="verse-number">21&ensp;</span>父たちよ、子どもたちを苛立たせてはいけません。自信を喪失させないためです。  
<span class="verse-number">22&ensp;</span>僕たちよ、陰日向なく、迎合せず、真摯の心で**神**を恐れ、すべてについて、肉の主人たちに従いなさい。  
<span class="verse-number">23&ensp;</span>そして何をしようが、人にするのではなく、**主**にするように、心を込めてしなさい。  
<span class="verse-number">24&ensp;</span>**主**から、相続の報酬を受けることを知っています。あなたがたは**主**キリストに仕えているからです。  
<span class="verse-number">25&ensp;</span>しかし、不正を行なう者には、その者の行なうことに、仮借ない報復があります。そして、それに差別の扱いはありません。  

## 第4章

主人たちよ、あなたがたにも天に**主人**がおられることを心得て、僕たちを正しく、そして、平等にものを与えなさい。  
<span class="verse-number">2&ensp;</span>祈りの時は、感謝をもって熱心に、警戒して、祈り続けなさい。  
<span class="verse-number">3&ensp;</span>それと同時に、御言葉を説くため、すなわち、キリストの奥義を話すため、**神**は私たちのために、御言葉の扉を開けてくださるように、私たちのために祈ってください。私もその奥義のために鎖につながれているのです。  
<span class="verse-number">4&ensp;</span>私が話さなければならない通り、明確に奥義を話せるよう、祈ってください。  
<span class="verse-number">5&ensp;</span>外部の人たちに対しては、知恵で歩み、時間を正しく使いなさい。  
<span class="verse-number">6&ensp;</span>あなたがたの言葉はいつも優雅さがあり、塩で味付けしなさい。一人一人にどう答えるべきかを知るためです。  
<span class="verse-number">7&ensp;</span>愛する兄弟、忠実な奉仕者、**主**にあって同労の僕であるテキコが、私についてのすべての情報をあなたがたに伝えます。  
<span class="verse-number">8&ensp;</span>私がテキコをあなたがたに遣わしましたのは、テキコがあなたがたの状態を知り、あなたがたの心を慰めるためです。正に、これが目的です。  
<span class="verse-number">9&ensp;</span>あなたがたの一人であり、忠実で愛する兄弟、オネシモを、共に遣わしました。彼らはここで起こっていることの一部始終を、あなたがたに明らかにします。  
<span class="verse-number">10&ensp;</span>私共々、囚人であるアリスタルコが、バルナバの従兄弟であるマルコと共に、あなたがたに挨拶を送ります。あなたがたはマルコについて指示を受けているのですから、あなたがたのところに行ったら、歓迎してください。  
<span class="verse-number">11&ensp;</span>そして、ユストと呼ばれているイエスも挨拶を送ります。割礼を受けている人の中、**神**の王国のために働く同労者は、この三人だけです。このアリスタルコ、マルコ、ユストは私には慰めなのです。  
<span class="verse-number">12&ensp;</span>あなたがたの仲間の一人で、キリストの僕であるエパフラスが、あなたがたに挨拶を送ります。あなたがたが**神**のご意志のすべてに、完全な者として立ち、全うされるため、エパフラスはいつも祈りによってあなたがたのため、熱心に働いています。  
<span class="verse-number">13&ensp;</span>なぜなら、あなたがたと、ラオデキアにいる人たち、そして、ヒエラポリスにいる人たちに対して、エパフラスは大変熱心なことを、私は証します。  
<span class="verse-number">14&ensp;</span>愛する医者のルカ、そしてデマスはあなたがたに挨拶を送ります。  
<span class="verse-number">15&ensp;</span>ラオデキヤにいる兄弟たち、そして、ヌンパと彼の家にある教会に、挨拶してください。  
<span class="verse-number">16&ensp;</span>そして、この手紙はあなたがたの間で読んだなら、ラオデキア人の教会にも読ませなさい。そして、あなたがたは、ラオデキアからの手紙を、読みなさい。  
<span class="verse-number">17&ensp;</span>そして、「**主**にあって与えられた奉仕を、全うするため、留意せよ。」と、アルキポに言いなさい。  
<span class="verse-number">18&ensp;</span>この挨拶状は、私パウロ自らの手によるものです。私の鎖を忘れないでください。恵みがあなたがたと共にありますように。アーメン。  


[^1]: ギリシャのストア派、スパルト派。
