# テモテ第一

### テモテへの手紙　第一

## 第1章

私たちの**救い主**である**神**と、私たちの希望である**主**イエス・キリストの命令により、イエス・キリストの使徒であるパウロより、  
<span class="verse-number">2&ensp;</span>信仰にあって真の息子であるテモテへ。私たちの**父**なる**神**と私たちの**主**であるキリスト・イエスより、恵み、哀れみ、平和がありますように。  
<span class="verse-number">3&ensp;</span>私がマケドニアに入っていた時、あの者たちが、他の教理を教えないよう、命じることができるように、エペソに残るようあなたに勧めました。  
<span class="verse-number">4&ensp;</span>そして、たわごとや、果てしなく続く系図の話しに耳を傾けてはいけないと勧めました。それらは、信仰による敬虔な教えを構築するのではなく、むしろ問題を持ち込むからです。  
<span class="verse-number">5&ensp;</span>さて、この命令の目的は愛です。清い心から、またよい良心から、偽りのない信仰から出る愛です。  
<span class="verse-number">6&ensp;</span>ある者たちはこれらのことから道を失い、空しい話に向きを変えてしまいました。  
<span class="verse-number">7&ensp;</span>彼らは己の言うことも、己が主張することをも理解せずに、律法の教師になりたがっています。  
<span class="verse-number">8&ensp;</span>しかし、もし人が合法的に律法を使うなら、律法は良いものであると、私たちはわかっています。  
<span class="verse-number">9&ensp;</span>律法は正しい人のためではなく、不法者らと不従順な者たちのため、つまり、**神**に敵対する者たちと罪人たちのため、邪悪な者と俗悪な者、父を殺す者たち、そして母を殺す者たち、人殺したちのために作られたことを知っています。  
<span class="verse-number">10&ensp;</span>また、性的な罪を犯す者たち、男色をする者たち、誘拐犯人たち、嘘つきたち、偽証者たち、その他、健全な教理に反対する者のため、律法が作られたと知っています。  
<span class="verse-number">11&ensp;</span>これは、私に任された、祝福された**神**の栄光ある福音による教理です。  
<span class="verse-number">12&ensp;</span>そして、私を能力のある者とされた、**主**キリスト・イエスに私は感謝します。忠実な者として認め、私を奉仕の場に置かれたからです。  
<span class="verse-number">13&ensp;</span>以前私は**神**を冒涜する者、迫害する者、暴言をはく者でしたが、知らずに、信じていない時行なったので、哀れみを得ました。  
<span class="verse-number">14&ensp;</span>そして、私たちの**主**の恵み、またキリスト・イエスと共にある信仰と愛は、途方もなく豊かになりました。  
<span class="verse-number">15&ensp;</span>キリスト・イエスは罪人たちを救うためにこの世に来られたとは、まことの言葉であり、あらゆる面で受け入れるのに価する言葉です。私はその罪人たちの先頭にいます。  
<span class="verse-number">16&ensp;</span>しかし、私がこの理由で哀れみを得たのは、イエス・キリストがあらゆる忍耐を先ず私に示し、永遠の命のためにキリストを信じようとする者たちのため、私を型とするためです。  
<span class="verse-number">17&ensp;</span>では、永遠の**王**に、不死の方、目に見えない方、知恵ある**神**に、永遠から永遠まで栄誉と栄光がありますように。アーメン。  
<span class="verse-number">18&ensp;</span>我が息子テモテよ、前もってあなたに関する預言により、私はこの任を任せます。それはこの預言によりあなたが有利な戦いをするためです。  
<span class="verse-number">19&ensp;</span>信仰とよい良心を持ち続けなさい。ある者たちは信仰とよい良心を拒み、信仰の船を難破させました。  
<span class="verse-number">20&ensp;</span>難破した中には、ヒメナオとアレキサンデルがいます。冒涜をしないことを学ぶように、私は彼らをサタンに引き渡しました。  

## 第2章

従って、何よりもまず先に、すべての人たちのために祈り、願い、とりなし、そして感謝をするように、私は強く勧めます。  
<span class="verse-number">2&ensp;</span>王たち、そしてすべて権威ある人たちのために、私たちがすべての敬虔と真摯で、穏やかで平和の生活をおくるためです。  
<span class="verse-number">3&ensp;</span>なぜなら、これは善いことであり、私たちの**救い主**である**神**の御前に、認められるからです。  
<span class="verse-number">4&ensp;</span>**神**はすべての人は救われ、そして、真理の知識に至ることを望んでおられます。  
<span class="verse-number">5&ensp;</span>なぜなら、**神**は唯一であり、そして、唯一の仲介者、つまり人としてのキリスト・イエスが**神**と人間との間におられます。  
<span class="verse-number">6&ensp;</span>定められた時に証されるように、キリストはすべての人のため、身代金としてご自分を与えられました。  
<span class="verse-number">7&ensp;</span>私はそのために説教者、使徒に指名されました。キリストにあって、私は嘘を言っていません。真理を言っています。また信仰と真理にある異邦人の教師に指名されました。  
<span class="verse-number">8&ensp;</span>ですから、罪を犯していない手を上げ、怒りや疑いを持たず、いかなる所でも男たちは祈ってほしいです。  
<span class="verse-number">9&ensp;</span>同様に女たちもたしなみと節度を保ち、編んだ髪、金、真珠、高価な衣服で身を飾ったりせず、つつましい衣服で自らを飾ってほしいです。  
<span class="verse-number">10&ensp;</span>むしろ**神**を敬うと言い表す女にふさわしく、善い行為で飾ってほしいです。  
<span class="verse-number">11&ensp;</span>女は静かに、すべて従順に学ぶべきです。  
<span class="verse-number">12&ensp;</span>女が男を教えたり、男の上に権威を持つことを私は許さず、女は黙っていることを許します。  
<span class="verse-number">13&ensp;</span>アダムが最初に造られ、次いでエバが造られたからです。  
<span class="verse-number">14&ensp;</span>また、アダムは騙されなかったが、その女は騙され、あやまちを犯しました。  
<span class="verse-number">15&ensp;</span>しかし、自制を持って、信仰、愛、そして聖なる内に留まり続ければ、女たちは子を生むことを通し救われます。  

## 第3章

「監督の地位を求めるなら、その者は、良い働きを求めている。」これは真実な言葉です。  
<span class="verse-number">2&ensp;</span>監督は非難されるところなく、一人の妻の夫であり、自らを制し、真摯な態度で、品位があり、客をよくもてなし、教える才能があり、  
<span class="verse-number">3&ensp;</span>葡萄液を好まず、暴力を好まず、金銭に執着せず、かえってやさしく、争いを好まず、貪欲でなく、  
<span class="verse-number">4&ensp;</span>自らの家庭を良く治め、子の尊敬を一身に受け、従わせている人であるべきです。  
<span class="verse-number">5&ensp;</span>（自らの家庭を治める方法のわからない人が、いったいどうやって、**神**の教会の世話をするのでしょうか。）  
<span class="verse-number">6&ensp;</span>プライドで膨らみ、悪魔と同じさばきに陥らないように、初心者であってはいけません。  
<span class="verse-number">7&ensp;</span>くわえて、監督は外の人たちの間で、良い証がなければなりません。非難に陥らず、また悪魔の罠にも陥らないためです。  
<span class="verse-number">8&ensp;</span>同じように、執事も尊厳があり、二枚舌を使わず、多量の葡萄液を好まず、金銭に貪欲でなく、  
<span class="verse-number">9&ensp;</span>清い良心に、信仰の奥義を持ち続けます。  
<span class="verse-number">10&ensp;</span>しかし、この人たちをまず細かく吟味し、非難されるところがなければ、執事として務めさせなさい。  
<span class="verse-number">11&ensp;</span>その妻たちも同様に尊厳があり、悪口を言う人ではなく、自らを制し、すべてのことにおいて誠実であるべきです。  
<span class="verse-number">12&ensp;</span>執事たちは一人の妻の夫であり、自らの子どもたちと家庭を良く治める者であるべきです。  
<span class="verse-number">13&ensp;</span>執事たちとして良く仕えて来た者たちは、己によい地位とキリスト・イエスにある信仰で強い自信を獲得するからです。  
<span class="verse-number">14&ensp;</span>近いうちにあなたのところへ行きたいので、私はこれらのことをあなたに書きます。  
<span class="verse-number">15&ensp;</span>しかし、もし遅くなれば、**神**の家で、すなわち生きる**神**の教会で、真理の柱と基礎である教会で、どうふるまうべきかをあなたが分かるように、私は書いています。  
<span class="verse-number">16&ensp;</span>そして、誰でも認めるように、敬虔の奥義は偉大です。**神**は肉において現れ、**御霊**によって義と宣言され、御使いたちに目撃され、異邦人たちの間に説かれ、世に信じられ、栄光のうちに上げられました。  

## 第4章

さて、**御霊**ははっきりと言っておられます。後の時代に、ある者たちはこの信仰から離れ、惑わす霊たちや悪霊たちの教えに気を取られ、  
<span class="verse-number">2&ensp;</span>偽善によってうそを言い、その者たちは自らの良心を熱い鉄で焼かれ、  
<span class="verse-number">3&ensp;</span>結婚を禁じるのです。そして、真理を知り、信じる者たちが、感謝を持って受けるべき、**神**により作られた食べ物を、その者たちは遠ざけるよう命じています。  
<span class="verse-number">4&ensp;</span>なぜなら、**神**の作られたすべての物は良い物であり、感謝をもって受け入れるなら、投げ捨てる物は何もありません。  
<span class="verse-number">5&ensp;</span>それは、**神**の御言葉と祈りによって聖別されているからです。  
<span class="verse-number">6&ensp;</span>これらのことを兄弟たちに定着させつつ、信仰の言葉をあなたが心して、従っていた良い教えの栄養で育った、イエス・キリストの良い奉仕者になります。  
<span class="verse-number">7&ensp;</span>しかし、俗悪なたわごと、それと老女の作り話は拒否し、敬虔に向け自らを鍛えなさい。  
<span class="verse-number">8&ensp;</span>なぜなら、体の鍛練は少々の益はあるが、約束された今の命と後で来る命のため、敬虔はすべてに有益です。  
<span class="verse-number">9&ensp;</span>これは真実な言葉であり、すべて受け入れるに値する言葉です。  
<span class="verse-number">10&ensp;</span>すべての人々、特に信じる人々の**救い主**である生ける**神**を、私たちは信頼しているから、それで、私たちはそのために労働し、侮辱されています。  
<span class="verse-number">11&ensp;</span>これらのことを命じ、教えなさい。  
<span class="verse-number">12&ensp;</span>誰にでもあなたの若さを軽んじさせてはいけません。かえって、言葉にも、行為にも、愛にも、霊にも、信仰にも、清さにも、信者たちの模範になりなさい。  
<span class="verse-number">13&ensp;</span>私が行くまで、読むこと、勧めること、教えることに専念しなさい。  
<span class="verse-number">14&ensp;</span>預言を通して、長老たちの按手によってあなたに与えられた賜物をなおざりにしてはいけません。  
<span class="verse-number">15&ensp;</span>あなたの成長がすべての人に明らかになるよう、これらのことを深く考え、全身全霊で当たりなさい。  
<span class="verse-number">16&ensp;</span>あなた自身、そして教理に心を配りなさい。これらのことに留まっていなさい。そうすれば、あなたは、あなた自身をも、あなたの言葉を聞く人たちをも救います。  

## 第5章

老人を叱責してはいけません。むしろ父として勧め、年下の男には兄弟として勧めなさい。  
<span class="verse-number">2&ensp;</span>年上の女性は母として、年下の女性は姉妹として、すべての清さを持って勧めなさい。  
<span class="verse-number">3&ensp;</span>本当の未亡人には礼をもって接しなさい。  
<span class="verse-number">4&ensp;</span>しかし、子ども、または、孫がいれば、どんな未亡人であれ、まずその子どもたちに家庭の中で**神**を敬うことを、また親に報いることを学ばせなさい。これは**神**の御前に良いことであり、**神**が受け入れてくださるからです。  
<span class="verse-number">5&ensp;</span>しかし、一人残された真の未亡人は、**神**に信頼を置き、夜も昼も**神**に願いと祈りを続けています。  
<span class="verse-number">6&ensp;</span>しかし、快楽を追い求める女は、生きながら死んでいます。  
<span class="verse-number">7&ensp;</span>それで、未亡人たちが非難されることのないように、これらのことを命じなさい。  
<span class="verse-number">8&ensp;</span>しかし、誰であれ、家族の者たちに、特に自らの家の者たちを顧みなければ、その者は信仰を否定したのであり、不信者より悪いです。  
<span class="verse-number">9&ensp;</span>六十歳以下の未亡人は名簿に載せてはいけません。載せるのは、一人の男の妻であった人で、  
<span class="verse-number">10&ensp;</span>行ないの評判のよい者でなければなりません。すなわち、子どもたちを育てあげたか、旅人を宿泊させたか、聖徒たちの足を洗ったか、苦しんでいる人たちを助けたか、数々の良い行為を苦労して行なった者であるかです。  
<span class="verse-number">11&ensp;</span>しかし、若い未亡人たちはキリストに背き、好色になると、結婚を望むので拒否しなさい。  
<span class="verse-number">12&ensp;</span>この若い未亡人たちは最初の信仰を捨てたのだから、裁かれます。  
<span class="verse-number">13&ensp;</span>おまけに、彼女たちは怠ける癖が身につき、家から家へと歩き回ります。そして、怠けるだけではなく、うわさ話を言う者になり、他人のことに口を出す者になり、言ってはいけないことを言います。  
<span class="verse-number">14&ensp;</span>そのため、若い未亡人たちは結婚し、子どもを産み、家庭を管理し、敵対する者に非難する余地を与えることのないように私は望みます。  
<span class="verse-number">15&ensp;</span>なぜなら、すでに、未亡人の中のある者たちは、サタンのほうへ向きを変えてしまっています。  
<span class="verse-number">16&ensp;</span>信じる男か女に、未亡人のいる者がいれば、その者に援助させなさい。教会が真の未亡人たちを助けることができるよう、教会の重荷になってはいけません。  
<span class="verse-number">17&ensp;</span>良く指導する長老たち、特に御言葉と教理で苦労する長老たちは、倍の尊敬に値する人であると思いなさい。  
<span class="verse-number">18&ensp;</span>「穀物を踏んで脱穀している雄牛に、口に袋をかけてはいけない」、また、「働き人はその者の賃金にふさわしい」、と聖書は言っているからです。  
<span class="verse-number">19&ensp;</span>二人、または三人の証人からではなければ、長老に対する訴えを受理してはいけません。  
<span class="verse-number">20&ensp;</span>他の者たちも恐れるよう、罪を犯している者たちを全員の前でとがめなさい。  
<span class="verse-number">21&ensp;</span>**神**と**主**であるイエス・キリスト、また選ばれた御使いたちの前で、私はテモテにおごそかに命じます。これらのことを偏見なく行ないなさい。不公平に行なってはいけません。  
<span class="verse-number">22&ensp;</span>誰とでも即、按手をしてはいけません。また、他人の罪を共に犯してはいけません。自らを清く保ちなさい。  
<span class="verse-number">23&ensp;</span>これからは水だけ飲まず、胃のため、よく病気になるのだから、葡萄液を少し用いなさい。  
<span class="verse-number">24&ensp;</span>ある者たちの罪は明らかで、先に裁きに行くが、その後から行く者もいます。  
<span class="verse-number">25&ensp;</span>同様に、きわめて目立つ善行もあるが、他の善行も隠れることはあり得ません。  

## 第6章

くびきの下にいる奴隷たちは、**神**の御名とその教理が冒涜されないよう、彼ら自身の主人に礼のすべてをつくすべきです。  
<span class="verse-number">2&ensp;</span>そして、信者の主人を持つ奴隷たちは、主人たちは兄弟だからと言って、見下げてはいけません。むしろ、益を受ける者たちは信者であり、愛されているのだからますますよく仕えなさい。これらのことを教え、勧めなさい。  
<span class="verse-number">3&ensp;</span>もし、異なることを教え、健全な言葉、すなわち私たちの**主**イエス・キリストの御言葉と、敬虔と一致する教理に同意しない者がいれば、誰であれ、  
<span class="verse-number">4&ensp;</span>その者は傲慢であり、何も知らず、そして、言葉についての討論と論争にとりつかれています。その結果、ねたみ、争い、ののしり、悪意の疑い、  
<span class="verse-number">5&ensp;</span>心が腐り、心理を失った者たちの無益な論争です。彼らは利益を敬虔と考えています。こういう者たちから身を引きなさい。  
<span class="verse-number">6&ensp;</span>しかし、満足を伴う敬虔は、大きな利益です。  
<span class="verse-number">7&ensp;</span>なぜなら、私たちはこの世界に、何も持たずやってきて、そして、何も持ち出すことができないのは間違いありません。  
<span class="verse-number">8&ensp;</span>そして、食べ物と着る物があれば、それで私たちは満足すべきです。  
<span class="verse-number">9&ensp;</span>しかし、金持ちになりたがる者たちは、誘惑と罠に陥ります。また、人たちを、崩壊と破滅の中で溺死させる、多くの愚かな有害な欲望にも陥ります。  
<span class="verse-number">10&ensp;</span>金銭を愛することは、諸悪の根源だからです。ある者たちは金銭を貪欲に求め、信仰からはずれ、数々の悲しみで自らを突き刺しました。  
<span class="verse-number">11&ensp;</span>しかし、**神**の男であるあなたよ、これらのことを避け、そして義、敬虔、信仰、愛、忍耐、柔和を追い求めなさい。  
<span class="verse-number">12&ensp;</span>信仰のよい戦いを戦い、永遠の命をしっかり握りしめなさい。このためにあなたも召され、多くの証人たちの前でよい告白を言い表したのです。  
<span class="verse-number">13&ensp;</span>すべてのものに命を与える**神**の前で、そして、ポンテオ・ピラトの前でよい告白を証したキリスト・イエスの前で、私はあなたに命じます。  
<span class="verse-number">14&ensp;</span>それは、私たちの**主**イエス・キリストの出現まで、無傷で、非の打ちどころのないままで、この命令を守りなさい。  
<span class="verse-number">15&ensp;</span>ご自分の時代に、王の**王**であり、主の**主**である祝福された唯一の**主権者**が、ご自分の出現を明らかになさいます。  
<span class="verse-number">16&ensp;</span>**主**のみが不死であり、近寄れない光の中に住み、いかなる者の目に触れず、またいかなる者も目にできない方です。その方に対して畏敬の念を、そして永遠の力がありますように。アーメン。  
<span class="verse-number">17&ensp;</span>この世の金持ちたちに命じなさい。高慢にならないよう、不確かな富に希望を置かず、楽しむため、豊かにすべてのものを私たちに与えてくださる、生ける**神**に希望を置きなさい。  
<span class="verse-number">18&ensp;</span>また、善を行ない、十分に善い行ないをし、喜んで与え、ためらわずに分かち合うように、金持ちたちに命じなさい。  
<span class="verse-number">19&ensp;</span>そうして、永遠の命をしっかり握りしめることができるように、やがて来るその時のため、彼ら自身のために良い土台を準備させなさい。
