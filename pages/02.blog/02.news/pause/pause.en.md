There have been several changes in my personal life recently that have kept me busy. Because of this, I will be taking a break over the next few months from updating this website with new content.

===

When this website was launched almost a year ago, I was learning new technologies and testing out software that I had never used before. It's now time to step back, review what I've learned, reconsider these systems, and architect a strategy and branding for the long-term.

Lord willing, the plans for the website this year are:

- A new name for the website, followed eventually by a new logo
- Evaluating backend technologies for the website, data storage, and distribution
- Using new software for audio recording and editing
- Publishing digital formats of the Bible
- Working with others to provide additional library content

Please pray for this work, as it is currently a one-man job. There are many things to do and only so many hours in a day.
