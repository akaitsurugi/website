# ピレモン

### ピレモンへの手紙

キリスト・イエスの囚人であるパウロと、私たちの兄弟テモテより、私たちの同労者である、愛するピレモンへ。  
<span class="verse-number">2&ensp;</span>また、愛するアピヤ、そして私たちの戦友アルキポ、そしてあなたの家にある教会へ、  
<span class="verse-number">3&ensp;</span>私たちの**父**なる**神**と**主**イエス・キリストから、あなたがたに恵みと平安がありますように。  
<span class="verse-number">4&ensp;</span>私は祈りで、常にあなたのことに言及し、私の**神**に感謝しています。  
<span class="verse-number">5&ensp;</span>**主**イエスに対して、またすべての聖徒たちに対して、あなたの持っている愛と信仰を聞き及んでいます。  
<span class="verse-number">6&ensp;</span>それは、あなたの内にあるすべてのよい行ないが知られることで、キリスト・イエスに向けての、あなたと共有している信仰が、力あるものとなるためです。  
<span class="verse-number">7&ensp;</span>私たちはあなたの愛に、大きな喜びと慰めを得ているのは、兄弟よ、聖徒たちの心は、あなたを通して鼓舞されているからです。  
<span class="verse-number">8&ensp;</span>ゆえに、ふさわしいことは何でも、キリストにあって、あなたに大いに大胆に命じることができるのが、  
<span class="verse-number">9&ensp;</span>愛のため、このように老齢なパウロ、そして今イエス・キリストの囚人ですが、あえて、お願いします。  
<span class="verse-number">10&ensp;</span>私は、鎖につながれた状態で、生んだ子、オネシモのために、お願いします。  
<span class="verse-number">11&ensp;</span>オネシモは以前、あなたにとって役立たずでしたが、今はあなたにも、私にも役に立っています。  
<span class="verse-number">12&ensp;</span>私はオネシモを送り返します。ですから、あなたはオネシモ、私の心であるこの人を受け入れてください。  
<span class="verse-number">13&ensp;</span>福音のため、鎖につながれた私の世話を、あなたに代わってオネシモにさせるため、私はオネシモを手元に置きたかったのです。  
<span class="verse-number">14&ensp;</span>しかし、あなたの許可なしで、私は何事もしたくありませんでした。それは、あなたの良い行ないが、義務からでなく、自発的である、必要があったのです。  
<span class="verse-number">15&ensp;</span>なぜなら、あなたがオネシモを永遠に受け入れるために、オネシモは多分しばらくの間、そのために出て行きました。  
<span class="verse-number">16&ensp;</span>もう奴隷としてではなく、奴隷を超越する、愛する兄弟として、特に私にとってはそうですが、あなたにとってはなおのこと、肉にあっても**主**にあっても、オネシモを受け入れてください。  
<span class="verse-number">17&ensp;</span>ですから、私を仲間として思っているのなら、オネシモを私だと思って、受け入れてください。  
<span class="verse-number">18&ensp;</span>しかし、もしオネシモがあなたに負い目があるなら、また、負債があるなら、それは私の帳簿に記載してください。  
<span class="verse-number">19&ensp;</span>私、パウロは、自分の手で書いています。私は支払いします。あなた自身が負債であることは、触れないでおきます。  
<span class="verse-number">20&ensp;</span>そうです、兄弟よ、**主**にあるあなたからの喜びを、私にください。**主**にあって私の心を鼓舞してください。  
<span class="verse-number">21&ensp;</span>私が言ったこと以上のことを、あなたはしてくれると知っています。それで、あなたが実行してくれると確信し、これを書きました。  
<span class="verse-number">22&ensp;</span>それで、私のために、宿舎を用意してください。あなたの祈りを通して、私があなたの所に、行かせて頂けると希望しているからです。  
<span class="verse-number">23&ensp;</span>キリスト・イエスにあって、私と共に囚人であるエパフラスが、あなたに挨拶を送ります。  
<span class="verse-number">24&ensp;</span>私の同労者、マルコとアリスタルコとデマス、ルカも挨拶を送ります。  
<span class="verse-number">25&ensp;</span>私たちの**主**イエス・キリストの御恵みが、あなたがたの霊と共にありますように。アーメン。
