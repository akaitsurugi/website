# ヨハネ第一

### ヨハネの手紙　第一

## 第1章

それは最初からのものでした。私たちが御言葉を聞いたものでした。私たちが私たちの目で見たものでした。私たちがしっかり見たものでした。私たちの手で触ったものでした。つまり、命の御言葉でした。  
<span class="verse-number">2&ensp;</span>そして、この命は明らかにされ、そして、私たちはそれを見て、証をし、また、**御父**と共にあった、この永遠の命を私たちに示されました。  
<span class="verse-number">3&ensp;</span>私たちが見たこと、聞いたことをあなたがたに伝えます。それはあなたがたが私たちと交流できるためであり、この私たちの交流は**御父**との交流であり、またあの**御子息**イエス・キリストの交流でもあります。  
<span class="verse-number">4&ensp;</span>そして、私たちがこれらのことをあなたがたに書くのは、あなたがたの喜びが満ちるためです。  
<span class="verse-number">5&ensp;</span>そして、私はあの方から聞き、またあなたがたに伝えるメッセージは、これです。すなわち、**神**は光であり、**神**には暗闇はまったくありません。  
<span class="verse-number">6&ensp;</span>もし、私たちがあの方と交流があると言いながら、暗闇の中を歩んでいるなら、私たちは嘘つきであり、真理を行なっていません。  
<span class="verse-number">7&ensp;</span>しかし、あの方が光の中におられるように、私たちも光の中を歩んでいれば、私たちは互いに交流があり、そして、あの方の**御子息**であるイエス・キリストの血は、私たちをすべての罪から清めます。  
<span class="verse-number">8&ensp;</span>もし、「私たちには、罪はない。」と言うならば、私たちは自分自身をだましています。また、私たちの中に真理がありません。  
<span class="verse-number">9&ensp;</span>私たちは自らの罪を言い表せば、あの方は、誠実で正しい方ですから私たちの罪を赦し、私たちをすべての不義から清めてくださいます。  
<span class="verse-number">10&ensp;</span>私たちは、「罪を犯したことがない。」と言えば、私たちはあの方を嘘つきとし、あの方の御言葉は、私たちのうちにはありません。  

## 第2章

我が子たちよ、あなたがたが罪を犯さないように、私はこれらのことを書きます。そして、もし誰であれ罪を犯せば、私たちには、唯一の**御父**と共にいる正しい方、つまり、イエス・キリストという助け主がおられます。  
<span class="verse-number">2&ensp;</span>そして、その方ご自身は、私たちの罪のため、また私たちの罪のためだけではなく、全世界の罪のための唯一の供え物です。  
<span class="verse-number">3&ensp;</span>そして、もしこの方の命令を守れば、私たちはこの方を知っていると、これによってわかるのです。  
<span class="verse-number">4&ensp;</span>この方を知っていると言いながら、この方の命令を守らない者は、嘘つきであり、そして真理はその者の中にはありません。  
<span class="verse-number">5&ensp;</span>しかし、その方の御言葉を守るその者の中には、**神**の愛はまことに全うされています。私たちはその方のうちにいると、これによって知ります。  
<span class="verse-number">6&ensp;</span>この方にとどまっていると言う者は、この方が歩まれたと同様に、自分自身もそう歩むべきです。  
<span class="verse-number">7&ensp;</span>兄弟たちよ、あなたがたに新しい命令を私は書いているのでありません。あなたがたが最初から持っている古い命令です。古い命令とは、あなたがたが最初から耳にしていた御言葉なのです。  
<span class="verse-number">8&ensp;</span>また、私はあなたがたに新しい命令を書きます。それは、その方にあって、またあなたがたにあって真理です。なぜなら、暗闇は過ぎ去り、真実の光がもうすでに輝いているからです。  
<span class="verse-number">9&ensp;</span>光の中にいると言いながら、自分の兄弟を憎む者は、今でも暗闇の中にいるのです。  
<span class="verse-number">10&ensp;</span>自らの兄弟を愛する者は、この光の中にとどまっています。また、その者の中につまずきになるものはありません。  
<span class="verse-number">11&ensp;</span>しかし、自分の兄弟を憎む者は、暗闇の中にいます。また、その者は暗闇の中を歩んでいます。そして、その暗闇はその者の目を盲目にしたから、彼は自分がどこへ行くのかわかりません。  
<span class="verse-number">12&ensp;</span>我が子たちよ。私が書くのは、この方の御名のゆえに、あなたがたの罪はすでに赦されているからです。  
<span class="verse-number">13&ensp;</span>父親たちよ、あなたがたは、最初からおられる方を知っているので、あなたがたに書きます。若者たちよ、あなたがたは、悪者を打ち破ったので、あなたがたに書きます。幼い子どもたちよ、あなたがたは唯一の**御父**を知ったのだから、あなたがたに書きます。  
<span class="verse-number">14&ensp;</span>父親たちよ、あなたがたは、最初からおられる方を知っているので、私はあなたがたに書きました。若者たちよ、あなたがたは強く、また**神**の御言葉はあなたがたのうちにとどまり、またあなたがたは悪者を打ち破ったので、あなたがたに書きました。  
<span class="verse-number">15&ensp;</span>世を、また世にあるものを愛してはいけません。もし誰であれ世を愛すれば、唯一の**御父**の愛はその者の中にはありません。  
<span class="verse-number">16&ensp;</span>すべて世にあるもの、つまり肉の欲望、目の欲望、生活の思い上がりは、唯一の**御父**から来たものではなく、世から来たものだからです。  
<span class="verse-number">17&ensp;</span>そして、世とその欲望は消え去るが、**神**のご意志を行なう人は、永遠にとどまります。  
<span class="verse-number">18&ensp;</span>我が子たちよ、今が最後の時です。また、反キリストが来るとあなたがたが聞いたように、今、多くの反キリストがいます。そのため、今が最後の時であると私たちは分かります。  
<span class="verse-number">19&ensp;</span>彼らは、私たちの中から出て行ったが、私たちの仲間ではありませんでした。なぜなら、彼らが私たちの仲間であったなら、私たちとともにきっと残ったでしょう。しかし、彼ら全員、私たちの仲間ではないことが明らかになるように、出て行ったのです。  
<span class="verse-number">20&ensp;</span>そして、あなたがたは**聖い方**から、油を注がれた[^1]のだから、すべてのことを知っています。  
<span class="verse-number">21&ensp;</span>あなたがたは真理を知らないから、あなたがたに書いたのではなく、あなたがたは真理を知っており、また真理から出た嘘はないからです。  
<span class="verse-number">22&ensp;</span>イエスはキリストであることを否定する者以外、嘘つきは誰か。唯一の**御父**を、そして**御子息**を否定する者こそが、反キリストです。  
<span class="verse-number">23&ensp;</span>**御子息**を否定する者は誰であれ、唯一の**御父**が共にはおられません。**御子息**を言い表す人は、唯一の**御父**が共におられます。[^2]  
<span class="verse-number">24&ensp;</span>したがって、あなたがたが初めに聞いたことを、あなたがたの中にとどまらせなさい。初めに聞いたことをあなたがたの中にとどまらせば、あなたがたは唯一の**御父**に、そして**御子息**にもとどまっているのです。  
<span class="verse-number">25&ensp;</span>そして、これはこの方があなたがたに約束された約束なのです。つまり、永遠の命なのです。  
<span class="verse-number">26&ensp;</span>あなたがたを惑わす者たちについて、これらのことをあなたがたに書きました。  
<span class="verse-number">27&ensp;</span>しかし、あなたがたがこの方から頂いた、油を注がれたことは、あなたがたの中にとどまっています。そして、いかなる人もあなたがたを教える必要はありません。しかし、あの油を注がれたことが、あなたがたにすべてのことは、嘘でない、真理を教えます。また、ちょうど油を注がれたことが、あなたがたを教えたように、あなたがたはキリストの中にとどまります。  
<span class="verse-number">28&ensp;</span>そして今、我が子たちよ、**主**の中にとどまりなさい。それは**主**が現われる時、自信を持ち、**主**の再臨の時、恥じることなく、御前にいるためです。  
<span class="verse-number">29&ensp;</span>もしあなたがたが、この方は正しい方と知っているならば、正しいことを行なうすべての人は、この方から生まれたことは、あなたがたは分かっています。  

## 第3章

**御父**が私たちにくださった、見事な愛を見よ。これは私たちが**神**の子と呼ばれるためです。このため、世は**神**を知らないから、世は私たちのことを知りません。  
<span class="verse-number">2&ensp;</span>愛する人たちよ。私たちはすでに**神**の子どもです。そして、私たちはどうなるかは、まだ啓示されていません。しかし、キリストが現われると、私たちはキリストに似るようになることは分かっています。なぜなら、キリストをありのままの姿で見るからです。  
<span class="verse-number">3&ensp;</span>そして、この希望を自らの中に抱く者は誰であれ、キリスト同様の聖さに自らを聖めます。  
<span class="verse-number">4&ensp;</span>誰であれ罪を犯す者は、不法をも行なっています。そして、罪は律法を犯すことです。  
<span class="verse-number">5&ensp;</span>それで、あなたがたは、キリストが私たちの罪を取り除くために現われたことが知っています。そして、キリストの内に罪がありません。  
<span class="verse-number">6&ensp;</span>キリストの内にとどまる人は誰であれ、罪を犯し続けません。罪を犯し続ける者は誰であれ、まだキリストを見ていないか、知らないからです。  
<span class="verse-number">7&ensp;</span>子どもたちよ、誰もあなたがたを人に惑わされてはいけません。キリストは正しいのと同様、正しいことを行ない続ける人は、正しいのです。  
<span class="verse-number">8&ensp;</span>罪を行ない続ける者は、悪魔から出ています。悪魔は最初から罪を犯しているからです。悪魔のすることを無効にするために、**御子息**は現われたのです。  
<span class="verse-number">9&ensp;</span>**神**から生まれたすべての人は、罪を犯し続けません。キリストの種がその者の内にとどまっているからです。そして、**神**から生まれたので、罪を犯し続けません。  
<span class="verse-number">10&ensp;</span>これによって、**神**の子たちと悪魔の子たちは明らかになります。義を行なわないすべての者、また自分の兄弟を愛していない者は、**神**からの者ではありません。  
<span class="verse-number">11&ensp;</span>なぜなら、私たちは互いに愛すべきである、とのメッセージは、私たちは初めから聞いていたのです。  
<span class="verse-number">12&ensp;</span>悪人から出て、自分の弟を無慈悲に殺したカインのようになるな。では、彼はなぜ弟を無慈悲に殺しましたか。カインの行為が悪で、弟の行為が善であったからです。  
<span class="verse-number">13&ensp;</span>私の兄弟たちよ。もし世があなたがたを憎むとしても、驚いてはいけません。  
<span class="verse-number">14&ensp;</span>兄弟たちを愛しているので、私たちは死から命に移ってきたことを、私たちは知っています。兄弟を愛さない者は、死にとどまります。  
<span class="verse-number">15&ensp;</span>自分の兄弟を憎む者は誰であれ人殺しです。そして、人殺しは誰であれ自分のうちに、永遠の命がとどまっていないと、あなたがたは知っています。  
<span class="verse-number">16&ensp;</span>私たちは、これによって**神**の愛を知っています。つまり、イエスは私たちのために、ご自分の命を投げ出してくださいました。それで、私たちも、兄弟のために命を捨てるべきです。  
<span class="verse-number">17&ensp;</span>しかし、世の富を持ちながら、自分の兄弟が困窮するのを見ても、兄弟から自分の心を閉じてしまう者が、どうして**神**の愛がその者の中にとどまるでしょうか。  
<span class="verse-number">18&ensp;</span>私の子どもたちよ。私たちは、言葉や舌によって愛するのでなく、行為と真理によって愛しましょう。  
<span class="verse-number">19&ensp;</span>そして、これによって私たちは真理から出たことがわかり、そして、**神**の御前に私たちの心を私たちは穏やかにします。  
<span class="verse-number">20&ensp;</span>それは、もし私たちの心が、私たちを責めれば、**神**は私たちの心より偉大であり、そして**神**はすべてをご存じだからです。  
<span class="verse-number">21&ensp;</span>愛する人たちよ。もし心にやましいところがなければ、**神**に対して率直でいれます。  
<span class="verse-number">22&ensp;</span>そして、願うなら私たちは何なりと、**神**から頂きます。**神**の命令を守り、**神**の目に喜ばれることを行なっているからです。  
<span class="verse-number">23&ensp;</span>そして、これこそが**神**の命令です。すなわち、私たちは**神の御子息**イエス・キリストの御名を信じ、**神**がくださった命令に従って、互いに愛し合うことです。  
<span class="verse-number">24&ensp;</span>また、**神**の命令を守り続ける人は**神**の内にとどまり、そして、**神**はその人の内にとどまり続けてくださいます。また、**神**が私たちにくださる**御霊**によって、**神**が私たちの中にとどまり続けていることがわかります。  

## 第4章

愛する者たちよ。多くの偽預言者が世に出ているから、すべての霊を信ぜず、その霊が**神**から出たかどうかを知るため、確かめなさい。  
<span class="verse-number">2&ensp;</span>イエス・キリストは肉体で来られた、と認める霊はすべて**神**から出ています。これによって、あなたがたはその霊は**神**の**御霊**と分かります。  
<span class="verse-number">3&ensp;</span>そして、イエス・キリストは、肉体で来られたと認めない者は、**神**から出ていません。また、来るとあなたがたが聞いたことのある反キリストの霊が正にこれであり、そしてそれはもうすでに世に来ています。  
<span class="verse-number">4&ensp;</span>子どもたちよ。あなたがたは、**神**から出た者であり、その霊たちに打ち勝っています。あなたがたの中におられる方は、世にいる者より偉大であられるからです。  
<span class="verse-number">5&ensp;</span>その霊たちは世から出ています。従って、その霊たちは世のことを話し、また世は彼らに耳を傾けます。  
<span class="verse-number">6&ensp;</span>私たちは**神**から出ています。**神**を知っている者は、私たちに耳を傾けます。**神**から出ていない者は、私たちに耳を傾けません。これによって、私たちは真理の霊と偽りの霊が分かります。  
<span class="verse-number">7&ensp;</span>愛する者たちよ。お互いに愛し合いましょう。なぜなら、愛は**神**から出ているからであり、また人を愛する者はすべて**神**から生まれ、**神**を知っています。  
<span class="verse-number">8&ensp;</span>**神**は愛であられますから、人を愛さない者は、**神**が分かりません。  
<span class="verse-number">9&ensp;</span>**御子息**を通して私たちが生きるために、**神**は唯一生まれた、ご自分の**御子息**を世に遣わされ、これにより**神**の愛が私たちに明らかにされました。  
<span class="verse-number">10&ensp;</span>愛はここにあります。すなわち、私たちが**神**を愛したのではなく、**神**が私たちを愛され、ご自分の**御子息**を、私たちの罪のなだめの供え物として、遣わされました。  
<span class="verse-number">11&ensp;</span>愛する者たちよ。もし**神**がこのように私たちを愛してくださったなら、私たちも互いに愛し合うべきです。  
<span class="verse-number">12&ensp;</span>誰も、いかなる時にも、**神**を熟視した者はいません。もし私たちが、互いに愛しているならば、**神**は私たちの中にとどまり、また**神**の愛は私たちの中に全うされています。  
<span class="verse-number">13&ensp;</span>**神**が私たちにご自分の**御霊**を与えてくださったこのことで、私たちは**神**にとどまり、そして**神**が私たちにとどまっておられることが分かります。  
<span class="verse-number">14&ensp;</span>そして、**御父**が**御子息**を世の**救い主**として遣わされたことを、私たちは見て分かり、そして証しています。  
<span class="verse-number">15&ensp;</span>誰であれ、イエスは**神の御子息**であられると言い表す人に、その人に**神**はとどまってくださり、そしてその人は**神**にとどまります。  
<span class="verse-number">16&ensp;</span>そして、私たちは、**神**が私たちの中に持っておられる愛を知り、そして信じました。**神**は愛であられます。そして、この愛にとどまる人は、**神**の中にとどまります。また、**神**はその者の中にとどまってくださいます。  
<span class="verse-number">17&ensp;</span>愛はこれによって私たちの間に全うされています。それは私たちが裁きの日に自信を持つためです。ちょうどあの方のように、私たちもこの世にいるからです。  
<span class="verse-number">18&ensp;</span>愛に恐れはありません。それどころか、完全な愛は、恐れを追い出します。なぜなら、恐れは刑罰を内に持っています。恐れる者は、完全な愛の内にいないからです。  
<span class="verse-number">19&ensp;</span>私たちは、あの方を愛しているでしょう。なぜなら、あの方こそは先に私たちを愛してくださったからです。  
<span class="verse-number">20&ensp;</span>もしある者が、「私は**神**を愛している」と言いながら、その者の兄弟を憎んでいれば、その者は嘘つきです。なぜなら、会ったことのある自分の兄弟を愛さない者が、どうして会ったことのない**神**を、愛することができますか。  
<span class="verse-number">21&ensp;</span>そして、私たちが持っているこの命令は、あの方からのものです。すなわち、**神**を愛する者は、自らの兄弟も愛すべきです。  

## 第5章

誰であれ、イエスがキリストであると信じる者は、**神**からの生まれです。そして、お生みになった方を愛するすべての者は、その方から生まれた者をも愛します。  
<span class="verse-number">2&ensp;</span>私たちが**神**を愛し、**神**の命令を守る時、私たちは**神**の子どもを愛していることが、これでわかります。  
<span class="verse-number">3&ensp;</span>なぜなら、**神**の命令を守ること、これこそが**神**の愛だからです。そして、**神**の命令は、重くはないのです。  
<span class="verse-number">4&ensp;</span>なぜなら、**神**から生まれたすべての者は、世に打ち勝っているからです。そして、私たちの信仰、これこそが世に打ち勝った勝利なのです。  
<span class="verse-number">5&ensp;</span>イエスが**神の御子息**であられると信じている者ではなければ、一体、誰が世に打ち勝つ者でしょうか。  
<span class="verse-number">6&ensp;</span>この方は、水と血を通して来られました。すなわちイエス・キリストです。水によるだけではなく、水と血によってです。そして、証をするのは**御霊**であられ、**御霊**は真実であられるからです。  
<span class="verse-number">7&ensp;</span>なぜなら、天に三人の証人、つまり、**御父**と**御言葉**と**聖霊**がおられるからです。そして、この三人はお一人であられます。  
<span class="verse-number">8&ensp;</span>そして、地上には、証をするものは三つあります。つまり、**御霊**と水と血です。そして、この三つは一つになりました。  
<span class="verse-number">9&ensp;</span>もし私たちが人間の証言を受け入れるなら、**神**の証言はもっと偉大です。これはご自分の**御子息**について証された**神**の証言だからです。  
<span class="verse-number">10&ensp;</span>**神の御子息**を信じる者は、その者の中にこの証言を持っています。**神**を信じない者は、**神**を嘘つきにしてしまいました。なぜなら、その者は**神**がご自分の**御子息**についてされたこの証言を、信じなかったからです。  
<span class="verse-number">11&ensp;</span>そして、この証言とは、**神**が私たちに永遠の命を与えてくださったこと、また、この命は**神の御子息**にある、と言うことです。  
<span class="verse-number">12&ensp;</span>**御子息**を持つ者は、命を持っています。**神の御子息**を持っていない者は、命を持っていません。  
<span class="verse-number">13&ensp;</span>私が、**神の御子息**の御名を信じるあなたがたに、これらのことを書いた理由は、あなたがたは永遠の命を持っていることを知るためであり、また**神の御子息**の御名を信じ続けるためです。  
<span class="verse-number">14&ensp;</span>そして、その方のご意志に従い、私たちが願うものは何でも、その方は聞いてくださると、これがその方に対する、私たちの持っている確信です。  
<span class="verse-number">15&ensp;</span>そして、私たちはその方が聞いてくださると分かれば、私たちは**神**に何を願っても、願ったものを私たちが**神**から頂いている、ということが分かります。  
<span class="verse-number">16&ensp;</span>もし誰であれ、その者の兄弟が、死に至らない罪を犯しているのを見て、願えば、**神**は、死に至らない罪を犯している者に命を与えてくださいます。死に至る罪があれば、祈れとは、私は言いません。  
<span class="verse-number">17&ensp;</span>すべての不義は罪です。死に至らない罪もあります。  
<span class="verse-number">18&ensp;</span>**神**から生まれた者は誰であれ、罪を犯し続けない、ということを私たちは知っています。しかし、**神**から生まれた者は、自分自身を守っておられるので、邪悪な者は彼に触れることはありません。  
<span class="verse-number">19&ensp;</span>私たちは**神**から出ており、また全世界は悪の中に横たわっていることを、私たちは知っています。  
<span class="verse-number">20&ensp;</span>しかし、私たちが真実であられる方を知るために、**神の御子息**が来られ、また私たちに理解力を与えられたことを、私たちは分かっています。そして真実の方、すなわち、**神の御子息**イエス・キリストの中に私たちはいます。この方は真実の**神**であられ、永遠の命であられます。  
<span class="verse-number">21&ensp;</span>子どもたちよ。自らを偶像から離しておきなさい。アーメン。
