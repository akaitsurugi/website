# ペテロ第二

### ペテロの手紙　第二

## 第1章

イエス・キリストの僕、そして使徒であるシモン・ペテロ、私たちの**神**と**救い主**イエス・キリストの義によって、私たちと同じ尊さの信仰を得た人たちへ。  
<span class="verse-number">2&ensp;</span>**神**と私たちの**主**イエスを知ることにより、幾重にも恵みと平安があなたがたにありますように。  
<span class="verse-number">3&ensp;</span>それは、栄光と徳により、私たちを召された**神**を知ることで、この方の**神**としての力は、命と敬虔に関するすべてのものを私たちに与えられたからです。  
<span class="verse-number">4&ensp;</span>このことで、私たちには途方もない、大きく、尊い約束が与えられました。それらを通し、あなたがたは**神**の性質を共有する者となり、この世にある欲望に起因する腐朽を逃れるためです。  
<span class="verse-number">5&ensp;</span>そして、この理由こそのため励んで、すべて勤勉に、あなたがたの信仰に徳を加え、徳に知識、  
<span class="verse-number">6&ensp;</span>知識に自制、自制に忍耐、忍耐に**神**を敬う心、  
<span class="verse-number">7&ensp;</span>**神**を敬う心に兄弟愛、兄弟愛に愛を加えなさい。  
<span class="verse-number">8&ensp;</span>なぜなら、これらのことがあなたがたのものになり、満ちれば、あなたがたは私たちの**主**イエス・キリストを知ることに、役立たずではなく、また実を結ばない者でもありません。  
<span class="verse-number">9&ensp;</span>なぜなら、これらのことに欠ける者は、盲目であり、近視であり、以前の罪は清められたことを忘れています。  
<span class="verse-number">10&ensp;</span>従って、兄弟たちよ、さらにあなたがたが召されたこと、選ばれたことを確かなものとするよう励みなさい。なぜなら、これらのことを行なえば、あなたがたは決してつまずくことはありません。  
<span class="verse-number">11&ensp;</span>なぜなら、このように、私たちの**主**、**救い主**であるイエス・キリストの永遠の王国に入ることは、あなたがたに十分に与えられます。  
<span class="verse-number">12&ensp;</span>このために、あなたがたはこれらのことを理解し、提示した真理に、しっかり立っていても、あなたがたがこれらのことを忘れたりしないよう、私は常に気をつけています。  
<span class="verse-number">13&ensp;</span>しかし、私がこの天幕にいる間、あなたがたを思い起こさせることにより、鼓舞することは、正しいことと思っています。  
<span class="verse-number">14&ensp;</span>私たちの**主**イエス・キリストが私に示してくださったように、私はまもなくこの天幕を畳むことを知っています。  
<span class="verse-number">15&ensp;</span>そして、私の死後、これらのことを、あなたがたが忘れないことを確実にするため、私は努力します。  
<span class="verse-number">16&ensp;</span>なぜなら、私たちの**主**イエス・キリストの力と来臨をあなたがたに教えた時、私たちは巧妙なたわごとに従った訳ではありませんでした。むしろ、私たちは**主**の尊厳の目撃者でした。  
<span class="verse-number">17&ensp;</span>なぜなら、「これは私の愛しい**息子**である。私は**息子**を喜びとする。」と言う声がすばらしい栄光より来て、イエスは**父**なる**神**より誉れと栄光を受けました。  
<span class="verse-number">18&ensp;</span>そして、私たちが**主**と共に聖なるあの山の上にいた時、天から来たのこの声を聞きました。  
<span class="verse-number">19&ensp;</span>それで、あの確かな預言者の言葉を私たちは受けました。これは、夜が明ける前、明けの明星があなたがたの心に昇るまで、暗いところを照らす灯りとして、注意するのはよいことです。  
<span class="verse-number">20&ensp;</span>聖書の預言は、いかなる個人の解釈もあり得ないと、まず知りなさい。  
<span class="verse-number">21&ensp;</span>なぜなら、むかし、預言は決して人間の決める意志から来たのではなく、**神**の聖なる人たちが、**聖霊**に動かされて、口にしたのです。  

## 第2章

しかし、国民の中に偽預言者たちが来たし、あなたがたの間にも、秘かに壊滅的な異端をもたらす偽教師たちがやって来ます。偽教師たちは、自分たちを贖われた**主**さえ否定し、早々と壊滅を自らにもたらします。  
<span class="verse-number">2&ensp;</span>そして、多くの人たちは彼らの壊滅の道に従います。偽教師たちを通して真理の道は冒涜されます。  
<span class="verse-number">3&ensp;</span>そして、貪欲によって、彼らは偽りの言葉で、あなたがたを搾取します。彼らの裁きは長い間、昔から怠惰ではなく、また彼らの壊滅は眠ることはありません。  
<span class="verse-number">4&ensp;</span>なぜなら、もし**神**は罪を犯した御使いたちを見逃さず、彼らを地獄に投げ込み、裁きに備えて、闇黒の鎖に繋がったなら、  
<span class="verse-number">5&ensp;</span>大昔の世界を見逃さず、八人の一人、義の説教者であるノアを救い、敬虔でない者たちの世界には洪水をもたらし、  
<span class="verse-number">6&ensp;</span>そして、ソドムとゴモラを灰燼にし、彼らを滅と定め、後の不敬虔に生活をする者たちのため、見せしめとし、  
<span class="verse-number">7&ensp;</span>不法者たちの汚い行ないに悩まされた、正しいロトを救い出されたなら、  
<span class="verse-number">8&ensp;</span>（なぜなら、その正しい人は、彼らの間に住み、日々、彼らの不法の行ないを見聞きして、正しい魂を苦しめました。）  
<span class="verse-number">9&ensp;</span>敬虔な人たちを誘惑から救い出す方法を、また罰を受ける不義の者たちを裁きの日に備える方法を**主**はご存じです。  
<span class="verse-number">10&ensp;</span>それは、特に肉による不潔な情欲に歩み、権威をさげすむ者たちです。彼らは横柄であり、得手勝手です。栄光ある人たちの悪口を言うのを恐れません。  
<span class="verse-number">11&ensp;</span>しかし、能力と力で、さらに優れている御使いたちでさえ、**主**の御前でその者たちをそしり、訴えることはしません。  
<span class="verse-number">12&ensp;</span>しかし、捕らえられ、破滅させられるためにつくられた、理性のない自然の野獣のように、この者たちは理解していない事の悪口を言い、自らの腐敗のうちに完全に滅ばされ、  
<span class="verse-number">13&ensp;</span>そして、不義の報酬を受け取ります。昼日中、享楽を是とする者たちのようです。この者たちはしみであり、汚点であり、あなたがたと宴会を共にしながら、自らの虚構を楽しみ、  
<span class="verse-number">14&ensp;</span>姦淫でいっぱいの目をし、その目は罪を止めることができず、揺れ動くたましいを誘惑します。彼らは欲望で鍛えられた心を持ち、呪われた子たちです。  
<span class="verse-number">15&ensp;</span>彼らは正しい道を捨てて、迷いに入り、不義の報酬を好むベオルの息子バラムの道に従いました。  
<span class="verse-number">16&ensp;</span>しかし、バラムはその悪のために咎められました。口を利かないロバが、男の声で話して、預言者の狂気を止めさせました。  
<span class="verse-number">17&ensp;</span>この者たちは水のかれた井戸であり、嵐に吹き飛ばされる雲であり、闇黒の闇がこの者たちのために、永遠に確保されています。  
<span class="verse-number">18&ensp;</span>なぜなら、彼らは虚しい、大言壮語し、間違いの中に住んでいる者たちから何とか逃れた者たちを、肉の欲を通して、好色で誘惑します。  
<span class="verse-number">19&ensp;</span>彼らはその人たちに自由を約束しながら、彼ら自身が腐朽の奴隷なのです。人は征服するものに、奴隷の状態にされるからです。  
<span class="verse-number">20&ensp;</span>なぜなら、もし**主**と**救い主**であるイエス・キリストの知識によって、世の汚れを逃れて後、またその汚れに絡みつかれ、征服されたなら、初めの状態より終わりの方が、彼らにとってなお悪い状態です。  
<span class="verse-number">21&ensp;</span>なぜなら、義の道を知り、彼らにもたらされた聖なる命令から向きを変えるより、それを知らなかった方がましです。  
<span class="verse-number">22&ensp;</span>しかし、あの真の諺通り、彼らにそれは起こりました。すなわち、「犬は自らの吐いた物の所に戻る。」そして、「めす豚は体を水で洗い、そしてどろの中で転がる。」  

## 第3章

愛する人たちよ、私は今ようやくこの第二の書簡をあなたがたに書いています。（両方の手紙を、記憶を呼び起こす方法として、あなたがたの清い心を揺り動かしたいのです。）  
<span class="verse-number">2&ensp;</span>それは以前、聖なる預言者たちによって話された言葉と、**主**である、**救い主**の使徒たちである私たちの命令を、心に留めてもらうためです。  
<span class="verse-number">3&ensp;</span>まず、このことを理解しなさい。すなわち、最後の日々に、あざける者たちがやってきて、自分たちの欲望のまま歩み、  
<span class="verse-number">4&ensp;</span>こう言います。「あの人の来臨の約束はどこにある。なぜなら、父祖たちが眠って以来、創造の始まりから、すべては変化のないまま流れている。」  
<span class="verse-number">5&ensp;</span>なぜなら、彼らはこのことについては、故意に無知です。すなわち、昔から、天は**神**の御言葉によって存在し、地は水から立ち上がり、水によってできています。  
<span class="verse-number">6&ensp;</span>この水により、かつて存在したあの世界は、水の下なになり、消滅しました。  
<span class="verse-number">7&ensp;</span>しかし、同じ御言葉によって、今、守られている天と地は火に焼かれるため、裁きの日、そして不敬虔な者たちが滅びるまで、確保されます。  
<span class="verse-number">8&ensp;</span>しかし、愛する人たちよ。この一つのことは無知であってはいけません。すなわち、**主**と共ならば、一日は千年と同じ、そして千年は一日と同じなのです。  
<span class="verse-number">9&ensp;</span>ある人々の是とするのと違い、ご自分の約束に関して、**主**は遅れたりしません。むしろ、**主**は私たちに対して忍耐深く、一人も滅びることを望まず、すべての人が悔い改めに至ることを望んでおられます。  
<span class="verse-number">10&ensp;</span>しかし、**主**の日は深夜の泥棒のようにやって来ます。そして、天は大音響と共に消滅し、万物の構成物は白熱で滅びます。そして、地球と地球にあるすべての業は焼き尽くされます。  
<span class="verse-number">11&ensp;</span>したがって、すべてこれらの物は破壊されるのだから、聖い生き方をし、敬虔でいるには、あなたがたはどういう人であるべきですか。  
<span class="verse-number">12&ensp;</span>**神**の日を待ち望み、その日が早めに来るように努力していますか。**神**の日のため、天は燃えて、消え失せ、そして万物の構成物は燃え尽きます。  
<span class="verse-number">13&ensp;</span>しかし、**主**の約束によって、私たちは義が住む新しい天、新しい地を待っています。  
<span class="verse-number">14&ensp;</span>愛する人たちよ。従って、これらのことを待ち望み、汚れも咎められることもなく、平和であると**主**が分かるように、勤勉でいなさい。  
<span class="verse-number">15&ensp;</span>そして、私たちの**主**の忍耐は、救いと考えなさい。私たちの愛する兄弟パウロは、パウロに与えられた知恵により、あなたがたにそう書いたとおりです。  
<span class="verse-number">16&ensp;</span>パウロはまた自分の全書簡で、これらのことを話しています。中には難しくて理解が困難なところがあります。聖書の別の箇所と同様に、無知な者、心定まらぬ者たちはパウロの書簡を歪曲し、自ら破滅に至ります。  
<span class="verse-number">17&ensp;</span>従って、愛する人たちよ。あなたがたは前もってこのことを知っているのだから、悪人たちの不法の過ちにより導かれ、あなたがた自らの動かぬ心を失わないように気をつけなさい。  
<span class="verse-number">18&ensp;</span>そして、私たちの**主**であり、**救い主**であるイエス・キリストの恵みと知識によって成長しなさい。今も永遠も、キリストに栄光がありますように。アーメン。
