# Akai Tsurugi Website

This is the source for the [Akai Tsurugi website](https://akaitsurugi.org/).

The site uses a flat-file based CMS called [Grav](https://getgrav.org/) to build the HTML pages from the Markdown, CSS, Twig, etc. files in this repo.

## Structure

- `images`: Site-wide images.
- `languages`: Localization variables.
- `pages`: The Markdown source files.
- `themes`: The `akai` theme, which contains CSS files, images, fonts, Twig templates, and JavaScript scripts that are used to lay out and complete the pages.

There are still some folders that I haven't uploaded yet (namely, `config`), as I haven't had the time to double-check them for sensitive information.
