# ペテロ第一

### ペテロの手紙　第一

## 第1章

イエス・キリストの使徒の一人であるペテロより、ポント、ガラテヤ、カパドキヤ、アジア、ビテニヤに散らされた、ディアスポラ*の選ばれた逗留者たちへ。  
<span class="verse-number">2&ensp;</span>その散らされたことは、以前から**父**なる**神**がご存じであったことで、**御霊**の聖別により起こったこと、そしてイエス・キリストへの従順と、キリストの血の振りかけのためです。あなたがたに、恵みと平安が幾重にもありますように。  
<span class="verse-number">3&ensp;</span>私たちの**主**イエス・キリストの**父**なる**神**は誉め讃えられますように。**神**は、ご自分の満ち溢れる哀れみにより、イエス・キリストの死者たちの中からの復活を通し、私たちが生きる希望を持てるように、私たちを再び生まれさせてくださいました。  
<span class="verse-number">4&ensp;</span>それは、朽ちることも汚れることもなく、また消え去ることもない受け継ぐ宝*を、あなたがたのため、天国に確保されました。  
<span class="verse-number">5&ensp;</span>終わりの時に、用意されている救いの啓示のため、私たちは信仰により、**神**の力に守られています。  
<span class="verse-number">6&ensp;</span>このためあなたがたは、少しばかり、必要であれば、いろいろな試練で、悲しみにあってきていますが、あなたがたはおおいに喜んでいます。  
<span class="verse-number">7&ensp;</span>それは、消えてなくなる黄金よりもっと尊い、あなたがたの信仰の験は、火によって試されても、イエス・キリストの現れの時、誉れと名誉と栄光に至るように認められるためです。  
<span class="verse-number">8&ensp;</span>あなたがたはイエスを見たことがなくても、愛しています。今イエスを見なくても、あなたがたは信じて、栄光に満たされ、言い表せない喜びで喜び、  
<span class="verse-number">9&ensp;</span>あなたがたの信仰の極み、つまりあなたがたの魂の救いを受けているからです。  
<span class="verse-number">10&ensp;</span>この救いに関しては、あなたがたへの恵みの預言をした、預言者たちが聞き取り、丹念に調べました。  
<span class="verse-number">11&ensp;</span>預言者たちの中におられたキリストの**御霊**が、キリストの苦難と、それに続く栄光をあらかじめ証し、それが何なのか、また、どのような時かを、預言者たちは調べていました。  
<span class="verse-number">12&ensp;</span>預言者たちが仕えていたのは、彼ら自身のためでなく、私たちのためであると、預言者たちらに明らかになりました。しかし、それは今、天から遣わされた**聖霊**によってあなたがたに福音を説いた人たちを通して、あなたがたに告げ知らされたことです。これらのことに対して、御使いたちは、よく調べたいと願っています。  
<span class="verse-number">13&ensp;</span>従って、あなたがたの心の帯を締め、身を慎み、イエス・キリストの現れの時にもたらされる、あなたがたに恵みに、全幅の希望を置きなさい。  
<span class="verse-number">14&ensp;</span>それは、無知であった時のように、昔の欲望に従っていたあなたがたではなく、従順な子どもとしてです。  
<span class="verse-number">15&ensp;</span>むしろ、あなたがたを召された方は、聖なる方なのだから、あなたがた自身もすべての行ないで聖なる人になりなさい。  
<span class="verse-number">16&ensp;</span>「わたしは聖なのだから、あなたがたも聖でありなさい。」と書いてあるからです。  
<span class="verse-number">17&ensp;</span>そして、差別せずに、一人一人のわざに応じて裁く方を**父**と呼ぶなら、ここに、異国人としている間、恐れを持ってふるまいなさい。  
<span class="verse-number">18&ensp;</span>あなたがたの父祖たちから、慣習で受け継いだ空しい行ないから、銀、または金のような、朽ちるものによってあなたがたは、買い戻されたのでなく、  
<span class="verse-number">19&ensp;</span>傷もなく、汚れもない子羊であるキリストの大切な血によって買い戻されたことを知っています。  
<span class="verse-number">20&ensp;</span>真にキリストは、世界の基礎の置かれる前に、あらかじめ知っておられましたが、この終わりの時代にあなたがたのため出現されました。  
<span class="verse-number">21&ensp;</span>キリストを死人たちの中から復活させられ、そしてその方に栄光を与えられた**神**を、キリストを通しあなたがたは**神**を信じているので、それによって、あなたがたの信仰と希望は**神**にあります。  
<span class="verse-number">22&ensp;</span>真摯な兄弟愛のため、**御霊**を通し、真理に従うことを通し、あなたがたは自らのたましいを清められたのだから、清い心で、熱心に互いを愛し合いなさい。  
<span class="verse-number">23&ensp;</span>あなたがたが新しく生まれたのは、腐る種からではなく、腐らない種からです。**神**の永遠に生きる、そして過ぎ去らない御言葉を通してです。  
<span class="verse-number">24&ensp;</span>なぜなら、「すべての肉は草のようであり、また人間のすべての栄光は草の花のようである。草は枯れ、その花は落ち去るが、  
<span class="verse-number">25&ensp;</span>**主**の御言葉は永遠に過ぎ去らない。」すなわち、あなたがたに述べ伝えられた福音とは、この言葉なのです。  

## 第2章

従って、すべての悪意、すべての欺き、偽善、ねたみ、またすべての悪口を捨て去り、  
<span class="verse-number">2&ensp;</span>生まれたばかりの乳児のように、理にかなった御言葉の混ざり物のない乳を求めなさい。それは成長するためです。  
<span class="verse-number">3&ensp;</span>（もし、実際に、あなたがたは、主は恵み深い方だと、味わったのならです。）  
<span class="verse-number">4&ensp;</span>人間に拒否されたが、**神**の選択であり、尊い生ける石としてのキリストのもとに行きながら、  
<span class="verse-number">5&ensp;</span>あなたがたも、生ける石として、霊的な家、すなわち、聖なる祭司職を建て上げられています。イエス・キリストを通し、**神**が受け入れられることのできる、霊的な捧げ物を捧げるためにです。  
<span class="verse-number">6&ensp;</span>従って、聖書の中にもあります。「見よ。すみの土台石をわたしはシオンに置く、選択されたものであり、尊いものである。そして、この方を信じる者は、決して辱めを受けない。」  
<span class="verse-number">7&ensp;</span>それで、信じるあなたがたにとって、この方は尊いが、不従順な者たちにとって、「家を建てる者たちが拒んだ石が、これこそすみの土台石になった。」ということです。  
<span class="verse-number">8&ensp;</span>そして、「つまずきの石、妨害の岩」になりました。彼らは御言葉に従わないので御言葉につまずきます。このつまずきは定められていたのです。  
<span class="verse-number">9&ensp;</span>しかし、あなたがたは選ばれた民族であり、王の祭司職であり、聖なる国民であり、暗やみからご自分のすばらしい光の中へ、あなたがたを呼び出された方の徳を説くための、あの方の持ちものとされた人民なのです。  
<span class="verse-number">10&ensp;</span>あなたがたは以前、どの民でもなかったのですが、今は**神**の民です。以前、哀れみを得ることはなかったが、今は哀れみを得ている人たちです。  
<span class="verse-number">11&ensp;</span>愛する人たちよ、他国人であり、逗留者であるあなたがたに私は勧めます。たましいと戦争する肉体の欲望を避けて、  
<span class="verse-number">12&ensp;</span>異邦人たちの間では、ふるまいを正しなさい。異邦人たちがあなたがたを悪を行なう者たちだと非難する時、あなたがたの善い行ないを見て、臨みの日には、異邦人たちが**神**をあがめるためです。  
<span class="verse-number">13&ensp;</span>従って、**主**のために、人の立てたすべての制度に自分を服従させなさい。最高統治者の王に、  
<span class="verse-number">14&ensp;</span>また、悪を行なう者たちを罰するために、また善を行なう者たちを賞賛するために、王に遣わされた総督にも従いなさい。  
<span class="verse-number">15&ensp;</span>善行で、愚かな者たちの無知を沈黙させるのです。なぜなら、これは**神**のご意志です。  
<span class="verse-number">16&ensp;</span>自由だからと言って、その自由を不道徳を隠すために使わず、**神**の僕として使いなさい。  
<span class="verse-number">17&ensp;</span>すべての人を礼をもって接しなさい。兄弟である人たちを愛しなさい。**神**を恐れなさい。王には礼をもって接しなさい。  
<span class="verse-number">18&ensp;</span>僕たちよ、全面的に恐れて、良い主人、優しい主人だけでなく、過酷な主人たちにも服従しなさい。  
<span class="verse-number">19&ensp;</span>なぜなら、**神**に対する良心のため、不正に苦しみ、悲しみを耐え忍ぶなら、それは推賞すべきことだからです。  
<span class="verse-number">20&ensp;</span>なぜなら、罪のため殴られ、それに耐えたと言って、一体何の手柄があるでしょう。しかし、善を行なっていて苦しむのなら、それを耐えたのは**神**の御前で推賞されるべきことです。  
<span class="verse-number">21&ensp;</span>なぜなら、あなたがたは、このために召されたのです。すなわち、キリストもまた、私たちのために苦しまれ、あなたがたが**主**の足跡に従うように、私たちに模範を残されたのです。  
<span class="verse-number">22&ensp;</span>キリストは何の罪も犯さず、その口には何の偽りも見いだされませんでした。  
<span class="verse-number">23&ensp;</span>キリストはののしられても、ののしり返しはなされませんでした。苦しまれても、脅迫されず、正しく裁く方にご自分を委ねられました。  
<span class="verse-number">24&ensp;</span>キリストが木にかけられ、ご自分の身に私たちの罪を負われました。それは、罪と係わりを断った私たちが*、義のため生きるためです。あなたがたはキリストのこのお受けになった傷でいやされたのです。  
<span class="verse-number">25&ensp;</span>なぜなら、あなたがたは迷子の羊のようであったが、今、あなたがたのたましいの羊飼いであり、監督者である方のもとに、帰ってきたのです。  

## 第3章

同様に、妻たちよ、あなた自身の夫たちに従いなさい。御言葉に従っていないとしても、夫たちは一言も言わず、妻たちの振る舞いによって獲得されるためです。  
<span class="verse-number">2&ensp;</span>恐れを伴う、あなたがたの慎み深い振る舞いを夫たちが見ることによってです。  
<span class="verse-number">3&ensp;</span>その振る舞いは外面的なもので、すなわち、頭髪を編み、黄金を身につけ、衣服を身につけることあってはいけません。  
<span class="verse-number">4&ensp;</span>むしろ、穏やかで静かで、滅びることのない心を、内に秘めたものを飾りとしなさい。それは**神**の御前ではとても尊いのです。  
<span class="verse-number">5&ensp;</span>なぜなら、昔、**神**に希望をたくした聖なる婦人たちも、このように、自らを飾り、各自の夫たちに従っていました。  
<span class="verse-number">6&ensp;</span>そのようにしてサラはアブラハムに従い、アブラハムを「主」と呼んでいました。もし、あなたがたがよい行ないをし、どんな脅かしをも恐れなければ、あなたがたはサラの娘たちです。  
<span class="verse-number">7&ensp;</span>夫たちは同様に、理解をもって妻たちと暮らしなさい。あなたがたより弱い器として、また共に命の恵みを受け継ぐ者として、妻たる者に敬意を払いなさい。それは、あなたがたの祈りが中断されないためです。  
<span class="verse-number">8&ensp;</span>最後に、あなたがた全員が、心を一つにし、互いを思いやり、兄弟として愛して、やさしい心を持って、礼儀正しく、  
<span class="verse-number">9&ensp;</span>悪に悪を持って報わず、侮辱に侮辱を持って報わず、それどころか、祝福しなさい。あなたがたは、この事のため召されたと知っています。すなわち祝福を受け継ぐためです。  
<span class="verse-number">10&ensp;</span>なぜなら、「命を愛し、幸せな日々を過ごしたいと求める者は、舌を悪から、また偽りを話すことから口を遠ざけなさい。  
<span class="verse-number">11&ensp;</span>その者は悪から顔をそむけ、善を行ないなさい。平和を捜し、これを追い求めなさい。  
<span class="verse-number">12&ensp;</span>なぜなら、**主**の目は義人に注がれ、**主**の耳は義人の祈りに傾けられているが、**主**の顔は悪を行なう人々から目を離さない。」  
<span class="verse-number">13&ensp;</span>そして、もしあなたがたが善行を追う者になれば、誰があなたがたに害を加えるのですか。  
<span class="verse-number">14&ensp;</span>しかし、あなたがたは義のために苦しんでも、祝福されています。「彼らの恐怖を恐れてはいけないし、また、取り乱してはいけない。」  
<span class="verse-number">15&ensp;</span>しかし、心の中で**主**なる**神**を聖なるものとして分け、あなたがたの中にある希望の理由を、あなたがたに説明を尋ねる人たちに、誰にでも、へりくだり、畏敬の心で、いつでも答弁できるように用意をしなさい。  
<span class="verse-number">16&ensp;</span>優れた良心を持ちなさい。それは、ある人たちがあなたがたを悪を行なう者として侮辱する時、キリストにある、あなたがたの善い行ないをののしる者たちが、恥じ入るためです。  
<span class="verse-number">17&ensp;</span>なぜなら、もし、善を行ない、苦しみを受けるのが**神**のご意志なら、悪を行なって、苦しむよりまさっています。  
<span class="verse-number">18&ensp;</span>なぜなら、キリストも一度、罪のため苦しみをお受けになりました。正しい方が正しくない者の代わりに、肉において殺され、**御霊**によって生かされました。私たちを、**神**のところに伴って行くためです。  
<span class="verse-number">19&ensp;</span>その**御霊**によって、キリストは牢獄にいる霊たちの所に行って、説かれました。  
<span class="verse-number">20&ensp;</span>以前、この霊たちは従わなかったのです。それは、ノアの時、箱船が準備されている間、一度は**神**が長い間、**神**は忍耐して待っておられた間です。水で救われたのはほんのわずか、すなわち八人でした。  
<span class="verse-number">21&ensp;</span>今、私たちを救う浸礼の呼応する型もあります。その浸礼は肉の汚れを取ることではなく、**神**に対して優れた良心での、イエス・キリストの復活を通しての返答です。  
<span class="verse-number">22&ensp;</span>キリストは天国に行かれ、**神**の右側におられます。御使いたちや権威ある者たちや権力ある者たちはキリストに隷属させらました。  

## 第4章

従って、キリストは肉において私たちのために苦しまれたので、あなたがたも同じ考えで武装しなさい。肉において苦しんだ人は罪を終わりにしたからです。  
<span class="verse-number">2&ensp;</span>それは、もう肉の残りの時を、人間の欲望のためでなく、**神**のご意志に従って生きるためです。  
<span class="verse-number">3&ensp;</span>なぜなら、私たちは過去の人生を、異邦人の心のままに十分に過ごしました。すなわち、好色、欲望、酩酊、飲み会、酒席、そして、忌まわしい偶像崇拝の道を歩んだのです。  
<span class="verse-number">4&ensp;</span>これらのことに関して、あなたがたが異邦人たちと共に、同じふしだらな生活の極みへ走らないことを、異邦人たちは不思議に思い、あなたがたの悪口を言います。  
<span class="verse-number">5&ensp;</span>生者たち、そして死者たちを裁く体制を整えられている方に、異邦人たちは決算することになります。  
<span class="verse-number">6&ensp;</span>このため、死んだ人たちにも福音が説かれたのは、肉体により人たちに裁かれるのに、霊によって**神**に従って生きるためです。  
<span class="verse-number">7&ensp;</span>しかし、すべての終わりは近づきました。ですから、真剣に、そして注意深く祈りなさい。  
<span class="verse-number">8&ensp;</span>とりわけ、「愛は数多くの罪を覆う」のだから、互いに心を込めた愛を持ちなさい。  
<span class="verse-number">9&ensp;</span>不平を言わず、互いにもてなしなさい。  
<span class="verse-number">10&ensp;</span>一人一人が賜物を受けたのだから、**神**の多様な恵みの執事としてその賜物で互いに奉仕しなさい。  
<span class="verse-number">11&ensp;</span>誰か話しをするなら、**神**のオラクル[^1]として話をさせなさい。誰か奉仕をするなら、**神**が与えられた能力で奉仕させなさい。**神**がイエス・キリストを通して、すべてのことに栄光を受けるためです。栄光と主権が永遠から永遠までイエス・キリストにありますように。アーメン。  
<span class="verse-number">12&ensp;</span>愛する人たちよ、あなたがたにやって来る火の試練のことを、たとえ不思議なことが、あなたがたに起こっているように、不思議と思ってはいけません。  
<span class="verse-number">13&ensp;</span>むしろ、キリストの苦しみを共にできるほど、喜びなさい。キリストの栄光があらわれる時も、あなたがたが、無上の喜びに歓喜できるためです。  
<span class="verse-number">14&ensp;</span>あなたがたはキリストの御名のため、叱責されても、あなたがたは祝福されています。なぜなら、**神**と栄光の**御霊**は、あなたがたの上に休息されるからです。彼らの側では、キリストは冒涜されるが、あなたがたの側ではキリストは栄光をお受けになります。  
<span class="verse-number">15&ensp;</span>しかし、あなたがたの中、何人も殺人者、泥棒、悪を行なう者、他人に干渉する者として、苦しむことがあってはいけません。  
<span class="verse-number">16&ensp;</span>しかし、クリスチャンとして苦しむ人がいたら、恥じることなく、その人にこのことで、**神**に栄光を捧げさせなさい。  
<span class="verse-number">17&ensp;</span>なぜなら、**神**の家で裁きが始まる時が来ました。そして、まず私たちの間で始まれば、**神**の福音に従わない人たちの結末は何でしょうか。  
<span class="verse-number">18&ensp;</span>では、「正しい人が何とか辛うじて救われるなら、不敬虔な者や罪人はどこに姿を現わすのか？」  
<span class="verse-number">19&ensp;</span>そして、そのように、**神**のご意志によって苦しむ人たちは、正しい行ないをしながら、自らのたましいを真実な創造者としての、**神**に委ねなさい。  

## 第5章

私はあなたがたの中の長老たちに勧めます。私も同じ長老であり、キリストの苦しみの証人であり、また、やがて明らかにされる、栄光を共にする者です。  
<span class="verse-number">2&ensp;</span>あなたがたの間にいる**神**の群れを、牧会しなさい。強制によらず、自発的に、不正な利益のためでなく、熱心に世話をする人として仕えなさい。  
<span class="verse-number">3&ensp;</span>また、委任された人たちの支配する者にならず、群れの模範になりなさい。  
<span class="verse-number">4&ensp;</span>そして、牧者の長が現れた時、あなたがたは消え去ることのない栄光の冠を受けます。  
<span class="verse-number">5&ensp;</span>同様に、若者たちよ、あなたがたの長老たちに自分から従いなさい。そして全員が、互いに従い、謙虚を身にまといなさい。「**神**は傲慢な者たちに敵対し、へりくだる人たちに恵みを与えられる。」とあるからです。  
<span class="verse-number">6&ensp;</span>ですから、**神**の力ある手のもとで、あなたがた自身、へりくだりなさい。時が来ると、あなたがたを**神**が高めてくださるためです。  
<span class="verse-number">7&ensp;</span>あなたがたの心配のすべてを、**神**に投げかけなさい。**神**はあなたがたを真剣に考えてくださっているからです。  
<span class="verse-number">8&ensp;</span>あなたがたの敵である悪魔は、吠えながら歩き回るライオンのように、むさぼり食い尽くせる者を、探し求めているのだから、気をゆるめず、用心深くありなさい。  
<span class="verse-number">9&ensp;</span>世にいるあなたがたの兄弟である人たちも、この同じ苦しみを経験していると理解し、信仰にしっかり立ち、悪魔に対抗しなさい。  
<span class="verse-number">10&ensp;</span>キリスト・イエスによって私たちをご自分の永遠の栄光に召された、すべての恵みの**神**は、あなたがたがしばらく苦しんでから、あなたがたを完全な者とし、確立し、強め、安定した者にしてくださいます。  
<span class="verse-number">11&ensp;</span>永遠から永遠まで、栄光と主権はキリストにあります。アーメン。  
<span class="verse-number">12&ensp;</span>私たちの忠実な兄弟であると、私は認めているシルワノを通し、私があなたがたに短く手紙に書いたのは、あなたがたが立っているのは、これこそが**神**の真の恵みであると勧め、証するためです。  
<span class="verse-number">13&ensp;</span>あなたがたと共に選ばれ、バビロンにいる婦人と、私の息子であるマルコからも、共に挨拶を送ります。  
<span class="verse-number">14&ensp;</span>愛の口づけで互いに挨拶をしなさい。キリスト・イエスにいるあなたがた全員に、平安がありますように。アーメン。  


[^1]: **神**からの啓示。
