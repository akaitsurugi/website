# コリント第一

### コリント人への手紙　第一

## 第1章

**神**のご意思を通し、イエス・キリストの一人の使徒として召されたパウロと、我らが兄弟ソステネ、  
<span class="verse-number">2&ensp;</span>コリントにある**神**の教会へ、キリスト・イエスによって聖別され、聖徒たちとして召された人々へ、いかなる所でも私たちの**主**であるイエス・キリストの御名を呼び求めるすべての人々へ。イエスは私たちの**主**であり、その人々の**主**です。  
<span class="verse-number">3&ensp;</span>私たちの**父**なる**神**と**主**イエス・キリストより、あなたがたに恵みと平安がありますように。  
<span class="verse-number">4&ensp;</span>キリスト・イエスによってあなたがたに与えられた御恵みゆえに、私はあなたがたのことで、私の**神**にいつも感謝しています。  
<span class="verse-number">5&ensp;</span>また、キリストにより、すべての言葉、すべての知識に、あなたがたが豊かにされたことを、私の**神**に私はいつも感謝しています。  
<span class="verse-number">6&ensp;</span>これは、キリストの証があなたがたの中で、確認されたと同じです。  
<span class="verse-number">7&ensp;</span>それは、あなたがたが、どの賜物にも不足な者とならず、私たちの**主**イエス・キリストの出現を渇望しています。  
<span class="verse-number">8&ensp;</span>そして、キリストは、あなたがたを最後まで確認して、私たちの**主**イエス・キリストの日に、あなたがたを咎められるところのない者にするためです。  
<span class="verse-number">9&ensp;</span>**神の御子息**であられる、私たちの**主**イエス・キリストとの交流へあなたがたを召された**神**は、真実でいらっしゃいます。  
<span class="verse-number">10&ensp;</span>では、私たちの**主**イエス・キリストの御名によって、私はあなたがた、兄弟に、お願いします。あなたがた全員、同じ事を話すこと、また、あなたがたの間に分裂がなく、むしろ、共通理解と、同じ判断で、完全に結びつくことです。  
<span class="verse-number">11&ensp;</span>なぜなら、あなたがたのことで、私の兄弟たちよ、クロエの家人たちより、あなたがたの間に争いがあると、私に伝えられたからです。  
<span class="verse-number">12&ensp;</span>それで、私はこう言います。あなたがた一人一人は言っています。「私はパウロに属する」、とか、「私はアポロに属する」、とか、「私はケパに属する」、とか、「私はキリストに属する」、と。  
<span class="verse-number">13&ensp;</span>キリストは分裂されていますか。または、パウロはあなたがたのために、十字架につけられたのですか。または、あなたがたはパウロの名において浸礼を授けられたのですか。  
<span class="verse-number">14&ensp;</span>あなたがたの中で、クリスポやガイオ以外、誰にも浸礼を授けなかったことを、私は**神**に感謝します。  
<span class="verse-number">15&ensp;</span>それは、誰も、私が私の名で浸礼を授けた、と誰も言えなくするためです。  
<span class="verse-number">16&ensp;</span>しかし、私はステパナの家人にも浸礼を授けました。しかし、そのほか、誰に浸礼を授けたかは、わかりません。  
<span class="verse-number">17&ensp;</span>キリストは私を、浸礼を授けに派遣されたのではなく、福音を説くために遣わされたのです。それは、キリストの十字架が無にならないためであり、言葉の知恵で語るためではありません。  
<span class="verse-number">18&ensp;</span>なぜなら、十字架の言葉は、滅びる人々にとって愚かしいことではあるが、救われている私たちにとっては、**神**の力だからです。  
<span class="verse-number">19&ensp;</span>それは、こう書いてあるからです。「私は、知者の知恵を滅ぼし、賢い者の知力を無にする。」  
<span class="verse-number">20&ensp;</span>知者はどこでいるのですか。律法学者はどこにいるのですか。この時代の論者はどこにいるのですか。**神**はこの世の知恵を愚かにされたのではありませんか。  
<span class="verse-number">21&ensp;</span>**神**の知恵により、世は知恵を通して**神**を知ることなく、信じる者たちを救うため、説かれた愚かさを通して、それは、**神**を喜ばせました。  
<span class="verse-number">22&ensp;</span>ユダヤ人たちはしるしを要求し、ギリシャ人は知恵を探し求めるからです。  
<span class="verse-number">23&ensp;</span>しかし、私たちは十字架につけられたキリストを説いているのです。それはユダヤ人たちにとってつまずくものであり、ギリシャ人たちにとって愚かさです。  
<span class="verse-number">24&ensp;</span>しかし、ユダヤ人たちにも、ギリシャ人たちにも、召された人たちにとって、キリストは**神**の力であり、**神**の知恵なのです。  
<span class="verse-number">25&ensp;</span>**神**の愚かさは人々より賢く、**神**の弱さは人々より強いからです。  
<span class="verse-number">26&ensp;</span>あなたがたの召されたことを見てみなさい。兄弟たちよ、肉による知者、力ある者、高貴な者は、大勢の者は召されていません。  
<span class="verse-number">27&ensp;</span>しかし、**神**は、知者たちを辱めるために、世の愚かなものを選ばれ、力ある者たちを辱めるために、世の弱いものを選ばれました。  
<span class="verse-number">28&ensp;</span>そして、世の底辺にいて、軽蔑されているものを、そして存在しないものを**神**は選ばれました。それは、存在するものを存在しないものにするためです。  
<span class="verse-number">29&ensp;</span>これは、どの肉も、**神**の御前に誇らないためです。  
<span class="verse-number">30&ensp;</span>しかし、あなたがたは、キリスト・イエスの内にあり、**神**からの者です。キリスト・イエスは我らのため、**神**からの知恵、そして義、そして聖別、そしてあがないとなってくださいました。  
<span class="verse-number">31&ensp;</span>「誇る者は、**主**にあって誇りなさい。」と書いてあるとおりのためです。  

## 第2章

そして、兄弟たちよ。あなたがたの所に行った時、**神**の証をするのに口先上手に、または、巧みな知恵で、私は説いたのではありません。  
<span class="verse-number">2&ensp;</span>あなたがたの間では、イエス・キリスト、つまり十字架につけられたイエス・キリストのほかは、私は何も知らない人間になると決めたからです。  
<span class="verse-number">3&ensp;</span>私は弱気になり、恐れ、震えて、あなたがたと共にいました。  
<span class="verse-number">4&ensp;</span>そして、私の言葉と私のメッセージは、人間の知恵で納得させる言葉ではなく、霊と力の証拠によるものでした。  
<span class="verse-number">5&ensp;</span>それは、あなたがたの信仰は人間の知恵にあるのではなく、**神**の力にあるからです。  
<span class="verse-number">6&ensp;</span>しかし、成人の間で、私たちは知恵を話します。今の時代の知恵でなく、無になろうとしている、今の時代の支配者の知恵でもなく、  
<span class="verse-number">7&ensp;</span>それどころか、私たちは、**神**の奥義による知恵を話します。それは、**神**が何代も前の時代に、私たちの栄光のため**神**が定められた知恵、つまり隠された知恵です。  
<span class="verse-number">8&ensp;</span>これは、この時代の支配者たちが誰も知らなかった知恵です。もしその知恵を知っていたならば、栄光の**主**を十字架につけることはありませんでした。  
<span class="verse-number">9&ensp;</span>しかし、こう書いてあります。「**神**を愛する者たちのために、**神**が準備してくださったものを、見た目はなく、聞いた耳はなく、また人の心に入ったことがない。」  
<span class="verse-number">10&ensp;</span>しかし、御自分の**御霊**を通して、**神**はそれらを私たちに啓示されました。それは、**御霊**はすべてのことを探知し、また**神**の深部をも探知するからです。  
<span class="verse-number">11&ensp;</span>なぜなら、その人の内にあるその人の霊以外、誰がその人のことが分かりますか。同様に、**神**の**御霊**以外、誰も**神**のことが分かっていません。  
<span class="verse-number">12&ensp;</span>それで、私たちが受けたのは、世の霊でなく、**神**からの**御霊**です。**神**から私たちに恵みで与えられたものを、私たちが知るためなのです。  
<span class="verse-number">13&ensp;</span>霊的なことは霊的なことに対比させながら、人間の知恵が教える言葉でなく、**聖霊**が教える御言葉で、私たちはこれらのことを話します。  
<span class="verse-number">14&ensp;</span>しかし、**神**の**御霊**のものは生まれたままの人間には愚かなので、受け入れませんし、また理解できません。**神**の**御霊**のものは霊的に認知されるからです。  
<span class="verse-number">15&ensp;</span>しかし、霊に属する人は、すべてのものを認知するが、その人自身は誰にも認知されません。  
<span class="verse-number">16&ensp;</span>「誰が、**主**に進言するほど、**主**の心を理解したのか。」しかし、私たちはキリストの心を持っています。  

## 第3章

そして、兄弟たちよ、私はあなたがたを霊の人としては、話しできず、肉の人として、キリストにあっては赤子として話しをしました。  
<span class="verse-number">2&ensp;</span>私はあなたがたに固形食を与えず、乳を与えました。それは、その時、あなたがたは未だ受け入れることができなかったからです。そして、今もまだ受け入れることができません。  
<span class="verse-number">3&ensp;</span>それはまだあなたがたは肉の人だからです。互いに妬み、争い、分裂があるのは、あなたがたは肉の人で、人間として歩んでいるだけではありませんか。  
<span class="verse-number">4&ensp;</span>なぜなら、一人は、「私はパウロ派だ」と言い、もう一人は、「私はアポロ派だ」と言う時、あなたがたは肉の人たちではないのですか。  
<span class="verse-number">5&ensp;</span>それでは、パウロとアポロとは何者ですか。あなたがたは、**主**がこの二人に与えられたことを通してあなたがたは信じた奉仕者だけです。  
<span class="verse-number">6&ensp;</span>私が植え、アポロが水をやり、**神**が成長させられたのです。  
<span class="verse-number">7&ensp;</span>ですから、植える者、水をやる者は何者でもありません。育ててくださるのは**神**です。  
<span class="verse-number">8&ensp;</span>ですから、植える者と水をやる者は一人であり、それぞれの働きに応じ、各自の報酬を受けます。  
<span class="verse-number">9&ensp;</span>ですから、私たちは共に働く、**神**の労働者です。あなたがたは**神**の畑であり、**神**の建造物なのです。  
<span class="verse-number">10&ensp;</span>私に与えられた**神**の御恵みにより、私は賢い建築家としてすでに据えた土台の上に、他の人が建てます。しかし、各自、その土台の上にどう建てるか、注意しなさい。  
<span class="verse-number">11&ensp;</span>据えられている土台以外、土台を据えることができないからです。つまりこの土台はイエス・キリストです。  
<span class="verse-number">12&ensp;</span>それで、もし誰かがこの土台の上に金、銀、宝石、木材、干し草、わらを使って建てるなら、  
<span class="verse-number">13&ensp;</span>各自の働きは明らかになります。その日が宣言します。なぜなら、火によって明らかになり、そしてその火は各自の働きを試し、それがどんな働きなのかを証明します。  
<span class="verse-number">14&ensp;</span>誰であれ土台の上の働きが残った者は、報酬を受けます。  
<span class="verse-number">15&ensp;</span>誰であれその者の働きが燃え落ちると、その者は損害を受けるが、その者自身は救われます。しかし、彼は火を通りぬけて救われるのです。  
<span class="verse-number">16&ensp;</span>あなたがたは**神**の神殿であり、**神**の**御霊**があなたがたの中に住んでおられることを知らないのですか。  
<span class="verse-number">17&ensp;</span>誰であれ**神**の神殿を汚すなら、**神**はその者を滅ぼします。なぜなら、**神**の神殿は聖なるものであり、その神殿はあなたがたなのです。  
<span class="verse-number">18&ensp;</span>自らを欺いてはいけません。もしこの時代、あなたがたの中で、自分が賢いと思う者がいるなら、その者は賢くなるために、愚か者になりなさい。  
<span class="verse-number">19&ensp;</span>世の知恵は、**神**に比べると愚かだからです。それは、「**神**は知恵者たちを、その者たちの悪賢さで捕まえられる。」と書いてあるからです。  
<span class="verse-number">20&ensp;</span>そしてまた、「知者の考えは無益であることを、**主**はご存じである。」とあります。  
<span class="verse-number">21&ensp;</span>それでは、誰も人間のことで誇ってはいけません。すべてはあなたがたのことだからです。  
<span class="verse-number">22&ensp;</span>パウロであれ、アポロであれ、ケパであれ、世であれ、命であれ、死であれ、今あるものであれ、後に来るものであれ、すべてあなたがたのことだからです。  
<span class="verse-number">23&ensp;</span>そして、あなたがたはキリストのものであり、キリストは**神**のものなのです。  

## 第4章

それで、私たちがキリストの僕、**神**の奥義の執事であると、人が勘定されるようにしなさい。  
<span class="verse-number">2&ensp;</span>そして次いで、忠実であることがわかることが、執事たちには要求されます。  
<span class="verse-number">3&ensp;</span>しかし、あなたがたや、また、人間の日[^1]で裁かれることは、私にとって、極めて些細なことです。事実、私は自分自身さえも裁きません。  
<span class="verse-number">4&ensp;</span>なぜなら、私は自分自身に責めがあると思いませんが、このことで私は義と認められません。私を裁く方は**主**だからです。  
<span class="verse-number">5&ensp;</span>したがって、**主**が来られ、暗やみに隠れたものを照らし、心のたくらみを明らかにされるその時まで、何も裁いてはいけません。そして、その時、一人一人に**神**からの賛辞があります。  
<span class="verse-number">6&ensp;</span>さて、兄弟たちよ、あなたがたのために、このことを、私とアポロに当てはめてみました。あなたがたは書かれた以上のことは考えず、だれ一人として、他人に対し思いあがらず、私たちを通して学ぶためです。  
<span class="verse-number">7&ensp;</span>なぜなら、誰があなたを他の人と違えて造ったのですか。そして、頂いていないなら、あなたがたは何を持っていますか。そして、もし本当にそれを頂いたのなら、あたかも頂いたものでないかのごとく、なぜ自慢するのですか。  
<span class="verse-number">8&ensp;</span>あなたがたはもう満たされています。もう豊かなのです。私たちがいなくとも、あなたがたは王たちのように支配しています。本当に、あなたがたが、支配できればよいと思っています。私たちも共に支配するためです。  
<span class="verse-number">9&ensp;</span>なぜなら、死刑を宣告された者たちとして、**神**は我々使徒たちを、最後に立たせられたのだ、と私は思います。つまり御使いにも人間にも、私たちを見せ物として世に示したのです。  
<span class="verse-number">10&ensp;</span>私たちはキリストのために愚か者ですが、あなたがたはキリストにあって賢いのです。私たちは弱いが、あなたがたは強いのです。あなたがたは傑出しているが、私たちは恥ずべき者たちです。  
<span class="verse-number">11&ensp;</span>現在まで、私たちは飢え、渇き、ぼろをまとい、そして殴られ、家はなく、  
<span class="verse-number">12&ensp;</span>自らの手で働き、労働しています。ののしられる時、私たちは祝福します。迫害される時、私たちは耐えます。  
<span class="verse-number">13&ensp;</span>侮辱される時、私たちは勧めます。私たちは世のごみのようになり、今まですべてのものの廃棄物になりました。  
<span class="verse-number">14&ensp;</span>あなたがたに、恥をかかせるため、私はこれらのことを書くのではなく、私の愛する子どもたちとしての、あなたがたに警告するのです。  
<span class="verse-number">15&ensp;</span>なぜなら、あなたがたはキリストにあって一万人の守役がいても、多くの父たちはいません。私はキリストにあって福音を通してあなたがたを生んだからです。  
<span class="verse-number">16&ensp;</span>ですから、私はあなたがたに勧めます。私をまねなさい。  
<span class="verse-number">17&ensp;</span>それで、**主**にあって忠実である、私の愛する息子、テモテをあなたがたに遣わしました。私がどこでも、どこの教会でも教えるキリストにある私の道を、テモテはあなたがたに、思い起こさせます。  
<span class="verse-number">18&ensp;</span>しかし、あなたがたのある者たちは、私はあなたがたの所に行かないと、思い上がっています。  
<span class="verse-number">19&ensp;</span>しかし、**主**の望みであれば、私はすぐあなたがたの所に行きます。そして、思い上がった者たちの言葉ではなく、その力のほどが分かります。  
<span class="verse-number">20&ensp;</span>**神**の王国は言葉にあるのではなく、力にあるからです。  
<span class="verse-number">21&ensp;</span>あなたがたは何が欲しいのですか。こん棒を持ってあなたがたの所に行きましょうか、または愛と優しさの霊を持って行きましょうか。  

## 第5章

あなたがたの間に性的な罪が行なわれていると、報告があるのは確かです。それは異邦人の間でさえ、名付けられていないほどの性的な罪です。すなわち、ある男が自分の父親の妻を所有している、と言うのです。  
<span class="verse-number">2&ensp;</span>そして、この行為をした者が、あなたがたの間から取り除かれることがないように、泣き悲しんだどころか、むしろ自慢しています。  
<span class="verse-number">3&ensp;</span>体は不在でも、霊は居るのですから、私はそこに本当に居るように、この行為をした者はすでに裁きました。  
<span class="verse-number">4&ensp;</span>私たちの**主**イエス・キリストの御名により、あなたがたが私の霊と、私たちの**主**イエス・キリストの力と共に、集まった時、  
<span class="verse-number">5&ensp;</span>このような男こそをサタンに引き渡すのが肉が滅んで、霊が**主**イエスの日に救われるためです。  
<span class="verse-number">6&ensp;</span>あなたがたの自慢は良くありません。パン種はほんの少しでも、塊全体をふくらませることを、あなたがたは知らないのですか。  
<span class="verse-number">7&ensp;</span>それで、あなたがたは新しい塊になるように、古いパン種を完全に一掃しなさい。あなたがたは真に種無しの塊だからです。私たちの過越であられるキリストは、私たちのためにいけにえにされたからです。  
<span class="verse-number">8&ensp;</span>同様に、また悪意や悪をもたず、古いパン種でなく、誠実、真理をもってこの祭りを守っていきましょう。  
<span class="verse-number">9&ensp;</span>性的に不道徳な者とあなたがたは交際しないよう、私は私の手紙に書きました。  
<span class="verse-number">10&ensp;</span>しかし、この世の性的に不道徳な者たち、貪欲な者たち、強奪する者たち、偶像崇拝者たちと、交際するな、と言う意味ではもちろんありません。そうであれば、人はこの世の外に出なくてはなりません。  
<span class="verse-number">11&ensp;</span>ただ、もし兄弟と呼ばれる者が、性的に不道徳な者、貪欲な者、偶像崇拝者、ののしる者、酔っぱらい、強奪する者なら、そんな者と交際してはいけません。また、そんな者と食事さえ共にしてはいけません。  
<span class="verse-number">12&ensp;</span>なぜなら、外の人たちを裁くことが、私のすべきことでしょうか。あなたがたが裁くのは内の者たちではないのですか。  
<span class="verse-number">13&ensp;</span>外の者たちは**神**が裁かれます。それで、このような悪人はあなたがたから遠ざけなさい。  

## 第6章

あなたがたの中で、他人と対立する事があり、聖徒たちの前に行かず、正しくない者たちの前に行き、あえて裁きを求める者がいますか。  
<span class="verse-number">2&ensp;</span>聖徒たちがこの世を裁くことを知らないのですか。そして、あなたがたによってこの世が裁かれるとしたら、あなたがたは最も小さい事を裁く価値もないのですか。  
<span class="verse-number">3&ensp;</span>私たちが御使いたちを裁くことを知らないのですか。この世のことはいうまでもないことではないですか。  
<span class="verse-number">4&ensp;</span>それで、この世に関する訴えがあれば、私たちは教会の中で最も軽視されている者たちを、裁く人の座に座らせましょう。  
<span class="verse-number">5&ensp;</span>私はあなたがたの恥であるこのことを言います。どうですか。兄弟たちの間を裁ける知者は、あなたがたの間に一人もいませんか。  
<span class="verse-number">6&ensp;</span>しかも、兄弟は兄弟を訴えます。そしてそれは未信者たちの前でです。  
<span class="verse-number">7&ensp;</span>それで、互いに訴えること自体、すでに、あなたがたにとって完全な失態です。あなたがたはなぜ、あえて不正を受け入れないのですか。なぜあえて、騙されないのですか。  
<span class="verse-number">8&ensp;</span>しかし、あなたがた自身が不正を行ない、かつ、欺きを行ないます。しかも、そんなことをあなたがたの兄弟たちへするのです。  
<span class="verse-number">9&ensp;</span>不正な者たちは、**神**の王国を相続することはないと知らないのですか。騙されてはいけません。性的な罪を犯す者、偶像崇拝者、姦淫をする者、同性愛者たち、男色者、  
<span class="verse-number">10&ensp;</span>泥棒、貪欲な者、酔っぱらい、悪口を言う者、強奪する者は、**神**の王国を相続することはありません。  
<span class="verse-number">11&ensp;</span>そして、あなたがたの中の幾人かは、こういう者でした。しかし、洗われ、また聖別され、また**主**イエスの御名において、私たちの**神**の**御霊**によって義と認められました。  
<span class="verse-number">12&ensp;</span>すべてのことは私にとって律法に合っていますが、その全部が役に立つとは限りません。すべてのことは私にとって律法に合っていますが、私は何事にも支配されることはありません。  
<span class="verse-number">13&ensp;</span>食べ物は胃のためであり、胃は食べ物のためですが、**神**は胃も食べ物も滅ぼされます。ですから、身体は不道徳な性行為のためのものでなく、**主**のためのものです。そして、**主**は身体のためです。  
<span class="verse-number">14&ensp;</span>そして、御自分の力を通して、**神**は**主**を復活させられ、私たちをも復活させてくださいます。  
<span class="verse-number">15&ensp;</span>あなたがたの身体は、キリストの一部であることを知らないのですか。それでは、私はキリストの一部を、売春婦の一部にしましょうか。そうなっては絶対にいけません。  
<span class="verse-number">16&ensp;</span>または、売春婦とつながっている者は、彼女と一体になっていることを知らないのですか。「この二人は一つの肉体になる。」と**神**は言われるからです。  
<span class="verse-number">17&ensp;</span>しかし、**主**とつながっている人は、**主**と一つの霊になります。  
<span class="verse-number">18&ensp;</span>性的な罪を避けなさい。人が犯すことごとくの罪は身体の外であるが、性的な罪を犯す者は、自分の身体に対し罪を犯しています。  
<span class="verse-number">19&ensp;</span>また、あなたがたの身体は、あなたがたの中におられる、**神**からの**聖霊**の神殿であることを知らないのですか。そして、あなたがたは、あなたがた自身のものでありません。  
<span class="verse-number">20&ensp;</span>あなたがたは値段を付けられて買われた者たちだからです。従って、**神**の物である、あなたがたの身体とあなたがたの霊で、**神**に栄光を捧げなさい。  

## 第7章

さて、あなたがたが私に書いたことについてですが、男が女を触れないことは良いことです。  
<span class="verse-number">2&ensp;</span>しかし、不道徳な性行為のゆえ、男は各自、自分自身の妻を持ち、また女は各自、自分自身の夫を持ちなさい。  
<span class="verse-number">3&ensp;</span>夫である者は自分の妻に、あるべき愛を与え、妻である者は同様に夫に与えなさい。  
<span class="verse-number">4&ensp;</span>妻である者は、自分自身の体を支配する権限はなく、夫にあります。また同様に、自分自身の体を支配する権限は夫にはなく、妻にあります。  
<span class="verse-number">5&ensp;</span>暫時、祈りと断食に専心するため、同意なしに、互いを騙し欺いてはいけません。そしてまた、共になりなさい。あなたがたの自制心のなさに乗じて、サタンがあなたがたを誘惑しないためです。  
<span class="verse-number">6&ensp;</span>しかし、私が言うこれは許可です。命令ではありません。  
<span class="verse-number">7&ensp;</span>すべての男が、私自身のようであることを私は望んでいるからです。しかし、それぞれ人は、**神**の賜物をもっています。この人はこんなふうに、あの人はあんなふうにです。  
<span class="verse-number">8&ensp;</span>しかし、私は結婚をしていない人々、それに、未亡人たちに言います。私のようなままでいることは、その人々にとって良いことです。  
<span class="verse-number">9&ensp;</span>しかし、自制できない者たちは、結婚させなさい。情念の燃える火より、結婚した方がましだからです。  
<span class="verse-number">10&ensp;</span>さて、結婚している人々に命じます。命じるのは私ではなく、**主**です。妻は夫から去るべきではありません。  
<span class="verse-number">11&ensp;</span>しかし、女がもし去るとしても、再婚しないままでいるべきであり、それとも、夫と和解をさせるべきです。そして、夫は妻を離婚してはいけません。  
<span class="verse-number">12&ensp;</span>そのほかの人々に言います。**主**ではなく、私が言うのです。兄弟の誰かに、信じていない妻があり、この女が共に住むことに同意するなら、離婚してはいけません。  
<span class="verse-number">13&ensp;</span>また、信じていない夫のいる妻は、この男が共に住むことに同意するなら、離婚してはいけません。  
<span class="verse-number">14&ensp;</span>信じていない夫は妻によって聖別され、信じていない者の妻は夫によって聖別されているからです。そうでなければ、あなたがたの子どもたちは汚れていますが、今は聖なる者たちです。  
<span class="verse-number">15&ensp;</span>しかし、信じていない者自身が去るのなら、去らせなさい。兄弟も姉妹もこういう場合、拘束の下にはいないのです。しかし**神**は私たちを平和へ召されたのです。  
<span class="verse-number">16&ensp;</span>なぜなら、妻よ、あなたは、自分の夫を救えるかどうか、どうしてわかりますか。または、夫よ、あなたはあなたの妻を救えるかどうか、どうしてわかりますか。  
<span class="verse-number">17&ensp;</span>しかし、**神**は一人一人に分け与えられたのだから、**主**が一人一人を召されたように、各自一人一人歩ませなさい。そして、私はすべての教会にこのように定めます。  
<span class="verse-number">18&ensp;</span>誰かが、割礼の状態で召されのですか。その者を割礼前の状態に戻してはいけません。誰かが、割礼のないままで召されたのですか。その者に割礼を受けさせてはいけません。  
<span class="verse-number">19&ensp;</span>割礼は無意味であり、無割礼も無意味なのです。そうでなく、問題は**神**の命令を守ることなのです。  
<span class="verse-number">20&ensp;</span>各自、一人一人は、召された召命にとどまりなさい。  
<span class="verse-number">21&ensp;</span>奴隷の状態の時、召されたのですか。それを心にかけてはいけません。でも、もし、自由になれることがあれば、その機会をむしろ利用しなさい。  
<span class="verse-number">22&ensp;</span>奴隷の状態で、**主**に召された人は、**主**の自由人だからです。同様に、自由人のままで召された人は、キリストの奴隷です。  
<span class="verse-number">23&ensp;</span>あなたがたは値を付けて買い取られたのです。人間の奴隷になってはいけません。  
<span class="verse-number">24&ensp;</span>兄弟たちよ、一人一人は、召されたその状態で**神**と共にとどまりなさい。  
<span class="verse-number">25&ensp;</span>さて、処女たちについてですが、私は**主**からの命令はありません。しかし、**主**の哀れみにより、**主**に忠実な者とさせていただいた一人として、私は私の判断を伝えています。  
<span class="verse-number">26&ensp;</span>目の当たりの危機であるなら、男はそのままの状態にどどまるのが良いと、私は思います。  
<span class="verse-number">27&ensp;</span>妻と結ばれていますか。自由になることを求めてはいけません。妻と結ばれていないのですか。妻を求めてはいけません。  
<span class="verse-number">28&ensp;</span>しかし、もしあなたが結婚したとしても、罪を犯したことにはなりません。そして、もし処女が結婚したとしても、その人は罪を犯したことにはなりません。しかし、そうすると、彼らは肉において問題が起こりますが、私はあなたがたを守りたいのです。  
<span class="verse-number">29&ensp;</span>しかし、兄弟たちよ、私はこのことを言います。時は短縮されたから、妻がいる者たちは、妻がいない者たちのように、  
<span class="verse-number">30&ensp;</span>そして、泣く者たちは泣かない者たちのよう、喜ぶ者たちは喜ばない者たちのよう、買う者たちは持っていない者たちのように、  
<span class="verse-number">31&ensp;</span>そして、世を己のために使う者たちは、使い切らないように。この世の有様は過ぎ去っていくからです。  
<span class="verse-number">32&ensp;</span>しかし、あなたがたは心配しないでほしいのです。結婚していない者は、**主**のこと、つまり**主**をどう喜ばせるかを、心配するからです。  
<span class="verse-number">33&ensp;</span>しかし、結婚している者は、世のこと、つまりどう妻を喜ばせるかを心配します。  
<span class="verse-number">34&ensp;</span>妻と処女は違います。結婚していない女は、体も霊も聖なるものであるように、**主**の物事を心配します。しかし、結婚している女は、夫を喜ばせるために、世の事を心配します。  
<span class="verse-number">35&ensp;</span>ついで、あなたがた自身の益のため、私はこれを言いますが、束縛するためではなく、適切なことのため、また雑念を持たず、**主**に仕えるためです。  
<span class="verse-number">36&ensp;</span>しかし、いかなる男であれ、自分の処女に対して不適切な振舞いをしていると思ったとしても、その処女が花の年頃を過ぎていて、そうすべきなら、その男の思うようにさせなさい。彼は罪を犯していません。結婚させなさい。  
<span class="verse-number">37&ensp;</span>しかし、その必要がなく、心かたく、しっかり立ち、自分の意志を支配する力を持っていて、しかも心中、断固とした決意で自分の処女を守る男は、良い行ないをしています。  
<span class="verse-number">38&ensp;</span>それに、女を結婚させる男はは良い行ないをしていますが、結婚させない男はもっと良い行ないをしています。  
<span class="verse-number">39&ensp;</span>夫が生きている間は、妻は法律に縛られています。しかし、もし女の夫が眠につけば、誰と結婚しようと自由ですが、**主**にある者に限ります。  
<span class="verse-number">40&ensp;</span>しかし、私の判断では、もしそのままとどまれば、女はもっと幸せです。そして、私も**神**の**御霊**を受けていると思います。  

## 第8章

さて、偶像に捧げられた物についてですが、私たち全員、知識を持っているのを、私たちは知っています。知識は人をふくらませるが、愛は建て上げます。  
<span class="verse-number">2&ensp;</span>そしてもし、何でも知っていると思う者がいるなら、その者は、知るべき事さえも知りません。  
<span class="verse-number">3&ensp;</span>しかし、**神**を愛する者であれば、その者は**神**に知られています。  
<span class="verse-number">4&ensp;</span>それで、偶像に捧げられた物についてですが、偶像は世には無であり、**神**は唯一の方しかおられないことを、私たちは知っています。  
<span class="verse-number">5&ensp;</span>なぜなら、それに天か地に、神と呼ばれるものがいたとしても（神も主も多くいるが）、  
<span class="verse-number">6&ensp;</span>私たちにとって、**父**なる**神**はお一人であり、すべてのものはこの方から出ており、そして私たちもこの方のためにいます。また、**主**イエス・キリストはお一人であり、すべてのものはその方を通してであり、そして私たちはその方を通して生きています。  
<span class="verse-number">7&ensp;</span>しかし、すべての者たちの中にこの知識があるわけはないのです。それで、ある者たちは、偶像に親しみの気持ちを持ち、これまで、偶像に捧げられたものとしてそれを食べます。ですからその者たちの良心は弱く、汚れているのです。  
<span class="verse-number">8&ensp;</span>しかし、食物が私たちを**神**の近くに引き寄せることはありません。食べても豊かにならないし、また食べなくても人後に遅れることはないからです。  
<span class="verse-number">9&ensp;</span>しかし、あなたがたのこの自由裁量が、弱い人たちにとってのつまずきの石にならないよう、気をつけなさい。  
<span class="verse-number">10&ensp;</span>なぜなら、もし知識があるあなたがたが、偶像の宮で食事をしているのを、判断力の弱い者が見れば、偶像に捧げられたものを食べるようと、勇気付けるではありませんか。  
<span class="verse-number">11&ensp;</span>すると、キリストが、この者のために死んでくださったあの弱い兄弟は、あなたの知識のせいで、滅びることになりませんか。  
<span class="verse-number">12&ensp;</span>しかし、このようにあなたがたは、兄弟たちに対して罪を犯し、彼らの弱い良心に傷をつけ、キリストに対して罪を犯しているのです。  
<span class="verse-number">13&ensp;</span>従って、もし食物が私の兄弟をつまずかせるなら、私は自分の兄弟をつまずかせないため、二度と再び決して肉は食べません。  

## 第9章

私は使徒ではないのですか。私は自由ではないのですか。私たちの**主**イエス・キリストに、私は会わなかったのですか。あなたがたは、**主**にあって私の働きではないのですか。  
<span class="verse-number">2&ensp;</span>私が他の人には使徒でないとしても、それでもあなたがたにとって使徒です。あなたがたは、**主**にあって私の使徒職の印です。  
<span class="verse-number">3&ensp;</span>私を取り調べる者たちへの弁明は、次の通りです。  
<span class="verse-number">4&ensp;</span>私たちは、飲食する権利がないのですか。  
<span class="verse-number">5&ensp;</span>他の使徒たち、**主**の兄弟たち、ケパと同様に、私たちは姉妹である妻を伴う権利がないのですか。  
<span class="verse-number">6&ensp;</span>または、私とバルナバだけは、仕事をしない権利がないのですか。  
<span class="verse-number">7&ensp;</span>自分の費用で、どんな時、誰が戦争に行きますか。ぶどう畑を作り、その果実を食べない者は誰ですか。または、群れを飼い、その乳を飲まない者は、誰ですか。  
<span class="verse-number">8&ensp;</span>私はただの男としてこれらのことを言っていますか。それともまた、律法も同じ事を言っていませんか。  
<span class="verse-number">9&ensp;</span>なぜなら、モーセの律法に、「穀物を脱穀している牛に、口輪でおおいをしてはいけない。」と書いてあります。**神**は牛のことを、心にかけてくださっているのですか。  
<span class="verse-number">10&ensp;</span>または、**神**はそれを完全に私たちのためだけに、言っておられるのですか。耕す者は希望の内に耕し、希望の内に脱穀する者は、その耕す者の希望の取り分のいただくべきです。これは完全に、私たちのために書かれています。  
<span class="verse-number">11&ensp;</span>私たちがあなたがたのために霊的なものを蒔いたのなら、あなたがたからの物質的なものを刈り入れるとしても、それはすごく負担なことでしょうか。  
<span class="verse-number">12&ensp;</span>もし他の人々があなたがたに対して、この取り分の権利があるのなら、私たちは、なおさらのことではありませんか。しかし、私たちはこの権利を行使したことなく、キリストの福音を妨げないよう、すべてのことを忍んでいます。  
<span class="verse-number">13&ensp;</span>聖なるものに仕える者たちは、神殿の物を食べ、また祭壇に仕える者たちは、祭壇の捧げ物の分配を食べることを知らないのですか。  
<span class="verse-number">14&ensp;</span>同様に、福音を説く者たちは、福音によって生計を立てる、と**主**は定められました。  
<span class="verse-number">15&ensp;</span>しかし、私自身はこれらのどれも使ってはいませんし、私にそうなるよう、これらのことを書いたわけではありません。私にとって、誰であれ、私の光り輝くものを無にするなら、私は死んだ方がましです。  
<span class="verse-number">16&ensp;</span>私は福音を説いても、光り輝くものは何もないからです。私に科せられた必然だからです。そうです、もし福音を説かなければ、それは、私には災です。  
<span class="verse-number">17&ensp;</span>なぜなら、喜んでこれを実行すれば、私は、賞を手にします。しかし、私の意に反したとしても、私はその摂理は与えられたのです。  
<span class="verse-number">18&ensp;</span>では、私の賞は何か。それは、私が福音を説く時、キリストの福音を代価なしで示すため、福音にある私の権威を私が悪用しないためなのです。  
<span class="verse-number">19&ensp;</span>なぜなら、私はすべての人から自由であるにもかかわらず、より多く勝ち取るため、私、自分自身を、すべての人にとっての奴隷としたのです。  
<span class="verse-number">20&ensp;</span>そして、ユダヤ人を勝ち取るため、ユダヤ人に対してはユダヤ人になり、律法の下にいる人々を勝ち取るため、律法の下にいる人々に対しては、律法の下にいる者と私はなりました。  
<span class="verse-number">21&ensp;</span>また、律法のない人々に対しては、私は律法のない者になります。（**神**に対しての律法がないのではなく、それどころか、私はキリストに対しての律法の下にいます。）それは律法のない人々を勝ち取るためです。  
<span class="verse-number">22&ensp;</span>弱い人々を勝ち取るため、私は弱くなりました。私は何とかして数人でも救うことができるように、すべての人に合わせて私は何にでもなりました。  
<span class="verse-number">23&ensp;</span>そして、私はこれを福音のためにしています。あなたがたと共に福音を共有する者となるためです。  
<span class="verse-number">24&ensp;</span>競争で参加する者は全員走るが、賞を受けるのは一人であることを知らないのですか。賞[^2]を獲得できるように走りなさい。  
<span class="verse-number">25&ensp;</span>そして、賞を受けるために競争する者たち全員は、すべてのことに節制します。それに、彼らは朽ちる冠を受けるためにやっていますが、私たちは朽ちることのない冠のためです。  
<span class="verse-number">26&ensp;</span>すなわち、私はこのように目標を持たずには走っていません。このように、空を打つような拳闘をしません。  
<span class="verse-number">27&ensp;</span>しかし、人に語った後、私自身が失格者にならぬよう、自分の体を律し、支配下においています。  

## 第10章

その上、兄弟たちよ、私たちの父祖たちは全員、雲の下にいたこと、また全員が海中を通り抜けて来たことを、あなたがたが知らぬままでいて欲しくはありません。  
<span class="verse-number">2&ensp;</span>雲と海の中で、全員モーセの方に浸されました。  
<span class="verse-number">3&ensp;</span>全員、同じ霊的な食べ物を食べました。  
<span class="verse-number">4&ensp;</span>そして、全員、同じ霊的な飲物を飲みました。父祖たちは、彼らについてきた霊的な岩から飲みました。その岩はキリストであられたからです。  
<span class="verse-number">5&ensp;</span>しかし、**神**はその者たちの大多数を喜ばれませんでした。そこで、彼らの体は荒野に散らされました。  
<span class="verse-number">6&ensp;</span>それで、彼らが悪事を渇望したと同様、私たちも彼らが渇望した悪事をしないために、これらの事は私たちの鑑になりました。  
<span class="verse-number">7&ensp;</span>それで、彼らの中のある者たちのように、偶像崇拝者になってはいけません。「民は食べたり飲んだりするために座り、遊ぶために立ち上がった。」と書いてあるとおりです。  
<span class="verse-number">8&ensp;</span>また、私たちは性的な罪を犯してはいけません。そして、彼らの中、犯した者、一日に二万三千人が倒れたのです。  
<span class="verse-number">9&ensp;</span>また、ある者たちがしたように、私たちはキリストを試みてはいけません。彼らの中、試みた者たちは、あの蛇らに滅ぼされました。  
<span class="verse-number">10&ensp;</span>また、ある者たちがしたように、不平を言ってはいけません。彼らの中のある者たちも、不平を言ってあの滅ぼす人に滅ぼされました。  
<span class="verse-number">11&ensp;</span>さて、これらのことはすべてその者たちに、鑑として行なわれたのです。そして、各時代の週末にいる私たちへの、警告のために書かれました。  
<span class="verse-number">12&ensp;</span>したがって、自分が立っていると思う者は、倒れないように気をつけなさい。  
<span class="verse-number">13&ensp;</span>あなたがたを襲う試みは人には常のものです。しかし**神**は信頼できる方であられるので、あなたがたが耐え忍ぶことができないほどの試みは、お許しになりません。しかも、その試みには、あなたがたが耐え忍ぶことができるよう、逃れる道が添えてあります。  
<span class="verse-number">14&ensp;</span>したがって、私の愛する者たちよ、偶像崇拝から逃れなさい。  
<span class="verse-number">15&ensp;</span>私はあなたがたが賢い人として話をしています。自分たちで、私の言うことを判断しなさい。  
<span class="verse-number">16&ensp;</span>私たちの祝福する、祝福のカップは、キリストの血の交流ではないのですか。私たちの裂くパンは、キリストの体の交流ではないのですか。  
<span class="verse-number">17&ensp;</span>なぜなら、私たちは大勢でも、一つのパンであり、一つの体だからです。なぜなら、私たち全員、その一つのパンを共に頂いているからです。  
<span class="verse-number">18&ensp;</span>肉に関してのイスラエルを見なさい。捧げ物を食べる者たちは、祭壇の交流をしていないのですか。  
<span class="verse-number">19&ensp;</span>それでは、私は何を言っているのでしょう。偶像は確かなものですか。また偶像に捧げられた物は確かなものですか。  
<span class="verse-number">20&ensp;</span>それどころか、異邦人たちが捧げる物は、**神**にではなく、悪霊に捧げているのです。私はあなたがたが悪霊と交流する者であって欲しくはありません。  
<span class="verse-number">21&ensp;</span>あなたがたは**主**のカップを飲んで、悪霊のカップを飲むことはできません。**主**の食卓に付き、悪霊の食卓には付けません。  
<span class="verse-number">22&ensp;</span>それとも、私たちは**主**にねたみをおこさせますか。私たちは**主**より強いのですか。  
<span class="verse-number">23&ensp;</span>すべてのことは私にとって合法であるが、すべてのことが役に立っているとは限りません。すべてのことは私にとって許されていますが、すべてのことが私を向上させるとは限りません。  
<span class="verse-number">24&ensp;</span>誰もが、自分の利益を求めずに、互いの利益を求めなさい。  
<span class="verse-number">25&ensp;</span>良心[^3]の故に、何も問わずに、市場の肉屋で売られている物は何でも食べなさい。  
<span class="verse-number">26&ensp;</span>地、そして、そこに満ち溢れているものは**主**のものだからです。  
<span class="verse-number">27&ensp;</span>もし信者でない者たちの一人が、あなたがたを招待したら、行きたければ行き、そして良心の故に何も問わずに、あなたがたの前に置かれた物は何であれ食べなさい。  
<span class="verse-number">28&ensp;</span>しかし、もし誰かあなたがたに、「これは偶像に捧げられた物だ。」と言ったとしたら、あなたがたにそう教えた人のためにも、また良心の故にも、食べてはいけません。地とその満ち溢れる物は、**主**のものだからです。  
<span class="verse-number">29&ensp;</span>「良心」、あなた自身の良心ではなく、他人の良心のことを、私は言っています。なぜなら、私の自由がなぜ他人によって裁かれるのですか。  
<span class="verse-number">30&ensp;</span>しかし、もし私が感謝をもって頂くのなら、なぜ私が感謝して頂くものを、人は悪く言うのですか。  
<span class="verse-number">31&ensp;</span>ですから、何を食べようが、飲もうが、また何をしようが、すべて、**神**の栄光のためにしなさい。  
<span class="verse-number">32&ensp;</span>ユダヤ人にも、ギリシャ人にも、**神**の教会にも、妨害するものになってはいけません。  
<span class="verse-number">33&ensp;</span>私も同様に、自分自身の益を求めず、多くの人が救われるため、多くの人の益を求めています。  

## 第11章

私がキリストをまねるように、あなたがたも私をまねなさい。  
<span class="verse-number">2&ensp;</span>さて、兄弟たちよ。私はあなたがたを誉めます。あなたがたはすべてのことにおいて私を忘れず、また私があなたがたに手渡したことを守っているからです。  
<span class="verse-number">3&ensp;</span>しかし、すべての男の頭はキリストであられ、女の頭は夫であり、キリストの頭は**神**であられることを知ってほしいのです。  
<span class="verse-number">4&ensp;</span>頭を覆って祈ったり預言したりする男は皆、自分の頭を辱めています。  
<span class="verse-number">5&ensp;</span>しかし、頭を覆うことなく、祈ったり預言したりする女は、自分の頭を辱めています。剃った頭と同じだからです。  
<span class="verse-number">6&ensp;</span>もし女が、頭を覆っていなければ、毛を短く切らせなさい。しかし、女が短く切るのも剃るのも恥ずかしければ、覆わせなさい。  
<span class="verse-number">7&ensp;</span>男は、**神**の写しと栄光であるから、当然、頭を覆ってはいけません。しかし、女は男の栄光です。  
<span class="verse-number">8&ensp;</span>男は女から来たのではなく、女は男から来たからです。  
<span class="verse-number">9&ensp;</span>また、男は女のために創造されたのではなく、女は男のために創造されたからです。  
<span class="verse-number">10&ensp;</span>このために、御使いたちがいるのだから、女という者は頭に権威の象徴を置くべきです。  
<span class="verse-number">11&ensp;</span>しかし、**主**にあって、男は女から独立しているのではなく、また女も男から独立しているのでもありません。  
<span class="verse-number">12&ensp;</span>女は男から来るように、男は女を通して来ます。しかし、すべては**神**から来るのです。  
<span class="verse-number">13&ensp;</span>自分たちの間で判断しなさい。女は頭を覆わず**神**に祈るのは、あるべき姿でしょうか。  
<span class="verse-number">14&ensp;</span>もし男が長い頭髪をしていれば、その者にとって恥であるのは、自然の教えるところではありませんか。  
<span class="verse-number">15&ensp;</span>しかし、もし髪の長い女であれば、覆う物として与えられているのだから、それはその女にとっては栄光です。  
<span class="verse-number">16&ensp;</span>しかし、もし議論を望む者がいても、私たちも**神**の教会もそんな習慣はありません。  
<span class="verse-number">17&ensp;</span>私がこれらの指示を与えても、あなたがたを誉めているのではありません。なぜなら、もっと良くするためではなく、もっと悪くするためにあなたがたは集まるからです。  
<span class="verse-number">18&ensp;</span>なぜなら、先ず第一にあなたがたは教会として集まる時、あなたがたの間に分派がある、と私は聞いています。そして、私はある程度、信じています。  
<span class="verse-number">19&ensp;</span>あなたがたの中で正しいと認められる人たちが、明白になるには、あなたがたの中に分派もなければなりません。  
<span class="verse-number">20&ensp;</span>したがって、あなたがたは一か所に集まる時、**主**の晩餐を食べるためではありません。  
<span class="verse-number">21&ensp;</span>そこで、食べる時、各自、自分の夕食を人に先んじて食べます。そのため、空腹の者もいれば、また、酔っている者もいます。  
<span class="verse-number">22&ensp;</span>何たることですか。あなたがたは飲み食いする家がないのですか。それとも、あなたがたは**神**の教会を見下げ、無一物の者たちを辱めるのですか。私はあなたがたに何を言えばよいのか。このことで誉めましょうか。私は誉めません。  
<span class="verse-number">23&ensp;</span>なぜなら、あなたがたに手渡した物は、私が**主**から受け取ったものです。**主**イエスが裏切られたその夜、**主**はパンを取られ、  
<span class="verse-number">24&ensp;</span>そして、**主**が感謝をし、そのパンを裂き、言われました。「取って、食べなさい。これはあなたがたのために裂かれた私の体です。私の記憶のために、このことをしなさい。  
<span class="verse-number">25&ensp;</span>そして同じように夕食後、イエスはカップを取り、言われました。「このカップは、私の血による新しい契約である。私を忘れないため、これを飲むたびにこうしなさい。」  
<span class="verse-number">26&ensp;</span>このパンを食べ、このカップを飲むたびに、**主**が来るまで、あなたがたは**主**の死を示すことになります。  
<span class="verse-number">27&ensp;</span>したがって、ふさわしくないままでこのパンを食べたり、**主**のカップを飲んだりする人は誰でも、**主**の体と血に対して罪があります。  
<span class="verse-number">28&ensp;</span>それで、自らを裁かせなさい。そうして人が、パンを食べるままに、カップを飲むままにさせなさい。  
<span class="verse-number">29&ensp;</span>なぜなら、ふさわしくないままで食べたり飲んだりする者は、**主**の体をわきまえず、自らを裁くからです。  
<span class="verse-number">30&ensp;</span>この理由で、あなたがたの中の多くは虚弱になり、病気になり、そして死んだ多くの者がいます。  
<span class="verse-number">31&ensp;</span>しかし、私たちが私たち自身を裁くなら、私たちは裁かれることはありません。  
<span class="verse-number">32&ensp;</span>しかし、私たちが裁かれる時、私たちは世と共に断罪されないため、私たちは**主**によって懲らしめられます。  
<span class="verse-number">33&ensp;</span>したがって、私の兄弟たちよ、食べるために集まった時、互いに待ちなさい。  
<span class="verse-number">34&ensp;</span>しかし、空腹の者がいれば、その者は家で食べさせなさい。裁きのために、あなたがたが集うことがないためです。残りの課題は私が行った時、指示します。  

## 第12章

さて、兄弟たちよ、霊的な賜物について、私はあなたがたが無知であってほしくありません。  
<span class="verse-number">2&ensp;</span>あなたがたは異邦人であり、どういうふうに導かれたとしても、物言わぬ偶像のもとに連れて行かれたことは、あなたがたは知っています。  
<span class="verse-number">3&ensp;</span>ですから、あなたがたに、私は知ってもらいます。**神**の**御霊**により話す者は、イエスを呪われた者と呼びません。また**聖霊**によらなければ、誰もイエスは**主**であると言えないのです。  
<span class="verse-number">4&ensp;</span>そして、賜物はいろいろありますが、同一の**御霊**です。  
<span class="verse-number">5&ensp;</span>そして、奉仕はいろいろありますが、同一の**主**です。  
<span class="verse-number">6&ensp;</span>そして、働きには種類はいろいろありますが、すべての中で、すべてを行なうのは同一の**神**です。  
<span class="verse-number">7&ensp;</span>しかし、**御霊**の啓示が一人一人に与えられているのは、全員の利益のためです。  
<span class="verse-number">8&ensp;</span>なぜなら、**御霊**を通してある者には智恵の言葉が与えられ、またある者には、同一の**御霊**によって知識の言葉が、  
<span class="verse-number">9&ensp;</span>そして、ある者には同一の**御霊**によって信仰が、またある者には同一の**御霊**によっていやしの賜物が、  
<span class="verse-number">10&ensp;</span>そしてまた、ある者には力あるわざ、ある者には預言、ある者には霊を判別する力、ある者には違う言語、ある者には言語の解釈の力が与えられています。  
<span class="verse-number">11&ensp;</span>しかし、これらすべてのことは、同一の同じ**御霊**が、御自分のご意志により一人一人に分け与えられます。  
<span class="verse-number">12&ensp;</span>なぜなら、多くの部分からなる体は一つであり、多くあるが、全部の部分で一体を成しているように、キリストも同じです。  
<span class="verse-number">13&ensp;</span>そしてまた、ユダヤ人であろうとギリシャ人であろうと、奴隷であろうと自由人であろうと、私たちは唯一の**御霊**によって一つの体の中に浸され、そして、私たちは唯一の**御霊**を飲むことになりました。  
<span class="verse-number">14&ensp;</span>そしてまた、体は多くの部分から成るもので、一部分ではありません。  
<span class="verse-number">15&ensp;</span>もし、「私は手ではないから、私は体の部分ではない。」と足は言っても、体の部分の一つではありませんか。  
<span class="verse-number">16&ensp;</span>そしてもし、耳が、「私は目ではないから、体のものではない。」と、言ったとしても、体の部分ではないのですか。  
<span class="verse-number">17&ensp;</span>もし体全体が目ならば、聞く所はどこですか。もし全体は聞く所ならば、嗅ぐ所はどこですか。  
<span class="verse-number">18&ensp;</span>しかし、**神**は今、その体の一部一部を、御自分の思うまま、この体に配置されました。  
<span class="verse-number">19&ensp;</span>そしてもし、各部がすべて一つの部分ならば、体はどこですか。  
<span class="verse-number">20&ensp;</span>しかし、さて、体には本当に多くの部分はあるが、体は一つです。  
<span class="verse-number">21&ensp;</span>そして、目は手に、「私はおまえは要らない。」と言えません。または、頭は足に、「私はおまえは要らない。」とも言えません。  
<span class="verse-number">22&ensp;</span>しかし、体の弱く見える部分は、なおさら必要です。  
<span class="verse-number">23&ensp;</span>そして、私たちがほかより立派でないと思っている体の部分に、ほかより立派な呼び方をし、そして、私たちの見苦しい部分には、他より上品な呼び方をします。  
<span class="verse-number">24&ensp;</span>しかし、私たちの立派な部分には、何も必要とするものはありません。しかし、欠落している部分に、**神**は立派さをもっと与え、この体をお作りになりました。  
<span class="verse-number">25&ensp;</span>それは、体に分裂がなく、この各部分が互いに同じ心配を共にするためです。  
<span class="verse-number">26&ensp;</span>そして、一つの部分が苦しめば、すべての部分は苦しみます。また、一つの部分は誉れを受ければ、すべての部分は共に喜びます。  
<span class="verse-number">27&ensp;</span>それで、あなたがたはキリストの体であり、一人一人はその部分です。  
<span class="verse-number">28&ensp;</span>そして、**神**は教会に置かれたのは、まず第一に使徒たち、第二に預言者たち、第三に教師たち、その後に力ある業、その後にいやしの賜物、補助、先導者、複数の言語です。  
<span class="verse-number">29&ensp;</span>全員、使徒ですか。全員、預言者ですか。全員、教師ですか。全員、力ある業を行なう人ですか。  
<span class="verse-number">30&ensp;</span>全員がいやしの賜物を持っているのですか。全員は複数の言語で話をしますか。全員、通訳しますか。  
<span class="verse-number">31&ensp;</span>しかし、最高の賜物を熱心に求めなさい。そして、私はあなたがたにさらにもっとすばらしい道を教えます。  

## 第13章

もし私が人間の言葉、御使いの言葉で話しても、愛がなければ、私は音を立てるどらか、または騒々しいシンバルになってしまいます。  
<span class="verse-number">2&ensp;</span>そして、私に預言の賜物があり、すべての奥義と知識を理解し、また山を動かせるほどの信仰のすべてがあったとしても、愛がなければ、無です。  
<span class="verse-number">3&ensp;</span>そして、私の全財産を、貧しい人たちに食事のために与えても、また自分を燔祭として捧げても、愛がなければ、その燔祭は私に何の益ももたらしません。  
<span class="verse-number">4&ensp;</span>愛は長期に耐え、思いやりがあります。愛はねたみません。愛そのものは自慢しませんし、高慢にはなりません。  
<span class="verse-number">5&ensp;</span>愛は無礼なことはせず、己の利益は求めず、怒らず、悪を考えず、  
<span class="verse-number">6&ensp;</span>不正を喜ばず、真理を喜び、  
<span class="verse-number">7&ensp;</span>何事をも忍び、何事をも信じ、何事をも希望し、何事をも耐えます。  
<span class="verse-number">8&ensp;</span>愛は決してしおれることはありません。しかし、今ある預言、それらは廃止され、今ある諸言語、それは使用されなくなり、今ある知識、それは廃止されます。  
<span class="verse-number">9&ensp;</span>なぜなら、私たちは一部しか知らない、一部しか預言しないからです。  
<span class="verse-number">10&ensp;</span>しかし、完全なるものが来た時、部分なるものは廃止になります。  
<span class="verse-number">11&ensp;</span>私が赤子であった時、赤子として話し、赤子として理解し、赤子として考えたが、成人になった時、赤子のものは捨てました。  
<span class="verse-number">12&ensp;</span>なぜなら、私たちは今、鏡で謎を見ているが、その時、顔と顔とを合わせます。今私が知っているのは、部分なのです。その時、私が知られているように、すべて、知るところとなります。  
<span class="verse-number">13&ensp;</span>そして今、残っているのは、信仰、希望、愛、この三つです。しかし、この中で最も偉大なのは、愛です。  

## 第14章

愛を追求しなさい。その上、熱心に、霊的な賜物を探し求めなさい。特に、預言ができるようになるためです。  
<span class="verse-number">2&ensp;</span>外国語で話す者は、人間にではなく、**神**に話しているからです。それで誰も理解できません。霊によってその者は奥義を話すのです。  
<span class="verse-number">3&ensp;</span>しかし、預言する者は、人たちを建て上げることを、鼓舞を、慰めを語ります。  
<span class="verse-number">4&ensp;</span>外国語で語る者は自らを建て上げるが、預言する者は教会を建て上げます。  
<span class="verse-number">5&ensp;</span>あなたがた全員が、外国語で語ることを私は望んでいるが、それよりも、あなたがた全員が、預言するのを、私は望んでいます。なぜなら、自ら通訳しない限り、外国語で語っても、教会の成長のためになりません。外国語で話す者より、預言する者のほうが優れているからです。  
<span class="verse-number">6&ensp;</span>しかし今、兄弟たちよ、もし私があなたがたの所に行き、外国語で話しても、啓示、知識、預言、教理によって説かなければ、あなたがたに何か利益がありますか。  
<span class="verse-number">7&ensp;</span>笛や立琴のような命のない物でさえも、音を出す時、音に違いがなければ、何が吹かれ、また、何がつま弾かれているのか、どうして分かるのですか。  
<span class="verse-number">8&ensp;</span>もし、ラッパが、不明確な音を出せば、誰が自ら、戦いの準備をしますか。  
<span class="verse-number">9&ensp;</span>同様に、あなたがたも、理解できる、安易な言葉を口にしなければ、何が話されたのかどうして理解されるでしょう。空に話すことになるからです。  
<span class="verse-number">10&ensp;</span>多分、世には、それは多くの種類の言語はあるでしょうが、その中に、意味のないものは何一つありません。  
<span class="verse-number">11&ensp;</span>したがって、もし私はその声の力を知らなければ、私は話す人にとって外国人であり、話す人は私にとって外国人です。  
<span class="verse-number">12&ensp;</span>同様に、あなたがたも霊的な賜物を熱心に求めているのだから、教会を築き上げるために、あなたがたも豊かになるように求めなさい。  
<span class="verse-number">13&ensp;</span>それで、外国語で説く者は、自ら通訳できるように祈りなさい。  
<span class="verse-number">14&ensp;</span>なぜなら、もし私が外国語で祈れば、私の霊は祈っても、私の理解力は豊かになりません。  
<span class="verse-number">15&ensp;</span>では、結論は何でしょう。私の霊で祈り、また私の理解力で祈ります。私の霊で聖歌を歌い、また私の理解力で聖歌を歌います。  
<span class="verse-number">16&ensp;</span>そうではなければ、あなたが霊で祝福しても、学問と無縁な所にいる人は、あなたの言うことがわからないのに、どうしてあなたの祝福に「アーメン」[^4]と言えますか。  
<span class="verse-number">17&ensp;</span>なぜなら、あなたは見事に祝福していますが、他の人は成長しません。  
<span class="verse-number">18&ensp;</span>あなたがたの誰よりも、私は複数の言語で話せることを、私の**神**に感謝しています。  
<span class="verse-number">19&ensp;</span>しかし、教会では、人たちに教えることができるように、外国語の一万の言葉で説くより、むしろ、私は自分の理解による五語で話したいのです。  
<span class="verse-number">20&ensp;</span>兄弟たちよ、知性では、子どもであってはいけません。しかし、悪意では赤子になり、知性では成人でありなさい。  
<span class="verse-number">21&ensp;</span>律法に書いてあります。「外国語、つまり他の唇で、この国民に私は話します。しかし、この方法を使っても、それでも彼らは私の言葉を聞こうとしない。」と**主**は言われます。  
<span class="verse-number">22&ensp;</span>それで、外国語は信じる者のためではなく、信じない者に対してしるしであるが、預言は信じない者のためではなく、信じる者たちのためです。  
<span class="verse-number">23&ensp;</span>したがってもし、教会全員が一カ所に集まった時、全員が外国語で話せば、学問がない者、あるいは信じない者が入って来れば、その者たちはあなたがたは狂っていると言いませんか。  
<span class="verse-number">24&ensp;</span>しかし、全員が預言すれば、信者でない者、または学問がない者が入って来れば、その者は全員に罪ありとされ、問いただされます。  
<span class="verse-number">25&ensp;</span>そして、このようにその人の心の秘密は現われます。それで、その人は地に身を倒し、**神**を礼拝し、**神**はまことにあなたがたの間におられると、口にします。  
<span class="verse-number">26&ensp;</span>では、どうでしょう、兄弟たちよ、あなたがたが集まった時はいつも、各自、詩編の歌があり、教えがあり、外国語があり、啓示があり、解釈があります。これらすべてを、人を築き上げるために実行しなさい。  
<span class="verse-number">27&ensp;</span>もし外国語を話す者がいれば、二人、多くても三人を交互に話しさせ、通訳は一人にさせなさい。  
<span class="verse-number">28&ensp;</span>しかし、もし通訳者がいなければ、教会では話す者は黙らせなさい。そして、その者を**神**と彼自身に話すようにさせなさい。  
<span class="verse-number">29&ensp;</span>預言者を二人か三人に話させ、そしてその他の者たちに判断させなさい。  
<span class="verse-number">30&ensp;</span>しかし、もし座っている別の誰かに何か啓示が与えられたら、先の一人を黙らせなさい。  
<span class="verse-number">31&ensp;</span>あなたがた全員は預言することができるのだから、全員が学び、全員が励ましを受けることができます。  
<span class="verse-number">32&ensp;</span>そして預言者たちの霊は、預言者たちに服します。  
<span class="verse-number">33&ensp;</span>なぜなら、聖徒たちの全教会で行なわれているように、**神**は混乱の源ではなく、平和の源であられます。  
<span class="verse-number">34&ensp;</span>教会では、女たちは話すことは許されていないから、あなたがたの女たちは黙らせなさい。また女たちは従順であるべきと、律法も言っています。  
<span class="verse-number">35&ensp;</span>そして、女たちは教会で話すことは恥なのだから、女たちが何か質問があるなら、家で各自の夫に質問させなさい。  
<span class="verse-number">36&ensp;</span>あるいは、**神**の御言葉は、あなたがたから出たのですか。または、あなたがただけに届いたのですか。  
<span class="verse-number">37&ensp;</span>もし誰か、自分を預言者である、または霊的な者であると思っているなら、私があなたがたに書くことは、**主**の命令であることを認めなさい。  
<span class="verse-number">38&ensp;</span>しかし、誰であれ無知の者がいれば、その者は無知のままにしておきなさい。  
<span class="verse-number">39&ensp;</span>ですから、兄弟たちよ、預言できるように望みなさい。そして、外国語で話すことを禁じてはいけません。  
<span class="verse-number">40&ensp;</span>すべてのことを適切に、秩序正しく行ないなさい。  

## 第15章

それに、兄弟たちよ、私が以前あなたがたに説いた福音を、明確に伝えます。あなたがたはこれを受け入れ、そしてそれに立脚しているのです。  
<span class="verse-number">2&ensp;</span>もし、訳も分からず信じたのでなく、私の説いた言葉をしっかりつかんでいれば、あなたがたもこの福音を通してもう救われているのです。  
<span class="verse-number">3&ensp;</span>なぜなら、私が受け取ったものを、まずあなたがたに手渡しました。それは、キリストは聖書に従って私たちの罪のために死なれ、  
<span class="verse-number">4&ensp;</span>そして埋葬され、そして聖書に従い三日目に復活されました。  
<span class="verse-number">5&ensp;</span>そしてキリストはケパで目撃され、それから十二人に目撃され、  
<span class="verse-number">6&ensp;</span>その後、五百人以上の兄弟たちに、同時に目撃されました。その中の大多数は現在も残っているが、眠りについている者もいます。  
<span class="verse-number">7&ensp;</span>その後、イエスはヤコブに目撃され、それから使徒の全員に目撃されました。  
<span class="verse-number">8&ensp;</span>最終的に、未熟児で生まれた、私のような者に目撃されました。  
<span class="verse-number">9&ensp;</span>なぜなら、私は使徒たちの中のもっとも小さい者であり、**神**の教会を迫害した者で、使徒と呼ばれる価値のない者だからです。  
<span class="verse-number">10&ensp;</span>しかし、私が今あるのは、**神**の恵みによるものです。そして、**神**の私への恵みは、無意味ではなかったのです。私は他の使徒たちより多く苦労しました。しかし、私ではありません。私と共にあった**神**の恵みがしたのです。  
<span class="verse-number">11&ensp;</span>従って、私であろうが、他の使徒たちであろうが、私たちがそう説き、そしてあなたがたはそう信じました。  
<span class="verse-number">12&ensp;</span>それで、キリストは死人たちの中から復活された、と説かれているのに、あなたがたの中のある者たちは、なぜ、死人の復活はないと言うのですか。  
<span class="verse-number">13&ensp;</span>しかし、もし死人たちの復活がなければ、キリストは復活されていません。  
<span class="verse-number">14&ensp;</span>そして、もしキリストが復活されていなければ、私たちの説くことは空虚であり、あなたがたの信仰も空虚です。  
<span class="verse-number">15&ensp;</span>それで、もし、死人たちは復活しないのなら、**神**はキリストを復活させられたと、証言しているのだから、私たちは**神**の偽証人だと露見します。  
<span class="verse-number">16&ensp;</span>もし死人たちが復活しないのなら、キリストも復活されていません。  
<span class="verse-number">17&ensp;</span>そしてもし、キリストが復活されていないなら、あなたがたの信仰は何の役にも立たず、今もあなたがたは自らの罪の中にいます。  
<span class="verse-number">18&ensp;</span>そしてまた、キリストにあって眠った人たちは、滅んでしまったことになります。  
<span class="verse-number">19&ensp;</span>もし、私たちがキリストの希望を、この人生のみに持っているとしたら、私たちはすべての人の中で、最も哀れむべきものです。  
<span class="verse-number">20&ensp;</span>しかし、キリストはもうすでに、死人たちの中から復活され、眠っている人たちの最初の収穫となられています。  
<span class="verse-number">21&ensp;</span>というのは、人によって死がもたらされ、また同じく、人によって死人たちの復活ももたらされたからです。  
<span class="verse-number">22&ensp;</span>アダムによってすべての人は死に、同じように、キリストによってすべての人が生きかえらせるからです。  
<span class="verse-number">23&ensp;</span>しかし、各自、自らの順によります。最初の収穫であるキリスト、それからキリストがお出でになることに関係する者たちです。  
<span class="verse-number">24&ensp;</span>それで、終わりの時が来ます。イエスが**父**なる**神**に王国を手渡す時です。すべての支配を、すべての権威を、そして力を、イエスが終わらせる時です。  
<span class="verse-number">25&ensp;</span>なぜなら、敵をすべてご自分の足で服従させたまで、イエスは統治しなくてはならないからです。  
<span class="verse-number">26&ensp;</span>滅ぼされる最後の敵は、死です。  
<span class="verse-number">27&ensp;</span>なぜなら、「この方はすべてのものをご自分の足で足下に服従させられた。」「すべてのものはこの方の下に服従させられた。」と言うことは、すべてのものを、下に服従させた方は含まれていないことは、明らかです。  
<span class="verse-number">28&ensp;</span>そして、すべてのものがこの方に足下に服従させられたその時、すべてのものを**御子息**の足下に服従させた方に、**御子息**も服従されます。**神**はすべての中に、すべてになられるためです。  
<span class="verse-number">29&ensp;</span>そうでなく、もし死人たちが決して復活しないなら、死人たちのために浸礼を受ける者たちは、何をすることになるのですか。なぜ死人たちのために人は浸礼を受けるのですか。  
<span class="verse-number">30&ensp;</span>また、私たちは、二十四時間いつもなぜ危険の状態ですか。  
<span class="verse-number">31&ensp;</span>私たちの**主**キリスト・イエスにある、あなたがたの誇りにより私は断言します。私は毎日死んでいます。  
<span class="verse-number">32&ensp;</span>私が人間のやり方で、エペソで野獣と戦ったとしたら、それは私に何の得があるでしょうか。死人たちが復活しないならば、「さあ、飲み食いしようじゃないか。明日は死ぬのだから」と。  
<span class="verse-number">33&ensp;</span>騙されてはいけません。「悪い仲間は、良き習慣を損なう。」  
<span class="verse-number">34&ensp;</span>義に目覚めなさい。罪を犯してはいけません。**神**の知識の無い者たちもいるからです。あなたがたを辱めるため、このことを私は言います。  
<span class="verse-number">35&ensp;</span>しかし、こう言う者はいるでしょう。「死人たちはどういうふうにして復活させられますか。そして、彼らはどういう体で来ますか。」  
<span class="verse-number">36&ensp;</span>愚かな者よ。あなたの蒔くものは、死ななければ、生かされません。  
<span class="verse-number">37&ensp;</span>そして、あなたがたの蒔くものは、肉体になるものを蒔くのではなく、大麦であれ、他の穀物であれ、ただの穀物です。  
<span class="verse-number">38&ensp;</span>しかし、**神**はご自分自ら望まれる通りに、蒔いたものに体を与えられます。また一粒一粒の種にそれぞれの体を与えられます。  
<span class="verse-number">39&ensp;</span>肉はすべて同じではなく、一つに人の肉、もう一つに動物の肉、そして魚の肉、鳥の肉があります。  
<span class="verse-number">40&ensp;</span>天国の体もあれば、地球の体もあります。しかし、天国の輝きは、地球の輝きとは違います。  
<span class="verse-number">41&ensp;</span>太陽に独自の輝きがあり、月には別の輝きがあり、そして星々にも別々の輝きがあります。輝きに関しては、星それぞれで違います。  
<span class="verse-number">42&ensp;</span>そして、死人たちの復活も同様です。体は腐って蒔かれ、腐りなく復活させられます。  
<span class="verse-number">43&ensp;</span>恥で蒔かれ、輝きで復活させられます。力なく蒔かれ、力で復活させられます。  
<span class="verse-number">44&ensp;</span>自然の体で蒔かれ、霊的な体で復活させられます。自然の体もあれば、霊的な体もあるのです。  
<span class="verse-number">45&ensp;</span>そして、こう書いてあります。「最初の人であるアダムは、生き物となった。」最後のアダムは、命を与える**御霊**となられました。  
<span class="verse-number">46&ensp;</span>しかし、霊的な人が最初ではなく、自然の人が最初で、霊的な人はその後です。  
<span class="verse-number">47&ensp;</span>最初の人は土からで、土埃で造られています。第二の人は天からの**主**でおられます。  
<span class="verse-number">48&ensp;</span>同様に、土埃で造られたこの人に、土埃で造られている者たちは同じで、天国の人は、天国にいる者たちと同じです。  
<span class="verse-number">49&ensp;</span>そして、私たち、土埃で造られた者たちには姿があると同様、私たちも天国の人の姿をまといます。  
<span class="verse-number">50&ensp;</span>それで、兄弟たちよ。私はこのことを言います。肉と血液は**神**の王国を相続することができず、または腐る者は腐らないものを相続できません。  
<span class="verse-number">51&ensp;</span>見よ。私はあなたがたに奥義を教えます。私たちは全員が眠るわけではありません。しかし、私たち全員は変えられます。  
<span class="verse-number">52&ensp;</span>一瞬、瞬きする間、最終のラッパで変えられます。このラッパが鳴り、死人たちは腐らず復活させられ、私たちは変えられるからです。  
<span class="verse-number">53&ensp;</span>この腐るものは腐らないものを身につけなければならず、そしてこの死ぬべきものは、不死を身につけねばならないからです。  
<span class="verse-number">54&ensp;</span>それで、この腐るものは腐らないものを身につけ、また死ぬべきものが死ないものを身につける時、「死が勝利に飲み込まれた。」と書いてある御言葉は実現します。  
<span class="verse-number">55&ensp;</span>「死よ、お前の刺はどこにあるか。ハデス[^5]よ、お前の勝利はどこにあるか。」  
<span class="verse-number">56&ensp;</span>死の刺は罪であり、罪の力は律法です。  
<span class="verse-number">57&ensp;</span>しかし、我らの**主**イエス・キリストを通して、我らに勝利を与えてくださる**神**に感謝すべきです。  
<span class="verse-number">58&ensp;</span>ですから、私の愛する兄弟たちよ、あなたがたの働きは決して**主**にあって無駄ではないことを知り、しっかり立ち、動じず、常に十二分に**主**の働きをしなさい。  

## 第16章

さて、聖徒たちのための献金については、私はガラテヤの諸教会に指示したように、あなたがたもそうしなさい。  
<span class="verse-number">2&ensp;</span>あなたがたが受けた祝福に応じた貯えを、私が行った時、献金せずともよいように、週の最初の日に各自それを取っておきなさい。  
<span class="verse-number">3&ensp;</span>そして、あなたがたの手紙によって、あなたがたが任命する人であれば、誰であれ、その[^6]募金を持たせ、私が行ったら、エルサレムに遣わします。  
<span class="verse-number">4&ensp;</span>しかし、もし私が同行するのが適切なら、その人たちと私は共に行きます。  
<span class="verse-number">5&ensp;</span>そして、私はマケドニアを通り、通過する予定ですから、その時、私はあなたがたの所に行きます。  
<span class="verse-number">6&ensp;</span>それで、私は留まるか、または、冬をあなたがたと過ごす可能性もあります。私がどこへ行こうが、あなたがたが、私を旅行に送り出してくれるためです。  
<span class="verse-number">7&ensp;</span>なぜなら、今のところ、私は途中ではあなたがたに会うことは望みません。しかし、**主**が許してくだされば、しばらくあなたがたと共に滞在することを望みます。  
<span class="verse-number">8&ensp;</span>しかし、五旬節まで、私はエペソに滞在します。  
<span class="verse-number">9&ensp;</span>それは、大きくて、収穫が期待できる門が私に開いていて、そして敵対する者が大勢いるからです。  
<span class="verse-number">10&ensp;</span>さて、もしテモテが行ったら、恐怖を与えることなく、あなたがたと暮らせるようしっかり守りなさい。テモテは、私と同様、**主**の働きをしているのですから。  
<span class="verse-number">11&ensp;</span>ですから、誰にもテモテを見下げさせてはいけません。そして、兄弟たちと私は共に、テモテを待っているのですから、平和裏にテモテを旅に送り出してください。  
<span class="verse-number">12&ensp;</span>さて、私たちの兄弟であるアポロのことですが、兄弟たちと共にあなたがたの所へ行くように、私は強く勧めました。しかし、今のところ、アポロは行く気はありませんでした。しかし、都合が良ければ、アポロは行きます。  
<span class="verse-number">13&ensp;</span>気をつけていなさい。信仰に堅く立ち、男らしく、強くありなさい。  
<span class="verse-number">14&ensp;</span>行なうことは、すべて愛をもって行ないなさい。  
<span class="verse-number">15&ensp;</span>兄弟たちよ、私はあなたがたに勧めます。知っての通り、ステパノの家の者たちは、アカヤの最初の収穫であり、聖徒たちへの奉仕に自らを定めました。  
<span class="verse-number">16&ensp;</span>そのような人たちに、そして、私たちと共に働き、労働する人たちに、あなたがたは従うことを勧めます。  
<span class="verse-number">17&ensp;</span>ステパナとポルトナトとアカイコが来たことを、私は喜んでいます。あなたがたの欠けていたことを補ったからです。  
<span class="verse-number">18&ensp;</span>それで、彼らは私の霊とあなたがたの霊を休息させてくれたのだから、そのような人たちは認めなさい。  
<span class="verse-number">19&ensp;</span>アジアの諸教会はあなたがたにご挨拶をします。アクラとプリスキラは、その家にある教会と共に、**主**にあって心からあなたがたにご挨拶をします。  
<span class="verse-number">20&ensp;</span>兄弟たちは全員あなたがたにご挨拶をします。聖なる口づけで、互いに挨拶をしなさい。  
<span class="verse-number">21&ensp;</span>このご挨拶は、私、パウロ自らの手によるものです。  
<span class="verse-number">22&ensp;</span>もし誰であれ**主**イエス・キリストを愛さなければ、その者は呪われるようにしなさい。**主**よ、おいでください。  
<span class="verse-number">23&ensp;</span>私たちの**主**イエス・キリストの恵みが、あなたがたと共にありますように。
