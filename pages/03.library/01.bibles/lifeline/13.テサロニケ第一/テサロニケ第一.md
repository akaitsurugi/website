# テサロニケ第一

### テサロニケ人への手紙　第一

## 第1章

パウロ、シラス、そして、テモテから、**父**なる**神**と**主**イエス・キリストにあってテサロニケの人たちの教会へ。私たちの**父**であられる**神**と**主**イエス・キリストより、あなたがたに恵みと平安がありますように。  
<span class="verse-number">2&ensp;</span>私たちは祈りの中で、皆さんのことに触れ、いつも、**神**に感謝しています。  
<span class="verse-number">3&ensp;</span>私たちの**神**と**御父**の御前に、あなたがたの信仰のわざ、愛の働き、私たちの**主**イエス・キリストへの希望の忍耐を絶えず、忘れずにいます。  
<span class="verse-number">4&ensp;</span>愛する兄弟たちよ、皆さんは、**神**の指名であると、私たちは知っています。  
<span class="verse-number">5&ensp;</span>なぜなら、私たちの福音は言葉だけではなく、力と**聖霊**と強い確信によってあなたがたに伝えられました。あなたがたのために私たちは力になったので、私たちはどんな種類の人間であったかは、あなたがたは知っての通りです。  
<span class="verse-number">6&ensp;</span>そして、多くの苦しみの中、**聖霊**の喜びをもって、御言葉を受け入れたから、あなたがたは私たちと**主**を手本にしました。  
<span class="verse-number">7&ensp;</span>ですから、マケドニアとアカヤにいるすべて信じる人にとって、あなたがたは模範になったのです。  
<span class="verse-number">8&ensp;</span>なぜなら、マケドニアとアカヤでだけではなく、すべての所であなたがたから**主**の御言葉は響き渡っています。あなたがたの**神**に対する信仰は広がったので、私たちは何も言わなくてもよいのです。  
<span class="verse-number">9&ensp;</span>なぜなら、私たちは、どんなふうにして、あなたがたの中に入ったかを、また、生ける、すなわち真の**神**に仕えるため、あなたがたがいかにして偶像から離れ、**神**の方に向きを変えたかを、  
<span class="verse-number">10&ensp;</span>そして、来るべき怒りから私たちを救い出してくださるイエス、つまり死者たちの中から蘇らせら方の、天からの**御子息**イエスを待つようになったかを、テサロニケの人たち自身が伝えているのです。  

## 第2章

それで、兄弟たち。私たちが、あなたがたを訪ねたことは、無駄でなかったと、あなたがた自身も知っているのです。  
<span class="verse-number">2&ensp;</span>しかし、以前ピリピで、私たちが苦しみと侮辱を受けたのは、知っての通りです。多くの論争のうちで、あなたがたに**神**の福音を恐れなく伝えるため、私たちの**神**は私たちを大胆にしました。  
<span class="verse-number">3&ensp;</span>なぜなら、私たちが勧めたのは、偽りや汚れから出たものでなく、また狡猾から出たものでもありません。  
<span class="verse-number">4&ensp;</span>しかし、福音に預かることを、私たちは**神**に認められているのですから、人を喜ばすのではなく、私たちの心を試す**神**を、喜ばすべく話しています。  
<span class="verse-number">5&ensp;</span>なぜなら、あなたがたは知っての通り、いかなる時も、へつらいの言葉を口にしなかったし、口実をもうけ、強欲なこともしていません。**神**こそが証人であられます。  
<span class="verse-number">6&ensp;</span>また、我々はキリストの使徒として、要求することができた時、私たちは栄光をあなたがたからであろうと、他の人々からであろうと、求めませんでした。  
<span class="verse-number">7&ensp;</span>しかし、母が自分の子たちを慈しみ育てるように、私たちはあなたがたの間で優しく接しました。  
<span class="verse-number">8&ensp;</span>このように、私たちには、あなたがたを恋い焦がれるようになりました。**神**の福音だけでなく、喜んで私たちの命も与えようとしました。なぜならば、私たちには、あなたがたは愛しい人となったからです。  
<span class="verse-number">9&ensp;</span>なぜなら、兄弟たちよ、私たちの労働と労苦を、あなたがたは忘れていないからです。それは、誰の重荷にもならないよう、私たちは昼夜の別なく働き、**神**の福音をあなたがたに宣べ伝えたからです。  
<span class="verse-number">10&ensp;</span>あなたがた、信じる人たちの間で、いかに敬虔に、正しくて、咎められることのないよう、私たちがふるまったことは、あなたがたと**神**が証人です。  
<span class="verse-number">11&ensp;</span>あなたがたは知っての通り、父が自らの子たちに接するのに、私たちはあなたがた一人一人に勧め、励まし、かつ証しました。  
<span class="verse-number">12&ensp;</span>それは、あなたがたを、ご自身の王国と栄光に呼んでくださる**神**に、ふさわしく歩むためでした。  
<span class="verse-number">13&ensp;</span>ですから、私たちも絶えず**神**に感謝しています。なぜなら、私たちから聞いた**神**の御言葉を、人間の言葉としてではなく、**神**の本当の御言葉として信じ、受け入れたあなたがたには、その御言葉は力を発揮します。  
<span class="verse-number">14&ensp;</span>したがって、兄弟たちよ、ユダヤにあってキリスト・イエスにある諸教会に倣う者にあなたがたはなりました。なぜなら、あの人たちがユダヤ人たちによって苦しんだのと同じことを、あなたがた同国人には苦しめられたからです。  
<span class="verse-number">15&ensp;</span>そのユダヤ人たちは、**主**イエスと彼ら自身の預言者たちを殺しただけではなく、私たちをも迫害しました。そして**神**を喜ばせず、すべての人に敵対し、  
<span class="verse-number">16&ensp;</span>彼らは、異邦人が救われるようにと、私たちが異邦人に語りかけるのを禁じました。そのように、ユダヤ人たちは自らの罪を膨らませ、ユダヤ人たちの頭上の、**神**の怒りは極限に達したのです。  
<span class="verse-number">17&ensp;</span>しかし、兄弟たちよ、暫時、私たちの姿は取り去られましたが、心は残っています。それどころか益々、皆さんの顔をいっそう見たいと言う思いに駆れました。  
<span class="verse-number">18&ensp;</span>ですから、私たち、特に私パウロは再度に渡り、あなたがたのところに行こうとしましたが、サタンは私たちを妨害しました。  
<span class="verse-number">19&ensp;</span>それに、私たちの希望は、または喜びは、はたまた、喜びの冠は何ですか。**主**イエス・キリストの再臨の時、**主**の御前にいるあなたがたではありませんか。  
<span class="verse-number">20&ensp;</span>なぜなら、あなたがたが、私たちの栄光であり、喜びなのです。  

## 第3章

ですから、私たちはもう耐えられなくなり、私たちだけがアテネに留まった方がよいと思いました。  
<span class="verse-number">2&ensp;</span>そして、信仰に関してあなたがたを揺ぎない者にし、あなたがたの信仰を勇気付けるため、私たちの兄弟、そしてキリストの福音に関しては、共に労働する者である、テモテを遣わしました。  
<span class="verse-number">3&ensp;</span>それは、この数々の苦難によって、誰も動揺しないためです。なぜなら、あなたがた自身、知っての通り、私たちはこのことに任命されているのです。  
<span class="verse-number">4&ensp;</span>それにまた、以前あなたがたと共にいた時、私たちは苦難に会うであろうと、あなたがたに言いました。正に、そうなったのは、あなたがたも知っての通りです。  
<span class="verse-number">5&ensp;</span>そのため、誘惑者があらゆる手段で、あなたがたを誘惑するのではないかと思い、私はもう耐えられなくなり、あなたがたの信仰を知ろうと、私たちの働きが無駄にならないため、人を遣わしました。  
<span class="verse-number">6&ensp;</span>しかし、あなたがたの信仰と愛の良い知らせを携え、すでにテモテは、あなたがたのところから、私たちのところに来ています。そしてまた、あなたがたはいつも、私たちのよい思いを保持し、私たちに会いたいと切望しています。それは私たちも同じです。  
<span class="verse-number">7&ensp;</span>したがって、兄弟たちよ、あなたがたの信仰によって、私たちのすべての苦しみと艱難の中にあって、あなたがたのことで、私たちは慰められました。  
<span class="verse-number">8&ensp;</span>なぜなら、あなたがたがしっかり**主**に立っているなら、私たちは、生きて行けます。  
<span class="verse-number">9&ensp;</span>なぜなら、私たちの**神**の御前に、あなたがたのために喜ぶ私たちのすべての喜びのために、**神**に感謝するために、あなたがたのために何をお返しできるでしょうか。  
<span class="verse-number">10&ensp;</span>あなたがたの顔を見て、あなたがたの信仰で欠けているところを完全なものにするため、私たちは昼も夜も切に祈っています。  
<span class="verse-number">11&ensp;</span>ですから、私たちの**神**と**父**ご自身と、私たちの**主**イエス・キリストが、私たちがあなたがたのところへ行く道を、指示してくださいますように。  
<span class="verse-number">12&ensp;</span>そして、私たちがあなたがたを愛しているように、あなたがたの互いの愛、またすべての人に対するあなたがたの愛が強まり、豊かになるように、**主**がそうしてくださいますように。  
<span class="verse-number">13&ensp;</span>それは、私たちの**主**イエス・キリストが、ご自身の全聖徒たちを連れ、来臨の時、私たちの**神**と**父**の御前に、あなたがたの心が聖なることに関しては、咎のないものとして、**主**が確定してくださるためなのです。  

## 第4章

後、少しですが、兄弟たちよ、**神**に喜んで頂くにはどう歩めばよいのかを、私たちから受け入れたのだから、今後も、益々歩み方を進むように、**主**イエスにあって、私たちはあなたがたに、願い、また勧めます。  
<span class="verse-number">2&ensp;</span>**主**イエスによって、私たちが皆さんに与えた命令は何か、皆さんは知っているからです。  
<span class="verse-number">3&ensp;</span>なぜなら、あなたがたが聖別されることこそ、つまりあなたがたが性的な罪を避けること、これこそが**神**のご意志なのです。  
<span class="verse-number">4&ensp;</span>聖別と誉れによって、一人一人が自らの器[^1]を所持する方法を知らなくてはなりません。  
<span class="verse-number">5&ensp;</span>それは、**神**を知らない異邦人のように、抑えられない性的な欲望に身を任せてはいけません。  
<span class="verse-number">6&ensp;</span>それは、このことに関して、人は誰も一線を超えて、自らの兄弟から騙し取ってはいけません。**主**はすべてこのことに関して、罰を与えられる方であられることは、私たちはあなたがたに警告し、証をしました。  
<span class="verse-number">7&ensp;</span>なぜなら、**神**は私たちを不潔のゆえではなく、聖別のため呼ばれたからです。  
<span class="verse-number">8&ensp;</span>それで、これを拒む者は人間を拒むのではなく、ご自身の**聖霊**を、私たちに下さった**神**を拒むことになります。  
<span class="verse-number">9&ensp;</span>しかし、兄弟愛について、私があなたがたに書く必要はありません。互いに愛し合うように、あなたがた自身は**神**から教えられているからです。  
<span class="verse-number">10&ensp;</span>そして、あなたがたは本当に、マケドニア全土にいるすべての兄弟をそのように愛しています。しかし、兄弟たちよ、益々そうするよう、あなたがたに勧めます。  
<span class="verse-number">11&ensp;</span>また、あなたがたは安穏な生活をするように努力し、自らの仕事を大切にし、私たちがあなたがたに命じたように、自らの手で働きなさい。  
<span class="verse-number">12&ensp;</span>それは外部の人々に対して、正当に歩み、何も不足がないようになるためです。  
<span class="verse-number">13&ensp;</span>しかし兄弟たちよ、眠についた者たちに関して、無知であって欲しくありません。希望のない人々のように、あなたがたが悲しまないためです。  
<span class="verse-number">14&ensp;</span>なぜなら、イエスは死んで復活されたと信じているなら、そのようにイエスによって眠った人々は、**神**が連れて来られます。  
<span class="verse-number">15&ensp;</span>それで、私たちは**主**の御言葉に基づき、このことをあなたがたに言います。つまり、**主**の来臨まで生きて留まる私たちは、眠っている人々より先になることは決してありません。  
<span class="verse-number">16&ensp;</span>それで、**主**ご自身は叫び声で、御使いの頭の声で、**神**のラッパで天から降って来られます。そして、キリストにある死者たちが先ず復活します。  
<span class="verse-number">17&ensp;</span>次いで、生きて留まっている私たちが、空中で**主**に会うために、私たちは先の人々と共に、雲の中に瞬時に引き上げられます。このようにして、私たちはいつまでも**主**と共にいます。  
<span class="verse-number">18&ensp;</span>ですから、今の一つ一つの言葉で、互いに慰め合いなさい。  

## 第5章

しかし兄弟たちよ、その時と時期について、私があなたがたに書く必要はありません。  
<span class="verse-number">2&ensp;</span>なぜなら、**主**の日は夜中の泥棒のように来ることを、あなたがた自身、よく知っているからです。  
<span class="verse-number">3&ensp;</span>なぜなら、人々が「平和で、安全だ」と言うその時、妊婦に出産の陣痛が来るように、突然の壊滅が人々の上に来ます。そして決して逃げおおせません。  
<span class="verse-number">4&ensp;</span>しかし、兄弟たちよ、泥棒が来るようにその日が人々を襲ったとしても、皆さんは暗やみの中にはいません。  
<span class="verse-number">5&ensp;</span>あなたがた全員、光の子であり、昼の子です。私たちは夜の者でも、闇の者でもありません。  
<span class="verse-number">6&ensp;</span>したがって、他の人々のように、眠っていてはいけません。かえって、警戒し、素面でいましょう。  
<span class="verse-number">7&ensp;</span>なぜなら、眠る人々は夜に眠り、酔う者たちは夜、酔います。  
<span class="verse-number">8&ensp;</span>しかし、我々昼の者は、信仰と愛の胸当てをつけ、救いの希望としての甲冑をかぶり、素面でいましょう。  
<span class="verse-number">9&ensp;</span>なぜなら、**神**は私たちを、怒りのために定められたのではなく、私たちの**主**イエス・キリストを通して、救いに預かるため、定められたのです。  
<span class="verse-number">10&ensp;</span>私たちが、目覚めていようが眠っていようが、**主**は私たちのために死んでくださったのです。イエスと共に生きるためです。  
<span class="verse-number">11&ensp;</span>ですから、あなたがたはすでに行なっているように、互いに慰め合い、互いに啓発しなさい。  
<span class="verse-number">12&ensp;</span>そして兄弟たちよ、皆さんにお願いします。皆さんの間で働く人々、そして、**主**にあってあなたがたの上に立ち、あなたがたを教え諭す人々を認めるよう、勧告します。  
<span class="verse-number">13&ensp;</span>また、その人々の働きのゆえ、愛をもって、特に高く評価し、尊敬しなさい。互いに平和を保ちなさい。  
<span class="verse-number">14&ensp;</span>さて、兄弟たちよ。無規律な生活をする者たちに警告しなさい。絶望をした者たちを元気付けなさい。弱い者たちを強固な者にし、すべての人に対して忍耐強くありなさい。  
<span class="verse-number">15&ensp;</span>いかなる人に対しても、悪に対し悪で報いず、皆さん自身にも、そして、すべての人のために、善なることを追求しなさい。  
<span class="verse-number">16&ensp;</span>どんな時でも、喜びなさい。  
<span class="verse-number">17&ensp;</span>絶えず祈りなさい。  
<span class="verse-number">18&ensp;</span>これはあなたがたのため、イエス・キリストにおける**神**のご意志なのだから、すべてのことについて感謝しなさい。  
<span class="verse-number">19&ensp;</span>**御霊**を拒否してはいけません。  
<span class="verse-number">20&ensp;</span>預言を軽視してはいけません。  
<span class="verse-number">21&ensp;</span>すべてのものを分析し、よいものをかたく握りしめなさい。  
<span class="verse-number">22&ensp;</span>いかなる姿形の悪も避けなさい。  
<span class="verse-number">23&ensp;</span>どうか、平和の**神**ご自身が、皆さんを完全に聖別してくださいますように。そして、皆さんの霊、魂、肉体が、私たちの**主**イエス・キリストの来臨の際、咎められるところがないように、保全されますように。  
<span class="verse-number">24&ensp;</span>あなたがたを召してくださる方は誠実な方であられます。皆さんをそうして下さるのもその方であられます。  
<span class="verse-number">25&ensp;</span>兄弟たちよ、私たちのために祈ってください。  
<span class="verse-number">26&ensp;</span>聖なる口づけで、兄弟たち全員に挨拶をしてください。  
<span class="verse-number">27&ensp;</span>聖なる兄弟たち全員に、この手紙を読み聞かせよ、と**主**によってあなたがたに命じます。  
<span class="verse-number">28&ensp;</span>私たちの**主**イエス・キリストの御恵みが、あなたがたと共にありますように。アーメン。  


[^1]: 二とおりの解釈があります。⑴妻、⑵自分の体。
